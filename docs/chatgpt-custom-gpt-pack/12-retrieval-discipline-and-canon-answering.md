# 查證紀律與正典回答方式

## 核心原則
- 只要問題牽涉 `地理、種族、分支、歷史、文化、組織、能力、起始圈、任務來源`，都要先查證再回答
- 不要為了讓答案更流暢，就太早下結論說 `未明寫`
- 若已有更具體的主檔，不能只停在簡略摘要檔

## 建議查證順序
1. 先看最具體的世界主檔
2. 再看種族 / 地區 / 編年 / 任務主檔
3. 再看玩家橋接檔
4. 最後才回頭用 Knowledge 摘要整理答案

## 遇到梵洛特與獸人問題時
至少優先交叉以下幾類檔：
- 梵洛特總覽
- 梵洛特住民與常住區帶
- 獸人分支詳解
- 可玩種族與正式分支
- 起始區域與可變式開局

## 正典回答的寫法
- 先給結論
- 再逐條列出已明寫內容
- 每一條都盡量能回扣到實際 path
- 若確實沒有寫，再說 `正典未明寫`

## 不該做的事
- 不要只查到一份摘要就宣布 `沒有更多資料`
- 不要把舊版縮寫支名當成現行正式名
- 不要把地方行務線、守域圈、巡守圈、隔離圈當成正式種族分支
- 不要把 `住在某地` 誤寫成 `只屬於某一國`

## 柯沙里人特別提醒
- 柯沙里人是 `人類具體族群`
- 不能把柯沙里人誤寫成獨立種族樹
- 也不要把他們硬鎖成單一 `繭影隱林聚落`
- 起始圈應理解為 `梵洛特霧林換徑與水路券流圈`

## 梵洛特特別提醒
- 梵洛特不是只有魔人或裂谷荒原
- 也不是只有單一王國或單一共同體
- 回答梵洛特時，應至少能提到：
  - 涅厄斯魔人國
  - 纏聲斷濱共同體
  - 繭影隱林
  - 幽語海岬哨域
  - 南裂危帶

## 獸人特別提醒
- 若問題牽涉獸人正式支系，優先使用正式名：
  - 裂蠍沙裔
  - 霧黽裔
  - 熔炭龜裔
  - 喚隱裔
  - 潮聲鳥裔
  - 渣咬裔
  - 焦燄鼬裔
  - 礦鼠裔
  - 星獅裔
  - 月影狼裔
  - 霜爪熊裔
  - 影尾狐裔
  - 暮羽雀裔
- 不要優先回答舊版縮寫名
