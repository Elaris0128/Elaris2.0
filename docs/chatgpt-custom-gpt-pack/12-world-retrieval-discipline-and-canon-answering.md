# 查證紀律與正典回答方式

## 核心原則
- 只要問題牽涉 `地理、種族、分支、歷史、文化、組織、能力、起始圈、任務來源、職業、物品、魔物`，都要先查證再回答
- 不要因為看到一份摘要檔就太早下結論說 `未明寫`
- 若已有更具體的主檔，不能只停在簡略摘要檔或玩家橋接檔
- 這份規則適用於 `整個 Elaris`，不是只適用單一大陸

## 建議查證順序
1. 先看最具體的世界主檔
2. 再看對應的種族 / 地區 / 組織 / 編年 / 任務主檔
3. 再看玩家橋接檔與起始規則檔
4. 最後才用 Knowledge 摘要整理答案

## 依題型的查證路線
### 問大陸 / 國家 / 海域 / 特殊區域
- 先看大區總覽
- 再看國家或海域主檔
- 再看起始圈、任務橋接與玩家規則檔

### 問種族 / 分支 / 器官 / 能力
- 先看種族主檔
- 再看分支詳解與特殊器官 / 能力檔
- 再看編年史與玩家能力橋接檔

### 問職業 / 任務 / 開局 / 生活方式
- 先看職業主檔或任務主檔
- 再看地區包與玩家橋接規則
- 最後才整理成玩家可理解的回答

## 正典回答的寫法
- 先給結論
- 再逐條列出已明寫內容
- 每一條都盡量能回扣到實際 path
- 若確實沒有寫，再說 `正典未明寫`
- 若只是暫時推定，必須明講那是推定，不可包裝成正典

## 不該做的事
- 不要只查到一份摘要就宣布 `沒有更多資料`
- 不要把地方制度圈、守域圈、巡守圈、隔離圈誤寫成正式種族分支
- 不要把 `住在某地` 誤寫成 `只屬於某一國`
- 不要讓某一塊大陸或單一議題蓋掉整個世界的回答視角

## 世界尺度提醒
- 賽勒亞、梵洛特、伊莎梅爾都應被視為完整大陸層級，不可只用單一國家代表整片大陸
- 六大洋與特殊區域也是世界主幹，不只是附屬背景
- 玩家起始圈、地方任務與文化職稱都應回扣到對應的大區與正典主檔
- 但也要記得：`完整大陸層級` 不等於 `多族混居大陸`；例如伊莎梅爾的正典主體就是精靈文明，只有巴布窟族是正典級共生特例
