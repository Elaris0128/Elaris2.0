# 起始區域與可變式開局

## 核心規則
玩家不是先挑城市開局，而是：

`種族 / 分支 -> 起始區域 -> AI 在該區域內隨機組開局`

## 重要目的
- 保住出身感
- 保住地方味
- 讓任務、派系、NPC 能自然接上
- 避免 AI 隨機把人丟到不合理的地方

## 人類與人類系常見起始圈
- 法奧人 -> 瑟恩聯邦國
- 瓦雷溫人 -> 阿瓦隆王國
- 法雷恩人 -> 法雷恩沿海諸港
- 極光徙裔 -> 賽勒亞-極冠嶺域
- 魈陽族 -> 魈陽族山陵圈
- 崖影族 -> 萬嶺幽陵山陵圈
- 巖穹族 -> 巖穹三聯域
- 薩魯恩人 -> 梵洛特-裂紋沙原
- 柯沙里人 -> 梵洛特-繭影隱林
- 諾魯因人 -> 幽語海

## 其他可玩種族起始圈
- 翼人 -> 瑟堤亞棲國 / 雲裂哨站
- 矮人 -> 杜羅山邦
- 地精 -> 杜魯摩克隱域
- 森馳裔 -> 森馳裔邊林圈

## 海人起始圈
- 潮霧之裔 -> 靈穹洋
- 熾潮之脈 -> 焰潮洋
- 鱗盾之裔 -> 幽語海或焰潮洋邊守域

## 精靈起始圈
- 雲瑟皇族 -> 雲瑟林域
- 雪華部族 -> 賽勒亞-雪華部族地
- 霜諾嵐族 -> 伊莎梅爾霜脊寒域
- 其餘精靈分支 -> 對應各自本域

## 獸人起始圈
- 十獸之環 -> 克羅嵐共誓域
- 月影狼裔 / 霜爪熊裔 / 影尾狐裔 / 暮羽雀裔 -> 暮鏡荒境
- 其他獸人支系 -> 對應自己的荒域、灘帶、坑脈或遷帶

## 魔人起始圈
- 曜血 -> 亡極徹淵・裂廊上層
- 鳴紗 -> 亡極徹淵・裂廊中層
- 噬淵 -> 亡極徹淵・裂廊中層
- 幽鬩 -> 亡極徹淵・裂廊下層
- 獰罡 -> 亡極徹淵・裂廊下層
- 垢金 -> 亡極徹淵・漩淵渦心
- 緲眠 -> 亡極徹淵・外環幽裂

## 可變式開局
AI 開局時應在起始區域內隨機組合：
- 第一個壓力
- 第一個目標
- 第一個互動對象
- 第一個麻煩
- 第一批可接任務

## 不要做成固定開局
以下都不應該發生：
- 每個諾魯因人都從同一個港口開始
- 每個海人都從同一種海難開始
- 每個獸人都一定先打架

## 例外
只有玩家背景明確寫出：
- 流亡
- 留學
- 調任
- 押運
- 遷徙
- 被販運 / 被救起

才可從非本族主居圈開始，但仍需有合理故事理由。
