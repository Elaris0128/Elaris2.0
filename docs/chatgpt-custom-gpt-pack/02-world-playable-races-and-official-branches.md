# 可玩種族與正式分支

## 讀法
- 玩家如果說 `海人 / 精靈 / 獸人 / 魔人`，AI 必須再追問正式分支
- 玩家如果已經說出 `諾魯因人 / 極光徙裔 / 柯沙里人 / 月影狼裔` 這種具體族群名，直接視為選擇完成
- 人類不建議只回答 `人類`；最好直接落到具體人類族群

## 特別正典
- 柯沙里人是 `人類具體族群`，不是獨立種族樹，也不是正式次級分支樹
- 海人只有三支，不可再擴成更多正式分支
- 諾魯因人與海人不設混血線
- 龍族、鳳族、神獸古族不是玩家可選種族
- `霧沼裔 / 炭龜裔 / 隱號裔 / 炭鼬裔` 不是現行優先正式名
- `夜角鹿裔 / 灰背犀裔 / 赤鬃鬣裔` 不應優先當成現行正式玩家支系回答，除非未來正典另行補回

## AI 的選種族判定
- 玩家說 `我要玩海人`：追問三支
- 玩家說 `我要玩獸人`：追問正式支系
- 玩家說 `我要玩十獸之環`：可再問要不要細選席位
- 玩家說 `我要玩諾魯因人 / 柯沙里人 / 月影狼裔`：直接視為種族選擇完成
