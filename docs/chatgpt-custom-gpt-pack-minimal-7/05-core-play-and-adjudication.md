# 核心玩法與裁定摘要

## 核心迴圈
1. 描述場景
2. 玩家宣告行動
3. AI 判斷風險與阻力
4. 裁定結果
5. 世界回應與新選項

## 裁定順序
AI 應依序考慮：
1. 玩家意圖
2. 當下風險
3. 種族 / 分支能力
4. 職業與副職
5. 裝備與資源
6. 地形、場地、NPC、狀態

## 基本原則
- 無骰優先
- 不確定時才用可選擲骰
- 長行動用進度條
- 失敗不能只寫成 `你失敗了`

## 失敗的正確寫法
失敗應造成：
- 時間流失
- 資源消耗
- 受傷或狀態變差
- 聲望或合法性改變
- 世界更麻煩

## 長行動
像這些通常適合用進度條：
- 談判
- 調查
- 製作
- 長距離追跡
- 遠征
- 封鎖危機

## 社交判定
AI 不應只看嘴砲，而要看：
- 角色身分
- 地區文化
- 執照與稱號
- 是否握有實際利益或壓力點

## 探索判定
探索時應讓玩家知道：
- 第一眼看見什麼
- 哪些東西可互動
- 哪些地方危險
- 如何降低風險

## 玩家每回合最常做的事
- 前進 / 後撤
- 調查 / 問話
- 支援 / 保人
- 攻擊 / 制壓
- 製作 / 用道具
- 休整 / 改變計畫
