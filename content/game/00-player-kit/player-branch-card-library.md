# 玩家分支卡庫

這份文件是給 `已經選了大種族，準備挑分支` 的玩家用的。

最適合這樣搭配：
- 先看 [玩家種族卡庫](C:/Users/rabbi/Desktop/Elaris/content/game/00-player-kit/player-race-card-library.md) 決定你要不要玩 `海人 / 精靈 / 獸人 / 魔人`
- 再來這裡挑 `分支卡`
- 如果想看正式完整名單，再查 [玩家種族分支清單](C:/Users/rabbi/Desktop/Elaris/content/game/00-player-kit/player-race-branch-library.md)

## 怎麼用這份卡庫

每張分支卡都看四件事：
- `玩法氣質`：這支最像什麼風格
- `擅長場域`：在哪些環境最有感
- `常見優勢`：這支角色最容易出彩的地方
- `常見弱點`：選了之後通常要注意什麼

## 海人分支卡

### 潮霧之裔
- 玩法氣質：霧潮、暗流、低光、潛渡
- 擅長場域：迷霧海、暗礁、低光潮窟、外海灰水道
- 常見優勢：改道、藏路、迷航判讀、低視野追跡
- 常見弱點：正面硬撐、超開闊明亮環境
- 適合職業：引水人、打撈手、外緣採樣手

### 熾潮之脈
- 玩法氣質：熱潮、危窗、爆發、穿壓
- 擅長場域：熱泉帶、焰潮洋、毒潮口、危材窗帶
- 常見優勢：短時爆發、高熱耐受、危地操作
- 常見弱點：長線消耗、精密潛伏
- 適合職業：危材處理者、船長、打撈手

### 鱗盾之裔
- 玩法氣質：守線、護群、礁防、共鳴
- 擅長場域：礁脊、守域圈、迎浪防線、群體行動
- 常見優勢：扛線、護送、群體協調、迎正面壓力
- 常見弱點：單走、需要大量偽裝的場合
- 適合職業：護衛、引水人、海岸守望者

## 最後怎麼決定

如果你已經選了大種族，但還在分支裡猶豫，可以這樣問自己：
- 我想玩 `群隊 / 守線 / 爆發 / 隱匿 / 調查 / 支援` 哪一種？
- 我比較常想去 `海上 / 高空 / 寒荒 / 森林 / 城市 / 深窟` 哪種場域？
- 我想要的是 `穩`、`狠`、`怪`、`靈`、`灰`、還是 `正` 的感覺？

通常只要先想清楚這三題，你就很容易挑到對味的分支。
