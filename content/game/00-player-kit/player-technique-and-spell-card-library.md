# 玩家起始戰技與術式卡庫

這份文件收的是 `新手最容易上手` 的戰技與術式。

## 一、戰技卡

### 破陣踏
- 類型：強襲
- 適合：近戰前衛、守誓戰士
- 作用：突進、壓線、逼退
- 常用能力：體魄 + 身法

### 承山架
- 類型：防禦
- 適合：重裝守衛、港巡兵
- 作用：扛衝擊、守人、穩位置
- 常用能力：體魄 + 心志

### 影返步
- 類型：機動
- 適合：機動斥候、巡翼哨手
- 作用：換位、脫離鎖定、迂迴
- 常用能力：身法 + 洞察

### 斜岔追痕
- 類型：追跡
- 適合：狩窗手、機動斥候
- 作用：追擊、維持目標線
- 常用能力：身法 + 洞察

### 折線護
- 類型：護衛
- 適合：重裝守衛、護行者
- 作用：保護隊友、拉安全角度
- 常用能力：體魄 + 身法

## 二、術式卡

### 尋脈引
- 類型：偵測術
- 適合：學院施術者、記錄學者
- 作用：找地脈、找殘留、找異常
- 常用能力：悟性 + 靈感

### 裂息矢
- 類型：放出術
- 適合：學院施術者、符印術者
- 作用：遠距壓制、打斷節奏
- 常用能力：靈感 + 靈核

### 冷息護皮
- 類型：防護術
- 適合：治療藥師、自然共鳴者
- 作用：降低熱傷、擦傷、毒刺傷
- 常用能力：悟性 + 靈核

### 靜域環
- 類型：場域術
- 適合：祭儀執行者、治療藥師
- 作用：穩定場面、保護傷者、降干擾
- 常用能力：心志 + 靈核

### 迷行霧幕
- 類型：遮蔽術
- 適合：自然共鳴者、契靈行者
- 作用：遮視線、保撤退、造模糊區
- 常用能力：靈感 + 心志

## 三、種族常見招式卡

### 潮骨伏息
- 所屬：諾魯因人
- 作用：降低存在感、短時守息

### 霧潮潛渡
- 所屬：潮霧之裔
- 作用：迷霧與暗流中的低聲移動

### 珊脈灼流擊
- 所屬：熾潮之脈
- 作用：熱潮環境下的短爆近擊

### 礁盾迎浪
- 所屬：鱗盾之裔
- 作用：迎擊、護線、穩住前方

### 裂風滑降
- 所屬：翼人
- 作用：高差切入、滑降轉線

- 承山架
- 折線護

### 斥候探路型
- 影返步
- 斜岔追痕

### 施法調查型
- 尋脈引
- 冷息護皮

### 儀式支援型
- 靜域環
- 迷行霧幕
