## 一、範例 A：海路救援型

### 創角思路
- 想玩：能救人、懂潮路、在海窟和港邊都很有用
- 選擇方向：`諾魯因人 + 引水人`

### 選卡流程
- 種族卡：`諾魯因人`
- 種族能力卡：`潮骨血息`
- 職業卡：`引水人`
- 能力傾向：`身法、洞察`
- 特質卡：`冷靜手穩`
- 缺陷卡：`不肯放棄失手者`
- 背景卡：`水道與濕地長成`
- 起始裝備包：`海路護行包`
- 起始資源包：`港口熟面包`
- 起始招式：`潮骨伏息`、`影返步`

### 完整角色紙示例
- 名稱：潮岑遙
- 種族：諾魯因人
- 種族分支 / 地方支系：無
- 出身地：幽語海寂岩孤島外緣泊棚
- 背景：水道與濕地長成
- 主職：引水人
- 副職：打撈手
- 文化職稱：守潮渡手

- 體魄：2
- 身法：4
- 心志：3
- 悟性：2
- 靈感：2
- 靈核：1
- 魅力：2
- 洞察：4
- 技巧：3

- 生命值（HP）：12
- 靈息儲量（VP）：6
- 專注值（Focus）：10
- 負荷上限（Load）：10
- 行動速度（Tempo）：快

- 種族起始傾向：冷潮、守息、辨壓
- 分支修正：無
- 特殊器官 / 體質：潮骨血息
- 常見弱點：極乾環境、長期離海、劇烈壓差

- 職業核心定位：判路、帶路、回潮脫困、短距救援
- 起始專長：潮路判向、泊道換位
- 起始裝備：海骨短槍、防滑短靴、耐潮披、繩鉤、防水包
- 起始裝備包：海路護行包
- 起始資源包：港口熟面包
- 常見任務面：帶路、救援、追查潮窟失蹤、短線護送

- 戰技 1：影返步
- 戰技 2：斜岔追痕
- 術式 1：無
- 術式 2：無
- 特殊招式 / 種族招式：潮骨伏息

- 主武器：海骨短槍
- 副武器：登船刀
- 防具：耐潮披、護臂
- 常備工具：繩鉤、防水包、小潮燈
- 消耗品：鹽止血布、乾糧、水囊
- 關鍵道具：舊潮路刻板
- 金錢：小額港幣與換手零錢
- 可動用人脈 / 權限：一位老碼頭工、一條平價補給線、一次低風險借泊

- 所屬組織：無正式公會席位
- 關鍵人脈：泊道老工「沉伯」
- 友善勢力：港邊熟面、少量引水人圈
- 敵對勢力：偽泊權掮客
- 初始聲望：港邊熟路但不算官面可信

- 地點：沿海外港
- 時間：回潮前
- 傷勢：無
- 精神壓力：低
- 異常狀態：無
- 進行中任務：第一份港邊失蹤委託

- 角色一句話定位：
  `來自幽語海外緣泊棚的諾魯因引水人，靠潮骨血息與潮窟回聲找人、帶路、把人活著拖回來。`

## 二、範例 B：城市調查型

### 創角思路
- 想玩：調查、查文簿、破譯、和官署與公會互動
- 選擇方向：`極光徙裔 + 記錄學者`

### 選卡流程
- 種族卡：`極光徙裔`
- 種族能力卡：`極光紋瞳`
- 職業卡：`記錄學者`
- 能力傾向：`悟性、洞察`
- 特質卡：`記憶穩定`
- 缺陷卡：`太容易追真相追到底`
- 背景卡：`學院旁聽生`
- 起始裝備包：`記錄學者包`
- 起始資源包：`公會熟面包`
- 起始招式：`尋脈引`、`靜域環`

### 完整角色紙示例
- 名稱：岑燈序
- 種族：極光徙裔
- 種族分支 / 地方支系：無
- 出身地：北境長夜觀測帶
- 背景：學院旁聽生
- 主職：記錄學者
- 副職：文簿官
- 文化職稱：抄錄見習師

- 體魄：1
- 身法：2
- 心志：3
- 悟性：4
- 靈感：3
- 靈核：2
- 魅力：2
- 洞察：4
- 技巧：2

- 生命值（HP）：9
- 靈息儲量（VP）：8
- 專注值（Focus）：12
- 負荷上限（Load）：8
- 行動速度（Tempo）：中

- 種族起始傾向：長夜、守望、低光辨識
- 分支修正：無
- 特殊器官 / 體質：極光紋瞳
- 常見弱點：強光、濕熱、曝曬

- 職業核心定位：讀文、比對、找破口、收束線索
- 起始專長：簿頁比對、殘痕判讀
- 起始裝備：記錄冊、抄錄筆、比對片、拓印布、保存匣
- 起始裝備包：記錄學者包
- 起始資源包：公會熟面包
- 常見任務面：調查、解謎、破譯、追簿、查名

- 戰技 1：無
- 戰技 2：無
- 術式 1：尋脈引
- 術式 2：靜域環
- 特殊招式 / 種族招式：極夜借光

- 主武器：短杖
- 副武器：小型護身刃
- 防具：學院披、輕護臂
- 常備工具：拓印布、比對片、折疊燈
- 消耗品：穩手墨、短時靜息粉
- 關鍵道具：借來的學院外證
- 金錢：中低額合法通貨
- 可動用人脈 / 權限：一位願意先聽你說話的公會熟人、一次小權限借閱

- 所屬組織：學院外圍講堂線
- 關鍵人脈：講堂抄錄員「聞燼」
- 友善勢力：記錄者與部分探索公會窗口
- 敵對勢力：討厭查帳的黑頁買手
- 初始聲望：年輕但不像外行

- 地點：阿瓦隆外港或王都外政廊
- 時間：陰雨下午
- 傷勢：無
- 精神壓力：低
- 異常狀態：無
- 進行中任務：失簿案

- 角色一句話定位：
  `來自長夜觀測帶的極光徙裔抄錄見習師，靠極光紋瞳與安靜的追簿本事，把所有被塗掉的名字一一找回來。`

## 三、範例 C：新手穩定戰鬥型

### 創角思路
- 想玩：不容易死、保隊友、戰鬥時好理解
- 選擇方向：`獸人・霜爪熊裔 + 重裝守衛`

### 選卡流程
- 種族卡：`獸人`
- 分支卡：`暮鏡荒境・霜爪熊裔`
- 種族能力卡：`獸人共通底卡`
- 分支能力卡：`霜爪熊裔`
- 職業卡：`重裝守衛`
- 能力傾向：`體魄、心志`
- 特質卡：`耐壓穩定`
- 缺陷卡：`太習慣自己扛`
- 背景卡：`冬聚地長大`
- 起始裝備包：`重裝壓線包`
- 起始資源包：`邊境照應包`
- 起始招式：`承山架`、`折線護`

### 完整角色紙示例
- 名稱：裂霜嵐
- 種族：獸人
- 種族分支 / 地方支系：霜爪熊裔
- 出身地：暮鏡荒境冬聚地
- 背景：冬聚地長大
- 主職：重裝守衛
- 副職：旅灶長
- 文化職稱：雪線扛盾手

- 體魄：4
- 身法：2
- 心志：4
- 悟性：1
- 靈感：1
- 靈核：1
- 魅力：2
- 洞察：2
- 技巧：2

- 生命值（HP）：14
- 靈息儲量（VP）：4
- 專注值（Focus）：9
- 負荷上限（Load）：12
- 行動速度（Tempo）：中偏慢

- 種族起始傾向：共獵、壓線、承壓
- 分支修正：寒地、守線、迎面承擊更穩
- 特殊器官 / 體質：霜爪厚掌、冬壓耐受
- 常見弱點：長距追逐、悶熱潮濕環境

- 職業核心定位：站前線、吃傷害、替隊友撐空間
- 起始專長：扛線護人、壓角不退
- 起始裝備：重盾、短槍、厚甲、固定樁、止血包
- 起始裝備包：重裝壓線包
- 起始資源包：邊境照應包
- 常見任務面：護送、守門、守線、拖撤同伴

- 戰技 1：承山架
- 戰技 2：折線護
- 術式 1：無
- 術式 2：無
- 特殊招式 / 種族招式：寒掌穩樁

- 主武器：短槍
- 副武器：手半劍
- 防具：厚甲、重盾
- 常備工具：固定樁、火絨袋、粗繩
- 消耗品：止血包、肉乾、暖石袋
- 關鍵道具：冬聚骨哨
- 金錢：不多，但足夠短時補給
- 可動用人脈 / 權限：一處可借火哨點、一位願意收留的守線熟人

- 所屬組織：無正式國家編制
- 關鍵人脈：冬聚守火者「嵐姨」
- 友善勢力：暮鏡冬聚地與部分北境哨線
- 敵對勢力：搶冬線資源的流竄盜群
- 初始聲望：扛得住、但不善官面

- 地點：北境或任何寒地前線
- 時間：冬季黃昏
- 傷勢：無
- 精神壓力：低
- 異常狀態：無
- 進行中任務：第一份護送或救援委託

- 角色一句話定位：
  `出身暮鏡荒境冬聚地的霜爪熊裔扛盾手，擅長站在最前面，把該撞過來的東西全都留在自己面前。`

## 四、第一次創角時怎麼抄最順

如果你真的只想快速開始：
- 想玩調查：直接抄 `範例 B`
- 想玩海路與救援：直接抄 `範例 A`
- 想玩穩定戰鬥：直接抄 `範例 C`

然後只改這五個地方就夠了：
- 名字
- 出身地
- 一項特質
- 一項缺陷
- 一件你特別想要的關鍵道具

這樣角色就已經很像你的，而不是完全照抄。
