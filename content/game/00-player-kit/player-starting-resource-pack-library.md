# 玩家起始資源包

這份文件收的是 `玩家創角時可直接挑選的起始資源包`。

這裡的資源不是只有錢，也包含：
- 人脈
- 證件
- 情報
- 工具權限
- 安全住處
- 可動用的交通與交接線

使用原則：
- 每個角色起始選 `1` 包
- 若背景和職業非常吻合，AI / 主持可再給 `1` 個小型附加資源
- 起始資源包不是永久免死金牌，而是 `第一階段劇情中的可動用優勢`

## 經濟型資源包

### 穩定現金包
- 內容：一筆安全可用的小額通貨、常見消耗補給、一次普通旅宿費
- 優勢：最泛用、最不挑背景
- 代價：沒有額外人脈與特殊通路

### 交易貨樣包
- 內容：數件可交換的小特產、估價筆記、熟門熟路的貨樣知識
- 優勢：更容易用物換物、打開交易話題
- 代價：現錢不一定多，可能吸引想壓價的人

### 高值單件包
- 內容：一件中價但不好立刻脫手的物件
- 優勢：必要時能換出大幫助
- 代價：平常可能反而成為負擔或被盯上

## 人脈型資源包

### 公會熟面包
- 內容：一位願意先聽你說話的公會熟人、一次借閱或借用小權限
- 優勢：很適合藥師、公會見習生、學者、探索者
- 代價：人情日後多半要還

### 港口熟面包
- 內容：一位碼頭熟人、一條平價補給線、一次低風險借泊
- 優勢：海路、走貨、問事都很好用
- 代價：港口風聲傳得快，你的行蹤較難完全隱密

### 邊境照應包
- 內容：一處能借火的哨點、一名願意暫時收留你的守線熟人
- 優勢：北境、荒境、長路線特別好用
- 代價：通常伴隨守夜、報訊或代辦責任

### 灰線接頭包
- 內容：一個能聯絡到灰線人物的暗號或老地方
- 優勢：急需黑市、消息、假名時很有用
- 代價：這條線通常不乾淨，也可能把麻煩帶回來

## 身分與權限型資源包

### 合法證明包
- 內容：基本身分證明、通行憑記、一次合法查驗快速通過機會
- 優勢：對大城、關口、公會前台很有幫助
- 代價：你的正式行蹤更容易留下紀錄

### 借名掛靠包
- 內容：一個能暫時掛靠的名字、商號或學籍外殼
- 優勢：方便進入本來進不去的地方
- 代價：若被拆穿，後果往往比沒有證件更糟

### 小型存取權包
- 內容：某個庫房、講堂、碼頭或工坊的低階進入資格
- 優勢：對調查、職業任務、裝備補給很有利
- 代價：只能用在特定範圍，而且很容易被追查

## 安全與後勤型資源包

### 安全落腳處包
- 內容：一處便宜但可靠的落腳點、一位默認你可以回來的人
- 優勢：休整、藏物、整理情報很穩
- 代價：地點一旦曝光，麻煩也會跟著來

### 旅運支援包
- 內容：一台車位、一張艙位、一條固定補給小路，三擇一
- 優勢：適合長路、海路、高空交通圈角色
- 代價：通常有時間窗口，錯過就沒了

### 緊急救助包
- 內容：一次可動用的醫療、撤離或短期收容資源
- 優勢：新手非常穩
- 代價：真正用掉之後就沒了，還可能欠人情

## 情報型資源包

### 舊案線索包
- 內容：一份殘破名單、半張地圖、一道舊口令、某件事的真名
- 優勢：很容易開支線、挖世界觀、接重大線
- 代價：也很容易讓危險更早找上你

### 地方熟路包
- 內容：一張局部熟路圖、一個短路捷徑、一條少人知道的繞路
- 優勢：逃跑、追路、避查驗都很好用
- 代價：離開該區後價值會大降

### 黑話與行話包
- 內容：某一圈人的術語、暗號、交接方式
- 優勢：更容易融入港務、公會、黑市或地方圈
- 代價：講錯地方可能反而更顯眼

## 推薦搭配

- 新手最穩：
  - `行旅生存包 + 穩定現金包`
  - `治療藥師包 + 緊急救助包`
- 海路角色：
  - `海路護行包 + 港口熟面包`
- 北境角色：
  - `冬聚熬夜包 + 邊境照應包`
- 灰線角色：
  - `灰線應急包 + 灰線接頭包`
- 學院 / 公會角色：
  - `基礎施術包 或 記錄學者包 + 公會熟面包`

## 裁定原則

- 裝備包回答的是：`你身上有什麼`
- 資源包回答的是：`你能叫到誰、借到什麼、動到哪條線`
- 如果玩家背景與資源包高度一致，AI / 主持可在第一個大場景裡額外給 `1` 次便利
