# 玩家種族卡庫

這份是玩家創角時的第一層清單。  
它回答的是：`你到底是在選一個大種族，還是在選一個已經很具體的族群名。`

## 直接可選的人類系族群
- `法奧人`
  傾向學城、都會、議會與制度社會。
- `瓦雷溫人`
  傾向王都、騎士、宮廷與大路秩序。
- `法雷恩人`
  傾向沿海港市、商路、航務與多文化混居。
- `極光徙裔`
  傾向高緯寒境、極光、風雪與長途遷徙。
- `魈陽族`
  傾向高崖、烈光、山陵與乾燥高地。
- `崖影族`
  傾向陰崖、索道、峽谷與隱伏路線。
- `巖穹族`
  傾向石城、斷崖、礦脈與高地工程。
- `薩魯恩人`
  傾向裂紋沙原、裂砂誓帶、乾地行旅與移動聚落秩序。
- `柯沙里人`
  傾向霧林溪谷、寨居水路、織紋銀飾與歌路文化。
- `諾魯因人`
  傾向幽語海、潮窟、水路法與潮骨血息文化。

## 直接可選的其他種族
- `翼人`
  高空巢域與風環秩序中的飛行種族。
- `矮人`
  山邦、鍛脈、礦工與深爐秩序的定居種族。
- `地精`
  隱域工巧、灰線技術與機敏群聚文化的種族。
- `森馳裔`
  樹海、葉脈、長途林徑與生態秩序中的種族。

## 需要再選分支的大種族
- `海人`
  先選海人，再選三支。
- `精靈`
  先選精靈，再選正式分支。
- `獸人`
  先選獸人，再選正式支系。
- `魔人`
  先選魔人，再選七血裔。

## 這份卡庫怎麼用
- 如果你已經看到的是 `諾魯因人` 這種具體名字，直接進下一步。
- 如果你選的是 `海人 / 精靈 / 獸人 / 魔人`，要先去挑分支。
