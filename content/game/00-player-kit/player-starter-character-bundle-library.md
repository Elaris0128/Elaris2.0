# 玩家開局角色包

這份是給想快速上手的玩家用的。
如果你不想從零慢慢拼，可以直接挑一包，再微調。

## 幽語海引路者包
- 種族：諾魯因人
- 職業：引水人
- 風格：帶隊、辨路、保人
- 建議特質：沉著、路感強
- 建議缺陷：多疑或舊傷
- 起始重點：潮路、繩索、錯燈判定

## 王都記錄學者包
- 種族：極光徙裔 或 法奧人
- 職業：學者 / 文簿官
- 風格：查證、抄錄、說理
- 建議特質：記憶強、條理清楚
- 建議缺陷：體能弱或不擅野外
- 起始重點：文件、見證、學城委託

## 暮鏡重裝守界包
- 種族：霜爪熊裔
- 職業：城衛兵 / 守墓者 / 巡哨手
- 風格：扛線、保人、守位
- 建議特質：硬撐、威懾
- 建議缺陷：笨重或直性
- 起始重點：寒地生存、換位保人、邊界壓力

## 暮鏡獵路斥候包
- 種族：月影狼裔 或 影尾狐裔
- 職業：路線踏勘手 / 狩窗手
- 風格：追跡、先手、拆局
- 建議特質：敏銳、沉默
- 建議缺陷：難信人或太愛單走
- 起始重點：獵路、邊境護送、先手情報

## 焰潮海人守域包
- 種族：熾潮之脈 或 鱗盾之裔
- 職業：外緣採樣手 / 港巡兵 / 船長
- 風格：守域、巡線、近海應變
- 建議特質：規矩感強、耐壓
- 建議缺陷：排外或太守舊規
- 起始重點：海人紅線、危材、灰線借路

## 靈穹航路護送包
- 種族：翼人 或 人類航務族群
- 職業：空艇航手 / 巡空護航者
- 風格：移動、護送、位置控制
- 建議特質：冷靜、方向感好
- 建議缺陷：怕久戰或討厭封閉地形
- 起始重點：高空航路、泊權、隊伍站位

## 星盞遺跡遠征包
- 種族：精靈分支、學者型人類、海人外來者都可
- 職業：探索者 / 古文讀解手 / 標本師
- 風格：探索、拆讀、封樣
- 建議特質：好奇、耐心
- 建議缺陷：過度深入或執著真相
- 起始重點：遺跡、競爭隊伍、樣本風險

## 杜羅山邦鍛脈工匠包
- 種族：矮人
- 職業：鍛匠 / 黑鍛匠 / 礦工
- 風格：製作、重裝、資源管理
- 建議特質：穩、耐操
- 建議缺陷：固執或慢熱
- 起始重點：工坊、脈線、器具與名聲

## 杜魯摩克灰釘技師包
- 種族：地精
- 職業：誓紋技匠 / 航具維養工 / 估價手
- 風格：工具流、拆改、應急
- 建議特質：手快、點子多
- 建議缺陷：太冒險或愛試爆
- 起始重點：工巧、仿品、灰線器具

## 魔人深淵外流包
- 種族：魔人血裔
- 職業：療傷手 / 情報掮客 / 守庫者
- 風格：高風險、高代價、社會壓力
- 建議特質：冷靜、意志強
- 建議缺陷：名聲差、難被信任
- 起始重點：外流身分、深淵規則、合法性問題
