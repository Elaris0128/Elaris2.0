# 玩家快速上手指南

這份文件給第一次進入 Elaris 的玩家。

## 一、你要先做的事
創角時只要先決定七件事：
1. 你是什麼種族
2. 你主要做什麼職業
3. 你選哪個分支或具體族群
4. 你最強的兩項能力是什麼
5. 你帶什麼特質
6. 你背什麼缺陷
7. 你是怎麼來到這裡的

## 三、每回合怎麼玩
你每回合只需要說：
- 你想做什麼
- 你想怎麼做
- 你最在意什麼結果

例如：
- `我想先觀察門上的誓印，不急著碰。`
- `我想說服他先讓我看樣本，不想驚動守衛。`
- `我用影返步繞到側面，再出手。`

## 四、AI 會怎麼處理
AI / 主持會做四件事：
1. 判斷你在做什麼類型的行動
2. 判斷這件事有多難、多危險
3. 給你結果
4. 更新你的狀態

## 五、最常見的角色方向

### 新手最穩
- 重裝守衛
- 治療藥師
- 學院施術者

### 最容易接世界互動
- 記錄學者
- 自然共鳴者
- 靈廚

### 最有風格
- 契靈行者
- 祭儀執行者
- 符印術者

## 六、能力值怎麼理解
- 體魄：硬不硬、能不能扛
- 身法：快不快、穩不穩
- 心志：撐不撐得住壓力
- 悟性：會不會學、能不能理解
- 靈感：感不感覺得到世界異常
- 靈核：撐不撐得住術式與靈息輸出
- 魅力：會不會說、能不能帶動氣氛
- 洞察：看不看得出問題
- 技巧：手穩不穩、會不會拆裝處理

## 七、種族與職業怎麼搭
- 種族給你 `底色`
- 分支給你 `細部風格`
- 職業給你 `玩法`
- 特殊體質給你 `風格`
- 招式給你 `實際手段`

## 八、第一次遊玩最推薦的組合

### 穩定探險型
- 種族：極光徙裔 / 翼人 / 矮人
- 職業：機動斥候 / 記錄學者 / 重裝守衛

### 海路生存型
- 種族：諾魯因人 / 海人
- 職業：引水人 / 打撈手 / 治療藥師

### 高風格劇情型
- 種族：魔人 / 精靈 / 柯沙里人
- 職業：契靈行者 / 符印術者 / 祭儀執行者

## 九、你不需要一開始就知道全部世界觀
- 先知道角色眼前看得到的事情就夠了
- 先會玩，比先背完設定重要
- 世界真相會在遊戲裡慢慢揭露
