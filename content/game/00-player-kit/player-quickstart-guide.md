# 玩家快速上手指南

這份文件給第一次進入 Elaris 的玩家。

## 一、先記住一件事
Elaris 最推薦的開局，不是讓你一開始就當成熟職業者。
更好的方式是：
- 先當白身
- 先當見習
- 先當跟行者
- 先在某個地方找活、接事、活下來
- 再慢慢長成你自己想要的生活與職業

所以你創角時選的 `職業卡`，預設更像：
- 你想往哪條路長
- 你現在最靠近哪條路
- 世界最可能怎麼命名你

不是一開始就等於你已經完整轉正。

## 二、你要先做的事
創角時先決定這七件事就夠了：
1. 你是什麼種族
2. 你選哪個分支或具體族群
3. 你想從哪種白身狀態開始
4. 你想往哪種生活或職業方向長
5. 你最強的兩項能力是什麼
6. 你帶什麼特質與缺陷
7. 你是怎麼來到這裡的

## 四、開局時會發生什麼
1. 你選完種族 / 分支後，會先落到對應起始區域
2. AI 會把你放在一個 `白身能合理出現的位置`
3. 你先碰到的是：
   - 小委託
   - 小麻煩
   - 地方人情
   - 第一批值得冒險的事
4. 你的正式職業，通常會在前幾章慢慢長出來

## 五、每回合怎麼玩
你每回合只需要說：
- 你想做什麼
- 你想怎麼做
- 你最在意什麼結果

例如：
- `我先看這條潮路到底安不安全，不急著往前。`
- `我想先幫忙搬貨，順便打聽誰在管這段碼頭。`
- `我不想硬碰，我先找比較懂這裡的人。`

## 六、AI 會怎麼處理
AI / 主持會做四件事：
1. 判斷你在做什麼類型的行動
2. 判斷這件事有多難、多危險
3. 給你結果
4. 更新你的狀態與世界回應

## 七、最適合白身開局的角色方向
### 穩定慢長型
- 種族：極光徙裔 / 諾魯因人 / 矮人
- 方向：引水人 / 記錄學者 / 鍛匠 / 療傷手

### 先求生再成名型
- 種族：獸人 / 海人 / 地精
- 方向：狩窗手 / 打撈手 / 港巡兵 / 外緣採樣手

### 高風格晚成型
- 種族：魔人 / 精靈 / 柯沙里人
- 方向：契靈行者 / 符印術者 / 情報掮客 / 祭儀執行者

## 八、你不需要一開始就知道一生要成為誰
- 先知道角色眼前看得到的事情就夠了
- 先會玩，比先背完設定重要
- 先讓世界命名你，再把職業寫死也不遲
- 世界真相與你的人生方向，都可以在遊戲裡慢慢揭露
