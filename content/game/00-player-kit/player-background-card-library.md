# 玩家背景卡庫

這份文件收的是 `玩家創角時可直接選用的背景卡`。

建議用法：
- 起始角色選 `1 張背景卡`
- 背景卡決定你 `從哪裡來、和誰熟、身上為什麼會有這些事`
- 背景比數值更像是 `你與世界的第一條連結`

## 世界常見背景卡

### 邊關長大的孩子
- 你從小就在守門、守夜、守線的環境長大
- 起始連結：邊關軍、城衛、關卡查驗官
- 常見好處：懂規矩、懂輪班、懂警訊
- 常見鉤子：舊長官、失守舊案、邊關調令

### 公會見習生
- 你曾正式或半正式地在某個公會做過見習
- 起始連結：冒險者、藥師、探索、誓紋、靈廚等公會
- 常見好處：知道行規、認識半專業人士
- 常見鉤子：考核沒過、被叫回去補試、舊導師來找

### 遠征殘隊成員
- 你曾參與一支後來出事的遠征隊
- 起始連結：倖存隊友、失蹤名冊、封存報告
- 常見好處：見過大場面，對危險不完全陌生
- 常見鉤子：遺失物回流、死者未死、有人想滅口

### 港邊跑腿出身
- 你從小在港務、泊位、貨棧與灰線消息間穿梭
- 起始連結：港巡兵、船長、司記、掮客
- 常見好處：認路、懂港話、知道怎麼問事
- 常見鉤子：假泊權、舊船票、被借過的名字

### 深山村落出身
- 你出身在遠離大城的山村、谷地或高原聚落
- 起始連結：村老、看火人、守墓者、採藥人
- 常見好處：自然生存、熟手藝、扛得住孤單
- 常見鉤子：祖地異變、回鄉求援、老路重開

### 水道與濕地長成
- 你從小在水網、濕地、潟谷或潮帶邊生活
- 起始連結：引水人、採藥人、沼地醫者、渡手
- 常見好處：判水、判味、判潮
- 常見鉤子：禁採樣本、失蹤採手、浮屍來歷

### 高空交通圈出身
- 你成長在空艇、橋港、懸鏈或高差交通帶
- 起始連結：空艇航手、巡翼哨手、風路技匠
- 常見好處：不怕高差、懂風窗、懂吊運
- 常見鉤子：橋索失事、舊票據、風路封鎖

### 冬聚地長大
- 你從小在季節聚散明顯的冬聚地或狩聚地生活
- 起始連結：守火者、旅灶長、狩窗手、送魂師
- 常見好處：共享資源、分配秩序、熬冬能力
- 常見鉤子：分肉舊帳、失路回尋、獵線重劃

### 學院旁聽生
- 你不一定有正式身份，但你混過學院或講堂
- 起始連結：講師、抄錄員、學者、門房
- 常見好處：會查書、會聽行話、懂規則漏洞
- 常見鉤子：被欠的人情、借來的名額、抄錯的禁頁

### 祀場服役者
- 你曾在神殿、祭場、安魂圈或供饌隊服役
- 起始連結：祭司、守墓者、供饌手、祈名手
- 常見好處：懂禮、穩場、知道禁忌
- 常見鉤子：祭物失蹤、神諭誤解、舊祭案

### 黑市熟面
- 你並不一定是罪犯，但你熟灰線
- 起始連結：情報掮客、黑市商、洗印手、偽封樣客
- 常見好處：懂不能明說的規矩
- 常見鉤子：假貨回流、被追債、舊主顧現身

### 旅團養大
- 你是被商團、藝團、遠行隊或護送隊養大的
- 起始連結：商人、巡演藝人、護衛、船長
- 常見好處：適應力強、不怕換地方
- 常見鉤子：舊團解散、團長失蹤、路上仇家

### 災後倖存者
- 你的人生被一場戰、災、異變或怪物事故切開過
- 起始連結：同災倖存者、收容者、災後官員
- 常見好處：知道局勢一壞會變多快
- 常見鉤子：舊災重現、責任追討、未被找到的東西

### 祖物保管家系
- 你出身在看守某件祖物、家印、舊器或封物的家系
- 起始連結：祖物看守人、文簿官、守誓者
- 常見好處：懂保存、懂家法、懂不能外流的東西
- 常見鉤子：祖物流失、真假難辨、有人來認領

### 海上長大
- 你是在船、港、礁、潮間與泊棚邊長大的
- 起始連結：漁戶、船長、引水人、港巡兵
- 常見好處：不怕晃、不怕海味、懂船上秩序
- 常見鉤子：老船舊債、港票問題、沉船消息

### 墓地與送魂圈長大
- 你從小就和死者、墓地與送魂儀式很近
- 起始連結：守墓者、送魂師、祭司、療傷手
- 常見好處：不容易怕屍與死亡場面
- 常見鉤子：無主遺骨、送魂錯誤、死者回名

### 工坊學徒
- 你長時間泡在工坊、鍛爐、拼裝台與庫房裡
- 起始連結：鍛匠、誓紋技匠、航具維養工
- 常見好處：懂修、懂料、懂工具
- 常見鉤子：被偷圖、出問題的成品、師承競爭

### 失路回還者
- 你曾經走失、迷航、誤入禁段，最後又活著回來
- 起始連結：回收隊、守路者、少數信你的人
- 常見好處：對危機與回路有特殊警覺
- 常見鉤子：那次你到底看見了什麼、誰沒有回來

### 被保下來的人
- 你年少時曾被某個重要人物、公會或聚落保過一命
- 起始連結：恩人、恩人的舊部、欠下的人情
- 常見好處：一條特別的人脈線
- 常見鉤子：對方來討回報、恩人出事、旁人誤會你的來歷

### 逃離原本人生的人
- 你不是去旅行，你是離開了原本不想待的地方
- 起始連結：舊家、舊師門、追你的人、少數知道真相的人
- 常見好處：韌性高，懂得重新做人設
- 常見鉤子：舊身份被翻出、追兵、被迫回頭

## 怎麼選最順

如果你不知道哪張背景卡最適合你，可以先想：
1. 你更想讓角色 `認識什麼人`
2. 你更想讓角色 `背著什麼舊事`
3. 你希望第一條主動找上你的劇情是 `工作 / 舊識 / 家鄉 / 災後 / 黑市 / 公會` 哪一種

通常這三題一答，背景卡就很容易挑出來。
