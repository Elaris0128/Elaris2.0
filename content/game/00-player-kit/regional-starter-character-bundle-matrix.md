# 地區對應的玩家開局角色包

這份是把 `起始區域` 和 `推薦角色包` 接起來。

## 幽語海
- 最適合：`幽語海引路者包`
- 次適合：`港邊失蹤調查型的學者或療傷手`

## 焰潮洋
- 最適合：`焰潮海人守域包`
- 次適合：`船長 / 港巡兵 / 灰線交易型角色`

## 暮鏡荒境
- 最適合：`暮鏡重裝守界包`、`暮鏡獵路斥候包`
- 次適合：`寒境療傷手 / 邊境護送者`

## 王都都市圈
- 最適合：`王都記錄學者包`
- 次適合：`關卡查驗官 / 城衛兵 / 商人`

## 港邊航務圈
- 最適合：`靈穹航路護送包`
- 次適合：`引水人 / 船長 / 稅簽官`

## 星盞遠征圈
- 最適合：`星盞遺跡遠征包`
- 次適合：`標本師 / 探索者 / 古文讀解手`

## 杜羅山邦
- 最適合：`杜羅山邦鍛脈工匠包`
- 次適合：`礦工 / 守庫者 / 估價手`

## 杜魯摩克隱域
- 最適合：`杜魯摩克灰釘技師包`
- 次適合：`情報掮客 / 航具維養工 / 偽封樣工`

## 亡極徹淵外環與裂廊帶
- 最適合：`魔人深淵外流包`
- 次適合：`療傷手 / 情報掮客 / 祖物看守人`
