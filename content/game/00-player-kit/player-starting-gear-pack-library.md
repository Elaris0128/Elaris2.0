# 玩家起始裝備包

這份文件收的是 `玩家創角時可直接挑選的起始裝備包`。

使用原則：
- 先依主職業選 `1` 包最相近的職業裝備包
- 若你的背景很強烈，也可以把其中 `1-2` 件換成符合背景的物件
- 如果你有副職業，可以再從對應領域補 `1` 件工具或副武器

## 換裝規則

- 可把裝備包內 `1` 件物品，換成同價值、同用途、同體積級別的地方版本
- 例如：
  - 普通披風 -> 耐潮披
  - 通用短刀 -> 海骨短刀
  - 普通火袋 -> 極地火袋
- 若背景非常鮮明，可再加 `1` 件背景紀念物，但它不應直接等於高戰力道具
