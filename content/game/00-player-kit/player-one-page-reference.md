# 玩家一頁速查表

## 先記住
- 你不需要一開始就當成熟職業者。
- 預設推薦：`白身 / 見習 / 跟行者 / 候補` 開局。
- 先活下來、先認識地方、先接第一批小事，再慢慢長成自己的生活與職業。

## 最短創角流程
1. 選 `種族`
2. 若需要，再選 `正式分支`
3. 選 `白身狀態`
4. 選 `想長成的職業方向`
5. 選 `兩項核心強項`
6. 選 `特質 / 缺陷 / 背景`
7. 選 `起始裝備包 / 資源包`
8. 選 `起始戰技 / 術式`

## 哪些種族要再選分支
- 海人
- 精靈
- 獸人
- 魔人

## 哪些不用再拆
這類已經是具體族群名：
- 諾魯因人
- 極光徙裔
- 柯沙里人
- 薩魯恩人
- 矮人
- 地精
- 翼人

## 白身開局怎麼想
你開局比較像：
- 家裡幫忙的人
- 跟著前輩跑的學徒
- 臨時短工
- 隊尾候補
- 還沒轉正的候補者

## 你每回合只要說三件事
- 你想做什麼
- 你想怎麼做
- 你最在意什麼結果

## AI / 主持會做什麼
1. 判斷你的意圖
2. 判斷阻力與風險
3. 給出結果
4. 更新你的狀態與世界回應

## 開局不是固定劇本
流程是：
`種族 / 分支 -> 起始區域 -> AI 在該區域內隨機組開局`

## 常見起始感受
- 幽語海：低聲、守路、海岬、送魂、潮窟
- 靈穹洋：航權、封樣、公會、懸流航務
- 焰潮洋：熱泉、危材、海人守域、借路風險
- 梵洛特：裂帶、隱林、谷寨水行、沿岸哨域與高壓生存
- 伊莎梅爾：精靈文明域、樹海、寒域、古道與責任網

## 遊戲裡最常見的三種行動
- 探索：找路、觀察、追跡、辨認異常
- 社交：問話、說服、壓價、立誓、借道
- 危險行動：搏鬥、突進、撤離、保人、搶先手

## 如果你不知道怎麼開口
直接說這種句子就可以：
- `我先觀察這裡有沒有不對勁。`
- `我想問問看誰真的知道這條路。`
- `我先幫忙做小事，順便打聽消息。`
- `我不想硬碰，我先找安全的做法。`

## 成長怎麼發生
- 不是靠等級條突然變強
- 而是靠：
  - 活下來
  - 被地方認識
  - 拿到信任 / 聲望 / 執照
  - 完成第一批真正像樣的事
  - 慢慢轉正成職業者
