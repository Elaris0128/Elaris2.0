# 玩家一頁速查表

## 一回合怎麼說
- `我想做什麼`
- `我怎麼做`
- `我最在意什麼結果`

## 創角時怎麼看種族
- 看到 `海人 / 精靈 / 獸人 / 魔人`：先選種族，再選分支
- 看到 `極光徙裔 / 諾魯因人 / 柯沙里人` 這種具體族群名：直接選
- 看不懂時直接查 [玩家種族與分支怎麼選](C:/Users/rabbi/Desktop/Elaris/content/game/00-player-kit/player-race-and-branch-selection-guide.md)
- 要列完整分支時，直接查 [玩家種族分支清單](C:/Users/rabbi/Desktop/Elaris/content/game/00-player-kit/player-race-branch-library.md)
- 想快速比較分支手感時，直接查 [玩家分支卡庫](C:/Users/rabbi/Desktop/Elaris/content/game/00-player-kit/player-branch-card-library.md)
- 想直接知道能用什麼能力時，查 [玩家種族能力卡庫](C:/Users/rabbi/Desktop/Elaris/content/game/00-player-kit/player-race-ability-card-library.md) 與 [玩家分支能力卡庫](C:/Users/rabbi/Desktop/Elaris/content/game/00-player-kit/player-branch-ability-card-library.md)

## 常見行動類型
- 觀察
- 詢問
- 交涉
- 潛行
- 攻擊
- 施法
- 使用物品
- 製作 / 處理
- 休整
- 撤退

## 九項能力速查
- 體魄：硬扛、負重、耐打
- 身法：移動、閃避、潛行
- 心志：抗壓、抗魅惑、穩定誓約
- 悟性：理解、解讀、學術、術式結構
- 靈感：感知靈息、異象、地脈
- 靈核：靈息容量、輸出、穩定度
- 魅力：說服、壓場、社交
- 洞察：看破、察覺、判斷
- 技巧：拆裝、精密處理、手工

## AI 判定結果五階
- 完全成功
- 成功但有代價
- 停滯但有收穫
- 失敗且局勢惡化
- 嚴重失敗

## 難度六階
- 自動成功
- 簡單
- 標準
- 困難
- 極難
- 傳聞級

## 風險五階
- 安全
- 曝露
- 危險
- 重危
- 災變

## 最常見的雙能力組合
- 潛行：身法 + 洞察
- 近戰強攻：體魄 + 身法
- 交涉：魅力 + 洞察
- 穩定施法：悟性 + 心志
- 引息施法：靈感 + 靈核
- 解讀古文：悟性 + 洞察
- 緊急處理：技巧 + 心志

## 什麼時候不用判定
- 沒阻力
- 沒風險
- 你明顯做得到
- 只是查自己已知資訊

## 什麼時候常會判定
- 有對手
- 有時間壓力
- 有風險
- 有隱藏資訊
- 失敗會改變局勢

## 新手最穩職業
- 重裝守衛
- 治療藥師
- 學院施術者

## 創角再補這三張卡
- 特質：決定你最常出彩的方式
- 缺陷：決定 AI 最容易從哪裡推劇情
- 背景：決定你和世界的第一條關係線

## 新手最實用招式
- 承山架
- 影返步
- 尋脈引
- 冷息護皮
- 靜域環
