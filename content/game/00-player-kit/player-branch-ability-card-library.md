# 玩家分支能力卡庫

這份文件收的是 `已經選了大種族之後，要再套上的分支能力卡`。

適用對象：
- 海人三分支
- 精靈十支
- 獸人正式分支
- 魔人七血裔

## 海人分支能力卡

### 潮霧之裔｜潮囊 / 霧鰭脈
- 被動：迷霧、暗流與低光潮帶中的潛行與改道更強
- 啟用：`霧潮潛渡`
  - 在潮霧、礁影或低光水域快速降低存在感並轉位
- 代價：乾燥、高曝曬與完全無濕度環境會壓低效果

### 熾潮之脈｜熾脈核 / 鰭刺束
- 被動：耐熱、耐壓與危窗爆發能力更強
- 啟用：`珊脈灼流擊`
  - 在高熱、高壓或危險水窗中做一次強行突破
- 代價：過度連發容易反噬體力與穩定度

### 鱗盾之裔｜鱗盾背甲 / 潮聲鰭刺
- 被動：守線、迎浪與群護能力更穩
- 啟用：`礁盾迎浪`
  - 短時間強化你在正面迎壓與保護同伴上的表現
- 代價：單走與高速灰線滲透不是這支的長處

## 精靈分支能力卡

### 雲瑟皇族
- 被動：王庭禮印、場面秩序與正式交涉更穩
- 啟用：`雲誓定場`
- 代價：灰線與失序污場中不易吃到完整優勢

### 雪華部族
- 被動：寒地照護、療養與長線守冬更強
- 啟用：`雪華回名息`
- 代價：爆衝、躁動、熱地久耗較不利

### 霜諾嵐族
- 被動：長夜、夢兆、夜哨與極寒判兆更準
- 啟用：`霜岚守夜視`
- 代價：喧鬧與高濕低地會壓低穩定

### 密蘭族
- 被動：濕地、花粉、水網和草藥判讀更有優勢
- 啟用：`密蘭隱汀步`
- 代價：乾燥硬石地較難發揮

### 歐洛賽族
- 被動：夜潮、潮影林與低調觀局更強
- 啟用：`歐洛潮影借位`
- 代價：過度明亮、過度公開的場面較不利

### 奧黎梵族
- 被動：祝福、穩場、正向引導更強
- 啟用：`晨藤安序歌`
- 代價：污穢場與惡意壓迫會較快干擾你

### 菲露涅族
- 被動：高枝守望、遠觀、風聽與吊橋行動更穩
- 啟用：`菲露聽風步`
- 代價：密閉地下與無高差空間較不利

### 維塔蕾族
- 被動：漂居、應變、浮橋與移動聚落的生活技術更強
- 啟用：`維塔漂橋轉身`
- 代價：軍紀過重或超固定流程會卡住你

### 瑟塔嶺族
- 被動：高原古道、長行與遠征穩定性更高
- 啟用：`瑟塔星泉定途`
- 代價：長期困於都城與繁瑣文場會壓低節奏

### 繭影族
- 被動：低光、保密、幻繭遮蔽與耐心觀察更強
- 啟用：`繭影靜覆`
- 代價：持續曝光與公開領導位會削弱表現

## 獸人分支能力卡

### 十獸之環・克羅嵐共誓支
- 被動：群隊行動、戰舞蓄勢與共誓壓線更穩
- 啟用：`共誓戰步`
- 代價：脫隊、單走與無法配合同伴時優勢下降
- 補充：如果再細選十獸席位，可再疊一層席位偏好

#### 十獸席位簡表
- `馭角`：護群與抗線更強
- `逐影`：追獵與聞痕更強
- `鷹目`：遠觀與判位更強
- `裂牙`：近身壓迫與咬線更強
- `斑影`：伏擊與混戰調位更強
- `隱耳`：警戒與偷聽風向更強
- `馭嶺`：高地行進與重踏更強
- `蟄蛇`：耐心、暗伏、反咬更強
- `裂痕`：巡領與邊線施壓更強
- `冥羽`：死線感、送亡與不祥預警更強

### 裂蠍沙裔
- 被動：伏沙、裂谷與乾熱裂地更有利
- 啟用：`裂谷藏針步`
- 代價：極寒與持續潮濕削弱你

### 霧黽裔
- 被動：煙谷、吐霧、濕地掩護更有利
- 啟用：`霧黽裂蹼伏線`
- 代價：大風與長期乾燥壓低表現

### 熔炭龜裔
- 被動：耐熱、厚甲、熔痕重壓更強
- 啟用：`熔炭壓線殼`
- 代價：超低溫與高機動追擊較吃虧

### 喚隱裔
- 被動：隱林、伏蹤與耐心等待更強
- 啟用：`喚隱繭伏`
- 代價：公開社交與硬碰硬較不擅長

### 潮聲鳥裔
- 被動：海崖回聲、風潮與遠訊判位更強
- 啟用：`潮聲回位鳴`
- 代價：完全封閉無聲場域會削弱優勢

### 渣咬裔
- 被動：荒地殘料利用、低資源求活更強
- 啟用：`渣咬硬熬息`
- 代價：細密貴重品處理與高禮制場合吃虧

### 焦燄鼬裔
- 被動：裂縫、高熱與短時高速轉位更強
- 啟用：`焦燄急轉尾`
- 代價：久戰與守線不如其他分支

### 礦鼠裔
- 被動：坑道、探礦、細裂道與聽震更強
- 啟用：`礦鼠碎步聽脈`
- 代價：寬闊平地與空中壓制較不利

### 星獅裔
- 被動：高地守勢、正面領隊與威壓更強
- 啟用：`星獅咆陣`
- 代價：灰線滲透與長期潛伏較不擅長

### 月影狼裔
- 被動：夜嚎、群追、失路回帶與雪原獵線更強
- 啟用：`月影回嚎`
- 代價：單走與節奏被切斷時優勢下降

### 霜爪熊裔
- 被動：守火、熬冬、扛線與照顧傷線更強
- 啟用：`霜爪守火勢`
- 代價：高速追逐與細密偽裝任務吃虧

### 影尾狐裔
- 被動：回尋、辨假兆、偽裝與灰區判斷更強
- 啟用：`影尾折返痕`
- 代價：正面硬扛與高調壓線較吃虧

### 暮羽雀裔
- 被動：守望、傳訊、先兆與風訊判位更強
- 啟用：`暮羽先聲警`
- 代價：被拖進重裝近戰時較難舒服發揮

## 魔人分支能力卡

### 曜血｜曜血核脈
- 被動：精準、血律與誓場壓制更強
- 啟用：`曜脈細律斬`
- 代價：核脈受創時會快速失穩

### 鳴紗｜魅息腺
- 被動：魅惑、幻術、紗幕掩護與操引更強
- 啟用：`紗魅引心絲`
- 代價：腺體受損時可能連自己都失控

### 垢金｜垢金核
- 被動：封存、契錄、重器承載與庫權操作更強
- 啟用：`沉金契壓印`
- 代價：核裂時會有嚴重反噬

### 幽鬩｜影心珠
- 被動：伏影、奪形、偷名與模糊身分更強
- 啟用：`影珠裂步奪形`
- 代價：長時間曝光會讓這支風格很難發揮

### 獰罡｜血爆囊
- 被動：近戰爆發、血焰壓場與重衝更強
- 啟用：`罡焰重衝`
- 代價：過度連續使用容易把自己也拖進失控

### 緲眠｜夢核
- 被動：低吟、夢寄、慢性牽引與安撫更強
- 啟用：`夢核低鳴罩`
- 代價：夢核失衡時可能反過來困住自己

### 噬淵｜噬源
- 被動：吞噬、轉質、極端環境壓迫與骨刃變形更強
- 啟用：`噬源裂口`
- 代價：這支的反噬風險是七血裔裡最高的之一

## 裁定原則

- 一般情況下，每個角色只套 `1 張種族能力卡 + 1 張分支能力卡`
- 沒有分支的種族，就只用種族能力卡
- `瓦雷溫人 / 法奧人 / 法雷恩人` 這三支不給固定先天能力卡，請改用 `背景 / 職業 / 特質`
- 如果玩家創角走的是新手模式，分支能力卡可先只拿 `被動 + 啟用技`
