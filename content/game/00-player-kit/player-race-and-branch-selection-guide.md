# 玩家種族與分支怎麼選

## 先記住一條
玩家創角時先選 `種族`。  
如果這個種族有正式分支，再往下選 `分支`。  
如果你選的是已經很具體的族群名，就不再往下拆。

## 需要再選分支的種族
- `海人`
- `精靈`
- `獸人`
- `魔人`

這四類都不是直接可用的最終答案。  
玩家說「我要玩海人」時，AI 還要再問分支。  
玩家說「我要玩諾魯因人」時，AI 不需要再問。

## 不再往下拆的常用玩家種族

### 人類系具體族群
- `法奧人`
- `瓦雷溫人`
- `法雷恩人`
- `極光徙裔`
- `魈陽族`
- `崖影族`
- `巖穹族`
- `薩魯恩人`
- `柯沙里人`
- `諾魯因人`

### 其他直接可選種族
- `翼人`
- `矮人`
- `地精`
- `森馳裔`

## 海人怎麼選
先選 `海人`，再選三支之一：
- `潮霧之裔`
- `熾潮之脈`
- `鱗盾之裔`

## 精靈怎麼選
先選 `精靈`，再從正式分支中挑一支。  
完整清單看：

## 獸人怎麼選
先選 `獸人`，再選正式支系。  
如果你選的是 `十獸之環`，可以再選一次內部席位。  
如果你選的是 `暮鏡荒境四獸裔`，則直接把那四支視為玩家可選的正式獸人支系。

### 十獸之環可再細選的席位
- `馭角`
- `逐影`
- `鷹目`
- `裂牙`
- `斑影`
- `隱耳`
- `馭嶺`
- `蟄蛇`
- `裂痕`
- `冥羽`

### 暮鏡荒境四獸裔
- `月影狼裔`
- `霜爪熊裔`
- `影尾狐裔`
- `暮羽雀裔`

## 魔人怎麼選
先選 `魔人`，再選七血裔之一：
- `曜血`
- `鳴紗`
- `垢金`
- `幽鬩`
- `獰罡`
- `緲眠`
- `噬淵`

## 不可玩家選擇的族類
- `龍族`
- `鳳族`
- `神獸古族`

這些可以作為世界觀角色、傳說存在、任務對象或災害級存在，但不放進玩家可選種族。

## AI / 主持的判定規則
- 玩家只說 `海人 / 精靈 / 獸人 / 魔人`：必須追問分支。
- 玩家說 `諾魯因人 / 極光徙裔 / 矮人 / 翼人`：直接視為種族選擇完成。
- 玩家說 `十獸之環`：可再追問是否細選席位。
- 玩家說 `月影狼裔 / 霜爪熊裔 / 影尾狐裔 / 暮羽雀裔`：直接視為獸人分支已完成。
