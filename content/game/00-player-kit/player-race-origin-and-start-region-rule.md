# 種族出身地與起始區域規則

## 核心原則
玩家不是先挑一個任意城市開局，  
而是先挑 `種族 / 分支`，再落到該族群對應的 `起始國家、海域或文化圈`。

## 開局流程
1. 選 `種族`
2. 如果該種族有正式分支，先選 `分支`
3. 依 [玩家起始區域與開局包對照表](C:/Users/rabbi/Desktop/Elaris/content/game/00-player-kit/player-start-region-and-opening-pack-matrix.md) 鎖定起始區域
4. 套用該區域的 [玩家地區專屬開局包](C:/Users/rabbi/Desktop/Elaris/content/game/00-player-kit/regional-opening-pack-library.md)
5. AI 只在那個地區內隨機抽 `第一問題 / 第一互動 / 第一麻煩 / 第一批任務`

## 這條規則的目的
- 保住種族與文化的出身感
- 讓每個角色的第一章有明顯地方味
- 避免 AI 把角色隨機丟到毫無關聯的區域
- 讓既有地區任務、NPC、派系與地方麻煩更容易接上

## 例外怎麼處理
只有在玩家背景明確寫出 `流亡 / 留學 / 調任 / 押運 / 被販運 / 遷徙` 這種例外時，AI 才可以讓角色從非本族主居圈開始。  
但就算有例外，也應該是 `相鄰文化圈`、`可合理抵達的海域` 或 `有明確故事原因的外派點`，不應該完全隨機。

## 種族與起始圈的讀法

### 已具體的人類族群
像 `諾魯因人 / 極光徙裔 / 柯沙里人` 這種名字，本身就已經包含出身圈。  
AI 不需要再猜測「這是哪一類人類」，而應直接落到對應區域。

### 需要細選的大種族
- `海人`：先分三支，再落到對應海域守域圈
- `精靈`：先分十支，再落到對應本域
- `獸人`：先分正式支系，再落到對應聚落圈
- `魔人`：先分七血裔，再落到對應亡極徹淵層帶

## AI 裁定規則
- 玩家只說 `我要玩海人`：不能直接開局，先追問三支。
- 玩家只說 `我要玩獸人`：先列獸人正式支系，再決定開局區域。
- 玩家已經說 `我要玩諾魯因人`：直接落 `幽語海`。
- 玩家已經說 `我要玩曜血`：直接落 `亡極徹淵・裂廊上層`。

## 任務橋接原則
如果該起始區域已經有既有任務池，AI 應優先從既有任務中抽第一批開局委託。  
如果該區域目前還沒有現成任務池，才用地區專屬開局包中的麻煩與第一目標先撐起第一章。
