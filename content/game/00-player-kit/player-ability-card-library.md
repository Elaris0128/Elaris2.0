# 玩家能力卡庫

這份文件把能力值壓成玩家最容易理解的卡片版本。

## 六大核心能力

### 體魄
- 你在什麼時候會用到：
  - 扛傷
  - 破門
  - 負重
  - 長途跋涉
- 最常搭配：
  - 身法
  - 心志
  - 技巧
- 適合誰：
  - 近戰前衛
  - 重裝守衛
  - 礦工

### 身法
- 你在什麼時候會用到：
  - 閃避
  - 潛行
  - 追擊
  - 崖地轉線
- 最常搭配：
  - 洞察
  - 體魄
  - 技巧
- 適合誰：
  - 機動斥候
  - 巡翼哨手
  - 引水人

### 心志
- 你在什麼時候會用到：
  - 抗恐懼
  - 抗魅惑
  - 維持誓約
  - 高壓時不崩盤
- 最常搭配：
  - 悟性
  - 靈核
  - 體魄
- 適合誰：
  - 守誓戰士
  - 祭儀執行者
  - 重裝守衛

### 悟性
- 你在什麼時候會用到：
  - 解讀古文
  - 理解術式
  - 學術推理
  - 複雜工藝
- 最常搭配：
  - 心志
  - 洞察
  - 技巧
- 適合誰：
  - 學院施術者
  - 記錄學者
  - 製藥師

### 靈感
- 你在什麼時候會用到：
  - 感知異象
  - 察覺靈息
  - 與地脈、場域、契靈互動
- 最常搭配：
  - 靈核
  - 悟性
  - 洞察
- 適合誰：
  - 契靈行者
  - 自然共鳴者
  - 符印術者

### 靈核
- 你在什麼時候會用到：
  - 承載術式
  - 穩住靈息輸出
  - 支撐高風險能力
- 最常搭配：
  - 靈感
  - 心志
  - 悟性
- 適合誰：
  - 學院施術者
  - 契靈行者
  - 魔人血裔角色

## 三項社會能力

### 魅力
- 你在什麼時候會用到：
  - 說服
  - 領導
  - 壓場
  - 安撫與談判

### 洞察
- 你在什麼時候會用到：
  - 看穿說謊
  - 讀局勢
  - 查異常
  - 抓細節

### 技巧
- 你在什麼時候會用到：
  - 開鎖
  - 拆裝
  - 精密處理
  - 現場修補

## 新手最常用的能力組合
- 潛行：`身法 + 洞察`
- 破門：`體魄 + 技巧`
- 勸說：`魅力 + 洞察`
- 穩定施法：`悟性 + 心志`
- 引息施法：`靈感 + 靈核`
- 急救：`悟性 + 技巧`
- 趕路：`體魄 + 心志`
- 查遺跡：`悟性 + 洞察`

## 起始分配建議
- 選 `2` 項最強
- 選 `2` 項次強
- 其餘維持普通
- 保留 `1` 項明顯弱點

## 記住這件事
能力值不是拿來限制創意的，  
而是幫 AI 判斷：`你比較擅長怎麼把創意做成現實。`
