# 玩家配方卡庫

這份是給玩家最快挑配方方向用的。

## 保命藥膏卡
- 類型：藥物
- 推薦職業：藥師、療傷手
- 難度：低
- 最重要條件：穩定藥泥 + 可封存容器

## 長途旅糧卡
- 類型：料理
- 推薦職業：靈廚、旅灶長
- 難度：低
- 最重要條件：耐放主糧 + 乾燥或煙熏設備

## 夜航靜燈卡
- 類型：器具
- 推薦職業：誓紋技匠、航具維養工
- 難度：中
- 最重要條件：穩定燈罩 + 遮光材料

## 封樣小瓶卡
- 類型：工具
- 推薦職業：藥師、標本師、外緣採樣手
- 難度：低
- 最重要條件：可封口瓶體 + 不漏液塞件

## 暫時保溫袋卡
- 類型：旅裝
- 推薦職業：商人、遠征後勤員、靈廚
- 難度：低
- 最重要條件：布層 + 保溫填料 + 束口件

## 低階武器補強件卡
- 類型：鍛件
- 推薦職業：鍛匠、黑鍛匠
- 難度：中
- 最重要條件：韌性材料 + 可淬火設備
