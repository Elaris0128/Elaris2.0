# 地區工坊與可用設備庫

## 王都 / 學城
- 有：正式藥灶、學術器材、封樣工具、細工台
- 優勢：精細製作、研究型成品、合法販售
- 缺點：流程慢、管制多

## 港邊 / 航務圈
- 有：旅灶、公用修補台、繩具架、簡易封樣桌
- 優勢：補貨快、做旅裝和海路補給很方便
- 缺點：深度研究型設備不足

## 工坊圈 / 鍛脈地
- 有：鍛台、冷淬槽、夾鉗、重型切割設備
- 優勢：武裝、補強件、耐用器具
- 缺點：精細藥物和食譜不一定好做

## 寒地 / 極地據點
- 有：暖室、保溫箱、乾燥架、冰封保存區
- 優勢：保存和極地裝備
- 缺點：液體與細工容易受溫差干擾

## 海島 / 外海棚
- 有：魚材處理台、煙熏架、鹽洗槽
- 優勢：海產、海怪素材、旅糧
- 缺點：精密器具稀少

## 灰線作坊
- 有：替代料、假封件、便宜工具
- 優勢：低成本、速度快
- 缺點：失敗率高、合法性差
