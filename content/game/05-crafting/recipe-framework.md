# 配方通用框架

這份是玩家版的製作母規則，能套用在：
- 製藥
- 料理
- 鍛造
- 工具組裝
- 探索素材處理

## 一張配方最少要有六件事
1. `主材`
2. `輔材`
3. `工具`
4. `場地`
5. `流程`
6. `目標型態`

## 配方公式
成品結果可用這個簡式判斷：

`成品分 = 主材適性 + 輔材協同 + 工具加值 + 場地加值 + 操作者相性 + 時間投入 - 風險值 - 相斥值`

## 主材適性
主材決定這件東西最像什麼。

常見主材：
- 藥材
- 怪物素材
- 礦材
- 料理主食
- 功能性樹脂 / 油 / 囊液
- 儀式性粉末 / 灰 / 血 / 符片

## 輔材協同
輔材不是越多越好，而是要和主材同方向。

最常見的協同方向：
- `穩定`
- `放大`
- `轉形`
- `保鮮`
- `導香 / 導味`
- `封存`
- `降毒`

## 工具加值
工具不只決定加值，也決定你能不能做這種東西。

常見工具：
- 煎煮鍋、滴定管、壓汁石、封瓶尺
- 鍛台、夾鉗、風箱、冷淬槽
- 廚刀、熬汁鍋、保溫盒、發酵罈
- 誓紋筆、對讀盤、穩式架

## 場地加值
場地通常比分數更重要，因為它會決定某些材料是否失活。

常見場地：
- 學城工坊
- 港邊公用灶
- 公會試驗室
- 船上簡易灶
- 邊地臨時棚
- 隔離封棚
- 極地暖室

## 目標型態
玩家做東西前，要先定成品長什麼樣。

常見型態：
- `固體`
- `液體`
- `氣體`
- `粉末`
- `膏體`
- `料理`
- `器具`

## 成功區間
- `穩定成功`：照預期完成，可直接使用或販售
- `缺陷成功`：有用，但耐久、保存或效果打折
- `特異成功`：主效果成立，並多出額外特性
- `副作用成功`：做成了，但附帶風險或額外代價
- `廢品`：沒達成目標，只能拆料或丟棄
- `反噬失敗`：污染、爆裂、誤燃、變質或傷人

## 玩家最快用法
1. 先定 `想做出什麼`
2. 再挑 `一個主材`
3. 補 `兩個同方向輔材`
4. 確認 `自己有沒有工具和場地`
5. 決定要拼穩定、拼效果，還是拼便宜

## 例子
### 想做保命藥膏
- 主材：止血藥泥
- 輔材：穩定樹脂、去腐鹽
- 工具：研槽、封盒
- 場地：藥師灶台
- 型態：膏體
- 目標：穩定成功優先

### 想做長途耐行糧
- 主材：乾肉或穀糧
- 輔材：保鮮鹽、功能性草粉、提神油
- 工具：熬鍋、乾燥架
- 場地：旅灶或港邊公用灶
- 型態：料理 / 固體
- 目標：保存與穩定優先

### 想做短兵器升級件
- 主材：礦材或獸骨件
- 輔材：韌筋膠、導式片
- 工具：鍛台、夾鉗、冷淬槽
- 場地：鍛工坊
- 型態：器具
- 目標：功能成功優先
