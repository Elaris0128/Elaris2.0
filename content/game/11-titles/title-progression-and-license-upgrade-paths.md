# 升階路線與執照升級路徑

這份文件用來把 `頭銜 / 資格 / 執照` 做成真正會成長的線，而不是一張拿到就結束的證件。

## 共通升階條件
- 能力達標
- 聲望達標
- 實務案例達標
- 無重大失格紀錄

## 冒險者線
- 見習牌 -> 正式誓章 -> 帶隊章 -> 危地臨行許可

常見升階條件：
- 至少完成數件正式委託
- 有一件中型危機處理紀錄
- 無棄隊或重大謊報

## 藥師線
- 助手證 -> 封樣手牌 -> 調製認證 -> 隔離棚執行許可

常見升階條件：
- 樣本處理穩定
- 無污染事故
- 能獨立完成中危材流程

## 探索線
- 跟隊印牌 -> 記錄准入 -> 勘界證 -> 高價遺跡開掘席位

常見升階條件：
- 座標與記錄可信
- 至少有一次高價值發現
- 未毀損關鍵遺跡

## 靈廚線
- 基礎危材處理牌 -> 試味保證牌 -> 宴席主理牌 -> 特危材調度權

常見升階條件：
- 無重大食安事故
- 能在高壓環境穩住危材流程
- 有正式宴席見證

## 升階失敗不一定是退回原點
常見結果：
- 延後審查
- 補案例
- 改成臨時牌
- 限制用途

這樣比較符合長團玩法，不會一失敗就把整條線砍掉。
