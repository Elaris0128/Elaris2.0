# 稱號、資格與執照索引

這一層處理角色在世界裡被 `怎麼認定`。

它不只包含稱呼，還包含：
- 你是否有資格做某件事
- 你能不能拿某種貨、進某種區、接某種案
- 你是否要對某個制度負責
- 失格之後世界會怎麼對你

## 這層能回答什麼
- 某角色拿的是榮譽稱號、正式資格、地方身分，還是制度職銜
- 某資格在哪些地區有效
- 沒有執照時會不會被查、被扣、被趕
- 頭銜會帶來哪些人脈、義務、敵意與曝光
- 同一張執照到了別國，是被完全承認、部分承認，還是直接不承認
- 一個高階稱號往往要先經過哪些資格、戰績與聲望條件

## 使用建議
- 玩家創角時不用一開始就持有很多頭銜。
- 真正重要的是 `一張能改玩法的資格`，而不是一堆只有名字的稱號。
- 高階頭銜應同時帶來權益、責任與壓力。
