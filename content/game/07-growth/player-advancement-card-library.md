# 玩家升階卡庫

## Purpose
這份文件把玩家從 `起步 -> 立足 -> 專精 -> 成名 -> 顯赫 -> 傳聞級` 的升階感，做成可直接對讀的卡。

## 升階卡怎麼看
每張卡會回答：
- 什麼時候差不多該升
- 升上去會帶來什麼
- 哪種代價會一起來
- 最適合哪類角色

## 1. 起步 -> 立足

### 升階感
- 你已經不只是被帶著跑的人
- 你能穩定完成一般工作

### 常見條件
- 完成 2-3 次正式任務
- 至少取得一條地方見證或正式證明
- 有一項主職表現足夠穩

### 常見收穫
- 可承接灰階任務
- 可擁有第一張地方執照或窗口身分
- 起始招式與工具用得更穩

### 常見代價
- 會開始被真正記住
- 失敗不再只算新手失誤

### 最適合
- 新手角色的第一個短期目標

## 2. 立足 -> 專精

### 升階感
- 你在某條路上開始被視為可靠

### 常見條件
- 主職或副職有明顯專長
- 完成一件需要專業判斷的任務
- 有一次被正式認可的專門表現

### 常見收穫
- 可開分支
- 可接青階任務
- 可合法碰較高風險材料、路權或專業窗口

### 常見代價
- 你的路線開始變窄
- 專精越深，外行的部分越容易落後

### 最適合
- 想開始有個人風格的角色

## 3. 專精 -> 成名

### 升階感
- 你的名字開始和一條路、一片地或一件事綁在一起

### 常見條件
- 完成一次區域級事件
- 取得跨地互認或正式高階資格
- 走出一條分支或融合玩法

### 常見收穫
- 可接赤階任務
- 更常拿到主動找上門的機會
- 擁有自己明顯的 `強項標籤`

### 常見代價
- 更常被對手、官面或組織記住
- 很難再完全匿名

### 最適合
- 想從「普通冒險者」變成「有名號的人」

## 4. 成名 -> 顯赫

### 升階感
- 你的存在本身就會改變一個區域的局勢

### 常見條件
- 持續在高風險事件中成功
- 取得正式職位、席位、指揮權或代表權
- 名聲已穩定到可以動員資源

### 常見收穫
- 能調度隊伍、人手、文件、船位、許可
- 黑階任務開始可能落到你頭上
- 玩家選擇會更像 `站隊`

### 常見代價
- 責任與曝光明顯上升
- 你失敗時會拖累別人

### 最適合
- 長團中後期的核心角色

## 5. 顯赫 -> 傳聞級

### 升階感
- 你不是進場參與事件，而是你進場就成為事件

### 常見條件
- 多次左右地區甚至跨區局勢
- 接上權柄回聲、災害級事件或世界級名錄
- 你的名字足以在沒有本人出現時仍影響判斷

### 常見收穫
- 能進入世界級故事
- 會碰到平常人碰不到的窗口
- 有資格接觸災階任務與傳說級敵手

### 常見代價
- 生活幾乎不可能回到普通
- 很多麻煩會提前找上門

### 最適合
- 世界級長線團、終章團

## 常見升階方向示例

### 穩定型
地方見證 -> 正式執照 -> 分支專精 -> 跨地互認 -> 組織正式職位

### 高風格型
災場生還 -> 體質深化 -> 第二路線穩定 -> 融合路線覺醒 -> 區域成名

### 世界互動型
組織信任 -> 高危材實績 -> 跨區委託 -> 顯赫席位 -> 權柄回聲觸及

## 升階時 AI / 主持最少要給玩家的感覺
- 你得到什麼
- 你失去什麼
- 世界開始怎麼看你
- 你下一步最可能走哪條路

## 玩家使用建議
- 不要把升階只看成數值
- 每次升階都問自己：`我現在更像什麼樣的人了？`
- 如果不知道選哪張，優先選能把你帶進下一章世界互動的那張
