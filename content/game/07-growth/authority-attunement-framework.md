# 權柄感應與承載通用框架

## Purpose
這份文件把 `凡人可能得到權柄機會` 這件事，正式接進玩家與重要 NPC 的成長層。

它回答的是：
- 玩家怎麼可能碰到權柄
- 權柄對成長有什麼意義
- 權柄不等於成神，那它在遊戲裡該怎麼表現

## 核心概念
- 玩家不會直接變成神。
- 但玩家可以成為某個權柄的 `回聲接觸者 / 顯現者 / 印記者 / 承載者 / 代行者`。
- 這是一條高風險、高敘事重量、非人人都會走到的特殊成長線。

## 五個常見階段

### 1. 權柄回聲
- 角色開始反覆碰到某一權柄的片段回應。
- 常見表現：夢兆變多、在特定場所反應變強、對某類儀式特別敏感。
- 這一階段通常還不穩定，像是被某份神聖力量短暫碰到。

### 2. 權柄顯現
- 權柄開始在角色身上與周圍出現 `可觀察的異常表現`。
- 常見表現：體溫、氣息、聲音、眼神、夢境、腳步痕、觸碰物的反應發生改變。
- 這是「別人已經能看出你不一樣」的階段，但還未必留下穩定印記。

### 3. 權柄印記
- 角色身上留下可辨識的神聖痕跡。
- 常見表現：誓痕、夢紋、潮痕、火印、輪印、守夜反應。

### 4. 權柄承載
- 角色能較穩定調用某類權柄的片段力量。
- 常見表現：某類儀式成功率提高、某些禁忌加重、可被正式宗教視為特殊對象。

### 5. 權柄代行
- 角色能在條件成立時，短時間代行某類權柄的作用。
- 常見表現：主持重大見證、開啟引渡、短暫鎮息失衡、替某片聖地完成回應。

## 權柄的六種具體表現
角色得到權柄，不應只用一句「你獲得神力」帶過。建議至少從這六個面向呈現：

### 1. 身體表現
- 皮膚、眼瞳、聲線、體溫、傷口癒合方式、睡眠狀態、氣味或呼吸節奏改變。
- 例：
  - 誓約類角色在立誓時掌心發熱
  - 記憶類角色在長夜中眼下出現淡紋
  - 火鍛類角色靠近工坊時體溫異常穩定

### 2. 感知表現
- 對某些人、地點、遺物、誓言、死者、夢境、風向、路徑或材料產生超出常人的感知。
- 例：
  - 能感到某份誓約是否穩定
  - 能察覺某片棲地是否失衡
  - 能從破碎器物上感到殘留成形意志

### 3. 術式表現
- 某些術式更容易成功，某些術式開始帶有神聖痕跡，或會自然偏向某種表現方式。
- 例：
  - 安魂術更穩
  - 路引術會自動修正方向
  - 契約術能留下更難偽造的見證痕

### 4. 場域表現
- 角色進入聖地、遺跡、井陣、墓地、誓石、渡口或工坊時，環境會對他產生回應。
- 例：
  - 誓石發熱
  - 夢井起霧
  - 渡口燈自亮
  - 爐火轉色

### 5. 社會辨識
- 宗教組織、地方祭司、守地者、異端或權柄研究者會把角色辨認成特殊對象。
- 例：
  - 被視為可見證者
  - 被要求進行守夜或送行儀
  - 被禁止碰某些聖物
  - 被偽宗教盯上，想利用其權柄外顯

### 6. 代價表現
- 權柄不是純收益，角色會出現新的禁忌、失衡風險與社會壓力。
- 例：
  - 不能輕易違誓
  - 不能無視亡者殘響
  - 不能穿越某些失衡區域而不受反噬
  - 若長期壓抑權柄，會出現夢魘、失溫、灼痛或迷路傾向

## 七大高位權柄的常見外顯

### 鑒誓者
- 常見外顯：掌心發熱、立誓時周圍安靜、能察覺偽誓與破誓。
- 常見能力感：對契約、見證、正名與合法性特別敏銳。

### 衡律者
- 常見外顯：看見紋路、陣式、文書與規則中的錯位感。
- 常見能力感：擅長校正、辨錯、認證與維持秩序穩定。

### 萬物母
- 常見外顯：靠近傷者、植群、幼體與棲地時，生命反應特別明顯。
- 常見能力感：對照護、生長、繁衍、安置與棲地恢復更敏銳。

### 歸輪者
- 常見外顯：在守夜、葬祭、墓地與臨終場景中，能感到魂息的流向。
- 常見能力感：對送行、引渡、安魂、輪轉與死後殘留特別敏感。

### 憶夢者
- 常見外顯：反覆做同型夢、看見舊痕、聽見殘聲、記憶忽然被喚醒。
- 常見能力感：對回憶、夢兆、先兆、遺失資訊與記名儀式更敏銳。

### 夜引者
- 常見外顯：在黑暗、霧中、夜路、海路或失向環境中仍能隱約感到「可走之路」。
- 常見能力感：對引路、避失、渡行、改道、守航與邊界跨越更敏銳。

### 鑄形者
- 常見外顯：能感到材料的穩定度、裂痕、火候與成形節點。
- 常見能力感：對鍛造、修補、重鑄、加工與成品見證更敏銳。

## 常見觸發條件
- 重大誓約成立或破裂
- 聖地回應
- 生還於重大災厄
- 承接某位昇格神或守地神的遺留
- 跨越高風險儀式
- 特定血脈與體質覺醒

## 常見效果
- 解鎖特殊儀式
- 解鎖某類神聖術式或誓印
- 獲得特殊通行或見證權
- 被宗教組織、異端、小教派或政權特別關注
- 某些地方 NPC 會以不同態度對待你
- 某些場域、材料與聖物會對你出現額外反應

## 常見代價
- 更高負荷
- 更嚴格禁忌
- 更高曝光
- 特定敵對勢力主動追索
- 一旦失衡，反噬比一般成長更重

## 玩家顯示建議
- 不要一開始就顯示成「你獲得神力」。
- 建議先用：
  - `你感覺某種權柄在回應你`
  - `你的身上開始出現不尋常的外顯`
  - `你的身上留下某種印記`
  - `某個宗教或聖地開始把你視為特殊承載者`
- 等到進一步穩定後，再顯示具體可用的權柄能力。

## 使用原則
- 權柄線應該稀少、重要、會改變角色與世界的關係。
- 它不只是能力獎勵，也應帶來責任、代價與世界回應。
- 若角色拿到權柄卻完全沒有 `外顯 / 場域反應 / 社會辨識 / 禁忌代價`，那通常代表這條線還沒寫完整。
