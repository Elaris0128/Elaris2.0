# 玩家成長卡庫

## Purpose
這份文件把玩家最常遇到的 `成長里程碑` 壓成可直接選讀的卡片。

它不是要玩家每次升成長都抽卡，而是讓玩家知道：
- 目前最適合追哪種成長
- 這種成長通常靠什麼得到
- 它會帶來什麼實際變化

## 成長卡怎麼看
每張卡至少回答：
- 適合哪個階段
- 主要來源
- 立刻得到什麼
- 代價或責任
- 下一步會開哪條路

## 起步期常見成長卡

### 1. 地方見證
- 適合階段：起步 -> 立足
- 主要來源：完成一次被正式記住的地方事件
- 立刻效果：當地說服、求助與詢問更容易成立
- 代價：你開始會被地方認出來
- 下一步：可接 `地方信任席位`

### 2. 第一專精開啟
- 適合階段：起步 -> 立足
- 主要來源：主職穩定使用、接受導師指點、完成第一個專門任務
- 立刻效果：可明確往一條職業分支深化
- 代價：其他副線會暫時放慢
- 下一步：可接 `分支資格試煉`

### 3. 災場生還
- 適合階段：起步 -> 立足
- 主要來源：從事故、魔物襲擊、遺跡崩塌或壞兆事件中活著回來
- 立刻效果：得到一條高壓環境經驗線
- 代價：創傷、後遺或對某種情境更敏感
- 下一步：可接 `耐壓者聲名`

### 4. 第一張正式證明
- 適合階段：起步 -> 立足
- 主要來源：完成見證、考核、查驗或授名
- 立刻效果：第一次得到 `紙面上的資格`
- 代價：之後更容易被追責
- 下一步：可接 `正式執照取得`

### 5. 路線熟門
- 適合階段：起步 -> 立足
- 主要來源：反覆跑某條路、某片海、某一圈城港
- 立刻效果：在那一圈世界的行動更穩
- 代價：離開熟圈時會明顯變難
- 下一步：可接 `跨地互認`

## 立足期常見成長卡

### 6. 正式執照取得
- 適合階段：立足 -> 專精
- 主要來源：公會、教團、工坊、官署或地方體系的認證
- 立刻效果：可以合法承接更高一級工作
- 代價：要接受抽查、章程與責任
- 下一步：可接 `跨地互認` 或 `高危材處理實績`

### 7. 地方信任席位
- 適合階段：立足 -> 專精
- 主要來源：多次幫助同一聚落、港區或哨線
- 立刻效果：地方 NPC 比較願意先信你
- 代價：地方出事你更難置身事外
- 下一步：可接 `地方代理人` 類型升階

### 8. 高危材處理實績
- 適合階段：立足 -> 專精
- 主要來源：安全處理一次危材、活性樣本、深樣或禁流半成品
- 立刻效果：更高階材料與裝具開始向你開窗
- 代價：失手代價會非常高
- 下一步：可接 `危材專家` 路線

### 9. 組織內部可信任度
- 適合階段：立足 -> 專精
- 主要來源：完成 2-3 次同組織任務且沒有翻車
- 立刻效果：能碰到更核心的消息或窗口
- 代價：組織也會更常找你做事
- 下一步：可接 `組織正式職位`

### 10. 第二條路線穩定化
- 適合階段：立足 -> 專精
- 主要來源：副職長期使用
- 立刻效果：可以開始準備融合或轉職
- 代價：資源與時間壓力上升
- 下一步：可接 `融合路線覺醒`

## 專精期常見成長卡

### 11. 融合路線覺醒
- 適合階段：專精 -> 成名
- 主要來源：兩條路線都夠穩，且完成一次強風格事件
- 立刻效果：可拿到辨識度很高的新玩法
- 代價：負荷與失衡風險同步上升
- 下一步：可接 `章節代表技`

### 12. 跨地互認
- 適合階段：專精 -> 成名
- 主要來源：不同地區都留下正式可靠紀錄
- 立刻效果：某些資格不再只限本地
- 代價：更容易被跨區勢力盯上
- 下一步：可接 `正式跨區委託`

### 13. 地區成名
- 適合階段：專精 -> 成名
- 主要來源：完成一件會被反覆提起的區域事件
- 立刻效果：名字本身會開門，也會惹麻煩
- 代價：低調變難
- 下一步：可接 `顯赫席位`

### 14. 災害處理經驗
- 適合階段：專精 -> 成名
- 主要來源：參與封鎖、撤離、巨獸、外流或深災事件
- 立刻效果：面對大型危機時判斷更準
- 代價：容易留下創傷、責任與追責
- 下一步：可接 `災階應對者`

## 成名期以上常見成長卡

### 15. 組織正式職位
- 適合階段：成名 -> 顯赫
- 主要來源：組織正式任命或代行
- 立刻效果：能動用人手、文件與制度資源
- 代價：自由行動變少
- 下一步：可接 `顯赫席位`

### 16. 顯赫席位
- 適合階段：成名 -> 顯赫
- 主要來源：一地或一線世界已把你當成可靠名字
- 立刻效果：進場就會影響局勢
- 代價：對手也會優先找你
- 下一步：可接 `傳聞級入口`

### 17. 權柄回聲觸及
- 適合階段：顯赫以上
- 主要來源：極少數高位事件、聖跡、律神回聲、古權柄接觸
- 立刻效果：得到特殊進展方向
- 代價：責任、禁忌與外顯變化
- 下一步：接 `權柄感應框架`

## 玩家使用建議
- 一次不要追太多條
- 初期最穩的是 `地方見證 + 第一專精開啟`
- 想玩世界互動感，優先追 `正式執照取得 / 組織內部可信任度`
- 想玩高風格，優先追 `災場生還 / 融合路線覺醒`
