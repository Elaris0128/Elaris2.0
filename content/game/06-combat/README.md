## 這一層能回答什麼
- 玩家一回合通常可以做什麼
- 什麼時候該先保人、換位、壓制、撤退，而不是只打輸出
- 地形、掩體、視野與站位怎麼改變局勢
- 遇到敵方首領或小隊時，AI 該怎麼把威脅講清楚
- 玩家看到不同敵人類型時，最先該拆的是什麼
