# 通用戰技與術式總表

## Purpose
這份文件整理 `玩家最容易接觸、各文明最常見、最適合作為底層能力模組` 的通用戰技與術式。

它不是收所有地方秘傳，而是回答：
- 哪些能力算世界通用
- 它們大概屬於什麼類型
- 常見由誰使用
- 前期玩家適不適合接觸

## 通用能力的判準
- 至少在兩個以上文明或機構裡有近似版本
- 不依賴單一神權、單一家系或單一血脈才能成立
- 可用不同文化名稱出現，但底層結構相近

## 資源標記
- `體勢`：主要吃體魄、身法與肉體負荷
- `專注`：主要吃心志、悟性與施放穩定
- `靈息`：主要吃靈核、靈感與儲量
- `誓耗`：主要吃見證、條件與誓約代價
- `媒材`：需要消耗器材、印材、藥材或場地

## 一、通用戰技

### 破陣踏
- 大類：體勢戰技 / 強襲型
- 作用：突進、破陣、撞開前線
- 常見使用者：近戰前衛、軍陣武士、守誓戰士
- 主要消耗：體勢
- 前期適性：高

### 承山架
- 大類：體勢戰技 / 抗衝型
- 作用：正面承受衝擊、穩住線位
- 常見使用者：重裝守衛、護衛型前線、港巡兵
- 主要消耗：體勢
- 前期適性：高

### 影返步
- 大類：步法戰技 / 迴避型
- 作用：脫離鎖定、切角換位、閃避接戰
- 常見使用者：機動斥候、情報行者、巡翼哨手
- 主要消耗：體勢 + 專注
- 前期適性：高

### 斜岔追痕
- 大類：步法戰技 / 追擊型
- 作用：追蹤、追擊、維持獵線
- 常見使用者：機動斥候、遠行獵手、狩窗手
- 主要消耗：體勢 + 專注
- 前期適性：高

### 斷環劍
- 大類：兵裝戰技 / 劍技型
- 作用：切斷對手節奏、破壞持握
- 常見使用者：近戰前衛、王庭誓衛、決鬥者
- 主要消耗：體勢
- 前期適性：中高

### 折線護
- 大類：守勢戰技 / 護衛型
- 作用：保護後排、拉出安全角度
- 常見使用者：重裝守衛、護衛隊、車隊護行者
- 主要消耗：體勢
- 前期適性：高

### 反鈎截
- 大類：守勢戰技 / 反擊型
- 作用：格擋後打斷第二手
- 常見使用者：城衛兵、港巡兵、關卡查驗官中的武備者
- 主要消耗：體勢 + 專注
- 前期適性：中高

### 追痕割
- 大類：狩獵戰技 / 追痕型
- 作用：利用傷口、氣味、血滴持續追敵
- 常見使用者：遠行獵手、狩窗手、採藥人中的危地採收者
- 主要消耗：專注
- 前期適性：中

### 剝取定手
- 大類：狩獵戰技 / 剝取型
- 作用：穩定切取素材、降低破壞率
- 常見使用者：打撈手、採樣手、標本師、藥師
- 主要消耗：專注 + 媒材
- 前期適性：中

### 地鳴借勢
- 大類：共鳴戰技 / 地脈借力型
- 作用：在適合場域放大一擊或穩住身位
- 常見使用者：重裝守衛、山地武士、守地護衛
- 主要消耗：體勢 + 靈息
- 前期適性：中

## 二、通用術式

### 尋脈引
- 大類：引導術 / 偵測型
- 作用：尋找地脈、殘留靈息、術式餘痕
- 常見使用者：學院施術者、記錄學者、遺跡勘界手
- 主要消耗：專注 + 靈息
- 前期適性：高

### 星點校準
- 大類：引導術 / 校準型
- 作用：穩定施術方位、修正遠距誤差
- 常見使用者：學院施術者、空艇航手、航標術士
- 主要消耗：專注
- 前期適性：中

### 裂息矢
- 大類：放出術 / 投射型
- 作用：遠距打擊、逼位、壓制
- 常見使用者：學院施術者、符印術者、軍用施法員
- 主要消耗：靈息
- 前期適性：中高

### 霧脈衝
- 大類：放出術 / 脈衝型
- 作用：擾亂視線、切敵節奏、驚退小群體
- 常見使用者：自然共鳴者、巡路施術者、水岸術士
- 主要消耗：靈息
- 前期適性：中高

### 燼紋附鋒
- 大類：附著術 / 武裝附著
- 作用：暫時強化武器
- 常見使用者：符印術者、誓紋技匠、戰法融合者
- 主要消耗：靈息 + 媒材
- 前期適性：中

### 冷息護皮
- 大類：附著術 / 身體附著
- 作用：降低熱傷、擦傷、毒刺傷害
- 常見使用者：治療藥師、自然共鳴者、寒地術士
- 主要消耗：靈息 + 媒材
- 前期適性：高

### 靜域環
- 大類：場域術 / 保護場
- 作用：穩定儀式、庇護傷者、降低干擾
- 常見使用者：祭儀執行者、守誓戰士、療傷手
- 主要消耗：靈息 + 專注
- 前期適性：中高

### 迷行霧幕
- 大類：場域術 / 迷導場
- 作用：遮斷視線、保護撤離、製造戰場模糊區
- 常見使用者：自然共鳴者、機動斥候中的施術支、契靈行者
- 主要消耗：靈息
- 前期適性：中

### 霜鹽轉質
- 大類：轉化術 / 性質改變
- 作用：把液體、鹽霧、薄冰轉為可用介質
- 常見使用者：寒地術士、旅糧保存者、港務防蝕師
- 主要消耗：靈息 + 媒材
- 前期適性：中

### 見證鎖
- 大類：封印術 / 檢證型
- 作用：封樣、鎖定、建立可追溯鏈
- 常見使用者：符印術者、治療藥師、誓紋技匠
- 主要消耗：靈息 + 誓耗 + 媒材
- 前期適性：中

### 逐客束印
- 大類：封印術 / 驅逐型
- 作用：逼退殘響、封口、隔離破口
- 常見使用者：祭儀執行者、守誓戰士、封印師
- 主要消耗：靈息 + 專注
- 前期適性：中

## 三、通用融合技與通用儀式術

### 誓燈導式
- 結構：引導術 + 儀式術
- 作用：照路、辨霧、穩定行進線
- 常見使用者：祭儀執行者、岬哨守燈手、巡路者

### 守關見證術
- 結構：封印術 + 場域術
- 作用：關卡檢證、辨印、維持合法通行
- 常見使用者：關卡查驗官、司燈者、誓紋技匠

### 潮行護航術
- 結構：場域術 + 引導術
- 作用：短航穩船、辨霧、保短程水路
- 常見使用者：引水人、船長、港泊管事

### 調膳安息火
- 結構：附著術 + 技藝術路
- 作用：把料理、藥膳、暖飲轉成兼具恢復與安撫的成品
- 常見使用者：靈廚、供饌手、治療藥師

## 四、玩家起始建議
- 戰技型主職：起始選 `2` 戰技 + `1` 防護或工具型能力
- 術式型主職：起始選 `2` 術式 + `1` 穩定或調查型能力
- 誓約型主職：起始選 `1` 戰技 + `1` 術式 + `1` 見證或保護型能力
- 技藝型主職：起始選 `1` 行動能力 + `2` 世界互動能力
