# 戰技與術式示例庫

## Purpose
這份文件提供 Elaris 通用的 `戰技 / 術式 / 融合技 / 神聖系儀路術` 範例，讓後續擴寫各國能力時，有可直接套用的共通參照。

## 玩家搭配檔案
若要把這些示例直接轉成玩家可選能力，請搭配：
- `universal-technique-and-spell-master-index.md`
- `technique-and-spell-profession-mapping.md`
- `player-technique-and-spell-selection-guide.md`

它不代表所有文明都會使用同樣名字，而是提供：
- 結構範例
- 常見用途
- 典型消耗與代價
- 適合掛接的職業與神聖來源

## 使用原則
- 先用 `technique-and-spell-taxonomy.md` 判斷大類與分支。
- 再用這份示例庫決定招式或術式長什麼樣。
- 國家、宗教、家系與地方版本，可以更改名稱、媒介與禁忌，但不要把底層結構改到失去共通性。

## 戰技示例

### 破陣踏
- 類型：`體勢戰技 / 強襲型`
- 常見用途：短距爆發、撞開陣線、壓迫單體
- 常見媒介：重靴、鎧甲重心、地面借力
- 常見代價：膝踝負荷高，連續使用容易失衡
- 常見適職：戰士、重裝衛、前鋒騎士

### 承山架
- 類型：`體勢戰技 / 抗衝型`
- 常見用途：硬接衝擊、替隊友吃下第一波撞擊
- 常見媒介：肩甲、盾臂、沉身呼吸
- 常見代價：肩背負荷與短暫遲滯
- 常見適職：護衛、誓約騎士、重裝兵

### 影返步
- 類型：`步法戰技 / 迴避型`
- 常見用途：脫離鎖定、切角、短線換位
- 常見媒介：輕甲、低重心、連續步點
- 常見代價：地形不穩時容易失步
- 常見適職：斥候、刺客、行路者

### 斜岔追痕
- 類型：`步法戰技 / 追擊型`
- 常見用途：追上轉彎目標、維持追蹤節奏
- 常見媒介：呼吸節拍、地表痕跡判讀
- 常見代價：對專注消耗較高
- 常見適職：獵手、巡路者、邊境追兵

### 斷環劍
- 類型：`兵裝戰技 / 劍技型`
- 常見用途：切斷連擊節奏、破壞敵方持握與誓印連動
- 常見媒介：單手劍、短弧揮切
- 常見代價：對節奏判斷要求高
- 常見適職：軍劍士、誓衛、決鬥者

### 雙鎚壓心
- 類型：`兵裝戰技 / 重兵型`
- 常見用途：壓制大體型魔物、擊碎護具與地表薄弱處
- 常見媒介：戰鎚、重錘、厚柄武裝
- 常見代價：體力與負荷消耗大
- 常見適職：山邦重兵、礦道護運者、破障手

### 折線護
- 類型：`守勢戰技 / 護衛型`
- 常見用途：在多人戰中替後排創造安全角度
- 常見媒介：盾具、披掛、隊列默契
- 常見代價：移動速度下降
- 常見適職：護衛、車隊衛、隘口守兵

### 反鈎截
- 類型：`守勢戰技 / 反擊型`
- 常見用途：格擋後立即打斷敵方第二手
- 常見媒介：短柄兵器、盾邊、手腕扭力
- 常見代價：判斷失誤時容易被反抓空檔
- 常見適職：城衛、決鬥兵、伴行守衛

### 追痕割
- 類型：`狩獵戰技 / 追痕型`
- 常見用途：沿傷口、氣味或血滴持續鎖定逃竄目標
- 常見媒介：狩刀、標記粉、短距感知
- 常見代價：對特定目標有效，換目標時節奏會斷
- 常見適職：獵手、魔物回收者、邊巡追跡員

### 剝取定手
- 類型：`狩獵戰技 / 剝取型`
- 常見用途：在不破壞珍貴素材的前提下完成切取
- 常見媒介：細刃、夾具、封樣袋
- 常見代價：戰鬥中使用風險高
- 常見適職：探索回收者、藥材獵手、誓獸處理員

### 契騎平潮
- 類型：`騎戰戰技 / 導騎型`
- 常見用途：穩定坐騎情緒、長距巡行、跨危地帶
- 常見媒介：韁印、呼哨、伴騎呼吸同步
- 常見代價：若坐騎驚失則反噬騎手節奏
- 常見適職：騎兵、馱隊首、草原伴行者

### 地鳴借勢
- 類型：`共鳴戰技 / 地脈借力型`
- 常見用途：在特定地形中放大一擊或穩住身位
- 常見媒介：踩點、地面紋理、短暫靈息灌注
- 常見代價：離開相應地形後效力顯著下降
- 常見適職：山地戰士、守地護衛、重裝前鋒

## 術式示例

### 尋脈引
- 類型：`引導術 / 偵測型`
- 領域：`共鳴系`
- 常見用途：找出地脈、殘留靈息、術式餘痕
- 常見媒介：導環、脈石、指尖印記
- 常見代價：過度偵測時容易頭痛與感知疲勞
- 常見適職：術士、探索師、導引員

### 星點校準
- 類型：`引導術 / 校準型`
- 領域：`空間系`
- 常見用途：穩定施術方位、修正長距離誤差
- 常見媒介：星盤、校刻尺、定位燈
- 常見代價：需要穩定站位與視野
- 常見適職：測繪術士、航標官、遠射施術者

### 裂息矢
- 類型：`放出術 / 投射型`
- 領域：`元素系`
- 常見用途：快速打擊、逼位、穿透輕護具
- 常見媒介：導杖、符矢、指向印
- 常見代價：連續放出會大幅消耗專注
- 常見適職：術士、軍用施法員、守城術手

### 霧脈衝
- 類型：`放出術 / 脈衝型`
- 領域：`自然系`
- 常見用途：擾亂視野、驚退小群體、切斷追兵節奏
- 常見媒介：霧瓶、潮珠、掌心導印
- 常見代價：在乾燥或高風環境下衰減很快
- 常見適職：林境術士、巡路者、水岸施術者

### 燼紋附鋒
- 類型：`附著術 / 武裝附著`
- 領域：`鍊成系`
- 常見用途：短時間提升兵器切割或灼痕效果
- 常見媒介：武器表面紋線、炭粉、熱印
- 常見代價：過載時可能反燒持有者手臂
- 常見適職：戰法融合者、鍛學術士、誓紋技師

### 冷息護皮
- 類型：`附著術 / 身體附著`
- 領域：`自然系`
- 常見用途：暫時降低高熱、擦傷或毒刺傷害
- 常見媒介：膏膜、冷霧、皮膚紋印
- 常見代價：持續太久會使末梢遲鈍
- 常見適職：藥師、寒地術士、旅隊醫者

### 靜域環
- 類型：`場域術 / 保護場`
- 領域：`誓環系`
- 常見用途：降低外部擾動、穩定儀式或短暫庇護傷者
- 常見媒介：圓環印記、誓燈、見證物
- 常見代價：維持期間施術者移動受限
- 常見適職：見證術士、守夜者、護送隊

### 迷行霧幕
- 類型：`場域術 / 迷導場`
- 領域：`自然系`
- 常見用途：遮斷追兵視線、保護撤離、擾亂敵方隊形
- 常見媒介：霧種、濕布、場界柱
- 常見代價：友軍若無標記也可能一起失向
- 常見適職：林地導引者、斥候術士、邊境守望者

### 霜鹽轉質
- 類型：`轉化術 / 性質改變`
- 領域：`元素系`
- 常見用途：把液體、鹽霧或薄冰轉成可用防護或保存介質
- 常見媒介：鹽片、冰印、低溫器皿
- 常見代價：環境條件不合時效力差
- 常見適職：寒地術士、旅糧保存者、港務防蝕師

### 熔紋換骨
- 類型：`轉化術 / 結構扭轉`
- 領域：`鍊成系`
- 常見用途：臨時修補金屬結構、扭正器具、穩定破損符架
- 常見媒介：熔紋筆、夾具、加熱印
- 常見代價：若材料失配，修補後會留下延遲性裂痕
- 常見適職：鍛造術士、工坊修護手、符架維護員

### 見證鎖
- 類型：`封印術 / 檢證型`
- 領域：`誓環系`
- 常見用途：封存樣本、鎖定違規物、維持證物可追溯性
- 常見媒介：見證印、封條、編號符片
- 常見代價：若見證鏈斷裂，封印可靠度會下降
- 常見適職：公證術士、調查員、誓紋技師

### 逐客束印
- 類型：`封印術 / 驅逐型`
- 領域：`共鳴系`
- 常見用途：逼退失衡殘響、排斥低階污染體、封住破口
- 常見媒介：符釘、場界線、短時高專注
- 常見代價：對大型或高權柄存在效果有限
- 常見適職：封印師、災害處理員、聖地守護者

### 安魂守夜式
- 類型：`儀式術 / 見證型`
- 領域：`死靈系`
- 常見用途：穩定亡者餘響、協助安渡、避免夜間異變
- 常見媒介：守夜燈、送亡布、哀歌節拍
- 常見代價：儀式主持者容易承受精神壓力與情緒殘留
- 常見適職：治喪者、守墓人、輪轉信仰者

### 開路祈燈式
- 類型：`儀式術 / 長時維持型`
- 領域：`誓環系`
- 常見用途：在高危路段、雪關、霧林或潮路建立短期安全導引
- 常見媒介：祈燈、路旗、巡路標柱
- 常見代價：需要多人輪值與穩定補材
- 常見適職：巡路者、邊關守夜隊、導引師

## 融合技與神聖系儀路術

### 誓燈導式
- 類型：`引導術 + 儀式術`
- 術學來源：`源神系 / 守夜祖靈系`
- 常見用途：照路、辨霧、穩定邊關或雪原行進線
- 常見代價：需要誓燈、見證物與持續守夜
- 常見傳承：法雷恩人、北境守關者

### 守關見證術
- 類型：`封印術 + 場域術`
- 術學來源：`源神系 / 律神系`
- 常見用途：維持關卡合法性、記錄進出、辨識偽證與偽印
- 常見代價：一旦場界被毀，補修成本很高
- 常見傳承：王國邊關、聯邦公證系統、雪關體系

### 巡路安息印
- 類型：`附著術 + 儀式術`
- 術學來源：`守地神 / 祖靈系`
- 常見用途：讓送亡隊伍平安穿越危地，避免亡者餘響失序
- 常見代價：只能在被承認的路線上穩定生效
- 常見傳承：墓守、旅路信仰、地方治喪體系

### 潮行護航術
- 類型：`場域術 + 引導術`
- 術學來源：`律神系 / 海潮支線`
- 常見用途：穩定船隊、減少霧潮迷失、保護港外短航
- 常見代價：離開水域後效力迅速下降
- 常見傳承：海人、航標官、沿岸商會

### 調膳安息火
- 類型：`附著術 + 技藝術路`
- 術學來源：`萬物母支線 / 靜燈照護系`
- 常見用途：把藥食、旅糧、暖飲轉成兼具恢復與安撫效果的成品
- 常見代價：對食材品質與火候控制要求很高
- 常見傳承：靈廚、照護團體、行旅醫帳
