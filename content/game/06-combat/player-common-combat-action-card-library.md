# 玩家常見戰鬥行動卡

這份卡庫收的是玩家最常說、AI 也最常裁定的戰鬥行動。

## 直攻
- 用途：單點輸出
- 風險：容易被反擊或吃掩體
- 最適合：敵人露出破綻時

## 壓制
- 用途：逼對方不敢輕鬆做事
- 風險：自己輸出變低
- 最適合：敵方施術者、弓手、首領讀條時

## 換位保人
- 用途：把高壓轉到自己身上
- 風險：自己暴露
- 最適合：隊友快倒或剛完成長行動時

## 牽制
- 用途：讓高威脅敵人暫時別去碰後排
- 風險：需要扛壓
- 最適合：重裝、守護、威嚇型角色

## 突進
- 用途：快速壓近高價值目標
- 風險：容易脫離隊友支援
- 最適合：敵後方弱皮或關鍵控制者

## 脫離
- 用途：保命、重整、拉怪
- 風險：若講不好，很像單純逃跑
- 最適合：高負荷、高傷勢、吃到壞狀態時

## 利用場景
- 用途：翻桌、關門、推落、熄燈、截索
- 風險：結果受場地與判定影響
- 最適合：純輸出不划算時

## 掩護長行動
- 用途：讓隊友抄錄、封樣、開門、醫治
- 風險：自己可能變成主要靶子
- 最適合：任務目標比打倒敵人更重要時
