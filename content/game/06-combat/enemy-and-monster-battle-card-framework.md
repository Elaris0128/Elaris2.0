# 敵人與魔物戰鬥卡框架

## Purpose
這份文件把 Elaris 玩家端的 `敵人卡 / 魔物戰鬥卡 / 首領模板 / 小隊遭遇卡` 統一成同一套格式。

它的用途是讓 AI / 主持能快速回答：
- 這次遇到的是什麼
- 對方大概會怎麼打
- 玩家最該注意什麼
- 打贏、逼退或拖過去後會留下什麼

## 核心原則
- 戰鬥卡是 `遭遇工具`，不是純百科摘要。
- 同一張卡應同時回答 `場景感`、`戰鬥方式` 與 `後果`。
- 敵人卡偏重 `人形對手與組織戰術`。
- 魔物卡偏重 `生態行為、要害與素材`。
- 首領卡比一般卡多一層 `階段變化`。

## 四種卡型

### 1. 一般敵人卡
適合：
- 巷戰
- 港邊衝突
- 遺跡黑掘小隊
- 巡邏兵、護衛、打手、掮客

### 2. 魔物戰鬥卡
適合：
- 野外遭遇
- 海上遇襲
- 高空掠食者
- 棲地失衡事件

### 3. 小隊遭遇卡
適合：
- 一次出現多種敵人
- 有陣地、有角色分工
- 有明顯戰術互補

### 4. 首領 / 災害卡
適合：
- 章節收尾
- 命名巨獸
- 名字會被世界記住的敵方人物
- 一旦失控會改變地區局勢的威脅

## 一般敵人卡最低欄位
- 名稱
- 類型
- 常見地區
- 常見出沒數量
- 推薦場景
- 威脅等級
- 戰鬥傾向
- 常用招式或手段
- 防守方式
- 要害或破綻
- 崩潰 / 撤退條件
- 常見掉落
- 任務鉤子

## 魔物戰鬥卡最低欄位
- 名稱
- 魔物分類
- 主棲地
- 常見遭遇數量
- 危險程度
- 行動徵兆
- 攻擊模式
- 主要壓力
- 要害
- 素材
- 是否可驅離 / 誘導 / 暫時封鎖
- 常見任務用途

## 首領卡追加欄位
- 階段一行為
- 階段二變化
- 失控條件
- 場地變化
- 特殊收尾方式
- 擊敗後世界後果

## 小隊遭遇卡追加欄位
- 隊伍構成
- 核心分工
- 第一回合傾向
- 第二波壓力
- 若玩家拖太久會發生什麼

## 威脅等級建議
- `低`：能造成麻煩，但單獨不太能壓垮隊伍。
- `中`：需要正面處理，否則容易有人受傷或任務翻車。
- `高`：需要動用招式、地形、裝具或分工。
- `極高`：若沒有明確計畫，正面硬打通常會吃大虧。
- `災害`：重點不只是打贏，而是怎麼活下來、撤離、封鎖或改變條件。

## 常見戰鬥傾向標籤
- `突進`
- `圍堵`
- `拖耗`
- `拉扯`
- `控場`
- `遠壓`
- `伏擊`
- `驚嚇`
- `召援`
- `環境利用`

## 常見要害類型
- 感知器官
- 脊柱與關節
- 呼吸與換氣部位
- 發光囊、毒囊、鳴器
- 薄膜、腹側、翼根、鰭根
- 指揮者、旗手、術式核心

## 戰鬥卡和玩家行動的對接
每張卡最好都能回答：
- 玩家能不能 `繞過`
- 玩家能不能 `談判`
- 玩家能不能 `引開`
- 玩家能不能 `局部勝利`
- 玩家若只想 `保人撤離`，最小成功條件是什麼

## AI / 主持使用順序
1. 先決定遭遇是 `一般敵人 / 魔物 / 小隊 / 首領`
2. 再決定場地優勢站在誰那邊
3. 先展示 `徵兆`，不要直接只說打起來了
4. 第一輪先給玩家看懂威脅
5. 若玩家描述聰明，優先讓他們改變卡上的壓力條件
