# 特質、血脈與特殊體質通用框架

## Purpose
這份文件重新定義 Elaris 中 `種族共通特徵 / 支系專屬構造 / 個體型特殊體質 / 後天特質` 的邊界。

## 玩家搭配檔案
若要把這些構造直接轉成能力與招式，請搭配：
- `README.md`
- `species-special-organs-and-ability-applications.md`
- `selected-race-organ-and-signature-move-library.md`

核心原則是：
- 不是所有差異都叫 `特質`
- 不是所有稀有能力都叫 `特殊體質`
- 某些能力其實屬於特定種族或人種才有的 `先天構造`

因此，這套系統不再把角色做成可隨意挑選的詞條列表，而是要忠於世界原有的生理與血脈設定。

## 核心修正
- `種族共通特徵`：描述生命大類的基本生理與天生傾向。
- `支系專屬構造`：描述特定人種、分支、血裔才會擁有的先天器官、脈絡、骨血結構或感官機制。
- `個體型特殊體質`：描述不來自種族天賦，而是由天賦、意志、災變、接觸特殊資源後形成的個體異常。
- `後天特質`：描述角色因環境、訓練、誓約、職業與事件養成的長期傾向。

## 四層結構

### 1. 種族共通特徵
- 回答角色屬於哪個生命大類。
- 例如：人類、矮人、精靈、翼人、海人、魔人、棲裔。
- 這一層只處理大類生命的共通體質，不處理稀有支系專屬器官。

### 2. 支系專屬構造
- 回答角色是否具備某個支系、血裔、分支限定的先天構造。
- 這一層最接近舊設定中的「血脈限定構造」。
- 可以是：
  - 特殊器官
  - 感官構造
  - 骨血結構
  - 皮膚與外覆結構
  - 先天脈絡與呼吸機制
- 例如：
  - 魔人的 `曜血核脈 / 魅息腺 / 垢金核 / 影心珠 / 血爆囊 / 夢核 / 噬源`
  - 極光徙裔的 `極光紋瞳`
  - 魈陽族的 `炁紋脈`
  - 柯沙里人的 `花隱膚與花藏息`
  - 諾魯因人的 `潮骨血息`
- 這一層不是「可學會」的能力，而是支系血脈的一部分。

### 3. 個體型特殊體質
- 回答角色是否出現不屬於種族先天構造的個體異常。
- 這一層對接舊設定的 `3.4 特殊體質`。
- 它明確不是種族限定天賦，而是獨立個體因：
  - 天賦異常
  - 意志極端化
  - 災變
  - 長期接觸特殊場域或資源
  - 靈核失衡
  所形成的稀有狀態。

### 4. 後天特質
- 回答角色因生活、訓練、誓約、職業與創傷養成了什麼長期傾向。
- 這一層才放：
  - 生態適應
  - 社會背景
  - 組織養成
  - 誓約後遺
  - 災後生還經驗

## 先定義什麼不算什麼

### 1. 支系專屬構造不等於特殊體質
- `極光紋瞳`、`炁紋脈`、`曜血核脈` 這類，屬於血脈或支系限定構造。
- 它們即使稀有，也不歸類為個體型特殊體質。

### 2. 個體型特殊體質不等於種族天賦
- `靈息染體`、`靈核障化`、`術式干擾` 這類，不應寫成某支系人人都可能有。
- 它們是個體異常，不是族群常規構造。

### 3. 後天特質不等於血脈
- `市集耳目`、`章程腦`、`災後倖存者` 這類，不應和先天構造混寫。

## 種族共通特徵的內容
- 基本生命型態
- 平均體貌與壽命輪廓
- 繁衍方式
- 一般感官與環境適性
- 與靈息、誓約、資源互動的共通傾向

## 支系專屬構造的定義

### 1. 適用範圍
- 只屬於特定種族分支、人種、血裔、混血穩定支系。
- 不是外人透過訓練就能學會。

### 2. 常見形式
- `器官型`：核脈、腺體、心珠、囊、內核
- `感官型`：特殊瞳紋、嗅覺器、聽感結構
- `骨血型`：高密度骨骼、血息循環、特殊血質
- `皮膚外覆型`：鱗甲、花隱膚、特殊表皮
- `脈絡機制型`：炁紋脈、裂脈感、巖息法這類先天機制

### 3. 常見特徵
- 通常有成年禮、啟脈儀式或血脈覺醒過程
- 外顯徵象通常穩定可辨識
- 常伴隨固定弱點或禁忌
- 容易成為家系、宗教、國家制度的重要分界

### 4. 使用原則
- 一個角色通常只有自己支系對應的構造
- 若是混血角色，是否同時具備雙構造，必須由世界設定明文支持
- 支系專屬構造應優先寫 `生理表現`，再寫 `能力效果`

## 個體型特殊體質的定義

### 1. 舊 canon 對應
這一層對應舊設定中的：
- `靈息染體`
- `靈息極化`
- `靈核障化`
- `術式干擾`
- `情緒共鳴`
- `靈息轉質者`

### 2. 核心特性
- 不來自種族天賦
- 不必綁定特定人種
- 可能出現在任何種族或支系個體身上
- 往往同時帶來潛力與風險

### 3. 常見來源
- 災變現場倖存
- 遺跡、共鳴點、異常資源長期暴露
- 高強度儀式
- 靈核先天失衡
- 強烈情緒與權柄回聲交纏

### 4. 使用原則
- 這一層是角色個體事件，不應覆蓋掉其種族或支系身份
- 若角色已有高風險支系構造，再追加個體型特殊體質時，必須明寫衝突與代價

## 後天特質的定義

### 1. 常見來源
- 生態與地理環境
- 組織訓練
- 職業文化
- 長期旅行與採集
- 誓約與受名
- 災後經驗與創傷

### 2. 常見分類
- `生態特質`
- `社會特質`
- `誓約特質`
- `事件特質`
- `訓練特質`

### 3. 使用原則
- 後天特質可以疊加
- 但應盡量描述生活與玩法影響，而不是只給數值加成

## 角色配置建議
- `1 個種族`
- `0 到 1 個支系專屬構造`
- `0 到 1 個個體型特殊體質`
- `1 到 3 個後天特質`

這樣能避免角色變成一堆互相衝突的被動詞條。

## 卡片模板建議

### 1. 支系專屬構造卡
- 名稱
- 所屬種族
- 所屬支系或血裔
- 構造類型
- 外顯徵象
- 核心功能
- 啟動或覺醒條件
- 弱點與代價
- 相關成年禮、信仰或制度

### 2. 個體型特殊體質卡
- 名稱
- 所屬大類
- 誘因或來源
- 主要表現
- 優勢
- 風險
- 常見誤判或社會影響
- 是否可穩定控制

### 3. 後天特質卡
- 名稱
- 類型
- 來源
- 核心效果
- 限制或代價
- 常見觸發場景
- 可成長方向
