# 血統、支系與體質框架

這份文件用來說清楚 Elaris 玩家端的 `種族 / 支系 / 特質 / 具名構造 / 能力方向` 到底怎麼分。

它的目的不是把所有種族都做成同一種規格，而是避免把 `文化傾向`、`地方習俗`、`正式分支`、`具名器官`、`代表招式` 混在一起。

## 這份文件回答什麼

- 哪些內容算 `正式種族層`
- 哪些內容算 `正式分支層`
- 哪些內容可以寫成 `具名特質 / 具名構造`
- 哪些內容只能寫成 `能力方向 / 玩法傾向`
- 玩家創角時應該先看哪一層
- AI 裁定時應該把哪些內容視為正典事實，哪些只能當風格提示

## 四個層級怎麼分

### 1. 種族層

這一層回答的是：
`你天生屬於哪一大類存在。`

例子：
- 人類
- 海人
- 精靈
- 獸人
- 魔人

種族層通常決定：
- 世界裡你被怎麼辨認
- 你大致可能擁有哪些生理或文化基底
- 你能不能再往下選正式分支

### 2. 正式分支層

這一層回答的是：
`這個大種族裡，你屬於哪一條被正典承認的支系。`

例子：
- 海人：`潮霧之裔 / 熾潮之脈 / 鱗盾之裔`
- 暮鏡荒境獸人：`月影狼裔 / 霜爪熊裔 / 影尾狐裔 / 暮羽雀裔`
- 人類具體支系：`極光徙裔 / 魈陽族 / 崖影族 / 巖穹族 / 薩魯恩人 / 柯沙里人 / 諾魯因人`

注意：
- 有些人類名稱本身就是最終支系，不能再硬拆第二層正式分支
- 地方行路系、守域圈、節市圈、職責圈，不等於正式種族分支

### 3. 具名特質 / 具名構造層

這一層回答的是：
`正典有沒有明寫、而且能被穩定指認的特殊體質、器官或構造。`

例子：
- 人類：`極光紋瞳 / 炁紋脈 / 崖攀肢 / 巖息法 / 裂脈感 / 花隱膚 / 花藏息 / 潮骨血息`
- 海人：`潮囊 / 霧鰭脈 / 熾脈核 / 鰭刺束 / 鱗盾背甲 / 潮聲鰭刺`

只有在正典已明寫時，才可以把它們當成正式名稱使用。

若正典沒有明寫，就不要自己替精靈、獸人、魔人或地方支系補新的器官名。

### 4. 能力方向 / 代表玩法層

這一層回答的是：
`這個種族或分支，通常在什麼環境、節奏、工作或戰鬥風格上特別有感。`

例子：
- 柯沙里人：`霧林溪谷換徑、曲水渡行、歌路記憶、節律型隱行`
- 薩魯恩人：`沙原辨路、裂帶感知、長旅耐久`
- 潮霧之裔：`潮窗判讀、霧潮移位、外緣採樣、海霧感知`

這一層可以很重要，但它不一定等於固定招式名，也不一定代表有具名器官。

## 哪些可以寫死，哪些不能

### 可以直接寫成正典事實

- 正式種族名
- 正式分支名
- 已明寫的具名特質
- 已明寫的具名構造
- 已明列的代表能力
- 已明列的起始圈與主要生活帶

### 只能寫成風格或傾向

- 某一族常見工作類型
- 某一族擅長的場景
- 某一族常見的戰鬥節奏
- 某一族在玩家端的玩法感受

### 不可以自行補完

- 新器官名
- 新血統名
- 新分支名
- 新正式招式名
- 新歷史設定
- 新社會制度，只因為看起來很合理

## 玩家創角時怎麼用

### 先決定你在選哪一層

- 若你選的是 `海人 / 精靈 / 獸人 / 魔人`，代表你還在大種族層，接下來還要選正式分支
- 若你選的是 `諾魯因人 / 柯沙里人 / 薩魯恩人`，代表你已經選到具體支系層

### 再決定你要的是哪種內容

- 想知道背景與起始圈：看 `種族層 + 分支層`
- 想知道你能拿什麼能力：看 `具名特質 / 構造層`
- 想知道實際玩起來像什麼：看 `能力方向 / 代表玩法`

### 最後才接職業

種族能力的作用是：
`改變玩法風格`

不是：
`直接取代職業`

所以一個角色的完整感，通常來自：
`種族 / 分支 + 職業 + 起始裝備 + 特質 / 缺陷 + 起始地區`

## AI 裁定時怎麼用

### 有具名特質或構造

可以當成：
- 穩定被動優勢
- 情境性加成
- 招式或主動能力的合法來源

### 只有能力方向

就把它當成：
- 玩家描述行動時的合理強項
- 場景優勢
- 風格提示

不要把它硬升級成新的正式名詞。

### 只有文化與地形傾向

就把它當成：
- 常見起始身份
- 常見生存環境
- 常見價值觀與應對方式

不要把文化描述誤判成生理能力。

## 幾條現在最重要的現行讀法

### 人類具體支系

- `極光徙裔 / 魈陽族 / 崖影族 / 巖穹族 / 薩魯恩人 / 柯沙里人 / 諾魯因人`
  已經是可直接使用的具體支系
- 其中具名特質已明寫者，可以直接進能力卡與裁定

### 海人正式三支

- `潮霧之裔 / 熾潮之脈 / 鱗盾之裔`
  既是正式分支，也有具名構造與能力方向

### 精靈、獸人、魔人

- 目前很多條目仍以 `正式分支 + 能力傾向` 為主
- 若沒有已明寫器官名，就不要把傾向誤寫成器官

### 柯沙里人

- 保持具名特質：`花隱膚`、`花藏息`
- 文化與生活帶偏 `霧林溪谷 / 谷寨水行 / 織紋銀飾 / 歌路記憶`
- 這些是生活與能力方向，不代表可以再拆出新的正式分支

## 一句話版判讀規則

- `有正式名 + 有具名特質`：可以當穩定能力來源
- `有正式名 + 只有能力方向`：當玩法與場景優勢
- `只有地方名稱或職責圈名`：不要當正式種族分支
- `沒有正典明寫`：不要硬補
