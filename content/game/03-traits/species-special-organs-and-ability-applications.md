# 種族特殊器官、能力與招式應用

## Purpose
這份文件把世界觀中的 `支系專屬構造` 壓成玩家與主持端都能直接使用的應用規則。

## 玩家使用原則
- 這份文件優先處理 `玩家可選或常見可用種族`。
- 像龍族、鳳族與其他神獸古族這種 `不可玩家選擇的高位古族`，不放在這份玩家向應用檔裡。
- 若後續要做龍族、神獸或災害級存在的能力，應另開主持人向或頭目向專章。

## 先說清楚
- 不是每個種族都有 `器官型` 特殊構造
- 有些種族比較偏 `環境適應`
- 有些則是明確的 `骨血、腺體、核囊、聽骨、換息膜` 這類先天構造
- 也有一部分人類系支族，會是 `具名地方體質`，例如 `極光紋瞳 / 炁紋脈 / 崖攀肢 / 巖息法 / 裂脈感 / 花隱膚 / 花藏息 / 潮骨血息`

## 玩家端應用時要回答的五件事
- 這個構造平常帶來什麼世界互動優勢
- 它在戰鬥裡能轉成什麼能力
- 它最怕什麼
- 它容易和哪些職業相性好
- 它會長出什麼代表招式

## 不同層級的差別

### 種族共通特徵
- 所有人都有
- 比較像底色

### 支系專屬構造
- 只屬於某支系或某種族分支
- 能穩定對應能力、抗性與招式
- 有些是器官，有些則是骨血、感官、換息、共鳴或先天記憶能力
- 人類系族群尤其常是 `地方適應 + 社會訓練 + 生境感官` 的組合，而不是新器官

### 個體型特殊體質
- 不是族群常規
- 更像突變、災變、過度暴露或意外

## 應用欄位建議
- 名稱
- 所屬種族或支系
- 構造類型
- 世界互動效果
- 戰鬥效果
- 對應能力值
- 對應職業
- 代表招式
- 風險與代價

## 不是所有種族都要做成招式包
- 法雷恩人這類，更適合寫成 `寒地適應 / 守夜視界 / 雪地穩定`
- 諾魯因人、海人、魔人這類，則更容易長出明顯的構造應用

## 與戰技術式的關係
- 有些構造偏向放大戰技
- 有些構造偏向放大術式
- 有些構造更適合長成獨立的 `種族招式`

例如：
- 極光徙裔的 `極光紋瞳` 會直接影響夜視、辨星與寒霧判路
- 魈陽族的 `炁紋脈` 會放大熱流承受與短時爆發
- 崖影族的 `崖攀肢` 會直接影響攀壁、抓索與崖地轉線
- 巖穹族的 `巖息法` 會放大高寒調息、力量穩定與高地感知
- 薩魯恩人的 `裂脈感` 更像沙脈微震與塌澤預判
- 柯沙里人的 `花隱膚 / 花藏息` 會放大耐毒、解毒與幻花環境適應
- 諾魯因人的 `潮骨血息` 比較偏守息、辨壓與低光行進
- 海人的 `潮囊 / 霧鰭脈 / 熾脈核 / 鰭刺束 / 鱗盾背甲 / 潮聲鰭刺` 會直接決定各分支的水域戰法與守域功能
- 魔人的器官則常決定整條血裔的核心戰法
- 翼人的 `風環平衡` 會直接影響滑翔、轉向與高流穩定
- 精靈的 `靈詩長記與靈誓伴共鳴` 會放大感知、儀式與同行能力

## 人類系補充原則
- 並不是所有人類系族群都應該有穩定的先天特殊能力。
- `瓦雷溫人 / 法奧人 / 法雷恩人` 這類，更適合寫成：
  - 文化訓練差異
  - 制度優勢
  - 長期生活環境造成的偏好
- `極光徙裔 / 魈陽族 / 崖影族 / 巖穹族 / 薩魯恩人 / 柯沙里人 / 諾魯因人`
  則可以合理寫成更鮮明的地方適應與常見招式模板。
- 矮人、獸人、地精則更常以 `先天感官與生境技能` 的形式表現
