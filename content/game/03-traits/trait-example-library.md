# 特質、血脈與特殊體質示例庫

## Purpose
這份文件提供第一批可直接引用的示例，但它不再把所有內容都當成可自由挑選的 trait。

這裡的示例分成三層：
- `支系專屬構造`
- `個體型特殊體質`
- `後天特質`

## 使用方式
- 角色先確認自己的 `種族` 與 `支系`
- 若該支系在世界設定中有專屬構造，再決定是否具備該構造
- 若角色另有災變、異常覺醒或靈核問題，再考慮個體型特殊體質
- 最後才補後天特質

## 一、支系專屬構造示例

### 1. 曜血核脈
- 類型：支系專屬構造 / 器官型
- 所屬：魔人 / 曜血
- 外顯徵象：胸骨深處有血脈核鳴，盛怒或高位血契時浮出暗紅脈光
- 核心功能：維持曜血裔的高壓血息循環與貴脈自復
- 弱點或代價：核脈一旦被重創，血流逆噬，自滅極快

### 2. 魅息腺
- 類型：支系專屬構造 / 器官型
- 所屬：魔人 / 鳴紗
- 外顯徵象：喉腔與舌根附近有異常魅息分泌反應
- 核心功能：支撐魅惑、惑聲與外貌穩定
- 弱點或代價：腺體受封後，魅惑崩解，外貌也可能失穩

### 3. 垢金核
- 類型：支系專屬構造 / 器官型
- 所屬：魔人 / 垢金
- 外顯徵象：腹腔深處有高密度礦性核體
- 核心功能：支撐吞金、礦質吸納與金屬性自強化
- 弱點或代價：核體碎裂時，會出現劇烈自腐

### 4. 影心珠
- 類型：支系專屬構造 / 器官型
- 所屬：魔人 / 幽鬩
- 外顯徵象：腦後或脊椎核心存在影性珠核
- 核心功能：支撐奪形、潛影與形態模擬
- 弱點或代價：心珠毀損時，形散魂滅

### 5. 血爆囊
- 類型：支系專屬構造 / 器官型
- 所屬：魔人 / 獰罡
- 外顯徵象：體內存在可高度壓縮怒血的囊狀結構
- 核心功能：提供爆發怒焰、狂戰與血爆能力
- 弱點或代價：囊體破裂時，容易血焰逆噬

### 6. 夢核
- 類型：支系專屬構造 / 靈魂核型
- 所屬：魔人 / 緲眠
- 外顯徵象：夢境與現實的邊界異常薄弱
- 核心功能：支撐夢魘術、寄魂與夢域操縱
- 弱點或代價：夢核若受割裂，會陷入長期夢魘或失魂

### 7. 噬源
- 類型：支系專屬構造 / 深層內核型
- 所屬：魔人 / 噬淵
- 外顯徵象：體內存在吞噬性極高的源性核心
- 核心功能：支撐吞噬、吸納與自體變形
- 弱點或代價：一旦被封鎖，可能出現自噬崩解

### 8. 極光紋瞳
- 類型：支系專屬構造 / 感官型
- 所屬：人類 / 極光徙裔
- 外顯徵象：瞳孔周圍浮現極光紋圈
- 核心功能：強化暗視、寒霧辨路與星紋記憶
- 啟動條件：部分族人先天可見，部分需長徙長老啟瞳
- 弱點或代價：強光與雪面高反照時易出現視壓負擔

### 9. 炁紋脈
- 類型：支系專屬構造 / 脈絡機制型
- 所屬：人類 / 魈陽族
- 外顯徵象：背脊或手臂可見淡炁紋
- 核心功能：提升熱流導引、煙炁掌控與熔誓鍛造適性
- 啟動條件：成年禮中的 `熔誓刻印`
- 弱點或代價：過度引炁時，紋脈可能灼裂

### 10. 崖攀肢
- 類型：支系專屬構造 / 肢體機制型
- 所屬：人類 / 崖影族
- 外顯徵象：手掌與足底肌理異常粗韌，趾骨彎鉤能力強
- 核心功能：強化攀崖、繩滑、無聲越壁
- 啟動條件：成年禮 `無聲攀徑`
- 弱點或代價：在平地厚鞋與拘束環境裡反而不易發揮

### 11. 巖息法
- 類型：支系專屬構造 / 呼吸機制型
- 所屬：人類 / 巖穹族
- 外顯徵象：啟息時肌理浮出嶺紋與冷霧光
- 核心功能：提高耐寒、感知與高嶺行道穩定度
- 啟動條件：嶺息誓禮後逐步穩定
- 弱點或代價：離開山息環境太久時，啟息效率下滑

### 12. 裂脈感
- 類型：支系專屬構造 / 足底感應型
- 所屬：人類 / 薩魯恩人
- 外顯徵象：足底與腿脈對地息微震異常敏感
- 核心功能：預判沙脈、裂谷與塌陷
- 啟動條件：自幼裸足修息
- 弱點或代價：在混亂震動環境中容易過載

### 13. 花隱膚與花藏息
- 類型：支系專屬構造 / 皮膚與內息雙機制型
- 所屬：人類 / 柯沙里人
- 外顯徵象：膚質光潤，對花粉與毒息反應異常穩定
- 核心功能：隔絕部分花毒、提升解毒與代謝能力
- 啟動條件：自幼接觸林中微毒與幻花息
- 弱點或代價：離開熟悉林息太久時，身體節律會變差

### 14. 潮骨血息
- 類型：支系專屬構造 / 骨血型
- 所屬：人類 / 諾魯因人
- 外顯徵象：骨質密、血色重、傷口封合快
- 核心功能：耐寒流、耐潮壓、深潛長息
- 弱點或代價：在極乾燥高熱環境裡恢復效率下降

### 15. 潮囊與霧鰭脈
- 類型：支系專屬構造 / 海人分支器官型
- 所屬：海人 / 對應分支
- 外顯徵象：胸腹近側有潮囊，鰭脈對潮壓與霧流敏感
- 核心功能：水壓調節、霧潮環境偵測
- 弱點或代價：囊體失衡時容易出現浮沉失調

### 16. 熾脈核與鰭刺束
- 類型：支系專屬構造 / 海人分支器官型
- 所屬：海人 / 對應分支
- 外顯徵象：體內有熱性核構造，鰭刺束硬化明顯
- 核心功能：偏熱流、防衛與高速破水
- 弱點或代價：在極寒深水環境中耗損更快

### 17. 鱗盾背甲與潮聲鰭刺
- 類型：支系專屬構造 / 海人分支外覆型
- 所屬：海人 / 對應分支
- 外顯徵象：背甲與鰭刺結構穩定發育
- 核心功能：防護、衝撞與潮聲感知
- 弱點或代價：脫水與乾裂會大幅削弱機能

## 二、個體型特殊體質示例

### 18. 灼痕體
- 類型：個體型特殊體質 / 靈息染體
- 來源：火性靈息殘留、災場生還、高熱靈場暴露
- 主要表現：體表或體內留有持續性灼痕靈息
- 優勢：對火性術式與高熱環境的耐受可能提升
- 風險：失控時容易反覆灼傷自身

### 19. 光痕體
- 類型：個體型特殊體質 / 靈息染體
- 來源：光性儀式、神聖場域、權柄回聲殘留
- 主要表現：體表、瞳底或靈息流動帶有持續光痕
- 優勢：對照明、見證與部分感知類術式更敏銳
- 風險：在遮蔽、潛伏與暗行場景中更難隱匿

### 20. 夢痕體
- 類型：個體型特殊體質 / 靈息染體
- 來源：夢域侵染、長期夢兆干涉、特殊神諭殘留
- 主要表現：清醒與夢境邊界偏薄
- 優勢：更容易接收夢兆、預感與意象記憶
- 風險：也更容易受夢魘、幻夢與失眠侵擾

### 21. 裂核體
- 類型：個體型特殊體質 / 靈核障化
- 來源：靈核結構受損、未穩定覺醒、過度施術
- 主要表現：靈核運轉不穩，常有突發性失衡
- 優勢：短時爆發可能異常高
- 風險：持續使用會引發反噬與崩核危險

### 22. 擾核體
- 類型：個體型特殊體質 / 靈核障化
- 來源：長期接觸干擾場域、遺器或共鳴事故
- 主要表現：靈核頻率易受外物牽動
- 優勢：對某些共鳴物品與場域反應特別快
- 風險：在高干擾區容易失眠、頭痛、施術失準

### 23. 情緒共鳴
- 類型：個體型特殊體質 / 情緒共鳴
- 來源：強烈創傷、極端意志、長期靈息壓抑
- 主要表現：情緒會直接放大術式、感知或場域反應
- 優勢：在情緒穩定且目標明確時，可大幅提高表現
- 風險：情緒失控時也最容易波及自身與旁人

### 24. 靈息轉質者
- 類型：個體型特殊體質 / 靈息轉質
- 來源：特殊資源、異常改造、長期權柄回聲侵染
- 主要表現：靈息性質逐漸轉成異質狀態
- 優勢：可對某些術式或材料產生罕見相性
- 風險：身體、靈核與社會認知都可能逐步失穩

## 三、後天特質示例

### 25. 久行筋骨
- 類型：後天特質 / 訓練型
- 來源：長年巡路、搬運、徒步與野外行旅
- 核心效果：長程移動更穩，旅途疲勞累積較慢
- 限制或代價：不代表短時間爆發也強

### 26. 濕地肺
- 類型：後天特質 / 生態型
- 來源：河港、湖岸、霧林與潮濕聚落長期適應
- 核心效果：對潮氣、濕冷與低度黴氣適應較佳
- 限制或代價：進入乾熱高塵環境時較不舒服

### 27. 菌境鼻息
- 類型：後天特質 / 生態型
- 來源：菌區、藥圃、發酵與腐殖環境長期生活
- 核心效果：更容易辨識藥味、菌味、腐敗與毒孢異常
- 限制或代價：對刺激性氣味特別敏感

### 28. 靜潮心守
- 類型：後天特質 / 精神型
- 來源：守夜、安魂、醫護與高壓事故處置
- 核心效果：面對災後現場與慌亂人群時更穩
- 限制或代價：情緒表達較慢，易被誤解成冷淡

### 29. 市集耳目
- 類型：後天特質 / 社會型
- 來源：長期混跡市場、碼頭、灰線與攤販人脈
- 核心效果：更容易察覺價格異常、風聲與門路
- 限制或代價：正式場合中常被視為不夠正統

### 30. 章程腦
- 類型：後天特質 / 社會型
- 來源：學院、官署、文契系統與抄錄訓練
- 核心效果：擅長閱讀條款、辨識權限與制度漏洞
- 限制或代價：在完全即興場景中反而慢半拍

### 31. 災後倖存者
- 類型：後天特質 / 事件型
- 來源：遺跡事故、崩線、獸潮或災變中生還
- 核心效果：面對類似災場時更能抓住生還重點
- 限制或代價：容易被特定氣味、聲響或場景觸發回想

### 32. 受名回音
- 類型：後天特質 / 誓約型
- 來源：正式受名、命名見證、祝護與誓網承認
- 核心效果：對真名、職責與命名系統有更穩定回應
- 限制或代價：一旦命名權威受損，反噬也較深

## 四、使用建議
- 不要把 `支系專屬構造` 當作一般 trait 隨意分配
- 不要把 `個體型特殊體質` 寫成某支系的普遍天賦
- 後天特質可以多一點，但仍應服務角色生活與玩法

## 五、典型配置示例
- 極光徙裔巡夜人：
  - `極光紋瞳`
  - `久行筋骨`
  - `靜潮心守`

- 魈陽族熔誓鍛匠：
  - `炁紋脈`
  - 工坊系後天特質

- 柯沙里林息採藥者：
  - `花隱膚與花藏息`
  - `菌境鼻息`

- 災場生還的抄錄官：
  - 支系本身未必有專屬構造
  - `裂核體` 或 `夢痕體`
  - `章程腦`
  - `災後倖存者`
