# 特質組合建議庫

這份建議庫不是硬性最佳解，而是幫玩家快速知道：
`某種族 / 分支 / 職業 / 能力` 通常會長成什麼風格。

## 諾魯因人 + 引水人
- 風格：冷潮航路、低光判位、帶隊穿越危水
- 推薦補強：穩定、專注、保人類型能力

## 極光徙裔 + 學者
- 風格：極地觀測、異光記錄、遠地調查
- 推薦補強：專注、長行動、低溫對應

## 月影狼裔 + 巡翼哨手 / 狩窗手
- 風格：追跡、警戒、夜間巡守
- 推薦補強：感知、移動、伏擊反制

## 霜爪熊裔 + 城衛兵 / 重裝守護
- 風格：撐線、護人、扛壓
- 推薦補強：耐久、威嚇、站位控制

## 潮霧之裔 + 外緣採樣手
- 風格：濕地樣本、低視野環境、微水路行動
- 推薦補強：採樣、回避、穩息

## 熾潮之脈 + 靈廚 / 危材處理
- 風格：熱潮區工作、危材分流、灶前高壓
- 推薦補強：耐熱、判材、抗污染

## 曜血 + 古文讀解手
- 風格：高風險深淵知識線
- 推薦補強：專注、壓力耐受、代價管理
