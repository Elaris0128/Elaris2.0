# 聲望與立場索引

這一層負責處理角色在 `國家 / 公會 / 聚落 / 地方人群 / 灰線勢力` 眼中的位置。

聲望不是單純的好感度，而是把下列幾種東西一起看：
- 公開立場
- 私下信任
- 嫌疑熱度
- 合法性壓力
- 是否有進入內圈的資格

## 這層能回答什麼
- 玩家在某個地區目前算是 `陌生 / 容忍 / 可合作 / 受信任 / 內圈`
- 玩家會不會被查驗、盯梢、封鎖、通緝
- 哪些任務、執照、裝備、禁區會因聲望改變
- 角色做了什麼會讓局勢升溫或轉好
- 某公會會不會給你進階資格、旁聽、內圈案源與專用設施
- 一次失敗會是單純掉好感，還是直接進入 `嫌疑 / 通緝 / 失格` 軸

## 建議實際追蹤的勢力數量
- 新手團：`3-5` 個就夠
- 中期團：`6-10` 個主要勢力
- 長團：只有真正會持續出手的勢力才值得長追
