# 背包與攜行索引

這個目錄收的是 Elaris 玩家端可直接使用的 `背包 / 容器 / 補給 / 負重 / 追蹤` 規則。

它回答的問題是：
- 角色到底能帶多少東西
- 東西放在哪裡會影響什麼
- 補給要怎麼配，才不會第一天就卡住
- 過重、帶錯容器、保存失敗時，玩法會怎麼變

## 這一層回答什麼
- `身上快速欄` 應該放什麼
- `隨身包 / 主背包 / 貨具 / 寄存` 怎麼分
- 哪些東西一定要用對容器
- 哪種旅程需要哪種補給包
- 超載時是變慢、變吵、變脆弱，還是全部一起發生

## 使用原則
- 初期遊戲可先只記 `身上 / 背包 / 貨具 / 寄存`
- 中期再加入 `容器類型 / 保存風險 / 查驗風險`
- 只有真的會改變局勢的東西，才需要精細追蹤
- 背包規則的目的不是折磨玩家，而是讓 `帶什麼` 真的會變成選擇
