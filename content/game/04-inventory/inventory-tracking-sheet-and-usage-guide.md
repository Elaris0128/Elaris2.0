# 背包追蹤表與使用指引

這份文件提供最簡單、最不拖節奏的背包追蹤方式。

## 一、玩家最簡版

只記四欄：
- `身上快速欄`
- `背包`
- `貨具`
- `寄存`

再補兩個狀態：
- `目前負重`
- `目前有沒有需要特殊保存的東西`

## 二、建議追蹤模板

### 身上快速欄
- 武器：
- 急救：
- 證件：
- 即抽即用工具：

### 隨身包 / 主背包
- 基礎補給：
- 專業工具：
- 消耗品：
- 回收物：
- 特殊樣本：

### 貨具 / 騎獸 / 車隊
- 大宗貨：
- 重型裝備：
- 長程補給：
- 暫時不急用的器物：

### 寄存
- 城鎮寄存：
- 公會保管：
- 私人藏點：

### 狀態
- 目前負重：`輕裝 / 標準 / 壓重 / 過重 / 崩帶`
- 特殊保存：`無 / 保鮮 / 封樣 / 防潮 / 防刺 / 灰線`

## 三、什麼時候一定要更新
- 打完一場大戰
- 回收大型戰利品
- 切換地區旅程
- 進城查驗前
- 把東西寄存、出售、封樣後

## 四、什麼時候不用每件都記
- 食水和普通乾糧若夠一兩天，可直接寫成 `短程補給`
- 一般衣物與雜貨可打包成 `日常行旅`
- 只有真正會改變玩法的物品才需要逐件列

## 五、最適合逐件列的東西
- 起始武器與防具
- 關鍵證件
- 專業工具
- 封樣樣本
- 高價值回收物
- 任務目標物
- 灰線物品

## 六、AI / 主持怎麼用

### 開場時
- 不要逼玩家列完全部
- 先問：`你身上快取什麼、背包裡大致是什麼、你現在偏輕裝還是偏重裝`

### 中途需要裁定時
- 先看玩家有沒有合理描述 `東西放哪裡`
- 再決定：
  - 能不能立刻拿到
  - 會不會先摔壞
  - 查驗時會不會很麻煩

### 遇到回收大件時
- 問這三句通常就夠：
  - `你們要立刻帶走，還是先做處理？`
  - `誰來扛？`
  - `你們願不願意為了這東西變過重？`

## 七、和角色紙怎麼接
- 角色紙上建議至少有：
  - 身上快速欄
  - 主背包
  - 貨具 / 寄存
  - 目前負重
  - 關鍵容器

## 八、實用判斷原則
- 玩家描述越清楚，越容易避免「早知道我其實有帶」的爭議
- 但不要把每次翻包都演成會計
- 追蹤的重點不是細帳，而是 `選擇的後果`
