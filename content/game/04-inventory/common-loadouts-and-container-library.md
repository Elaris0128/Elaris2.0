# 常用容器與旅裝示例庫

## Purpose
這份文件提供第一批 `容器 / 貨具 / 旅裝配置` 的通用示例，讓角色、NPC、商店與任務可以直接引用。

## 使用方式
- 容器可直接用於角色卡、商店貨架、探索任務與掉落保存描述。
- 旅裝組可用在開局配備、NPC 裝備、護送隊與探索隊配置。

## 常用容器示例

### 1. 三層旅人背包
- 類型：普通背包
- 可放物品傾向：旅糧、衣物、繩具、常備工具
- 優點：分層清楚，適合中短程旅行
- 限制：防潮與防撞能力普通
- 保存條件影響：無
- 合法性影響：無

### 2. 硬殼封樣盒
- 類型：封樣盒
- 可放物品傾向：樣本、危險果實、脆性礦片、誓痕樣本
- 優點：防撞、防漏、可貼封條
- 限制：容量小，攜帶時占手
- 保存條件影響：提供基礎緩震與隔離
- 合法性影響：有封條時更容易通過正式查驗

### 3. 低溫保鮮匣
- 類型：保鮮匣
- 可放物品傾向：藥材、鮮果、料理半成品、活性食材
- 優點：能延長可食與可加工時間
- 限制：需要冰片、冷源或冷地支撐
- 保存條件影響：低溫、避光
- 合法性影響：攜帶高風險食材時較易獲得店家收購

### 4. 防刺籐筐
- 類型：防刺筐
- 可放物品傾向：帶刺果實、危險藤種、爆殼種子、海膽葡萄
- 優點：外殼有緩衝與挑刺層
- 限制：體積大，不適合戰鬥中快速取用
- 保存條件影響：減少刺殼互碰與爆裂
- 合法性影響：高風險食材公開運輸時較常被要求使用

### 5. 文契防潮筒
- 類型：防潮筒
- 可放物品傾向：通行證、票據、地圖、印契文書
- 優點：防潮、防折、防泥水
- 限制：不適合放厚型物件
- 保存條件影響：乾燥、避污
- 合法性影響：通行與查驗速度較穩

### 6. 草藥分袋帶
- 類型：腰包與工具帶
- 可放物品傾向：藥草、粉包、小瓶、止血布、去刺夾
- 優點：快取方便，適合醫療與靈廚
- 限制：容量小，容易沾染氣味
- 保存條件影響：依袋別提供簡單分類
- 合法性影響：無

### 7. 騎獸雙掛貨鞍
- 類型：騎獸貨具
- 可放物品傾向：旅裝、大宗補給、工藝零件、中量成品
- 優點：長途效率高，便於商旅與護送
- 限制：需騎獸適應，狹路與斷橋不便
- 保存條件影響：依附掛容器而定
- 合法性影響：過關時更容易被全面查驗

### 8. 灰貨雙底箱
- 類型：硬殼箱
- 可放物品傾向：文書夾層、少量禁制物、走私樣本
- 優點：夾層隱藏性強
- 限制：一旦被拆查，後果較嚴重
- 保存條件影響：一般
- 合法性影響：高風險灰線容器

## 常用旅裝配置

### 1. 短程城鎮行旅組
- 適用：一到兩日移動、城市間往返、普通委託
- 身上快速欄：短刃、通行證、錢袋、傷藥布
- 隨身包與腰掛：水袋、小糧包、火種、簡易繩
- 主背包：替換衣、燈具、小毯、雜貨盒
- 備註：最常見的平民與新手配置

### 2. 三日探索組
- 適用：遺跡外圍、山道、野外短探
- 身上快速欄：武器、封燈、止血包、訊號片
- 隨身包與腰掛：封樣盒、工具刀、粉筆、乾糧
- 主背包：濾水、備燈、繩索、保鮮匣、替換布
- 備註：適合探索者、公會委託者

### 3. 採集與安全處理組
- 適用：藥材、魔植、危險果實、小型樣本
- 身上快速欄：去刺夾、薄手套、小刀、標記牌
- 隨身包與腰掛：草藥分袋、急救布、小瓶
- 主背包：防刺籐筐、封樣盒、防潮筒、保鮮匣
- 備註：靈廚、藥師、採集者常用

### 4. 邊境守夜組
- 適用：寒地巡邏、雪關看守、長夜哨戒
- 身上快速欄：誓燈、見證片、短兵、應急藥包
- 隨身包與腰掛：熱飲囊、標記釘、哨音片
- 主背包：保暖披毯、封道繩、乾糧、文契筒
- 備註：特別適合法雷恩式邊境配置

### 5. 商旅押貨組
- 適用：車隊、船隊、長線批貨
- 身上快速欄：票據、護身武器、印契牌
- 隨身包與腰掛：小帳本、封條、應急金、訊號哨
- 主背包：替換衣、基礎補給、修補包
- 騎獸或貨具：雙掛貨鞍、標準貨箱、冷藏窖箱
- 備註：查驗與護送風險高

## 保存與查驗提醒
- 活性素材優先進 `封樣盒 / 保鮮匣 / 防刺筐`
- 文契、地圖與票據優先進 `防潮筒`
- 灰線貨若用錯容器，容易在查驗時直接暴露
- 能不能安全帶回城，很多時候比素材本身值多少更重要
