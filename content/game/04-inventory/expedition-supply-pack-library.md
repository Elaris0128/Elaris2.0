# 常用補給包與遠征裝載庫

這份文件提供玩家與 AI 直接選用的補給包模板。

## 使用原則
- 補給包回答的是 `這趟行程你帶哪些類型的東西`
- 它和起始裝備包不同，重點是旅程與任務需求
- AI 可以直接把這份當成開局後的第一批購物與整備建議

## 一、短程城市委託包
- 適用：王都、港邊、學都、單日或雙日任務
- 核心內容：水袋、小糧包、簡易藥布、短繩、文契筒、基礎照明
- 建議容器：三層旅人背包
- 最怕缺什麼：通行票據與快速止血物

## 二、三日遺跡外圍包
- 適用：遺跡前庭、山道外圈、短探查
- 核心內容：乾糧、濾水具、封燈、粉筆、繩索、封樣盒、備用燈料
- 建議容器：主背包 + 硬殼封樣盒
- 最怕缺什麼：標記工具與回收容器

## 三、藥材採集包
- 適用：藥師、採藥人、濕林與沼地採收
- 核心內容：草藥分袋、小瓶、止血布、去刺夾、薄手套、防刺籐筐、保鮮匣
- 建議容器：草藥分袋帶 + 防刺籐筐
- 最怕缺什麼：分類袋與保鮮條件

## 四、海路護送包
- 適用：港邊航務、幽語海、靈穹洋、焰潮洋
- 核心內容：防潮筒、封樣繩、短鉤索、保暖披肩、止暈物、夜航靜燈、訊號片
- 建議容器：文契防潮筒 + 防潮雙層旅箱
- 最怕缺什麼：文書保護與夜航照明

## 五、寒地守夜包
- 適用：北境、天冠洋、極冠嶺域、伊莎梅爾寒域
- 核心內容：暖膏、熱飲囊、替換手套、封道繩、哨片、保暖披毯、雪盲遮鏡
- 建議容器：主背包 + 快速欄
- 最怕缺什麼：保暖層與備用火種

## 六、高空巡守包
- 適用：曦環浮境、雲裂哨站、維爾高嶺
- 核心內容：風窗尺、掛鉤索、輕量繫帶、路引片、急救小包、靜燈罩
- 建議容器：快抽兵繫帶 + 輕背包
- 最怕缺什麼：掛鉤與補綁材料

## 七、工坊與礦道整備包
- 適用：杜羅山邦、裂星階、杜魯摩克隱域
- 核心內容：礦燈、敲擊錘、支架釘、繃帶、濾塵布、工序記板、小修補包
- 建議容器：防潮雙層旅箱或工坊拆件框
- 最怕缺什麼：修補件與粉塵防護

## 八、灰線快進快出包
- 適用：灰釘荒境、港邊灰線、黯鉚外線
- 核心內容：抹味密封囊、雙底箱、小額應急金、假副票、短刃、快修線材
- 建議容器：灰貨雙底箱
- 最怕缺什麼：撤退預案和合法掩護

## 九、深淵邊界巡守包
- 適用：亡極徹淵邊環、南渦接面、封存回收
- 核心內容：封存釘、抑味囊、記錄板、隔離布、簡易去活化粉、備用名錄片
- 建議容器：硬殼封樣盒 + 抹味密封囊
- 最怕缺什麼：隔離材料與記錄備份

## 十、長程商旅押貨包
- 適用：多日商道、跨海航路、隊伍護送
- 核心內容：通關文書、票據副本、修補包、應急糧、替換衣、帳簿、貨封條
- 建議容器：騎獸雙掛貨鞍或標準貨棧箱
- 最怕缺什麼：文書副本與快速現金

## 十一、玩家快速選法
- 要跑 `城市 / 官面 / 文書`：短程城市委託包
- 要跑 `遺跡 / 山道 / 短探`：三日遺跡外圍包
- 要跑 `採集 / 藥材 / 危果`：藥材採集包
- 要跑 `海路 / 護送 / 夜航`：海路護送包
- 要跑 `寒地 / 守夜 / 越冬`：寒地守夜包
- 要跑 `灰線 / 快取 / 高風險`：灰線快進快出包

## 十二、AI 使用建議
- 先看地區開局包決定 `哪種補給最合理`
- 再看職業決定 `要不要多帶專業工具`
- 若玩家背景偏外派、護送、長途任務，就把補給包升一級
