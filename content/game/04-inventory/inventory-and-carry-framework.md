# 背包、攜行與容器通用框架

## Purpose
這份文件定義 Elaris 的 `背包 / 攜行 / 容器 / 倉儲 / 保存` 規格。

它的目標是讓探索、採集、旅行、商店與掉落物，能真正接到角色身上，而不是素材和道具永遠像漂浮清單。

## 核心概念
- 角色能帶多少，不只看重量，也看體積、易碎性、污染性與保存條件。
- 容器不只是額外格子，而是會改變物品狀態、合法性與安全性。
- 特殊素材若沒有正確處理，可能會失值、腐壞、爆裂、污染或違法。

## 五層攜行結構

### 1. 身上快速欄
- 武器、急救、通行證件、即時可拿取的工具。
- 適合戰鬥、追逐與緊急使用。

### 2. 隨身包與腰掛
- 最常見的冒險者攜行層。
- 適合放補給、簡易工具、少量素材與替換品。

### 3. 主背包與載具包
- 適合長途旅行、採集、遺跡回收與多日補給。
- 取用較慢，但可承載更多。

### 4. 騎獸、車隊與貨具
- 適合商旅、重裝探索、公會補給與大型回收。
- 會額外牽涉護送、道路風險與合法查驗。

### 5. 倉儲與寄存
- 城市倉庫、公會保管、港倉、家族庫房、祕密藏點。
- 適合放高價值、違禁或短期無法帶走的東西。

## 保存條件

### 常見保存需求
- 乾燥
- 避光
- 低溫
- 緩震
- 密封
- 去活化
- 去刺或去毒
- 與其他物品隔離

### 常見失效狀態
- 腐敗
- 破損
- 潮化
- 漏液
- 共鳴失穩
- 氣味外洩
- 刺殼爆裂

## 攜行與探索的關係
- 高價值素材不一定能立刻帶走，可能要先做初步處理。
- 特殊地點的掉落若未正確封存，可能在回程途中失去價值。
- 大型回收應同時考慮路線、護送、人手與合法查驗。

## 合法性與查驗
- 某些物品即使能帶走，也未必能公開攜帶。
- 常見需要額外標記的物品：
  - 活性樣本
  - 武裝級素材
  - 封印物
  - 受污染資源
  - 未去刺或未處理的危險食材
