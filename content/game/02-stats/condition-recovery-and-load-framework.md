# 狀態異常、回復與負荷通用框架

## Purpose
這份文件定義 Elaris 的 `當前狀態 / 狀態異常 / 回復 / 負荷` 通用規格。

能力值回答的是角色 `能做到什麼`，這份文件回答的是角色 `現在還撐不撐得住`。

它的目標是讓戰鬥、探索、施術、誓約、生態事故與長途旅行，都能使用同一套狀態語言。

## 核心原則
- Elaris 的危險不只有生命值下降，也包括專注崩潰、靈息失穩、負荷累積與誓約回響。
- 狀態異常不應只服務戰鬥，也應服務旅行、生態、儀式與社交壓力。
- `負荷` 不是普通 debuff，而是高爆發、融合技、重裝、長途行軍與極端行動的共同壓力軸。
- 回復也不只靠睡覺或藥水，還應包括照護、安撫、導引、儀式與安全環境。

## 四條核心狀態軸

### 1. 生命狀態
反映肉身是否還能正常活動。

常見描述：
- 完整
- 輕傷
- 中傷
- 重傷
- 瀕危

### 2. 靈息狀態
反映靈息流動是否充足、穩定、可安全調用。

常見描述：
- 穩定
- 偏低
- 緊絀
- 枯竭邊緣
- 失穩

### 3. 專注狀態
反映角色是否還能穩定維持判斷、施術、儀式、見證或高精度行動。

常見描述：
- 穩定
- 分神
- 吃緊
- 崩散邊緣
- 已崩

### 4. 負荷狀態
反映肉身、靈核、術式與誓印承受壓力的程度。

常見描述：
- 低
- 中
- 高
- 警戒
- 臨界

## 負荷層級的意義

### 低
- 可正常活動
- 可承受一般戰技、施術與旅途壓力

### 中
- 已開始累積壓力
- 短時間再爆發會更容易出現失手與破綻

### 高
- 不適合再持續高輸出
- 行動判斷、穩定度與回復速度開始明顯下滑

### 警戒
- 任一強行爆發都可能導致反噬、昏厥、誓裂或施術失穩

### 臨界
- 已接近崩潰
- 繼續壓榨通常不該只用「扣數值」帶過，而應進入事故或失控敘事

## 狀態異常大類

### 1. 肉體異常
常見於戰鬥、跌落、寒熱、毒傷、疲勞與野外環境。

常見例子：
- 流血
- 骨裂
- 扭傷
- 凍僵
- 脫水
- 劇痛

### 2. 精神異常
常見於高壓、恐懼、幻術、長期失眠、目睹失衡事件與誓約壓迫。

常見例子：
- 恐慌
- 恍惚
- 執念化
- 崩潰
- 驚悸
- 麻木

### 3. 靈息異常
常見於施術過度、引息錯位、共鳴污染、地脈失衡與外來強制導入。

常見例子：
- 靈息紊亂
- 引息斷流
- 導式錯位
- 靈核震盪
- 共鳴過載

### 4. 誓約異常
常見於誓印反噬、契約衝突、誓語違逆、見證失衡與儀式出錯。

常見例子：
- 誓印灼痛
- 契約反咬
- 誓語沉壓
- 見證污染
- 誓裂前兆

### 5. 環境異常
常見於霧林、濕地、極寒、熱脈、礦毒、菌區與王者級領域壓力。

常見例子：
- 方向感錯亂
- 低視野壓迫
- 潮毒附著
- 孢粉侵擾
- 領域壓迫

## 狀態強度層級
任一狀態異常可再用強度層級表示：

### 輕度
- 仍可行動
- 但效率下降或風險上升

### 中度
- 已開始限制選項
- 需要處理，不然容易惡化

### 重度
- 強烈干擾行動
- 通常需要專業處理或安全休整

### 危急
- 若不立刻處理，會很快演變成失敗、昏厥、死亡或長期後遺

## 常見狀態來源
- 戰鬥受傷
- 長途旅行
- 缺水缺糧
- 睡眠不足
- 施術過度
- 融合技反震
- 誓印壓迫
- 共鳴污染
- 王者級魔物領域
- 生態毒性與採集失手

## 回復方式

### 1. 自然休整
- 睡眠
- 靜養
- 安全紮營
- 降低活動強度

適合：
- 疲勞
- 輕中度負荷
- 輕度精神壓力

### 2. 醫療與照護
- 包紮
- 正規醫療
- 安養
- 監護

適合：
- 肉體傷勢
- 失血
- 發燒
- 中重度生理異常

### 3. 靈息回復
- 冥想
- 地脈穩息
- 靈藥補助
- 誓器導引

適合：
- 靈息偏低
- 導式亂流
- 靈核震盪後的穩定化

### 4. 專注回復
- 安靜環境
- 情緒平復
- 誓文、樂曲、節律導引
- 可靠同伴陪護

適合：
- 分神
- 驚悸
- 施術後的精神吃緊
- 見證或審查後的壓力消解

### 5. 負荷回復
- 停止爆發行動
- 降低裝備壓力
- 緩和引息
- 改以低風險方式行動

適合：
- 高負荷
- 警戒狀態
- 重裝或長途行軍後的壓力累積

### 6. 特殊處理
- 去刺
- 去毒
- 孢粉淨化
- 導印校正
- 誓裂封止
- 樣本隔離

適合：
- 特定環境異常
- 高危採集事故
- 共鳴污染與誓約事故

## 回復節奏建議

### 短休
- 回一點專注
- 回一點低階負荷
- 不處理重傷與深層異常

### 長休
- 大幅改善疲勞、專注與一般負荷
- 可處理部分中度異常
- 仍未必能直接解除誓約異常與深層污染

### 專業處理
- 適合中重度異常
- 通常需要：
  - 店鋪
  - 組織
  - 地方設施
  - 特定職業

## 失控與崩潰觸發
當角色處於下列狀況時，應考慮引入失控、事故或敘事性後果：
- 高負荷以上仍連續爆發
- 專注已崩仍強行維持術式
- 靈息失穩時硬拉高輸出
- 誓印異常時仍繼續加碼契約效果
- 極端環境下缺乏正確保護與導引

## 主持使用建議
初期可先用簡化版本：
- 生命：完整 / 輕傷 / 中傷 / 重傷
- 靈息：穩定 / 偏低 / 緊絀 / 失穩
- 專注：穩定 / 分神 / 吃緊 / 崩散
- 負荷：低 / 中 / 高 / 警戒 / 臨界

若進入中後期，再加上：
- 異常種類
- 強度層級
- 特定恢復方法

## 設計原則總結
- 狀態系統應讓風險看起來像世界的一部分，不只是數值懲罰。
- 真正讓 Elaris 有辨識度的，是靈息、誓約、共鳴與負荷共同構成的壓力感。
- 回復應同時反映職業、地方設施、生態素材與社會資源，而不是只靠一瓶萬用藥。
