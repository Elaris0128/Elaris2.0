## 這層能回答什麼
- 角色當下是疲勞、受傷、失溫、失神，還是單純壓力太高
- 一個狀態會怎麼影響行動、施術、戰鬥與長行動
- 野外短休、城內療養、特殊照護各能回多少
- AI 怎麼快速給出合理修正，而不是每次臨時猜
- 玩家怎麼用一張表記住自己現在到底中了什麼
- 休息一次到底應該先降哪種壓力
