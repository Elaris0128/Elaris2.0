# 種族起始傾向與隱性修正

## Purpose
這份文件把 `玩家選擇種族` 之後，會固定帶來哪些起始能力傾向、隱性被動與世界反應整理清楚。

這些修正是 `輕量固定傾向`，不是把角色鎖死的絕對命運。

## 使用原則
- 所有人先從同一套基礎能力架構開始。
- 再套用 `種族起始傾向`。
- 再套用 `種族-分支` 的地方修正。
- 最後才由玩家決定自己的個人強弱項。

## 修正層級

### 1. 主種族修正
- 給 `2 項固定起始傾向`
- 給 `1 到 2 項隱性被動`
- 給 `1 項世界反應或環境限制`

### 2. 種族-分支修正
- 給 `1 項額外起始傾向`
- 給 `1 項地方被動`
- 若世界設定允許，開放 `支系專屬構造`

補充：
- 人類系多半直接把 `瓦雷溫人 / 法奧人 / 法雷恩人` 這些正式入口當作種族使用
- 因此人類系通常不先疊一層泛稱 `人類` 的主種族修正
- 而是直接採用下方的人類系種族入口修正

### 3. 特殊構造與特殊體質
- 不屬於一般種族修正
- 需另外判定

## 古典主種族固定傾向

### 魔人
- 起始傾向：`心志 +1`、`靈核 +1`
- 隱性被動：
  - `禁制感應`：更容易察覺崩解痕、禁制與毀誓殘留
  - `高壓施術耐受`：面對高風險術式與禁制時，較不容易立刻崩潰
- 世界反應：
  - 在高制度文明中更容易被預設警戒

### 森馳裔
- 起始傾向：`身法 +1`、`靈感 +1`
- 隱性被動：
  - `長程步脈`：長距離移動時，疲勞累積較慢
  - `生境親和`：更容易察覺群行、遷徙、牧行與共生節奏
- 世界反應：
  - 在原野、草原、牧路與林野共同體中較容易被視為懂路者

### 矮人
- 起始傾向：`體魄 +1`、`悟性 +1`
- 隱性被動：
  - `礦脈判讀`：更容易辨認礦脈、岩層與鍛材價值
  - `重工穩手`：重裝、鍛造與器械維護時更穩
- 世界反應：
  - 在工坊、山地堡線與契約工事體系中較容易被信任

### 地精
- 起始傾向：`悟性 +1`、`技巧 +1`
- 隱性被動：
  - `拆裝直覺`：更容易理解機巧、陷阱、零件與可回收結構
  - `灰線嗅感`：較容易捕捉市價異動與非公開門路
- 世界反應：
  - 官署與正規軍政體系通常較慢給予完全信任

### 海人
- 起始傾向：`身法 +1`、`靈感 +1`
- 隱性被動：
  - `潮向判流`：更容易讀懂潮汐、水壓與回流
  - `濕域行息`：在水域、潮霧與海風環境中的行動更穩
- 世界反應：
  - 在深內陸與極乾環境中，生活節律較難完全舒展

### 精靈
- 起始傾向：`悟性 +1`、`靈感 +1`
- 隱性被動：
  - `長記憶`：對長期知識、禮制、地名與誓名記憶更穩
  - `靈誓伴路徑`：更容易進入與靈誓伴、守林、靈詩與古儀相關的學習線
- 世界反應：
  - 在古林、儀式文明與守地神脈絡中辨識度更高
  - 重裝與高負荷近戰起步通常較慢

### 翼人
- 起始傾向：`身法 +1`、`洞察 +1`
- 隱性被動：
  - `風路感`：對風向、高差、滑流與空間落點更敏銳
  - `高域視野`：更容易察覺遠距動向與上方威脅
- 世界反應：
  - 在狹窄地形、重甲路線與長期重載時較難充分發揮

### 獸人
- 起始傾向：`體魄 +1`、`心志 +1`
- 隱性被動：
  - `群獵感`：更容易理解狩獵隊列、圍獵路線與威脅節奏
  - `祖骨緊繃`：面對恐嚇、血戰與高壓場景時更不易退縮
- 世界反應：
  - 在大城與高制度區常承受較強偏見
  - 在邊境、狩獵文化與氏族共同體中更容易獲得尊重

## 分支級固定傾向

### 使用原則
- `魔人 / 精靈 / 海人 / 獸人` 在主種族修正之外，還會再疊一層分支修正。
- 這一層主要用來表現各自真正的所長。
- `翼人 / 森馳裔 / 矮人 / 地精` 目前先停在主種族層，不再向下拆固定數值。

## 種族起始傾向的實際用法

### 1. 套用時機
- 在創角分配個人強弱項之前先套用
- 這會形成角色的自然起點

### 2. 玩家仍可逆著慣性培養
- 可以把精靈養成重裝近戰
- 也可以把獸人養成高悟性學者
- 但應承認他是逆著種族慣性成長出來的特殊例

### 3. 支系專屬構造不等於再加一輪起始點
- 構造主要改變玩法方向、環境適性、能力上限與弱點
- 不建議單純再疊更多固定數值
