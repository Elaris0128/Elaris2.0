# 職業分支與融合框架

## Purpose
這份文件補強 Elaris 的通用職業系統，定義世界共通的 `分支`、`融合職業` 與 `特殊職業` 邏輯。

## 職業層級

### 1. 通用譜系
代表最上層的世界通用職能來源。

- 武戰系
- 術法系
- 誓約系
- 技藝系

### 2. 通用職業類
代表明確的遊戲功能，例如：
- 重裝守衛
- 學院施術者
- 契靈行者
- 治療藥師
- 靈廚

### 3. 基礎分支
代表同一職業類往不同路線細分的子系。

例如 `重裝守衛` 可分成：
- 堡壘守衛
- 護衛型前線
- 反衝鋼衛

### 4. 融合分支
代表兩條不同譜系交會後形成的複合職業。

例如：
- 武戰 + 術法
- 武戰 + 誓約
- 術法 + 技藝
- 誓約 + 技藝

### 5. 特殊職業
代表需要特定條件、勢力、體質、誓約或地區傳承才能成立的高辨識職業。

## 四大譜系的世界整體分類

### 武戰系
重點在肉身、兵裝、位置、追擊、守勢與臨場反應。

常見主幹：
- 前衛型
- 守衛型
- 獵手型
- 斥候型
- 騎戰型
- 軍陣型

### 術法系
重點在引息、術式理解、結構控制與範圍變化。

常見主幹：
- 元素導引型
- 符印構式型
- 心念操控型
- 自然共鳴型
- 空間干涉型
- 召契型

### 誓約系
重點在誓言、代價、契約、儀式、審律與條件性力量。

常見主幹：
- 守誓型
- 審律型
- 契靈型
- 祭儀型
- 誓印型

### 技藝系
重點在工藝、醫療、調和、記錄、情報、商契與馴養。

常見主幹：
- 鍛造型
- 藥療型
- 靈膳型
- 修典型
- 情報型
- 商契型
- 馴養型

以 `靈廚` 為例，可往下細分成：
- 旅灶型：擅長旅糧、野營鍋與長途補給
- 藥食型：擅長安撫膳、恢復膳與照護型料理
- 宴席型：擅長儀式宴、交涉席與高價料理
- 危材處理型：擅長處理帶刺、帶毒、爆裂或共鳴不穩的特殊食材

## 分支設計原則
- 每個通用職業類至少可以往 2 到 4 條分支發展。
- 分支應代表玩法差異，而不只是名稱差異。
- 分支可以偏重戰鬥、探索、支援、社交或生產。

## 融合職業原則
融合職業不是把兩個職業名字接在一起，而是形成穩定的新玩法結構。

### 常見融合範型
- `武戰 + 術法`：戰法融合職
- `武戰 + 誓約`：誓武職
- `武戰 + 技藝`：工戰 / 狩工職
- `術法 + 誓約`：誓術職
- `術法 + 技藝`：鍊成 / 修典職
- `誓約 + 技藝`：誓器 / 聖療 / 審契職

### 融合職業範例
- 符刃使
- 誓武者
- 契獸騎士
- 靈藥調和師
- 靈膳調理師
- 誓宴廚
- 誓器鍛匠
- 修典術官
- 審契行者

## 特殊職業原則
特殊職業應至少滿足其中兩項：
- 特定地區文化
- 特定勢力或學派
- 特定種族或血統適性
- 特定誓約或儀式資格
- 特定體質或力量條件

特殊職業不應取代通用分類，而應建立在通用分類之上。

## 建議的職業記錄欄位
- 所屬譜系
- 通用職業類
- 基礎分支
- 可融合方向
- 進階條件
- 文化職稱範例
- 常見勢力與社會定位
- 對應能力值傾向
- 對應戰技 / 術式 / 誓約 / 技藝模組

## 設計原則總結
- 先定世界整體分類，再定各國叫法。
- 分支負責細化玩法，融合負責做出高辨識特色。
- 特殊職業應稀有但合理，不應讓通用職業失去意義。
- 後續每個國家新增地方職稱時，都必須回掛到這套框架。
