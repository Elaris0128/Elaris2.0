# 玩家職業卡庫

這份是玩家創角最常直接挑的起始職業卡。

## 引水人
- 玩法定位：帶路、辨潮、保隊伍不走錯
- 最擅長：危路、潮路、霧路、撤退線
- 最怕：完全陌生的內陸制度場景
- 建議主能力：感知 / 意志
- 建議副能力：社交 / 生存
- 最適合起始：幽語海、港邊航務圈

## 探索者
- 玩法定位：探路、進遺跡、找入口、帶回發現
- 最擅長：探索、決策、風險判斷
- 最怕：長時間純行政卡關
- 建議主能力：感知 / 體能
- 建議副能力：知識 / 生存
- 最適合起始：星盞、殞星海、天脈

## 藥師
- 玩法定位：處理藥材、穩定樣本、補隊伍狀態
- 最擅長：製藥、封樣、照護
- 最怕：高壓正面硬碰
- 建議主能力：知識 / 精細
- 建議副能力：意志 / 感知
- 最適合起始：霧羽、王都、病潮區

## 靈廚
- 玩法定位：補給、危材料理、穩定長途行動
- 最擅長：把危險食材變成可用資源
- 最怕：完全沒有時間準備的連續急戰
- 建議主能力：精細 / 感知
- 建議副能力：社交 / 生存
- 最適合起始：暮牙海、港邊、長途遠征

## 城衛兵
- 玩法定位：扛線、保人、守位、壓迫
- 最擅長：守門、卡路、護送
- 最怕：長時間高機動追逐
- 建議主能力：體能 / 意志
- 建議副能力：感知 / 社交
- 最適合起始：王都、邊境、暮鏡荒境

## 誓紋技匠
- 玩法定位：器具、結構、修補、技術支援
- 最擅長：誓紋器、機件、工具流解法
- 最怕：沒有設備的純野外長線
- 建議主能力：精細 / 知識
- 建議副能力：感知 / 社交
- 最適合起始：杜魯摩克、工坊圈、港務圈

## 文簿官
- 玩法定位：文件、記錄、流程、制度交涉
- 最擅長：查資料、過程序、做正規連結
- 最怕：完全無文書可用的荒野局
- 建議主能力：知識 / 社交
- 建議副能力：感知 / 意志
- 最適合起始：王都、學城、港務查驗圈

## 商人
- 玩法定位：交易、議價、人脈、補貨
- 最擅長：買賣、換資源、壓價、找熟路
- 最怕：純粹只看正面輸出的局
- 建議主能力：社交 / 感知
- 建議副能力：知識 / 精細
- 最適合起始：港邊、王都、市場圈、灰線圈

## 療傷手
- 玩法定位：救急、穩傷勢、把人活著帶走
- 最擅長：應急醫護、撤離、休整安排
- 最怕：重裝硬推型場景
- 建議主能力：意志 / 精細
- 建議副能力：知識 / 感知
- 最適合起始：寒荒、災場、外流圈

## 路線踏勘手
- 玩法定位：先探、找路、定可走與不可走
- 最擅長：追跡、獵路、高風險地帶導航
- 最怕：被迫長時間留在單一室內空間
- 建議主能力：感知 / 生存
- 建議副能力：體能 / 意志
- 最適合起始：暮鏡荒境、邊海、山路圈

## 打撈手
- 玩法定位：回收、拖取、判斷能拿什麼
- 最擅長：海上回收、沉物判斷、危險拆取
- 最怕：內陸純社交局
- 建議主能力：體能 / 感知
- 建議副能力：精細 / 生存
- 最適合起始：暮牙海、殞星海、港邊航務圈

## 學者
- 玩法定位：讀懂、辨真、整理、推理
- 最擅長：典籍、異象、制度與歷史
- 最怕：沒有任何資訊來源的亂戰
- 建議主能力：知識 / 意志
- 建議副能力：社交 / 感知
- 最適合起始：王都、學城、遺跡遠征圈

## 最短挑卡法
- 想穩：`引水人 / 城衛兵 / 療傷手`
- 想查：`學者 / 文簿官 / 探索者`
- 想做東西：`藥師 / 靈廚 / 誓紋技匠`
- 想跑商與人脈：`商人`
- 想走高風險路線：`打撈手 / 路線踏勘手`
