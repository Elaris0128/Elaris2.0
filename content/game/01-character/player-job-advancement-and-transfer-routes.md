# 玩家職業升階與轉職路線

這份是把玩家角色從開局一路長到中後期的職業路線講清楚。

## 先分清三種變化
- `升階`：同一主職越做越專，地位、責任與工具都變強。
- `轉職`：主職重心改變，玩法會明顯換軌。
- `融合`：主職不換，但副職逐漸變成固定第二支柱。

## 玩家最常用的六階段
1. `起步`
   第一次正式拿這份工作吃飯、活命或掛名。
2. `立足`
   已能獨立完成常規委託，不再只是學徒或跑腿。
3. `熟練`
   有穩定成果，別人會主動找你處理這類事。
4. `專精`
   開始形成個人風格，會被某個地區、公會或派系記住。
5. `成名`
   已有代表事蹟，能接到高風險、高價值或高政治性的工作。
6. `顯赫 / 傳聞級`
   你不只是做事的人，而是會改變一條線、一座城、一片海域局勢的人。

## 主職升階路線

### 引水人
- 起步：港邊副引、霧路帶渡、潮窗指路
- 立足：正式引水、夜航領路、危潮撤線
- 熟練：多港承認、能帶陌生船團過險帶
- 專精：深霧引水、軍港保線、極地冰路引航
- 成名：一地首席引水、航權顧問、跨洋保線人

### 探索者
- 起步：外圈探路、標記入口、帶樣本回來
- 立足：能獨立完成短程遺跡 / 山路 / 廢墟探索
- 熟練：會做風險判斷、撤退規劃和第二方案
- 專精：深遺跡、禁帶、深洋、極地任一方向
- 成名：能帶隊進入只有少數人生還過的地方

### 藥師
- 起步：藥材辨識、簡方穩傷、樣本保管
- 立足：能獨立調配常規藥物和緊急穩定劑
- 熟練：開始處理危材、病潮、封樣與副作用平衡
- 專精：病潮藥師、沼藥藥師、極地照護藥師、深海藥師
- 成名：公會授權、地方唯一、戰地指定或王都特聘

### 靈廚
- 起步：旅糧、保命粥、耐行食
- 立足：能處理危材與功能性膳食
- 熟練：開始以食物處理長行動、休整、交涉與恢復
- 專精：危材料理、儀式膳食、記憶宴席、極地補給
- 成名：能靠一桌飯扭轉局勢、和解衝突、穩住遠征隊

### 城衛兵 / 守衛型
- 起步：值勤、查驗、站位保護
- 立足：巡守、護送、城門判斷與短時壓制
- 熟練：能帶小隊、調整站位、處理混亂群眾與突發事件
- 專精：重裝守衛、邊境封線、王都近衛、極地哨守
- 成名：一地首席守衛、邊軍保線官、王庭認證護衛

### 商人 / 文簿官 / 交涉型
- 起步：跑價、抄單、議價、送票簿
- 立足：能獨立成交、過文件、保一筆貨安全入港
- 熟練：開始掌握人脈、信用、違約與補救
- 專精：高價貨、灰線貨、長途航貨、政治敏感貨
- 成名：不是賣東西，而是能調動一整條供應線

## 常見轉職方式

### 保留主職，補強副職
- 引水人 -> 加 `療傷手`
- 探索者 -> 加 `學者`
- 城衛兵 -> 加 `文簿官`
- 商人 -> 加 `情報掮客`

這類轉法最穩，角色會更好用，但仍保留原本核心身分。

### 主職換軌，但保留舊專長
- 城衛兵 -> 港巡兵 / 船長 / 護航者
- 探索者 -> 遺跡勘界手 / 古文讀解手
- 藥師 -> 隔棚照護員 / 療傷手 / 危材處理師
- 商人 -> 估價手 / 稅簽官 / 灰線撮合者

這類轉法最適合角色已累積明確地區經驗，想讓玩法更聚焦。

### 從民生職走向高風險職
- 靈廚 -> 遠征後勤員
- 採藥人 -> 藥師
- 漁戶 -> 打撈手
- 鍛匠 -> 誓紋技匠 / 黑鍛匠

這類轉法通常要靠一次事件、一次保薦或一次生還經驗。

## 文化升階
除了規則上的升階，Elaris 很多職業還有 `文化升階`。

常見形式：
- 從 `學徒` 升到 `掛名者`
- 從 `候補` 升到 `正職`
- 從 `副引` 升到 `正式引水`
- 從 `未受印` 升到 `受印技匠`
- 從 `臨時照護` 升到 `封棚照護員`

這類升階有時不直接加數值，但會帶來：
- 新工具
- 新開局資源
- 新委託層級
- 新執照或合法性
- 更高曝光與更重責任

## 玩家何時適合轉職
- 當主職已無法回答你想玩的玩法
- 當你經歷了一個足夠改變人生方向的大事件
- 當你想把副職變成角色第二根骨架
- 當地區、執照或勢力身份逼你改變工作方式

## 最短用法
1. 先定主職現在在 `起步 / 立足 / 熟練` 哪一段
2. 再決定你想補短板、換方向，還是把副職固定化
3. 最後把升階寫成一個世界事件，例如：
   - 拿到正式執照
   - 保住一整艘船
   - 成功帶隊撤出遺跡
   - 活過一次病潮處置

## 這份檔案回答什麼
- 玩家角色什麼時候該升階
- 什麼情況適合加副職
- 什麼情況適合轉職
- 升階是規則成長，還是文化身份升級，還是兩者一起來
