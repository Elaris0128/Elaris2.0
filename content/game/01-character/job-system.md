# 職業架構

## Purpose
這份文件定義 Elaris 文字冒險中的職業系統。職業不只是戰鬥分工，而是角色如何使用靈息、知識、武器、誓約與社會身分的方式。

## 世界觀原則

Elaris 的職業架構必須符合以下設定：
- 魔法與戰技並非完全分開，而是可協同應用
- 個體差異來自靈息感知、內在靈核、術式理解、身體訓練與特殊體質
- 同一個人可以是學術施術者、戰法融合者、誓約使用者或偏向民生職系
- 職業不等於種族，種族影響傾向，但不決定全部可能性
- 不同國家與文化會對相同職能賦予不同名稱、禮制與社會印象

## 系統定位

職業採用雙層模型：
- `通用職業種類`：負責規則與玩法定位
- `文化職業名稱`：負責世界觀名稱與地區風格

玩家最終的職業呈現方式為：
- `主職業種類` + `副職業種類` + `文化職業名稱` + `專精路線`

### 通用職業種類
代表角色實際扮演的功能類型，是整個遊戲系統的底層骨架。

它決定：
- 主要成長屬性
- 核心技能組
- 常見戰鬥與非戰鬥行動
- 主要裝備或施展媒介

### 文化職業名稱
代表同一種職能在不同地區、國家、教派、公會或軍政體系中的正式稱呼與社會身分。

它主要影響：
- 對話中的職稱顯示
- NPC 對角色的第一印象
- 裝備命名與儀式語彙
- 部分地區聲望、通關方式與入門門檻

原則上：
- `文化職業名稱` 不應直接改寫底層規則
- 若有特殊效果，應透過專精、背景或勢力身份實作

### 專精路線
代表角色在中後期逐步形成的獨特分支，例如誓武者、符刃使、港誓審查官、巡禮修典師等。

## 通用職業種類與文化職銜的關係

同一個通用職業種類，可以在不同文化中擁有不同名稱。

例如：
- 同樣是守護型前線武戰職，在王國可能被稱為王庭誓衛，在北境可能被稱為寒境守衛
- 同樣是學院施術職，在學術都市可能被稱為御式學士，在教權社會可能被稱為誓典術官
- 同樣是契靈型誓約職，在民間可能是契靈行者，在祭儀體系中可能成為召契祭官

因此後續設計時要先決定：
1. 這個角色的底層職能是什麼
2. 這個角色屬於哪一個文化圈或國家
3. 那個文化會如何稱呼這種人

## 四大通用職業譜系

### 1. 武戰系
以身體、兵器、戰技、位置掌控與臨場反應為核心。

適合：
- 無法穩定構築術式，但能使用戰技者
- 專精近戰、護衛、追擊、重擊、偵察者
- 與魔法有協同需求的前線角色

常見通用職業種類：
- 近戰前衛
- 重裝守衛
- 機動斥候
- 遠行獵手
- 軍陣武士

### 2. 術法系
以靈息感知、術式結構、引息與釋式控制為核心。

適合：
- 能穩定感知靈息並編織術式者
- 學院型、教廷型、遺法研究型角色
- 偏重範圍、控制、轉化、支援者

常見通用職業種類：
- 學院施術者
- 符印術者
- 元素引導者
- 自然共鳴者
- 遺法研究者

### 3. 誓約系
以誓言、誓印、靈契、教義與約束性力量為核心。

適合：
- 擅長誓環系統、契約、教廷、守誓體系者
- 能在政治、宗教、儀式、召喚與制約間切換者
- 在 Elaris 世界中特別有辨識度

常見通用職業種類：
- 守誓戰士
- 契靈行者
- 祭儀執行者
- 聖職審律者
- 誓印守護者

### 4. 技藝系
以知識、工藝、醫療、鍊成、情報與民生專業為核心。

適合：
- 不一定是純戰鬥角色，但能深度影響世界互動
- 可連動鍊金、誓器、藥劑、公會、商路與情報系統
- 很適合文字冒險的非戰鬥路線

常見通用職業種類：
- 鍊成工匠
- 治療藥師
- 靈廚
- 記錄學者
- 情報行者
- 商旅判契者

### 靈廚定位
- 所屬譜系：技藝系
- 核心定位：料理、旅糧、藥食、宴席與靈息調膳
- 常見媒介：灶具、鍋具、刃具、保鮮器皿、調和盒、低階引息廚具
- 世界互動價值：
  - 可把危險食材、魔植果料與探索素材轉成安全成品
  - 可支撐長途旅行、戰前補給、照護飲食與儀式膳食
  - 可把料理變成交涉、安撫、恢復與地方文化互動的一部分

靈廚不只是會做飯的人，而是懂食材靈息、處理風險、調整食效，並把食物變成世界互動工具的專業職類。

## 職業與魔法 / 戰技對應

### 純戰技型
- 靈息感知低或不穩
- 依賴體式、灌注、擊型與殘效
- 偏重體能、反應、穩定輸出

### 純術式型
- 靈息感知穩定
- 能使用引息、導式、轉式、釋式
- 偏重範圍、控制、術式精度

### 戰法混合型
- 可在戰技與術式間切換
- 對體力與靈核耐性要求更高
- 是 Elaris 中後期最有代表性的高階路線之一

### 誓約特化型
- 重視誓言、契約、代價與條件交換
- 可能不追求爆發傷害，但極擅長限定性強化與特殊效果

## 主副職規則

### 主職業種類決定
- 主要成長屬性
- 核心技能組
- 戰鬥與非戰鬥行動優勢
- 主要裝備或施展媒介

### 副職業種類提供
- 補足短板
- 擴展世界互動能力
- 打開額外任務解法
- 形成更鮮明的角色風格

### 文化職業名稱提供
- 世界觀稱呼
- 國家制度中的定位
- 社會認知與身份感
- 地區化風格與歷史感

### 範例組合
- 主職業種類：重裝守衛 / 副職業種類：聖職審律者
- 主職業種類：機動斥候 / 副職業種類：情報行者
- 主職業種類：學院施術者 / 副職業種類：鍊成工匠
- 主職業種類：近戰前衛 / 副職業種類：契靈行者
- 主職業種類：治療藥師 / 副職業種類：自然共鳴者
- 主職業種類：靈廚 / 副職業種類：自然共鳴者
- 主職業種類：靈廚 / 副職業種類：商旅判契者

## 專精路線範例

### 戰法融合
- 符刃使
- 雷光步鬥者
- 誓火衝鋒者

### 誓約與契靈
- 誓武者
- 契獸騎士
- 誓印審律者

### 知識與工藝
- 誓器鍛匠
- 靈藥調和師
- 靈膳調理師
- 遺文修典師

### 情報與灰色行動
- 暮環密探型職系
- 港誓查驗官型職系
- 邊境路印追跡者

## 玩家建立角色時的職業流程

1. 先決定角色比較接近哪一條通用職業譜系
2. 再決定主要使用的是戰技、術式、誓約還是技藝
3. 選擇主職業種類與副職業種類
4. 依出身國家、文化與勢力決定職業名稱
5. 最後選擇中後期可能走向的專精路線

## 設計原則

- 規則應先綁定通用職業種類，不要直接綁在文化名稱上
- 文化職業名稱可以有世界觀風味，但底層功能要清楚
- 不要把所有職業都做成只會打架
- 每個職業都應有世界互動價值
- 後續若新增地區特色職業，應先對應到某個通用職業種類
