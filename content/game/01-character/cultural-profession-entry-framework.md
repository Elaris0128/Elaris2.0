# 文化職業章節框架

## Purpose
這份文件定義 `文化職業` 的正式建檔方式。

在 Elaris 中，玩家與 NPC 先看到的應該是 `文化職業名稱`，而不是抽象的通用職業類別。
通用職業負責玩法定位，文化職業則負責世界觀、制度、社會印象與實際進入門檻。

## 核心原則
- 文化職業是主名，通用職業是備註。
- 每個文化職業都應明確寫出 `獲得條件`。
- 文化職業至少要能對應到一種軍事、諜報、宗教、學術、公會或民間功能。
- 文化職業最好綁定一到兩個 `專屬戰技 / 術式 / 儀式術`。
- 若某職業存在穩定傳承，應再標出 `流派方向`。

## 建檔欄位
- 文化職業名稱
- 類別
- 對應通用職業
- 常見所屬機構
- 獲得條件
- 常見媒介 / 裝備
- 專屬戰技 / 術式
- 流派方向
- 社會印象

## 獲得條件的常見來源
- 出身：血緣家學、支系傳承、部族成年禮、宿地或家隊認可
- 見習：公會見習、軍隊侍從、學徒、文書或雜務內訓
- 儀式：賜名、守誓、受印、尋伴禮、入教
- 能力：通過考核、完成試煉、能穩定使用指定媒介
- 關係：有合法保薦、被組織吸納、立功轉正

## 專屬能力的寫法
- `專屬戰技`：偏肉身、兵裝、隊列、狩獵、護衛、騎戰
- `專屬術式`：偏引導、封印、場域、附著、儀式、導印
- `專屬儀路`：偏宗教、祖靈、命名、安魂、路權、見證

## 流派的寫法
可按：
- 正統主流派
- 邊地變體派
- 組織內訓派
- 家系秘承派
- 灰線變體

## 實作原則
- 玩家創角時，先選國家 / 文化，再看可進入哪些文化職業。
- 若玩家不滿足完整條件，可先以 `學徒 / 見習 / 候補 / 未受印` 版本進場。
- 專屬能力不一定在開局全拿到，可以綁任務、授印、立功、拜師或成年禮。
