# 玩家職業卡框架

這份文件定義 `玩家職業卡` 應該長什麼樣。

## 一張職業卡至少要有
- 職業名
- 玩法定位
- 最擅長的事
- 最容易卡住的點
- 建議主能力
- 建議副能力
- 常見起始裝備傾向
- 常見起始資源傾向
- 最適合的起始區域或任務型
- 常見升階方向

## 玩家拿職業卡時最先要看
1. 這張卡讓我在隊伍裡最像誰
2. 這張卡在第一場最能解什麼問題
3. 這張卡最怕什麼局面

## 職業卡的三層資訊

### 1. 玩法定位
回答這張職業卡在遊戲裡到底偏：
- 戰鬥
- 調查
- 探索
- 支援
- 製作
- 社交

### 2. 起始形狀
回答玩家第一場通常會拿什麼：
- 大概的起始裝備
- 大概的起始資源
- 第一批最合理的任務型

### 3. 後續發展
回答這張卡之後最常怎麼長：
- 升階方向
- 轉職方向
- 融合方向

## 主職與副職的關係
- `主職` 決定你在桌上的核心功能
- `副職` 決定你多一種解法或多一種社會身分
- 起始時不用把副職做得跟主職一樣重

最實用的起始比重通常是：
- `主職 70%`
- `副職 30%`

## 玩家挑卡的最短方法

### 想先穩定開局
- 選 `引水人 / 城衛兵 / 療傷手 / 商人`

### 想先打與保人
- 選 `城衛兵 / 巡空護航者 / 港巡兵`

### 想先查與探索
- 選 `探索者 / 學者 / 古文讀解手 / 路線踏勘手`

### 想先製作與工具流
- 選 `藥師 / 誓紋技匠 / 黑鍛匠 / 靈廚`

### 想先吃社交與灰線
- 選 `商人 / 文簿官 / 情報掮客`

## 設計原則
- 職業卡是給玩家快速抓玩法，不是完整職業百科
- 一張卡應該讓玩家三分鐘內就知道自己大概怎麼玩
- 文化職稱、國家制度和完整職業史，留給其他檔案補
