# 玩家物品與購物索引

這一層處理玩家端最常碰到的物品問題：
- 買什麼
- 去哪買
- 帶什麼比較合理
- 缺貨時怎麼辦
- 一件裝備怎麼從入門一路升到專精

## 這層能回答什麼
- 一件東西大概算便宜、中價還是高價
- 在不同地區，哪些東西容易買到
- 玩家該先補消耗品、補給，還是先升級主裝
- 黑市、港務、學城、公會、工坊各自會賣什麼
- 起始區的第一筆錢，最適合先花在哪裡
- 哪些服務比直接買新裝備更划算
- 缺貨時應該換替代品、延後、議價，還是轉黑市
