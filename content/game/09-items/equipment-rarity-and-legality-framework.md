# 裝備、稀有度與合法性通用框架

## Purpose
這份文件補強 Elaris 的通用物品系統，專門處理 `裝備` 這一層的世界共通規格。

它的重點是把下列概念拆清楚：
- 什麼算裝備
- 裝備分哪些類型
- 角色能裝在哪些位置
- 稀有度和製作品質有什麼差別
- 裝備在世界中的合法性與流通限制怎麼判定

## 核心原則
- 裝備是物品的一部分，但因為會直接影響戰鬥、探索、身分與職業表現，所以需要獨立規格。
- `稀有度` 不等於 `品質`。稀有代表取得難度與世界存量，品質代表做工、穩定性與實際表現。
- `合法性` 不只看殺傷力，也看來源、用途、技術敏感度與政治風險。
- 同一件裝備可以在不同國家有不同的流通限制，但底層合法性標籤應一致。

## 什麼算裝備
在 Elaris 中，裝備指的是會長時間被角色攜帶、穿戴、掛裝、持用或配置在特定槽位上的物件。

常見判定方式：
- 會佔用角色身上固定位置
- 會改變戰鬥、探索、儀式或專業表現
- 需要維護、修繕或校正
- 常與職業、階級、勢力身份或通行資格綁定

## 裝備大類

### 1. 武器
用於直接輸出、壓制、反制與戰場威懾。

常見子類：
- 單手武器
- 雙手武器
- 長柄武器
- 遠程武器
- 投射發射器
- 特殊戰鬥型廚刃與工刃

### 2. 防具
用於承受衝擊、切割、穿刺、環境危害與靈息干擾。

常見子類：
- 輕甲
- 中甲
- 重甲
- 軟護具
- 抗潮 / 抗寒 / 抗熱專用護具

### 3. 盾具與防護件
偏重格擋、遮護、隊伍掩護與區域防守。

常見子類：
- 圓盾
- 長盾
- 折盾
- 臂盾
- 特殊防護板

### 4. 誓器與術式媒介
用於引息、誓印、導式、封樣、共鳴校正與儀式操作。

常見子類：
- 誓印器
- 導印片
- 術杖 / 術板 / 導晶柄
- 校正環
- 封樣媒器

### 5. 工具型裝備
會被長時間配置在身上，並直接影響採集、製作、醫療或專業職能。

常見子類：
- 採集刀具
- 修繕工具組
- 醫療工具袋
- 廚具與旅灶具
- 鍛造腰具
- 勘探與測氣裝備

### 6. 穿戴飾件
不一定是主戰裝備，但會提供功能、身份或儀式性影響。

常見子類：
- 披帶
- 戒飾
- 項飾
- 臂環
- 記章
- 護符

### 7. 乘騎與魔物配裝
給坐騎、戰獸、誓獸或合作魔物使用的裝備。

常見子類：
- 鞍具
- 韁具
- 護額與頸護
- 負載框
- 騎乘識別件

### 8. 特殊遺裝
屬於古代遺構、王者級素材產物、特殊勢力秘裝或封存裝備。

常見子類：
- 古遺戰裝
- 被封存誓器
- 高敏感導印裝
- 失衡樣本裝具

## 稀有度
延續通用物品分類的五階寫法：

### 1. 常見
- 量產、常見、容易維修
- 一般平民、旅人、低階從業者常見

### 2. 稀有
- 不一定很強，但取得有門檻
- 可能需要專門材料、職人或地方渠道

### 3. 罕見
- 世界存量低
- 通常與高階素材、名匠工法、特殊勢力或特定地區有關

### 4. 特殊
- 具明顯背景、專屬用途、特殊機制或高度敏感性
- 不一定唯一，但已不適合普通流通

### 5. 唯一
- 幾乎不可複製
- 多半和王權、古代遺構、傳奇人物或唯一儀式結果綁定

## 品質等級
品質等級描述的是做工與性能，不等於稀有度。

### 1. 標準
- 可穩定使用的正常版本

### 2. 精製
- 做工更好，耐久與手感更穩

### 3. 名匠
- 出自知名工匠、特定工坊或專門傳承

### 4. 古遺
- 來自古代技術或已失傳工法

### 5. 異質
- 帶有共鳴、誓痕、活材、王者級素材或不穩特性

## 合法性

### 1. 合法
- 可正常持有、販售、修繕與攜帶

### 2. 管制
- 可持有，但通常要登記、限地區或限用途

### 3. 需許可
- 需機構、軍方、公會、教團或地方權力核可
- 未經申請即持有可能會被扣留或追查

### 4. 禁制
- 原則上不可持有、流通或研究
- 只要出現就可能引來查扣、獵捕或封存處置

## 合法性判定常見因素
- 是否具高殺傷力
- 是否涉及禁術、封印或失衡樣本
- 是否能偽造身分、通行或契約
- 是否屬於軍規裝備、情報裝備或宗教專裝
- 是否來自禁區、封存遺構或危險共鳴帶

## 耐久與維護
裝備應延續通用耐久標籤，但更強調維護需求。

常見耐久寫法：
- `易耗`
- `常備`
- `長久`
- `遺構級`

常見維護需求：
- 清理
- 磨利
- 校正
- 重新導印
- 重綁 / 重鉚
- 穩定化處理

## 設計原則總結
- 先分裝備類型與槽位，再看稀有度與合法性。
- 稀有度負責回答難不難取得，品質負責回答做得好不好。
- 合法性一定要能接回世界秩序，而不只是系統限制。
- 後續商店、任務、掉落、角色卡與勢力查驗都應引用這套標準。
