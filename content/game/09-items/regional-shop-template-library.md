# 地區商店模板庫

這份文件提供可直接抽用的地區商店模板。

## 1. 王都都市圈
- 常見店：正規器材行、藥棚、禮裝店、文件代辦
- 常見貨：標準裝備、學城器具、正式藥品、衣裝與紙本
- 隱藏貨：執照代辦、內線推薦、特許配補

## 2. 港邊航務圈
- 常見店：補給舖、泊具店、濕寒藥棚、海路飯鋪
- 常見貨：繩、燈、鉤、鹽貨、防潮披、海上醫包
- 隱藏貨：夜航貨、換牌件、熟客靠泊服務

## 3. 幽語海圈
- 常見店：潮窟補給棚、守路雜鋪、潮骨藥棚
- 常見貨：止暈、守路香條、深窟燈、緊口容器、暖布
- 隱藏貨：地方認可路牌、深窟樣本處理件

## 4. 焰潮洋圈
- 常見店：熱泉補材舖、珊骨工料舖、隔離棚窗口
- 常見貨：防熱、防鹽、珊骨件、隔離用品、熱潮醫包
- 隱藏貨：守域特許件、深洋隔離材、黑線轉手件

## 5. 暮鏡荒境圈
- 常見店：荒境皮毛鋪、獵窗補給棚、寒地旅灶
- 常見貨：厚披、獸脂、風罩、陷具、冰傷藥
- 隱藏貨：部族認可護符、荒境追跡件

## 6. 杜羅山邦與杜魯摩克隱域
- 常見店：礦料鋪、鍛件鋪、工料行、齒輪修配點
- 常見貨：金屬件、工匠工具、耐用容器、修補件
- 隱藏貨：工坊內部級件、未公開工料、特殊鑿具

## 7. 學城與研究圈
- 常見店：器材行、抄錄舖、標本行、封樣窗口
- 常見貨：測量具、記錄具、導印片、封樣瓶、對照盤
- 隱藏貨：內部清單、特定研究件、未公開標準品

## 使用建議
- 開局第一間店不需要很複雜，只要決定 `店型 + 3 件常備 + 1 件隱藏貨` 就夠了。
- 同一地區的商店應該有相似風格，但不一定賣完全一樣的東西。
