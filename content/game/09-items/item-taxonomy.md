# 物品分類通用框架

## Purpose
這份文件定義 Elaris 的世界通用物品分類。物品不只依材質分類，也必須依 `用途`、`使用場景`、`合法性` 與 `來源` 來分類，這樣後續商店、掉落物、裝備、素材與任務物件才不會混亂。

## 核心原則
- 資源是原料層，物品是用途層。
- 同一份資源可以被做成不同物品。
- 同一種物品也可能來自不同地區與不同資源鏈。
- 物品分類應優先服務遊戲互動，而不是只服務倉儲統計。


## 物品命名建議
- 物品名稱優先使用 `通俗名`，讓玩家與居民一看就知道它大概是什麼。
- 若需要更深世界觀感，可以在內文補 `學術名 / 古稱 / 地方別稱`，不必把正式名稱做得太拗口。
- 尤其是常見消耗品、素材、工具、食物與商店貨品，越通俗越有利於遊戲節奏。
- 命名時可優先從 `外觀 / 用途 / 使用方式 / 危險性 / 產地` 下手。

## 一級分類

### 1. 生活用物品
用於日常居住、飲食、清潔、照明、搬運、書寫與民生工作。

常見子類：
- 容器與炊具
- 食材與料理
- 飲品與旅糧
- 衣物與保暖用品
- 燈具與照明媒介
- 繩索、布匹、箱籠、簡易工具
- 文書與書寫用品

### 2. 戰鬥用物品
直接參與戰鬥或提高戰鬥效率。

常見子類：
- 武器
- 防具
- 盾具
- 投擲物
- 戰場消耗品
- 戰鬥型誓器

### 3. 探索用物品
用於旅行、生存、導航、開路、偵察與地下遺構探索。

常見子類：
- 地圖與導引器
- 水袋與補給套件
- 採集工具
- 攀爬與渡水器材
- 封燈、訊號、測深、測氣器具

### 4. 醫療與恢復物品
用於療傷、穩定狀態、解毒、安撫、恢復靈息或專注。

常見子類：
- 傷藥
- 解毒與抗病用品
- 安撫與鎮靜用品
- 靈息補助品
- 專注恢復用品

### 5. 工藝與生產物品
用於鍛造、鍊成、修典、製藥、建造、農耕或養殖。

常見子類：
- 鍛造工具
- 鍊成器皿
- 修典與抄寫器具
- 農耕與水利用品
- 養殖與馴化器材

### 6. 誓約與儀式物品
和宗教、誓約、封印、祭祀、認證與儀式流程直接相關。

常見子類：
- 誓印與誓環媒介
- 祭儀用品
- 封印器具
- 驗證物與認證符
- 契約承載物

### 7. 身分與交易物品
本身就是權利、資格、流通與交易的一部分。

常見子類：
- 貨幣
- 印記
- 通行證
- 配額票據
- 契約副本
- 可交易憑證

### 8. 馴養與生態互動物品
用於和魔物、生態鏈、棲地與採集環境互動。

常見子類：
- 餌料
- 馴養繫具
- 採孢與授粉器材
- 濾水與淨化用品
- 驅逐與安撫用品

### 9. 特殊 / 禁制 / 遺物
不適合進入一般民生流通的高風險或高價值物件。

常見子類：
- 古代遺物
- 被封存道具
- 禁制研究物
- 高階誓器
- 危險樣本容器

## 二級分類標籤
每種物品至少再補以下標籤中的數項：
- 使用場景：生活 / 戰鬥 / 探索 / 儀式 / 工藝
- 體積：微型 / 輕型 / 標準 / 大型 / 車載
- 耐久：一次性 / 易耗 / 常備 / 長久 / 遺構級
- 合法性：合法 / 管制 / 需許可 / 禁制
- 稀有度：常見 / 稀有 / 罕見 / 特殊 / 唯一
- 啟動方式：直接使用 / 引息啟動 / 誓印啟動 / 儀式啟動
- 主要使用者：平民 / 專業職 / 軍職 / 術者 / 誓約者 / 特定勢力

## 物品與資源的關係
- 每個物品都應能往回對到一個或多個資源來源。
- 若是特殊物品，應能往回對到某個環境、生態鏈、工藝流程或特殊地點。
- 後續商店與掉落物設計，建議都至少補 `來源資源` 與 `加工層級`。

## 探索素材流通
- 探索點掉落物不應只停在 `戰利品` 這一層，還應接到 `收購端 / 初步處理 / 加工方向 / 可製成物`。
- 同一份素材在不同地區可以有不同合法性與價格，設計時應先決定 `誰敢收`，再決定 `值多少`。
- 若素材會因時間、潮氣、共鳴、毒性或爆裂性而失值，應把 `初步處理` 視為物品設計的一部分。

## 物品模板最低欄位
- 名稱
- 一級分類
- 二級標籤
- 用途
- 使用條件
- 消耗或耐久
- 來源資源
- 常見收購端
- 初步處理
- 可製成物
- 常見產地或勢力
- 法律與流通限制

## 設計原則總結
- 先分用途，再分材質與來源。
- 生活用、戰鬥用、探索用應該是最穩定的三大外層。
- 同一件物品可以有多重標籤，但應有一個主分類。
- 後續商店、任務、掉落與角色職業都應引用同一套物品分類。
