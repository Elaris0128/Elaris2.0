# 玩家裝備升級路線

這份文件幫玩家判斷：`下一件該換什麼`。

## 一、旅人通用路線
1. 先有能活的披具與鞋
2. 再補一件穩定主手
3. 再補照明與容器
4. 最後才是漂亮但不必要的稀貨

## 二、近戰路線
1. 制式主武
2. 基礎護具
3. 副手或盾
4. 對應環境防護
5. 專精戰裝或特化武器

## 三、海路路線
1. 防潮披
2. 防滑鞋 / 鉤具
3. 夜航燈罩
4. 航務副牌與合格容器
5. 專門海戰或海上生存裝

## 四、遺跡路線
1. 穩定照明
2. 繩索與鉤具
3. 封樣與記錄器
4. 測氣 / 測深 / 探路件
5. 專業遺跡套裝

## 五、寒地路線
1. 保暖披具
2. 防雪視具
3. 暖源與火袋
4. 厚手套與防裂護具
5. 極地專用載具或雪行件

## 六、術式 / 誓器路線
1. 先有穩定媒介
2. 再補基礎保命裝
3. 再補校正與記錄器
4. 最後才追高級導印與特殊件

## 七、AI / 主持建議
- 每完成 `1-2` 次重要任務，應讓玩家有一個明顯可升級點。
- 升級不一定是更強，也可以是更穩、更合法、更方便。
