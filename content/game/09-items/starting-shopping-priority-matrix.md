# 起始購物優先順序矩陣

這份矩陣回答新玩家最常問的問題：
`我第一筆錢到底該先買什麼？`

## 通用優先順序
1. 保命與回復
2. 照明與視野
3. 攜行與容器
4. 職業核心工具
5. 主裝升級
6. 風格化加購

## 依開局風格細分

### 城市調查型
1. 記錄用品
2. 備用衣裝與體面物件
3. 小額打點與通行支出
4. 證件保管容器

### 港邊與海路型
1. 防潮
2. 止暈
3. 夜航照明
4. 繩索與鉤具

### 遺跡探索型
1. 照明
2. 容器與標記物
3. 基礎醫療
4. 固定與攀附工具

### 寒地求生型
1. 保暖
2. 熱源
3. 乾糧與補鹽
4. 雪盲防護與備用布層

### 灰線交涉型
1. 乾淨的證件與包裝
2. 可丟棄替代品
3. 消音與低調裝具
4. 備用現金與人情費

## 哪些東西不急著買
- 漂亮但不實用的副裝
- 你暫時用不到的高級專業器具
- 沒有維護能力的昂貴主武
- 只在極特殊場景才用得到的單次道具
