# 第一場任務挑選指南

這份指南專門回答 AI 一個問題：
`第一場該挑哪種任務，最不容易把團跑死或跑散。`

## 最穩的第一場任務特徵
- 目標單純
- 區域味道強
- 可在一場內看到結果
- 同時能留下第二場鉤子

## 優先順序
1. 護送 / 帶路 / 補給
2. 失蹤 / 查驗 / 小調查
3. 地方衝突 / 人情債
4. 高風險灰線或封鎖事故

## 不建議第一場就上的任務
- 世界級遠征
- 多勢力大型政治局
- 必須高度依賴執照與高聲望的任務
- 一失敗就直接封團的高壓深層事故

## AI 的最短挑法
- 若玩家偏新手：選低風險、短線、區域味道強
- 若玩家偏老手：可在第二問題裡加灰線或轉折
- 若玩家全員偏戰鬥：任務仍要有明確理由，不要只丟怪
