## 這一層回答什麼
- 玩家在告示板上會看到什麼
- 玩家手上的任務到底屬於哪一型
- 任務怎麼分快速危險度與報酬層級
- 失敗後世界通常怎麼回應
- 不同區域最常長出哪種任務
- 玩家選好起始區域後，AI 應該優先接哪批既有任務
- 同一條任務要怎麼升溫、反轉、延長成下一章
- 玩家怎麼用一張表追已接委託、時限與違約風險
- AI 在第一場時應該怎麼挑一張 `最不容易卡住` 的起始任務
