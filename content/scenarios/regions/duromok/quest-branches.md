# 杜魯摩克隱域任務分支導引

## 分支重點
- `黑火熄燈`：可走 `守備止損 / 議會查印 / 工坊追因`。
- `失契主印`：可選 `內部保密 / 公開對質 / 反釣偷印者`。
- `老坑回聲`：可偏 `封坑安全 / 深入回收 / 追查舊案`。
- `裂隙貨單`：可走 `正帳對照 / 灰線追貨 / 留線放長`。
- `索籠墜落`：可選 `先救人 / 先保貨 / 先抓責任人`。
- `晨穗拆件客`：可偏 `合作拆帳 / 釣海路 / 反查來源`。
- `爆坑下層的靜火`：可選 `撤離 / 取樣 / 借機引蛇出洞`。
- `灰鉤封樣錯碼`：可走 `守程序 / 查內鬼 / 借錯碼釣大魚`。
- `齒燈外哨的假警鐘`：可選 `重建信任 / 反設假鐘 / 直接清耳`。
- `黑齒市集的死票據`：可偏 `市契清查 / 黑市側查 / 追背書人`。
- `裂熔工坊的逆鍛件`：可選 `封技 / 偷留 / 正式立案再改規矩`。
- `第十三份契約`：最後會整合你在隱王、議會、暗齒與市契中的立場。
