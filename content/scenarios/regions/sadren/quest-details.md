# 薩德原任務細化包

## 使用方式
- 本檔整理 [questlines.md](questlines.md) 對應任務線的摘要、分支與接軌方向。
- 每條任務線都已拆成獨立任務檔，請同步參照 `../../quests/` 目錄。

## 五柱共議的裂痕
- 類型與起點：國家主線，起點為 `風勒原心`。
- 核心矛盾：水權、古路與柱間權責看似能以共議解決，但每一柱都背著自己要保的人與路。
- 主要牽涉人物：烏嵐赫、蘇芮瑪、風策局代表、夢牧長者。
1. 玩家會先看見五柱互相禮讓，接著才看見每一次禮讓都在消耗信任。
2. 中段會在缺水、錯路訊與旅獸調度間逼玩家判斷誰在讓危機失控。
3. 玩家若站邊太早，會直接失去部分草原支持。
4. 收尾時會自然牽到 `旱季前的第五柱`。
- 主要分支：強化共議、扶某一柱、讓外部危機逼出新秩序。
- 可延伸學習：草原議誓、旅路補給、水權仲裁。
- 後續接軌：可帶動 `第二順位`、`假風訊`、`逐風舊路`。

## 枯井前的第二順位
- 類型與起點：民生線，起點為 `日蹄井村`。
- 核心矛盾：補水順位被動過，表面只是名單微調，實際足以決定誰先活過旱前。
- 主要牽涉人物：井前分水人、旅隊領頭者、被擠到後位的老家戶、敢偷偷改順序的書寫童。
1. 玩家先要查清楚順序是誰改、何時改、改後影響哪幾個聚點。
2. 中段會發現改順位不只是私心，也可能是在為某條軍路或護送線讓水。
3. 若玩家當場公開名單，現場可能立刻失控。
4. 結尾要在救急、公平與長線後果間選擇。
- 主要分支：優先救命、恢復秩序、保一條更大路線。
- 可延伸學習：水囊調配、井權規矩、旅隊協調。
- 後續接軌：可連 `五柱共議的裂痕`。

## 夢牧靜地的無歸者
- 類型與起點：祖靈線，起點為 `夢牧金汀`。
- 核心矛盾：送魂儀式總差最後一步，像有什麼在把死者留在靜地邊緣。
- 主要牽涉人物：送魂引者、夢牧守靜人、失去親人的騎手、能感到異常回聲的孩子。
1. 玩家先要在靜地中學會分辨思念與真正的異常殘響。
2. 中段會追出某名無歸者生前在古路上遺留了未完成的誓。
3. 若玩家粗暴處理，可能會打斷整片靜地的安撫秩序。
4. 結尾可讓死者安行、留下為祖靈，或把未竟之事轉成新的任務。
- 主要分支：安魂、追誓、封靜地。
- 可延伸學習：送魂禮、夢行安撫、祖靈禁路辨識。
- 後續接軌：可連死後世界、輪迴與草原古路線。

## 沙鬃鳴徑的倒號火
- 類型與起點：軍事線，起點為 `沙鬃鳴徑`。
- 核心矛盾：一組信標次序被刻意顛倒，只在真正懂草原報號的人眼裡才看得出異常。
- 主要牽涉人物：驛火守號手、原線騎隊長、風策局暗查員、故意報錯的新兵。
1. 玩家先確認哪個號序錯了，錯了會把哪支隊伍引到哪裡。
2. 中段會帶出薩德原如何用少量信標調動大片騎隊。
3. 若玩家反應慢，真隊伍可能已被導去假集結點。
4. 結尾能改變某條戰備路線與軍方信任。
- 主要分支：迅速糾號、放錯號引蛇、掩下風波。
- 可延伸學習：草原號火、騎隊調度、風向報號。
- 後續接軌：可連 `假風訊`、阿瓦隆軍需線。

## 踏曜幼駒失群
- 類型與起點：地方線，起點為 `踏曜平野聚`。
- 核心矛盾：幼駒失群像是草原常事，卻總在某些固定方向上失去隊列。
- 主要牽涉人物：年輕牧手、老馴手、追蹤好手、對路線太熟的陌生旅者。
1. 玩家從足跡、糞熱與草壓方向開始追失群路線。
2. 中段會發現有人在用幼駒測試一條舊路是否還能安全通行。
3. 村聚內部會爭論要不要追究外來者責任。
4. 收尾可保住幼駒，也可能放出更深的徙路秘密。
- 主要分支：找回幼駒、抓試路人、借此重開古路。
- 可延伸學習：旅獸辨識、馴養基礎、草痕追蹤。
- 後續接軌：可連 `逐風舊路`。

## 雲環草聚的靜病
- 類型與起點：照護線，起點為 `雲環草聚`。
- 核心矛盾：旅獸突然失去方向感的病正在擴散，最可怕的是患者看起來仍然很平靜。
- 主要牽涉人物：野祿契、草藥照護人、旅隊頭、誤把靜病當成詛咒的老人。
1. 玩家先要辨認這是疾病、毒植、還是被動過的旅糧。
2. 中段會把玩家帶到採集線、旅灶線與灰尾換市的藥物流通。
3. 若處理慢，整條旅路會因此失去穩定補給獸。
4. 結尾能建立病防流程，也可能讓恐慌先一步蔓延。
- 主要分支：醫病優先、追來源、封路。
- 可延伸學習：旅獸照護、藥食調理、低擾安撫。
- 後續接軌：可連 `野灶契膳`、瑟堤亞安撫樣本線。

## 風策局的假風訊
- 類型與起點：情報線，起點為 `風勒原心`。
- 核心矛盾：有人用假預警逼聯盟提前挪動補給與騎隊，讓真正的目標可以在空窗中被碰觸。
- 主要牽涉人物：風策局信訊官、邊哨跑騎、被牽連的柱間使者、看得出假風味道的老導路人。
1. 玩家先從幾次過於準時的預警發現異常。
2. 中段會進入風訊節點、傳馬、口耳與旗號對照。
3. 玩家若想反用假風訊，就要承擔自己也成為誤導源的風險。
4. 收尾能拆掉假線，也可反轉做更大的埋伏。
- 主要分支：拆訊、反放、壓訊。
- 可延伸學習：風訊辨別、旗號快判、騎訊調度。
- 後續接軌：可連克羅嵐與阿瓦隆邊線。

## 野灶契膳
- 類型與起點：非國家組織線，起點為 `灰尾換市` 或 `日蹄井村`。
- 核心矛盾：一批旅糧被動過手腳，而問題不只在食材，還在於它們本來要送給哪一支路隊。
- 主要牽涉人物：靈廚公會旅灶手、換市秤商、長路護隊、對食安比對很敏銳的老旅人。
1. 玩家先要分辨食材變質、惡作劇與精準投藥的差別。
2. 中段會帶出靈廚公會與地方旅灶間的磨擦。
3. 若選錯處理方式，可能傷到整體對公會的信任。
4. 結尾能建立合法膳糧檢線，也可把問題壓回灰線交易。
- 主要分支：公會收束、地方自治、灰線反查。
- 可延伸學習：旅糧調製、危材辨識、契膳規格。
- 後續接軌：可連 `灰尾換市黑秤`。

## 逐風舊路
- 類型與起點：探索線，起點為 `逐風路標群`。
- 核心矛盾：被抹去的古路開始重新出現在路標、夢裡和旅獸慣性中。
- 主要牽涉人物：祁原、老路引者、風柱見證人、想搶先開路的商隊代表。
1. 玩家先從舊標殘刻與旅獸反應確定古路真的還在。
2. 中段會碰到風蝕殘跡、錯位界標與古送魂井。
3. 玩家可把舊路開成新商路，也可能把它封成只留少數人知道的徙路。
4. 古路若重開，會影響整個東境北南路權。
- 主要分支：重開、封存、只保秘密通行。
- 可延伸學習：導路、古標識讀法、長程護送。
- 後續接軌：可連克羅嵐 `雷痕界標` 與東境跨國大路線。
