# 薩德原任務分支導引

## 分支重點
- `五柱共議的裂痕`：核心分支在 `水權 / 古路 / 柱間權責`，會決定五柱是否共同認你。
- `被改過的路標石`：可走 `立即修路 / 查錯標者 / 借路標查更大古路`，是北境入口型任務。
- `枯井前的第二順位`：可選 `保公平 / 保活命 / 查誰改順位`，會改變地方信任與補水秩序。
- `夢牧靜地的無歸者`：可走 `祖靈安渡 / 追無歸原因 / 保持祕密`，會打開夢牧與送魂線。
- `灰尾換市黑秤`：可偏 `查黑秤 / 查灰貨 / 借換市釣走貨人`，會外溢到非國家組織與灰線。
- `沙鬃鳴徑的倒號火`：可走 `軍備速修 / 追改號者 / 借假火號反查`，是風策局與邊線交界。
- `踏曜幼駒失群`：可選 `救幼獸 / 保聚落安定 / 查試路人`，會接草原營地與護送線。
- `雲環草聚的靜病`：可走 `先照護 / 查病源 / 保路線不斷`，會影響旅隊與地方醫療。
- `風策局的假風訊`：可偏 `配合內查 / 追誰放風 / 借假訊釣更大網`。
- `野灶契膳`：可選 `保旅人 / 查食材 / 查運糧路`，會串到灰尾換市和靈廚公會線。
- `逐風舊路`：可走 `重開古路 / 保持封存 / 借舊路追先民秘密`。
- `旱季前的第五柱`：會整合水權、古路、祖靈與風策局，決定你能否進入草原高層調停線。

## 主要接軌
- `風勒原心`：五柱、風策局、國家主線。
- `日蹄井村 / 灰尾換市 / 沙鬃鳴徑`：民生、交易、軍事與灰線。
- `夢牧靜地 / 逐風路標群`：祖靈、古路與深層草原傳承。
