# 塔里伏嶺任務分支導引

## 分支重點
- `裂風快信`：可走 `保訊優先 / 保人優先 / 借信釣線`。
- `暗灣失泊`：可選 `救船 / 追貨 / 查泊位內鬼`。
- `共火背契`：可偏 `護共同體 / 抓冒名者 / 借背契摸更大網`。
- `赫石裂響`：可走 `封礦 / 深探 / 借裂響找遺構`。
- `斷路送骨`：可選 `護骨返鄉 / 查死因 / 讓真相先到`。
- `岩蜃迷帶`：可偏 `保路權石 / 先護人 / 追藏石者`。
- `長風訊腳的假回條`：可走 `清系統 / 保名聲 / 反做假訊`。
- `暗灣沉箱`：可選 `合法回收 / 灰線轉手 / 放長線`。
- `共火外環夜賭局`：可偏 `止賭 / 借局查人 / 讓局自己爆`。
- `灰嶺記契的假收容`：可走 `護收容者 / 查買賣網 / 保共同體門檻`。
- `收屍棚外的第三串腳印`：可選 `追腳印 / 保路棚 / 掩住恐慌`。
- `不受封的誓火`：會整合你在共火圈、訊腳、工棚與灰嶺記契裡的站位。
