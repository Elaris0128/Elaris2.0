# 暮牙海區域包

## 區域定位
- 暮牙海是一片由假燈、沉船、黑流、回收權與危材料理共同構成的危海。
- 這裡不只是海盜與打撈者的地盤，也是 [靈廚公會主總部與漂影島下層潮窟群](/C:/Users/rabbi/Desktop/Elaris/content/world/03-special-regions/12-six-oceans/duskfang-sea-spirit-chef-headquarters.md) 的核心海域。
- 玩家在這裡最常碰到的，不是單純戰鬥，而是 `認貨 / 認燈 / 認責 / 認味`。

## 主題焦點
- 偽燈、假旗與黑流斷路
- 沉船打撈、認領權與夜拍
- 靈廚公會的危材料理、配方祕本與試味誓
- 替味假貨、黑宴交易與灰線供應鏈
- 近岸淺棚、沿岸小島與短窗危材採收
