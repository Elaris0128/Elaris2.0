# 靈穹洋玩法補充

- 可把無名小洲當成一次性資源點、夜泊點或假燈陷阱。
- 具名小島適合做航段節點，不一定都要當最終目的地。
- 啟誓嶼最適合接長線外務、公會內鬥與跨洋護送案。
- 鯨背、白汐、靜藻三線最適合做海怪、生態與樣本責任戲。
- 啟誓嶼內圈則最適合接 `程序衝突 / 內部賣訊 / 申訴翻案 / 宿層互保` 這種制度戲。
- 如果想把總部演得更像真正樞紐，就多用 `權限階序 / 中立調停 / 警報閉泊 / 夜警黑格` 這幾條線。
- 回航弧棚、雙燈候洲、藥封轉匙洲、影頁沉步礁這四點，最適合拿來演 `外環等待 / 候泊分流 / 跨公會切責 / 先讀權之爭`。
- 無名島群不必都變成地圖節點，很多時候把它們當 `老水手才知道的功能地點` 反而更有味道。
