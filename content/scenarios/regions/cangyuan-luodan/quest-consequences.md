# 蒼淵據地與洛丹崖營任務後果與接軌

## 核心原則
- 這裡的任務後果最常改變 `路權 / 祖地信任 / 夜路安全 / 外客可見度 / 邊營緊張度`。
- 若玩家長期偏某一核，另一核會更難完全信你。

## 任務後果
- `祖地陌印`：成功會提升祖地與長老議席信任；失敗會讓外來者更難靠近祖地。
- `撤路先亮`：成功穩住軍心與撤路系統；失敗會讓夜哨反應失真。
- `寂索偽繩`：成功恢復靜索橋鎮可信度；失敗會讓護送成本提高。
- `崖營假手勢`：成功可清理邊營滲透；失敗會讓夜弩隊更保守甚至誤傷。
- `霧谷導路`：成功可打開霧谷更多安全線；失敗會讓霧谷對外全面封閉。
- `低火回名`：成功能提升葬送與祖靈線信任；失敗則會讓失名者議題惡化。
- `影火耳的空消息`：成功可進入更深情報線；失敗讓石光對玩家保留。
- `沉霧哨棚雙簿`：成功穩住哨棚補給；失敗會讓西境邊線文書更混亂。
- `崖隱市的斷價草`：成功可穩住崖市交易；失敗會讓特殊資源黑價飆升。
- `無聲墜隊`：成功可恢復邊路信任；失敗會讓危路進一步封閉。
- `祖地缺刻`：成功可打開更深歷史線；失敗會讓某段歷史被正式埋掉。
- `雙核之間的第三條路`：會決定玩家更被哪一核視為「自己人」。
