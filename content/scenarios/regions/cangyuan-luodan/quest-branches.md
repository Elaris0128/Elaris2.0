# 蒼淵據地與洛丹崖營任務分支導引

## 分支重點
- `祖地陌印`：可走 `護祖地 / 追外印 / 借印查舊路`。
- `撤路先亮`：可選 `穩軍心 / 抓試探者 / 反做假訊`。
- `寂索偽繩`：可偏 `保人命 / 保路權 / 保真正目的地`。
- `崖營假手勢`：可走 `查內鬼 / 反模仿 / 夜間反伏`。
- `霧谷導路`：可選 `保護外客 / 保霧谷祕密 / 追路上伏手`。
- `低火回名`：可偏 `先安魂 / 先找名 / 先查死因`。
- `影火耳的空消息`：可走 `追消息 / 追送信人 / 追收信人`。
- `沉霧哨棚雙簿`：可選 `查假簿 / 借雙簿釣人 / 保哨線`。
- `崖隱市的斷價草`：可偏 `保市場 / 保種源 / 查截價人`。
- `無聲墜隊`：可走 `救人 / 收證 / 守住邊線面子`。
- `祖地缺刻`：可選 `公開缺刻 / 先補記 / 追查誰怕它被看見`。
- `雙核之間的第三條路`：會整合你在祖地、軍線、情報與路權之間的站位。
