# 盧米昂起始任務

## 1. 斷掉的一盞副燈
- 地點：灰燈塔聚
- 核心：某座副塔熄燈後，官方記錄卻聲稱它整夜正常
- 重點：夜訊真假、守夜輪值、可能的偽訊滲透

## 2. 寒牙集市的調包燃材
- 地點：寒牙集市
- 核心：冬倉中一批高品質燃材被偷換成劣貨
- 重點：補給優先序、內外勾結、極寒環境下的小作弊會造成多大後果

## 3. 雪橋下的無名貨箱
- 地點：雪橋村
- 核心：冰下卡著一批失蹤旅隊的貨箱，裡面可能有偽牌或禁運物
- 重點：橋線安全、安名程序、誰想讓這件事永遠埋在冰裡

## 4. 關隘夜封後的送亡隊
- 地點：霜道關隘
- 核心：一支合法送亡隊在封關後沒有如期抵達
- 重點：邊關秩序、安魂優先權、真假送亡牌
