# 盧米昂任務細化包

## 使用方式
- 本檔整理 [questlines.md](questlines.md) 對應任務線的摘要、分支與接軌方向。
- 每條任務線都已拆成獨立任務檔，請同步參照 `../../quests/` 目錄。

## 誓燈斷鏈
- 類型與起點：守夜與封關主線，起點為 `冰冠港 / 霜道關隘`。
- 核心矛盾：一段本該穩定的誓燈鏈在封關前夕接連熄滅，顯示有人在用寒國最基礎的秩序當掩護。
- 主要牽涉人物：赫芙琳、莫恩、赫里斯、凜修。
- 主要分支：維持寒國秩序優先。 / 先救被卡在燈鏈外的人。 / 私下保留能再利用的暗線。。
- 可延伸學習：守夜輪值與燈鏈修護、封關牌照與寒國程序禮。
- 後續接軌：可接 `lumiang-double-seal-gate.md`。；會暗示星紋古道的假引路問題。。

## 寒蓮短窗
- 類型與起點：水系採集與寒河風險線，起點為 `雪橋村 / 寒蓮伏流洞`。
- 核心矛盾：寒河蓮窗只開很短，卻有人在採行窗口外還能拿到完整蓮粉。
- 主要牽涉人物：瑪爾薇、玦因、寒盞柚、橋名叔。
- 主要分支：保護村落採行權。 / 交給公會進行正規採集。 / 用秘密樣本換取更深線索。。
- 可延伸學習：寒水採集、藥食處理、送亡禮前素材準備。
- 後續接軌：可接雪華的蓮系採行線。。

## 寂裂豎井的回音

- 核心矛盾：封井後仍有敲擊聲從深井傳出，像有人在黑暗裡繼續挖掘不該挖的東西。
- 主要牽涉人物：伊柏、科瑟克、礦聲弈、凍耳兒。
- 主要分支：封井保密。 / 有限重開並由國家接管。 / 與探索公會合作。。
- 可延伸學習：裂井測錄、寒地繩降、礦聲辨識。
- 後續接軌：可接杜羅礦脈與工件線。。
