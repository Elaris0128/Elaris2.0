# 阿瓦隆任務分支導引

## 分支重點
- `王冠下的四重誓`：核心分支在 `王室 / 教廷 / 侯系 / 暮環司`，決定你後面更容易碰哪一批高層人物。
- `誓港的幽影貨櫃`：可選 `港務正查 / 商路私談 / 灰線釣魚`，會影響東海誓港貨源與盤查強度。
- `禁誓森林的暗號石`：可偏 `軍方速處 / 情報暗查 / 地方封路`，後續會影響霜誓關與山道安全感。
- `失蹤的前鋒`：可選 `救人優先 / 真相優先 / 軍紀優先`，決定前鋒舊事是否被公開。
- `偽幣之流`：可走 `查假券 / 查放券的人 / 反釣偽券網`，後續會接瑟恩票據線。
- `王糧署的空倉`：可選 `公開救濟 / 暗修帳目 / 追查上層動手者`，會改變南線物價與民心。
- `白鹿溪的夜光藥草`：可走 `保村規矩 / 保藥材供應 / 釣外地收購者`，後續會打開地方藥師線。
- `灰盾舊旗`：可選 `替老兵討名 / 先壓風波 / 借旗釣舊案人脈`，會改變老兵與王室態度。
- `教鐘下的空見證`：可走 `教義解法 / 文書解法 / 安魂解法`，會影響教廷與平民對你的信任。
- `暮環港線`：可選 `正式合作 / 有限互用 / 保持距離`，會決定暮環司之後怎麼用你。
- `燈岬沉貨名單`：可偏 `合法回收 / 漁聚私保 / 追查源頭`，後續會接海線探索。
- `誓痕之原的第二面旗`：會整合前面多條線的立場，最後決定你在阿瓦隆被誰正式承認。

## 主要接軌
- `銀鋒城`：政治、宗教、文契、王室書官線。
- `東海誓港`：港務、票據、灰貨、海線探索。
- `霜誓關與暮環關隘`：邊線軍務、情報與暗號。
- `誓痕之原`：國家級高位事件與授旗儀式。
