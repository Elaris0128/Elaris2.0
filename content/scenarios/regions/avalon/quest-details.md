# 阿瓦隆任務細化包

## 使用方式
- 本檔整理 [questlines.md](questlines.md) 對應任務線的摘要、分支與接軌方向。
- 每條任務線都已拆成獨立任務檔，請同步參照 `../../quests/` 目錄。

## 王冠下的四重誓
- 類型與起點：國家主線，起點為 `銀鋒城`。
- 核心矛盾：王權、教權、侯系與暮環司都聲稱自己在守護阿瓦隆，但四條秩序彼此掣肘，讓外敵和內線都能鑽空。
- 主要牽涉人物：凱洛恩・鋒塔、雷維斯、梅列絲、塔維恩，以及至少一名代表侯系利益的封臣。
1. 玩家先在王都看見四方說法彼此矛盾，同一件事會被軍政、教庭、侯府與情報司解釋成不同危機。
2. 玩家在三條中線中選擇先追哪一方：邊防補給、教誓見證、港線密耳。
3. 中段會發現真正的問題不是單一叛徒，而是各權力機構都在保留自己的白手套。
4. 末段會把玩家推向 `誓痕之原的第二面旗` 與東境跨國主線。
- 主要分支：扶正王室、平衡四權、讓暮環司擴權、或讓侯系暫時壓過教庭。
- 可延伸學習：宮廷禮儀、誓禮見證、軍令流程、暮環司暗線應對。
- 後續接軌：能直接牽到 `第二面旗`、`教鐘下的空見證`、`暮環港線`。

## 失蹤的前鋒
- 類型與起點：軍事調查線，起點為 `霜誓關`。
- 核心矛盾：一支巡前鋒隊在名冊上被整隊抹去，像從來沒有存在過。
- 主要牽涉人物：邊關副統領、關口書記官、一名倖存侍從、灰盾舊軍系老兵。
1. 玩家先查到裝備補給曾經被領走，但前鋒隊在軍令系統裡不存在。
2. 玩家需要在山路、舊營地與補給簿之間回推路線。
3. 玩家會發現有人想掩蓋的不是失蹤本身，而是前鋒隊看到的路標異常。
4. 結尾可選擇把真相交回軍方、交給王都、或交給暮環司。
- 主要分支：保全軍中名譽、公開軍紀問題、私藏證據換暮環司信任。
- 可延伸學習：邊關軍制、雪地斥候、前線醫護流程。
- 後續接軌：可連到 `燈岬沉貨名單`、`王冠下的四重誓`。

## 王糧署的空倉
- 類型與起點：民生線，起點為 `誓禾鎮`。
- 核心矛盾：糧帳完整，冬儲卻實際短缺，表示有人在合法流程內把糧食抽空。
- 主要牽涉人物：糧署巡核官、誓禾鎮穀倉長、地方護送騎、飢寒邊村代表。
1. 玩家先從倉口重量、封條與車隊行跡看出糧食被分批換出。
2. 玩家會被迫在保全冬糧、追人與避免地方恐慌之間做取捨。
3. 中段會牽出王都採買價與港口轉賣價之間的異常差額。
4. 末段可能要在鎮上公開帳冊，也可能暗查到更高層。
- 主要分支：先救人、先抓人、先保制度。
- 可延伸學習：糧秤與配發、車隊護送、地方官署交涉。
- 後續接軌：可接 `偽幣之流`、`第二面旗`。

## 白鹿溪的夜光藥草
- 類型與起點：地方採集線，起點為 `白鹿溪村`。
- 核心矛盾：夜光藥草本應只在安定季節發亮，現在卻在禁採時段大片出光。
- 主要牽涉人物：村中採藥師、外地藥材商、守溪見證者、偷偷帶路的年輕獵戶。
1. 玩家先學會安全採收與辨別被動過的藥圃。
2. 玩家會發現外地收購者不只為藥，而是在找溪床下的另一種誓痕礦物。
3. 若處理失當，會引來夜行魔植與溪畔巡鹿。
4. 最後要決定是保護村子的秘密，還是讓公會介入正規採集。
- 主要分支：村內封存、合法收購、灰線外流。
- 可延伸學習：靈藥採集、初階調膳、草木安撫。
- 後續接軌：可接靈廚公會、藥師公會與瑟堤亞樣本線。

## 灰盾舊旗
- 類型與起點：老兵線，起點為 `灰盾屯村`。
- 核心矛盾：一面早該封存的舊軍旗被高價收購，像有人在試探舊軍誓脈是否仍在。
- 主要牽涉人物：灰盾老兵首、旗紋收藏商、曾在邊線服役的教庭見證者。
1. 玩家先從旗布纖維、修補痕和收藏商說詞判定這不是普通古物買賣。
2. 玩家追到的會是老兵間未講完的一場舊敗仗。
3. 中段可能觸發私鬥、舊創傷或軍功記錄重審。
4. 收尾時，玩家能讓舊旗回村、上交軍庫，或拿去換更深情報。
- 主要分支：榮譽結案、制度重審、灰線交易。
- 可延伸學習：老式旗語、軍旅禮節、歷史軍陣辨識。
- 後續接軌：可連 `失蹤的前鋒`、`第二面旗`。

## 教鐘下的空見證
- 類型與起點：宗教線，起點為 `銀鋒城`。
- 核心矛盾：一次公開誓禮裡應留下名字與見證的人，最後在神殿記錄中完全空白。
- 主要牽涉人物：塔環司祭、誓錄修士、當夜值守的鐘樓童、被空見證牽連的家族。
1. 玩家先從誓禮流程、鐘聲節點與見證筆錄找出缺漏。
2. 玩家會發現有人不是偽造紀錄，而是刻意把一場見證變成「沒有被承認過」。
3. 若玩家驚動教庭，會引發對誓禮正統的爭論。
4. 結尾可選擇維持教庭面子，或揭露更深的見證腐蝕。
- 主要分支：護教、正本清源、將證據送往外部法理體系。
- 可延伸學習：誓禮程序、教義辯證、見證紋印識別。
- 後續接軌：可連 `王冠下的四重誓` 與法奧／法雷恩信仰系統。

## 暮環港線
- 類型與起點：諜報線，起點為 `東海誓港`。
- 核心矛盾：暮環司懷疑誓港某條耳目鏈已被外地勢力反向利用。
- 主要牽涉人物：塔維恩、港線耳錄官、夜班碼頭監哨、偽裝成搬運頭目的外線接頭者。
1. 玩家要先學會分辨「表面情報」與「被故意餵給暮環司的情報」。
2. 中段會在小船、碼頭票據與夜搬貨線之間做交叉驗證。
3. 玩家可能得選擇讓暮環司保住臉面，或公開港線有滲漏。
4. 任務完成後會直接改變阿瓦隆與瑟恩、瑟堤亞的港線關係。
- 主要分支：配合暮環司清線、雙面放線、私留通道。
- 可延伸學習：耳錄術、口風測試、港區暗號。
- 後續接軌：可連 `幽影貨櫃`、`偽幣之流`、東境跨國情報線。

## 燈岬沉貨名單
- 類型與起點：海線探索線，起點為 `燈岬漁聚`。
- 核心矛盾：近岸沉船名單與王都貨單對不上，表示有人在事後重寫了失事記錄。
- 主要牽涉人物：燈岬老領航、沉貨打撈手、港務抄錄員、想回收貨箱的祕密委託者。
1. 玩家要先在潮間、舊燈塔與海圖抄本之間拼出失事時序。
2. 打撈過程中會遇上潮蝕魔物與被封存的箱號。
3. 真相可能指向港線黑帳，也可能指向軍需轉運。
4. 結尾可把沉貨交回港務、私賣、或保留作為跨國證據。
- 主要分支：海民優先、港務優先、情報優先。
- 可延伸學習：近岸航圖、打撈安全、鹽潮封樣。
- 後續接軌：可連 `誓港的幽影貨櫃`、`第二面旗`。
