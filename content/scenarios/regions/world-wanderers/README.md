## 類型
- 小型任務：適合單次旅行、臨時插入、快速建立人物印象
- 大型任務：適合跨區、多站點、需要長路或多方勢力介入
- 特殊任務：適合稀有個體、公會評議、黑市重案或高價遺物事件
