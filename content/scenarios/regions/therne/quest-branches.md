# 瑟恩任務分支導引

## 分支重點
- `七曜席次裂縫`：核心分支在 `議席平衡 / 危機整合 / 城邦利益`，會決定聯邦高層怎麼看你。
- `失頁的條契簿`：可走 `法務修補 / 外務追查 / 密錄暗線`，後續會接公證與跨國偽文書。
- `星軌全權令`：可偏 `提前授權 / 拖延觀望 / 條件式啟動`，直接影響聯邦對危機的反應。
- `港盟偽券`：可選 `保票據信任 / 釣商路網 / 借偽券查港務`，會接阿瓦隆與河路線。
- `學院樣本黑格`：可走 `學院內查 / 公會外查 / 封存壓案`，後續會接瑟堤亞樣本線。
- `希恩勒斯安養簿`：可選 `保病患 / 查刪簿 / 追外流`，會改變照護共同體對你的態度。
- `艾綴塔雙版本協議`：可走 `外交修補 / 公開對證 / 暗中換本`，後續會牽動聯邦外務。
- `曦紙里的刪名抄手`：可偏 `保匿名 / 揭黑工 / 釣更高層委託`，是地方制度切口。
- `觀測丘的錯時星圖`：可走 `學術解讀 / 宗教解讀 / 危機封口`，會改變觀測圈對你的信任。
- `珀恩達的糧配爭議`：可選 `民生優先 / 程序優先 / 政治交換優先`，會改變地方穩定度。
- `密錄局的雨夜查封`：可走 `配合內查 / 保外部證據 / 反查被利用者`，會決定密錄局如何看你。
- `協章裂口`：會整合制度、學術、情報三條大線，決定你能否進聯邦核心程序。

## 主要接軌
- `瑟彌亞`：議席、學院、密錄局、程序核心。
- `洛菲達 / 河紋橋聚`：票據、水路、涉外文本。
- `艾綴塔 / 希恩勒斯 / 觀測丘`：法理、安養、觀測與地方學術線。
