# 瑟恩任務細化包

## 使用方式
- 本檔整理 [questlines.md](questlines.md) 對應任務線的摘要、分支與接軌方向。
- 每條任務線都已拆成獨立任務檔，請同步參照 `../../quests/` 目錄。

## 七曜席次裂縫
- 類型與起點：國家主線，起點為 `瑟彌亞`。
- 核心矛盾：七曜席次名義上是聯邦平衡機制，實際上每一席都在利用程序爭取危機時的額外權限。
- 主要牽涉人物：伊瑟菈、娜薇、露黎恩、學院與橋稅派系代表。
1. 玩家會先看見一場合法卻彼此拆台的議決。
2. 接著必須在議席記錄、地方回報與緊急授權命令之間判讀誰在扭曲程序。
3. 當危機真正發生時，玩家手上的證據會決定誰有資格接手統整權。
4. 最終會導向 `協章裂口` 與東境跨國制度主線。
- 主要分支：扶正某一席、強化聯席制衡、讓密錄與危機體系擴權。
- 可延伸學習：聯邦法理、議事程序、危機授權結構。
- 後續接軌：能帶動 `港盟偽券`、`雙版本協議`、`密錄局的雨夜查封`。

## 港盟偽券
- 類型與起點：經濟線，起點為 `河紋橋聚` 或 `瑟彌亞`。
- 核心矛盾：表面上是小額偽券，實際上是在測試聯邦票據信譽能被拉扯到什麼程度。
- 主要牽涉人物：賽莫、河紋橋券驗員、中立商會聯席代表、私印工坊主。
1. 玩家從幾張幾乎完美的港盟通券開始追查。
2. 中段會牽涉到河港保管箱、票據對保與跨國商路信用。
3. 玩家若處理不慎，會直接傷到聯邦商路名聲。
4. 結尾可能變成刑案、外交案，或被當成更大跨國網的前哨。
- 主要分支：保聯邦信用、放長線釣更大票據網、讓某地方自行吸收損失。
- 可延伸學習：票據辨識、商路估值、公證封樣。
- 後續接軌：可連阿瓦隆 `偽幣之流`、杜羅器具箱線。

## 學院樣本黑格
- 類型與起點：學術線，起點為 `奧芙瑞亞`。
- 核心矛盾：列管樣本被悄悄換出正規封存鏈，卻還保有學院內部標記。
- 主要牽涉人物：樣本庫講師、封格管理員、外借名義的研究生、靈息學會外部顧問。
1. 玩家先接觸一個看似合規、其實編號前後不連的樣本箱。
2. 中段要調查誰能進封存格、誰能寫臨時授權、誰在幫外部單位走便利線。
3. 若玩家追得太快，會逼出學院自保與聯邦外調的衝突。
4. 真相可能通往瑟堤亞樣本線，或指向杜羅器具與容器來源。
- 主要分支：護學院名聲、清出內線、轉交聯邦正式調查。
- 可延伸學習：封樣規則、研究倫理、容器安全。
- 後續接軌：可連 `被偷帶出去的安撫樣本`、`星軌全權令`。

## 希恩勒斯安養簿
- 類型與起點：醫療線，起點為 `希恩勒斯`。
- 核心矛盾：一段病歷被整批抽走，像有人要讓某群病患從制度記錄中消失。
- 主要牽涉人物：安養院主理、值夜醫護、文書保管人、失名家屬代表。
1. 玩家先從藥單與護理班表的不匹配發現問題。
2. 中段要判斷這是掩護醫療疏失、保護特殊病患，還是有人藉制度洗掉身分。
3. 若追到夜班路線，會發現有人把人送出了希恩勒斯。
4. 結尾會決定玩家站在安養體系、家屬，還是聯邦檢查方。
- 主要分支：護病人、護制度、揭露真相。
- 可延伸學習：基礎醫療記錄、照護倫理、病歷保全。
- 後續接軌：可連 `刪名抄手`、`失名的邊緣`。

## 艾綴塔雙版本協議
- 類型與起點：外交線，起點為 `艾綴塔`。
- 核心矛盾：同一份協議同時有兩個可被承認的版本，只差在最後兩段限制條款。
- 主要牽涉人物：外約院公證師、涉外使節、翻譯校印師、密錄局旁聽員。
1. 玩家需分辨哪個版本是真，或兩者都是真只是送達順序不同。
2. 中段會牽涉翻譯慣例、使節回執與河港傳件流程。
3. 玩家若站錯邊，可能直接得罪某外地政體。
4. 收尾常會把玩家推向聯邦如何看待「合法欺瞞」。
- 主要分支：護聯邦對外名分、優先求穩、用假版本釣出幕後。
- 可延伸學習：條約語義、涉外禮節、譯本鑑定。
- 後續接軌：可連 `失頁的條契簿`、阿瓦隆文契線。

## 曦紙里的刪名抄手
- 類型與起點：地方線，起點為 `曦紙里`。
- 核心矛盾：有人專門替人刪改姓名紀錄，讓人能合法地換殼進出聯邦流程。
- 主要牽涉人物：街巷抄手、紙材行掌櫃、追名債主、被刪名者的舊同伴。
1. 玩家先要摸清曦紙里的紙、墨、抄手與夜間取件節奏。
2. 玩家逐步會發現這不是一人犯罪，而是某條小規模但高效率的地下文書服務。
3. 某些被刪名者並非惡徒，會讓玩家很難只靠制度下判。
4. 結尾可能牽向醫療、情報或邊境偽證網。
- 主要分支：取締、收編線人、放走特定對象。
- 可延伸學習：筆跡辨識、紙墨來源、街區口風。
- 後續接軌：可連 `安養簿`、`密錄局的雨夜查封`。

## 觀測丘的錯時星圖
- 類型與起點：神秘線，起點為 `星誓觀測丘`。
- 核心矛盾：觀測圖和歷年校正表持續出現危險偏差，像有某種時序被擰動了一點點。
- 主要牽涉人物：觀測院教師、夜班記錄生、序律編院的律象師、外部求兆者。
1. 玩家先在冷靜的學術場域中發現「每個人都在否認眼前數據」。
2. 中段會確認是器具偏差、人為篡改，還是真有異常天象。
3. 若玩家把消息擴散，將引發恐慌或宗教化解讀。
4. 收尾可走學理收束，也可讓這條線留成更大的神秘伏筆。
- 主要分支：封存研究、公開警示、秘密追查異象來源。
- 可延伸學習：星圖閱讀、儀器校準、律象推演。
- 後續接軌：可連 `七曜席次裂縫`、死後世界與夢兆體系。

## 珀恩達的糧配爭議
- 類型與起點：民生線，起點為 `珀恩達`。
- 核心矛盾：糧配制度被地方勢力拿來換取政治支持，表面合法，實際致人於寒季風險。
- 主要牽涉人物：配給官、倉務監核、地方代議者、碼頭運糧隊長。
1. 玩家先看見的是公平表格，卻碰到完全不公平的實際發放。
2. 中段要在民怨、法規與寒季前時限之間取捨。
3. 追得更深會發現票據、配糧與議席支持互為交換。
4. 結尾能改變某地民心與聯邦名聲。
- 主要分支：先救寒戶、先扳倒地方議系、先保制度運作。
- 可延伸學習：配給規則、倉儲核查、群眾安撫。
- 後續接軌：可連 `港盟偽券`、東境糧配與商路爭議。

## 密錄局的雨夜查封
- 類型與起點：情報線，起點為 `瑟彌亞` 或 `河紋橋聚`。
- 核心矛盾：表面查違禁文書，實際是密錄局藉雨夜查封把更深一層的人逼出來。
- 主要牽涉人物：露黎恩、外勤查封官、河港聯絡人、一名自稱只是搬運文書的年輕跑腿。
1. 玩家需先決定是配合密錄局、旁觀，還是暗中搶先拿走關鍵物。
2. 查封現場會同時出現合法搜檢與程序越界。
3. 事件之後，玩家會被不同派系記住是幫手還是障礙。
4. 結尾能直接把玩家送進更深的聯邦情報線。
- 主要分支：助查、反查、雙面操作。
- 可延伸學習：文書查封、情報辨別、危機保密。
- 後續接軌：可連 `協章裂口` 與東境跨國灰線網。
