# 瑟堤亞任務分支導引

## 分支重點
- `受名之網的裂線`：核心分支在 `受名秩序 / 外來承認 / 誓網穩定`，會決定棲國是否讓你更靠近中心。
- `海膽葡萄的安全採收`：可走 `保採收 / 保市場 / 查誰想偷技巧`，會打開霧林採集與照護線。
- `沉默的回聲廊`：可選 `學庭內解 / 導引外解 / 暫時封存`，會改變學庭對你的信任。
- `被偷帶出去的安撫樣本`：可走 `追樣本 / 保對外合作 / 借樣本釣灰線`，會接瑟恩與阿瓦隆。
- `失名的邊緣`：可偏 `接回身份 / 正式切斷 / 放線追偽名者`，是身份與通行核心線。
- `繭心塢的過早甦醒`：可走 `照護優先 / 研究優先 / 封鎖優先`，會改變成長與照護圈態度。
- `露藤棲村的共棲病`：可選 `村級穩定 / 查病源 / 查外來接觸者`。
- `綠苞溪聚的採樣者`：可走 `保生態 / 抓採樣者 / 查誰收樣本`，會接學庭與跨國樣本線。
- `灰霧眠灣的長睡者`：可偏 `照護 / 查夢兆 / 查誰被允許留下`，會連到庇護與安養組織。
- `棲枝前哨的空牌`：可選 `保前哨秩序 / 查偽牌 / 借空牌查身份網`。
- `暗響監察部的靜追`：可走 `配合監察 / 保持距離 / 借監察查別線`，決定監察部如何標記你。
- `遺失的命名線`：會整合受名、身份、監察與外部樣本網，決定你在棲國的正式承認層級。

## 主要接軌
- `渥恩城`：受名、監察、誓網與高層秩序。
- `霧蘚長廊 / 綠苞溪聚`：採集、樣本、生態與外來交易。
- `繭心塢 / 灰霧眠灣 / 棲枝前哨`：照護、安養、身份與邊界問題。
