# 瑟堤亞任務細化包

## 使用方式
- 本檔整理 [questlines.md](questlines.md) 對應任務線的摘要、分支與接軌方向。
- 每條任務線都已拆成獨立任務檔，請同步參照 `../../quests/` 目錄。

## 受名之網的裂線
- 類型與起點：國家主線，起點為 `渥恩城`。
- 核心矛盾：受名秩序不是官僚名冊，而是整個棲地誓網的一部分；一條裂線同時傷到身份、信任與共棲平衡。
- 主要牽涉人物：賽盧柯・翠蘚脈、枝銘、誓網維序者、對外窗口記錄官。
1. 玩家先會把裂線理解成行政錯誤，後來才知道那是整片誓網的疼痛點。
2. 中段要在外界理解與棲國內部秩序之間充當翻譯者。
3. 玩家若站錯邊，會被視為在用外部制度硬套棲裔秩序。
4. 收尾會自然導向 `遺失的命名線`。
- 主要分支：護誓網、護個體、護對外穩定。
- 可延伸學習：受名禮、導印、低擾通行規矩。
- 後續接軌：可帶動 `失名的邊緣`、`棲枝前哨的空牌`。

## 沉默的回聲廊
- 類型與起點：學庭異常線，起點為 `語孢學庭`。
- 核心矛盾：應該被導回的回聲遲遲不散，像有誰在學庭內持續留下一段不願結束的思念。
- 主要牽涉人物：學庭導聲者、誓網記錄員、失去導師的學習者、對回聲過敏的外來研究員。
1. 玩家先得學會不把瑟堤亞的回聲現象誤判成單純鬼魅。
2. 中段會查到某段回聲其實與一名被撤名的個體有關。
3. 若用外界的強制淨化方式，會直接傷到學庭結構。
4. 結尾可導回回聲、保存為教學案例，或追入更深的受名歷史。
- 主要分支：安撫導回、保存研究、強行切斷。
- 可延伸學習：回聲聆聽、導印安撫、記憶封存。
- 後續接軌：可連 `遺失的命名線`、`憶夢者` 的在地詮釋。

## 失名的邊緣
- 類型與起點：身份線，起點為 `受名記錄庭`。
- 核心矛盾：有人仍活著，卻在記錄中已經被撤出，問題不只是程序，而是整個共同體是否還承認其存在。
- 主要牽涉人物：枝銘、撤名審議者、被失名者的照護同伴、想把此事外賣的灰線中介。
1. 玩家先得理解「失名」在棲國不是單純除籍，而是被從共棲秩序中抽離。
2. 中段會追查撤名原因是否正當、是否被人利用。
3. 若玩家只用外部法理論理，容易錯失真正的誓網傷口。
4. 收尾可能恢復其名、暫留邊緣身分，或確認撤名有其必要。
- 主要分支：復名、觀察留置、徹底撤離。
- 可延伸學習：受名文書、安撫照護、邊緣個體接觸規範。
- 後續接軌：可連 `受名之網的裂線` 與死後世界路徑。

## 繭心塢的過早甦醒
- 類型與起點：成長線，起點為 `繭心塢`。
- 核心矛盾：某個不該提早甦醒的個體開始回應外界，整片棲地都因此變得緊張。
- 主要牽涉人物：繭塢照護者、苔翹、早醒個體的伴護者、擔心被外界看見的長老。
1. 玩家要先判定早醒是自然變異、外擾，還是有人刻意刺激。
2. 中段會帶出瑟堤亞如何看待成長、等待與共同體節奏。
3. 若處理不慎，可能造成個體永久失衡。
4. 結尾可讓個體回眠、穩定甦醒，或成為特殊存在。
- 主要分支：保守安養、有限喚醒、冒險培育。
- 可延伸學習：照護、棲地節律、個體成長觀察。
- 後續接軌：可連 `露藤棲村的共棲病`。

## 露藤棲村的共棲病
- 類型與起點：地方線，起點為 `露藤棲村`。
- 核心矛盾：一種低擾卻持續的病在共棲生命間傳遞，像是病本身知道哪些個體最不能被打斷。
- 主要牽涉人物：村中照護者、流動藥師、受名魔物看護、帶入外來草料的旅隊。
1. 玩家要區分共棲病與普通感染、情緒性失衡與樣本污染。
2. 中段會牽涉草料、安撫樣本與照護節律。
3. 玩家若引入過度侵入式療法，將遭到本地強烈反感。
4. 結尾可建立新的照護流程，也可能迫使村子暫時封棲。
- 主要分支：照護優先、查源優先、封村優先。
- 可延伸學習：低擾醫護、棲地病理、共棲觀察。
- 後續接軌：可連 `被偷帶出去的安撫樣本`。

## 綠苞溪聚的採樣者
- 類型與起點：生態線，起點為 `綠苞溪聚`。
- 核心矛盾：有人採得太乾淨，乾淨到像知道每一個節點的恢復時間與安全上限。
- 主要牽涉人物：溪聚採樣者、外來研究商、苔翹、對節點感知敏銳的幼棲裔。
1. 玩家先從採樣區的恢復速度與殘留手法看出對方很懂這裡。
2. 中段會發現對方採的不只是素材，還是在繪製一張棲國資源弱點圖。
3. 玩家可選擇當場阻止、追向下游，或故意放掉少量樣本換大線。
4. 結尾會影響瑟堤亞對外研究合作態度。
- 主要分支：封樣保護、引合作、設局反追。
- 可延伸學習：生態採樣、節點辨識、封樣流程。
- 後續接軌：可連瑟恩研究線與阿瓦隆港線。

## 灰霧眠灣的長睡者
- 類型與起點：安養線，起點為 `灰霧眠灣`。
- 核心矛盾：旅者睡著後遲遲不醒，卻又不像受害，更像被某種過度溫柔的安撫困住。
- 主要牽涉人物：眠灣照護者、外來旅者家屬、枝銘、想把長睡者運走的商旅代表。
1. 玩家先要判斷這是保護性長睡、危險沉眠，還是有人借眠灣做掩護。
2. 中段會碰到夢境、回聲與照護倫理的邊界。
3. 若玩家強行喚醒，可能反而讓人受創。
4. 收尾能讓眠灣名聲被理解，也可能被外界誤認成危險區。
- 主要分支：守眠、喚醒、轉移。
- 可延伸學習：安眠照護、夢兆分辨、低擾轉運。
- 後續接軌：可連死後世界與輪迴宗教詮釋。

## 棲枝前哨的空牌
- 類型與起點：邊境線，起點為 `棲枝前哨`。
- 核心矛盾：通行牌完整回來了，人卻沒有回來，表示有人懂得如何讓瑟堤亞的邊界承認「牌已回，而人未返」。
- 主要牽涉人物：前哨導印官、邊巡隊、失蹤者伴行者、想避免事態外流的長老。
1. 玩家先從牌面導印、進出時間與巡路記錄查起。
2. 中段會追到外緣低擾通行與外界私採樣本的灰線。
3. 如果把空牌當成單純失蹤案，會漏掉邊境承認機制被測試的事實。
4. 結尾能補強邊界流程，也能順藤摸向外部組織。
- 主要分支：護邊界、救人優先、設局反抓。
- 可延伸學習：導印查驗、低擾通行、邊境照護。
- 後續接軌：可連瑟恩與薩德原外緣路線。

## 暗響監察部的靜追
- 類型與起點：情報線，起點為 `渥恩城`。
- 核心矛盾：某條偽導印路徑像是故意留給監察部看的，真正的問題躲在追查行動本身後面。
- 主要牽涉人物：監察部靜追官、枝銘、外部票據中介、被誤當線人的流浪者浮蘚。
1. 玩家需先判定哪些線索是真的漏出，哪些是特意投餵。
2. 中段會遇到監察部內部對是否向外部借力的分歧。
3. 若玩家和監察部合作過深，會被記錄成可接觸外人。
4. 結尾能查出真正的外流路，也可能只留下下一輪追擊的起點。
- 主要分支：協助靜追、獨自追線、放假線誘出更高層。
- 可延伸學習：靜追節奏、導印辨識、樣本保密。
- 後續接軌：可連 `被偷帶出去的安撫樣本`、東境跨國灰線網。
