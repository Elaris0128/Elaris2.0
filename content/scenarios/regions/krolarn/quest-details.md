# 克羅嵐任務細化包

## 使用方式
- 本檔整理 [questlines.md](questlines.md) 對應任務線的摘要、分支與接軌方向。
- 每條任務線都已拆成獨立任務檔，請同步參照 `../../quests/` 目錄。

## 十獸席次重劃
- 類型與起點：國家主線，起點為 `獸魂裂丘`。
- 核心矛盾：十獸席次一旦重劃，就會重新定義誰能代表古誓、獵場和邊界說話。
- 主要涉入獸席：`馭角 / 裂痕 / 逐影 / 冥羽`
- 主要牽涉人物：哈魯格、葛薩、骨語占兆者、伴獸大族的領席。
1. 玩家會先遇到席次爭奪被包裝成榮譽試煉。
2. 接著發現試煉、邊界糾紛與貨路問題被同一群人綁在一起。
3. 中後段將逼玩家站在舊誓秩序或新席次改革一側。
4. 結尾自然導向 `裂丘的假骨兆`。
- 主要分支：守住古席、支持新席、讓外敵壓力逼成妥協。
- 可延伸學習：骨歌禮、氏族議誓、伴獸指揮。
- 後續接軌：可帶動 `骨語錯兆`、`雷痕界標`。

## 河牙誓貨
- 類型與起點：商路線，起點為 `河牙會埠`。
- 核心矛盾：一批標記完整的貨物帶著不屬於任何氏族的見證痕，表示有人在繞過克羅嵐的認領秩序。
- 主要涉入獸席：`逐影 / 鷹目 / 斑影 / 裂痕`
- 主要牽涉人物：河牙誓貨主簿、碼頭契獸看守、外地搬貨頭、環影部隊外哨。
1. 玩家先從貨標、繩結和伴獸反應看出不對。
2. 中段會發現某些貨其實是邊界上被重新洗名的戰利品或禁用品。
3. 若玩家在河埠鬧大，將牽出不同氏族對商路認領的爭議。
4. 結尾會決定河牙未來更偏開放還是封閉。
- 主要分支：依法認領、讓貨回原主、用貨釣線。
- 可延伸學習：誓貨標識、伴獸嗅判、河埠交涉。
- 後續接軌：可連瑟恩水運線與薩德原換市線。

## 骨語錯兆
- 類型與起點：宗教線，起點為 `獸魂裂丘`。
- 核心矛盾：兩位骨語者對同一戰兆說出相反預言，動搖了整個裂丘的先兆威信。
- 主要涉入獸席：`冥羽 / 蟄蛇 / 裂痕`
- 主要牽涉人物：老骨語者、年輕骨語者、戰前指揮首、質疑祭儀的青年獵隊。
1. 玩家要先判斷這是占兆失準、骨具遭動，還是有人故意操縱信仰解讀。
2. 中段會帶出骨庫管理與氏族記憶的不穩。
3. 若玩家公開支持其中一位骨語者，另一派會視其為挑釁。
4. 收尾可讓裂丘恢復秩序，也可讓裂痕保留下來成為後患。
- 主要分支：護傳承、求真相、促成新制。
- 可延伸學習：骨歌、占兆禮、祖靈禁忌。
- 後續接軌：可連 `假骨兆`、獸人神話與祖靈層。

## 霧骨灘的活藤索
- 類型與起點：地方採集線，起點為 `霧骨灘村`。
- 核心矛盾：用來編繩的活藤索連續失效，像是被懂行的人先挑掉了最好的一批。
- 主要涉入獸席：`隱耳 / 蟄蛇 / 冥羽`
- 主要牽涉人物：濕地採繩師、外地收購者、灘邊守望者、懂魔植的人類旅者。
1. 玩家先學會在危險濕地辨別可採與不可採的藤類。
2. 中段會發現外人挑走的並非普通材料，而是能製作高級束具的心藤。
3. 玩家若想反追，必須進濕地深處面對異常魔植與潮沼掠食者。
4. 結尾可為村子留住資源，也可建立更高價的合法交易。
- 主要分支：保本地、引公會、私換更大利益。
- 可延伸學習：濕地採集、束具編製、伴獸護裝。
- 後續接軌：可連誓獸育契公會與杜羅束具需求。

## 藍塹靜影
- 類型與起點：情報線，起點為 `影息藍塹`。
- 核心矛盾：某條巡線只剩回程足跡，說明有人能在不留下進路痕跡的情況下進出克羅嵐邊線。
- 主要涉入獸席：`逐影 / 鷹目 / 隱耳 / 蟄蛇`
- 主要牽涉人物：環影部隊前哨、藍塹獵隊、踏跡師、沉默倖存者。
1. 玩家先從腳印、風向與伴獸不安反應中確定這不是自然風蝕。
2. 中段要在藍塹伏線、偽足跡與靜默路標間找出真正滲透線。
3. 玩家可能會發現內部有人在養這條路。
4. 結尾能讓邊線收緊，也能保留暗門做更深佈局。
- 主要分支：封線、反追、留線養魚。
- 可延伸學習：踏跡、邊巡手語、夜行埋伏。
- 後續接軌：可連 `環影斷訊`、阿瓦隆港線與薩德原草路。

## 巨蹄營的伴獸傷契
- 類型與起點：誓獸線，起點為 `巨蹄踐痕原營鎮`。
- 核心矛盾：伴行巨獸身上的契傷不像野外受創，更像有人在試驗切斷羈約反應。
- 主要涉入獸席：`馭角 / 馭嶺 / 裂牙`
- 主要牽涉人物：伴獸醫、營鎮教頭、誓獸育契外來照護師、疑心最重的騎手。
1. 玩家先要安撫傷獸，學會判讀契傷與普通傷痕。
2. 中段會揭開有人想把伴獸變成無主軍用資源。
3. 若處理失當，會引發營鎮集體對外來照護體系的不信任。
4. 結尾能強化克羅嵐與誓獸育契公會的合作，或把公會趕出此線。
- 主要分支：護獸優先、揪內鬼優先、壓下風波優先。
- 可延伸學習：誓獸安撫、騎獸醫護、契傷辨識。
- 後續接軌：可連 `石蹄失幼` 與薩德原旅獸病線。

## 石蹄失幼
- 類型與起點：地方照護線，起點為 `石蹄窩聚`。
- 核心矛盾：一隻幼獸失蹤後，原本只是家內悲劇，卻逐漸變成氏族間的相互指控。
- 主要涉入獸席：`馭角 / 馭嶺`
- 主要牽涉人物：失幼家族、年輕放牧者、老年守窩人、想藉題發作的氏族對手。
1. 玩家先從足跡、餵食殘痕與窩邊咬痕追查去向。
2. 中段會發現幼獸可能是自己被引走，而不是被偷走。
3. 玩家若處理得太官樣文章，會被地方視為不懂伴獸情感。
4. 收尾能修補氏族關係，也可能留下長期怨結。
- 主要分支：找獸、找兇手、保氏族面子。
- 可延伸學習：幼獸照護、氏族談判、窩聚規矩。
- 後續接軌：可連 `伴獸傷契`。

## 狼歌坡的禁唱夜
- 類型與起點：祖靈線，起點為 `狼歌坡村`。
- 核心矛盾：被禁止的骨歌又在夜裡響起，而唱的人明顯熟知禁段。
- 主要涉入獸席：`逐影 / 斑影 / 冥羽`
- 主要牽涉人物：守歌長者、夜哨、秘密學歌者、對禁唱有私人執念的老獵手。
1. 玩家先需判定唱歌者是在求祖靈、報復，還是在傳遞訊號。
2. 中段會牽出某段骨歌被禁並非因邪惡，而是因其曾導致氏族分裂。
3. 玩家可選擇保留禁令、有限復唱，或公開真相。
4. 結尾常伴隨地方聖地、祖靈或骨庫的劇烈回應。
- 主要分支：維持禁唱、破禁求真、作為誘餌抓人。
- 可延伸學習：祖靈禮、骨歌誦法、夜間祭儀。
- 後續接軌：可連 `骨語錯兆`。

## 雷痕界標
- 類型與起點：邊境線，起點為 `雷痕界丘`。
- 核心矛盾：界標被挪了一點點，剛好足以改變一片獵場與誓路的歸屬。
- 主要涉入獸席：`裂痕 / 鷹目 / 逐影`
- 主要牽涉人物：界標看守、邊界斥候、鄰境使者、擅長地勢記憶的老獵人。
1. 玩家先要證明界標真被挪過，而不是地形改變。
2. 中段會牽出誰能證明「這裡原本在哪裡」這個困難問題。
3. 某些人想要的不是土地，而是重新定義誰擁有穿越權。
4. 最終可能升為外交、軍事或祖誓糾紛。
- 主要分支：恢復原位、重談界誓、拿界標做交換。
- 可延伸學習：地勢判讀、界標禮制、邊巡談判。
- 後續接軌：可連薩德原 `被改過的路標石` 與東境跨國邊界主線。
