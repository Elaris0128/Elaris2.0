# 克羅嵐任務分支導引

## 分支重點
- `十獸席次重劃`：核心分支在 `氏族名分 / 戰力平衡 / 誓界重劃`，會決定聯盟高層是否把你當外人。
- `試峰下消失的獵旗`：可走 `守試煉 / 查偷旗 / 釣更深試營共謀`，會接獵手與試峰線。
- `河牙誓貨`：可選 `保商路 / 保誓貨 / 查見證痕來源`，後續會接瑟恩水路與灰貨線。
- `骨語錯兆`：可走 `信骨語 / 查骨語 / 借錯兆查會盟操手`，會改變祖靈圈對你的態度。
- `霧骨灘的活藤索`：可偏 `保村採集 / 查外地買家 / 導向濕原異變`，是地方生態切口。
- `藍塹靜影`：可走 `巡線搜人 / 追假足跡 / 先保邊線`，會改變環影部隊對你的信任。
- `巨蹄營的伴獸傷契`：可選 `保戰獸 / 查契傷 / 釣營鎮內鬼`，後續會打開育契與軍線。
- `石蹄失幼`：可走 `救幼獸 / 保氏族關係 / 追市場黑流`，會外溢到地方與育契支線。
- `狼歌坡的禁唱夜`：可偏 `祖靈安撫 / 公開破禁 / 追唱歌的人`，是禁忌與傳承線。
- `雷痕界標`：可選 `立即修界 / 追調界的人 / 借偏移查邊界手腳`，會接跨國邊線。
- `環影斷訊`：可走 `救影耳 / 查暗碼 / 借斷訊釣大網`，後續會與跨國情報線相連。
- `裂丘的假骨兆`：會整合氏族、祖靈、影線三層，決定你能否站上聯盟真正的會盟圈。

## 主要接軌
- `獸魂裂丘`：高位會盟、骨語、氏族名分。
- `河牙會埠`：誓貨、水路、環影前線。
- `霧骨灘 / 巨蹄營 / 試峰營地`：地方、生態、軍陣與試煉線。
