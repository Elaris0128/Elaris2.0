# 克羅嵐玩法深化包

## 用法
- 本檔整合克羅嵐的 `人物掛點 / 導師入口 / 地方宗教與組織 / 季節與環境玩法`。
- 很適合拿來把氏族政治、祖靈、伴獸與濕原邊界故事真正做出差異。

## 河牙會埠
- 主要獸席：`逐影 / 鷹目 / 裂痕 / 斑影`
- 常駐人物：哈魯格、河牙誓貨主簿、環影部隊外哨、中立商會河埠接頭者與伴獸看守。
- 導師與資格：可學誓貨標識、伴獸護貨、河埠交涉與邊河查驗；要先證明自己懂得尊重氏族認領秩序。
- 地方宗教與組織：誓貨見證場、冒險者公會河運委託點、環影部隊外埠耳目、中立商會聯席水館。
- 季節與環境玩法：春潮與夏霧時最容易出現錯貨、暗筏與濕地掠食者壓力。
- 任務串接：`河牙誓貨`、`環影斷訊`、跨境水路與薩德原換市線。

## 獸魂裂丘
- 主要獸席：`十獸共席`，但以 `馭角 / 裂痕 / 冥羽` 最常主導節奏
- 常駐人物：哈魯格、葛薩、骨語占兆者、各大族領席與戰前見證者。
- 導師與資格：能學骨歌、氏族議誓、祖靈禁忌與伴獸戰陣，但必須先被某席承認可入場。
- 地方宗教與組織：十獸環誓會議、骨語祭地、祖靈裂丘、戰前誓場。
- 季節與環境玩法：大型會盟季最像火藥桶；平時則更適合做暗流、試探與未爆裂痕。
- 任務串接：`十獸席次重劃`、`骨語錯兆`、`裂丘的假骨兆`。

## 斷風試峰營地
- 主要獸席：`馭角 / 裂牙 / 馭嶺`
- 常駐人物：試煉教頭、年輕氏族挑戰者、伴獸調教師與榮譽見證人。
- 導師與資格：能學狩戰、短突、榮譽試煉規矩與伴獸協同；入場先看膽量，其次看可信度。
- 地方宗教與組織：試煉營地、冒險者公會高風險試峰委託點、少量誓獸育契臨時站。
- 季節與環境玩法：強風季最適合試煉也最危險；雨後則偏向山坡滑落與視線失真。
- 任務串接：`試峰下消失的獵旗`、青年競逐與名譽線。

## 雷痕界丘
- 主要獸席：`裂痕 / 鷹目 / 逐影`
- 常駐人物：界標看守、老獵人、邊巡者與懂風暴節律的年長戰士。
- 導師與資格：可學地勢判讀、界標禮制、邊巡辨誓與風暴前兆。
- 地方宗教與組織：界誓石、邊巡火點、氏族界標守望圈、環影部隊邊界耳目。
- 季節與環境玩法：暴雷季是最大賣點，地表雷痕與泥地變形都會讓玩家很難確認真界線。
- 任務串接：`雷痕界標`、邊界外交與古誓爭議。

## 影息藍塹
- 主要獸席：`逐影 / 鷹目 / 隱耳 / 蟄蛇`
- 常駐人物：環影前哨、踏跡師、沉默倖存者與假足跡製造者。
- 導師與資格：能學夜巡、靜默通行、濕地掩行與偽跡辨識；通常只開給被試過的人。
- 地方宗教與組織：環影部隊隱哨、邊地補給縫、少量探索公會測地點。
- 季節與環境玩法：藍塹在濃霧與夜雨時最有壓迫感，敵人和盟友都可能只留下半條線索。
- 任務串接：`藍塹靜影`、`環影斷訊`。

## 霧骨灘村
- 主要獸席：`隱耳 / 蟄蛇 / 冥羽`
- 常駐人物：濕地採繩師、藥婦、灘邊守望人與懂魔植的人類旅者。
- 導師與資格：可學濕地採集、束具編製、潮汙安全與灘地逃生。
- 地方宗教與組織：灘邊祖靈小壇、藥婦圈、少量藥師公會合作採材點。
- 季節與環境玩法：霧重時適合做失足、錯藤與濕原怪異，旱時則偏向資源爭奪。
- 任務串接：`霧骨灘的活藤索`、地方生計與公會收購線。

## 巨蹄踐痕原營鎮
- 主要獸席：`馭角 / 裂牙 / 馭嶺`
- 常駐人物：伴獸醫、營鎮教頭、長程護隊與誓獸育契外來照護師。
- 導師與資格：能學伴獸醫護、隊伍補給、營地秩序與騎乘隊列協調。
- 地方宗教與組織：氏族營誓圈、誓獸育契公會臨站、冒險者公會長程護送據點。
- 季節與環境玩法：遷徙旺季最能展現營鎮壓力；病季與傷契事件時則更偏照護與信任危機。
- 任務串接：`巨蹄營的伴獸傷契`。

## 石蹄窩聚
- 主要獸席：`馭角 / 馭嶺`
- 常駐人物：守窩人、幼獸照護者、家族長輩與對外人戒心很重的青年。
- 導師與資格：可學幼獸照護、窩聚規矩與小族談判。
- 地方宗教與組織：地方祖靈窩壇、誓獸育契公會偶發巡護點。
- 季節與環境玩法：春季最適合幼獸故事，寒季則更像食物與安全的壓力測試。
- 任務串接：`石蹄失幼`。

## 狼歌坡村
- 主要獸席：`逐影 / 斑影 / 冥羽`
- 常駐人物：守歌長者、夜哨、學歌者與對禁唱執念很深的舊戰士。
- 導師與資格：能學骨歌節奏、祖靈禮、夜間守坡與民間故事記憶。
- 地方宗教與組織：骨歌坡祭地、祖靈坡壇、少量祖燼安魂盟來訪記錄人。
- 季節與環境玩法：月明夜是這裡最強的演出時機；風雨夜則更偏向禁歌與誤聽。
- 任務串接：`狼歌坡的禁唱夜`、祖靈與地方傳承支線。
