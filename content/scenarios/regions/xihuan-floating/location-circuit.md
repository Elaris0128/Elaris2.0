# 曦環浮境地方迴路

- 環心主線：幽璨懸谷 -> 鳴羽台 -> 晨光垂岬
- 外務主線：暮渦曦岑 -> 螺環泊庭（外環附屬浮台群） -> 幽璨懸谷
- 資源主線：凌宵潤澤 -> 環淵碎脈 -> 浮鷹棲巢
- 外環巡補線：暮渦曦岑 -> 迎風簿棲 -> 歇翼台層 -> 螺環泊庭
- 守脈照護線：凌宵潤澤 -> 晨光垂岬 -> 曬羽垂場
- 外客靜容線：暮渦曦岑 -> 螺環泊庭 -> 灰泊靜棚
- 補律復位線：鳴羽台 -> 暖羽歇欄 -> 低浮潤塘
- 稚翼養成線：晨光垂岬 -> 繫風稚坪 -> 鳴羽台
- 白欄緩衝線：鳴羽台 -> 白欄退翼庭 -> 凌宵潤澤
- 來源判讀線：暮渦曦岑 -> 螺環泊庭 -> 客源風盤台
- 成翼爭議線：晨光垂岬 -> 爭巢試序坪 -> 鳴羽台
