# 星紋古道任務細化包

## 使用方式
- 本檔整理 [questlines.md](questlines.md) 對應任務線的摘要、分支與接軌方向。
- 每條任務線都已拆成獨立任務檔，請同步參照 `../../quests/` 目錄。

## 夜路前的最後一次校碑
- 類型與起點：修碑與路權主線，起點為 `星碑宿地`。
- 核心矛盾：一塊決定整條夜路能不能開放的主碑在大雪前出現錯句，所有人都在等最後一次校碑。
- 主要牽涉人物：星衡、祈碑、路裁、夜柝。
- 主要分支：保守封路。 / 修好後有限開放。 / 讓特定勢力優先使用。。
- 可延伸學習：校碑術、夜路開段規則、路權談判。
- 後續接軌：可接北境商路與東境跨境運輸線。。

## 白口失路洞裡還有活人嗎
- 類型與起點：回尋與抉擇線，起點為 `白口失路洞`。
- 核心矛盾：一支失蹤已久的旅隊突然傳出求援聲，卻只有在最不該進洞的時辰才聽得到。
- 主要牽涉人物：白舟、朔圖、探段師深禾、失碑僧。
- 主要分支：救人優先。 / 封洞優先。 / 記錄優先。。
- 可延伸學習：迷洞辨位、回尋程序、長路生存。
- 後續接軌：可接暮鏡迷宮與北境探索主線。。

## 集市上流出的假祈安牌
- 類型與起點：騙徒與護符線，起點為 `徙路集市`。
- 核心矛盾：假的祈安牌在市集流通，讓夜路出事的人越來越多，卻也顯示大家對真正儀式的焦慮。
- 主要牽涉人物：祈碑、券旅司南、假引路聶狐、短禱修師蘄言。
- 主要分支：全面取締。 / 建立正規售牌。 / 放線追主使。。
- 可延伸學習：祈安短儀、市集辨偽、旅人心理安定。
- 後續接軌：可接偽宗教與騙教組織線。。

## 碑旋迷宮缺了一片記錄
- 類型與起點：迷宮與拓印線，起點為 `碑旋迷宮`。
- 核心矛盾：碑旋迷宮的導航紀錄缺了關鍵一片，讓新舊版本的出口完全對不上。
- 主要牽涉人物：星笠、朔圖、拓影女、夜站灰耳。
- 主要分支：公共記錄優先。 / 保密。 / 交易。。
- 可延伸學習：拓印判讀、迷宮記錄整理、探索協作。
- 後續接軌：可接探索公會與收藏黑市線。。

## 誰在夜裡改了木牌
- 類型與起點：危段管理與灰線線，起點為 `夜行守望站`。
- 核心矛盾：高危雪段的木牌一夜之間被改掉，讓原本安全的回路變成逼死人的抄近道。
- 主要牽涉人物：夜柝、路裁、假牌追查人槭墨、路毛童。
- 主要分支：恢復正式路冊。 / 臨時開放替代路。 / 放任灰線自解。。
- 可延伸學習：危段判級、路牌管理、應急改道。
- 後續接軌：可接盧米昂封關與暮鏡邊荒危線。。

## 古道王駝帶錯了人
- 類型與起點：王者級伴路獸與失路線，起點為 `徙路集市 / 北脊雪道`。
- 核心矛盾：向來只帶對人的王駝這次卻帶著整隊旅人走向禁段，顯示古道秩序本身正在偏移。
- 主要牽涉人物：驛蘭、織鞍、白舟、無宿歌者。
- 主要分支：相信王駝。 / 強行拉回。 / 分線探索。。
- 可延伸學習：伴路獸理解、隊伍分線、雪道求生。
- 後續接軌：可接北境王者級魔物與旅隊信任線。。

## 星碑宿地的斷句
- 類型與起點：地方傳承線，起點為 `星碑宿地`。
- 核心矛盾：一段本來每天都會被誦出的宿地斷句突然被所有人念錯，像有人把它從大家的嘴裡偷走了。
- 主要牽涉人物：星衡、默碑、拓階叔、岑舟。
- 主要分支：恢復原句。 / 保留錯句。 / 只讓守路人知道真句。。
- 可延伸學習：口傳修補、宿地禮制、封印句識別。
- 後續接軌：可接神話、祖靈與迷宮線。。

## 夜行守望站少了一盞回路燈
- 類型與起點：夜路運作線，起點為 `夜行守望站`。
- 核心矛盾：一盞負責把夜路接回宿地的回路燈被整座拆走，像有人想讓某支隊伍永遠走不到盡頭。
- 主要牽涉人物：夜柝、北脊汛鷹手、驛蘭、夜站灰耳。
- 主要分支：制度優先。 / 救命優先。 / 秘密雙線運作。。
- 可延伸學習：回路燈管理、守望接力、夜路短儀。
- 後續接軌：可接盧米昂燈鏈與北境跨路網。。

## 霜河渡票與假引路人

- 核心矛盾：一群假引路人靠偽渡票與模仿古道口令把旅人導向收費死路。

- 可延伸學習：票據查驗、人流分流、路口審查。
- 後續接軌：可接盧米昂裂冰稅票與東境偽券線。。

## 拓印人不敢說出的第十三碑
- 類型與起點：隱碑與高階探索線，起點為 `碑影帳場`。
- 核心矛盾：有傳聞說古道其實有第十三碑，但所有見過的人都不願意把它畫下來。
- 主要牽涉人物：星笠、朔圖、岑舟、拓影女。
- 主要分支：封存。 / 公開。 / 私留。。
- 可延伸學習：高階碑文解讀、禁碑處理、知識保密。
- 後續接軌：可接宗教、神話與大陸古路史。。

## 星紋古道上的流亡修行者
- 類型與起點：修行與收容線，起點為 `星碑宿地 / 北脊雪道`。
- 核心矛盾：一名帶著流亡誓紋的修行者一路替人補路，卻也一路招來追捕者與舊宗教糾紛。
- 主要牽涉人物：岑舟、蘄言、宿誓見證妤岚、無宿歌者。
- 主要分支：庇護。 / 中立協調。 / 換取秘密古路。。
- 可延伸學習：流亡生存、宿地協商、夜路修行。
- 後續接軌：可接人類宗教與北境修行者線。。

## 凍河中的蓮燈貨
- 類型與起點：水路貨流與魔植線，起點為 `霜河渡口 / 徙路集市`。
- 核心矛盾：一批應該送往盧米昂與雪華的蓮燈貨在凍河中發出不正常的亮度，像貨物自己在找路。
- 主要牽涉人物：驛蘭、回桑、凍河蓮燈販、票渡衡。
- 主要分支：截貨封樣。 / 安全改裝後放行。 / 利用它追出買家。。
- 可延伸學習：水路貨流辨識、魔植封樣、凍河運輸。
- 後續接軌：可接盧米昂與雪華的蓮系資源線。。
