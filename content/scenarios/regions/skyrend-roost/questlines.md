# 雲裂哨站任務線總表

## 任務線
1. 斷風標：治安 / 空路，起於 雲裂哨站。關鍵風標被人刻意破壞，導致空路亂流加劇。 詳見 ../../quests/skyrend-broken-wind-beacon.md。
2. 禁飛闖入者：軍事 / 調查，起於 雲裂哨站。有人闖入禁飛區後消失，卻留下內部人才知道的訊號。 詳見 ../../quests/skyrend-no-fly-intruder.md。
3. 墜羽回聲：神秘 / 安魂，起於 碎羽外環。一段來自墜者的回聲在外環反覆出現。 詳見 ../../quests/skyrend-fallen-feather-echo.md。
4. 鞍市掉包案：工藝 / 犯罪，起於 飛鱗鞍市。高價飛具構件在運途中被精準掉包。 詳見 ../../quests/skyrend-saddle-swap.md。
5. 碎羽夜望：巡守 / 生存，起於 碎羽外環。外環夜巡隊在同一夜裡失去三個觀測點。 詳見 ../../quests/skyrend-shattered-feather-nightwatch.md。
6. 斷繫封匣：回收 / 秘密，起於 碎羽外環。一只封匣從高處墜落，內容與來源都不對勁。 詳見 ../../quests/skyrend-broken-tether-sealcase.md。
7. 回羽偽印：身分 / 政治，起於 雲裂哨站。返站翼人的名錄印記疑似被偽造。 詳見 ../../quests/skyrend-return-feather-false-mark.md。
8. 傾斜的懸台：工程 / 災害，起於 雲裂哨站。一座重要懸台開始傾斜，像是結構被人動過。 詳見 ../../quests/skyrend-floating-platform-tilt.md。
9. 延遲的空名：宗教 / 社會，起於 雲裂哨站。一名墜者的名字遲遲不能入冊。 詳見 ../../quests/skyrend-delayed-skyname.md。
10. 盲雲道：探索 / 路線，起於 飛鱗鞍市。一條被封閉的古空路忽然出現可通行跡象。 詳見 ../../quests/skyrend-blind-cloud-lane.md。
11. 鞍市龍影：神獸 / 異象，起於 飛鱗鞍市。一種不該在中央出現的龍裔影子被多名工匠目擊。 詳見 ../../quests/skyrend-drake-shadow-saddle.md。
12. 裂外第二巢：探索 / 跨區，起於 雲裂哨站。外環外側可能存在一處失落已久的古翼據點。 詳見 ../../quests/skyrend-second-roost-beyond-rift.md。

## 跨區提示
- 中央空路能自然接上北境、外海群島、曦環浮境與翼人使命線。
