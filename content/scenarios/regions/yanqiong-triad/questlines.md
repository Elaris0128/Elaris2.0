# 巖穹三臺盟任務線總表

## 任務線
1. 碎碑之夜：政治 / 信仰，起於 梵穹台。主碑夜間被人改刻，三臺盟高層各說各話。 詳見 ../../quests/yanqiong-broken-stele-night.md。
2. 失聯的橋望：調查 / 救援，起於 獨風橋廊。一名橋望守夜人與他的記錄同時消失。 詳見 ../../quests/yanqiong-missing-bridgewatch.md。
3. 風口的假柱：軍事 / 治安，起於 獨風橋廊。有人在危段豎起偽風柱，引導重車走向錯路。 詳見 ../../quests/yanqiong-windpass-false-column.md。
4. 鬼影糧車：民生 / 謎案，起於 石霧憩。夜裡有人看見早該墜谷的糧車再度通過宿站。 詳見 ../../quests/yanqiong-ghost-supply-train.md。
5. 被偷換的橋件：工程 / 犯罪，起於 獨風橋廊。主索構件被換成劣質品，像是有人在等一次大事故。 詳見 ../../quests/yanqiong-swapped-bridge-parts.md。
6. 逆風兆：神秘 / 觀測，起於 梵穹台。風向與碑文預示產生罕見對衝。 詳見 ../../quests/yanqiong-counterwind-omen.md。
7. 借名石：社會 / 身分，起於 梵穹台。一塊本屬亡者的名石出現在活人的旅契裡。 詳見 ../../quests/yanqiong-loaned-name-stone.md。
8. 三臺誤刻：學術 / 政治，起於 梵穹台。關鍵碑拓出現互相矛盾的版本。 詳見 ../../quests/yanqiong-three-terraces-misprint.md。
9. 裂脊暗脈：生態 / 採集，起於 石霧憩。宿站附近出現侵蝕構件的暗脈物質。 詳見 ../../quests/yanqiong-broken-ridge-darkvein.md。
10. 空糧倉：民生 / 經濟，起於 石霧憩。帳面充足的糧站打開後竟是空的。 詳見 ../../quests/yanqiong-stonemist-empty-granary.md。
11. 遠脊來信：外交 / 跑腿，起於 獨風橋廊。一封急信必須避開官方路線送達。 詳見 ../../quests/yanqiong-far-ridge-letter.md。
12. 天梁下的第四條路：探索 / 跨區，起於 梵穹台。傳說失落的第四條高空古道再度露出入口。 詳見 ../../quests/yanqiong-fourth-road-under-skybeam.md。

## 跨區提示
- 中央的道路、橋路與古道秘密能自然接上西境、北境與翼人高空路線。
