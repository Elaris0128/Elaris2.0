# 繭影隱林新手總覽

## 這裡適合什麼開局
- 適合從 `白身 / 見習 / 跟行者 / 跑腿 / 渡點學徒` 類型切入。
- 最自然的第一步通常不是打鬥，而是 `認路、認人、守規矩、處理地方小麻煩`。
- 若玩家來自柯沙里人活動圈，開局會更偏 `霧林溪谷、谷寨水行、歌路記憶、節市與換徑`。

## 最常見的起手壓力
- 有路卻不能走，因為今天不再被允准。
- 有名字卻不能直接喊，因為場合與誓序不對。
- 有貨或藥要送，但送法會決定你是幫了地方還是壞了地方。

## 最推薦的入門任務味道
- `失伴與帶回`
- `外路失準`
- `名字與記憶`
- `換徑與節市糾紛`
- `小規模藥材與危粉問題`
