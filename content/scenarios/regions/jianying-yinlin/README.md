## 讀法
- 繭影隱林的任務很少只是單純跑腿，通常都和 `名字、路權、允准、歌路記憶、節市秩序、共夢與守誓` 有關。
- 玩家在這裡最常學到的，不是怎麼衝進去，而是怎麼被地方秩序允許走下去。
- 若角色與柯沙里人、繭影族或喚隱裔相關，特別適合從 `花霧換徑`、`粉語渡徑`、`靜潭誓穴` 這幾個入口開始。
