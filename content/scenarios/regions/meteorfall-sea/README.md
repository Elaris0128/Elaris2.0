# 殞星海區域包

## 區域定位
- 殞星海是一片把 `隕晶、沉船、古物、海怪異化與錯路機運` 全疊在一起的危海。
- 這裡的重點不是單純打怪，而是 `算潮窗 / 判真假 / 決定值不值得冒險`。

## 主題焦點
- 碎晶、古物與沉物認領
- 錯燈、假圖與潮窗誤算
- 海怪因隕晶與逆流而異化
- 黯鉚灰線與近岸白線公證的對撞
- 近岸斷崖淺海與小島短泊規矩
- 海盜、黑市買手、商船與艦隊同時搶窗口
