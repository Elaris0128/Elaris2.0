# 杜羅任務分支導引

## 分支重點
- `五環裂鍛`：核心分支在 `環權平衡 / 技術外流 / 工坊合法性`，會決定五環怎麼看你。
- `未刻名的鐵箱`：可走 `查貨路 / 查押運 / 借箱釣更大流向`，會接白炬遠堡與赤鐵隘城。
- `石契裂紋`：可選 `法契修補 / 查誰動手 / 借裂紋查舊案`，後續會打開碑庭與法典線。
- `白炬錯號`：可走 `保邊線 / 追錯號人 / 利用假火號反查`，是軍事與密廷的交界。
- `深廊歸還錄`：可偏 `保回收隊 / 查多出的東西 / 封鎖深廊`，會引向古爐與探索線。
- `灰鍊封坑名單`：可走 `保礦營秩序 / 查死人名 / 借名單釣非法回收者`。
- `砧尾家業`：可選 `護家業 / 抓濫用者 / 挖更高層訂單`，是地方與工坊線交會點。
- `奔嶺谷的車轍`：可走 `查車隊 / 查路線 / 查誰讓它繞過查驗`，會接物流灰線。
- `工坊資格影章`：可偏 `廢章追查 / 壓住產業風波 / 順藤摸瓜查假工坊`。
- `幽廊冷爐`：可走 `技術回火 / 宗教式封爐 / 探索深處`，是神秘工藝線。
- `密廷假線`：可選 `配合密廷 / 借密廷查別人 / 保持獨立`，決定情報線對你的定義。
- `石契法典的斷頁`：會整合五環、碑庭、回收線，決定你能否觸碰杜羅核心法契。

## 主要接軌
- `靈爐懸心`：工坊、資格、五環主線。
- `石契碑庭`：法契、見證、斷頁問題。
- `白炬遠堡 / 赤鐵隘城 / 深廊宿聚`：軍事、物流、探索與密廷線。
