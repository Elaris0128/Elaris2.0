# 杜羅任務細化包

## 使用方式
- 本檔整理 [questlines.md](questlines.md) 對應任務線的摘要、分支與接軌方向。
- 每條任務線都已拆成獨立任務檔，請同步參照 `../../quests/` 目錄。

## 五環裂鍛
- 類型與起點：國家主線，起點為 `靈爐懸心`。
- 核心矛盾：五環各自都有不能讓別環知道的缺口，但越隱瞞，越容易把杜羅整體推向失衡。
- 主要牽涉人物：鐸格蘭、塔格魯、岩律院環長、爐影密廷觀察者。
1. 玩家先看到的是工坊、軍備、法契、礦務與外運彼此甩責。
2. 接著會發現所謂裂鍛不是單次事故，而是不同環長期累積的偷步與讓步。
3. 玩家可逐環拆問題，也可先保某一環。
4. 收束時會自然推向 `石契法典的斷頁`。
- 主要分支：平衡五環、強壓一環、讓密廷暗中整頓。
- 可延伸學習：鍛序、環籍規則、工坊權責。
- 後續接軌：可帶動 `石契裂紋`、`工坊資格影章`、`密廷假線`。

## 石契裂紋
- 類型與起點：法契線，起點為 `石契碑庭`。
- 核心矛盾：一段高位石契出現非自然裂紋，說明有人正在用極高技巧繞過杜羅的承認結構。
- 主要牽涉人物：碑庭契判官、石刻修補師、外借契印工、岩律院審定者。
1. 玩家先要辨認裂的是石、印，還是承認本身。
2. 中段會從碑面追到曾借閱、修補、轉寫過契文的人。
3. 若玩家證明裂紋是人為，將撼動杜羅對誓契可靠性的自信。
4. 最終可能導向法典、工坊資格，甚至高位權柄承認問題。
- 主要分支：低調修補、公開重審、以假裂釣真兇。
- 可延伸學習：誓契讀法、石紋修補、工藝法理。
- 後續接軌：可連 `石契法典的斷頁`。

## 深廊歸還錄
- 類型與起點：探索線，起點為 `深廊宿聚`。
- 核心矛盾：回收隊帶回來的東西比登錄冊多一件，但誰都不承認那件東西在隊伍裡。
- 主要牽涉人物：回收隊長、登錄官、深廊老宿員、對古物過敏的鍛匠。
1. 玩家需先判定多出來的是實物、名錄錯漏，還是某種被帶回來的「影」。
2. 中段會牽涉深廊採回規則與回收隊分利慣例。
3. 某些人想銷毀多出物，某些人想留下研究。
4. 收尾會影響探索公會與杜羅官方的信任。
- 主要分支：歸還、封存、私留。
- 可延伸學習：回收流程、古物封樣、深廊生存。
- 後續接軌：可連東境探索線與瑟恩樣本研究線。

## 灰鍊封坑名單
- 類型與起點：礦務線，起點為 `灰鍊礦屯`。
- 核心矛盾：封坑名單上出現本不該活著的人名，表示封坑並沒有把所有人都留在過去。
- 主要牽涉人物：坑口監工、名單保管人、失名礦工家屬、想再開坑的黑工頭。
1. 玩家從封坑令、死者名冊與坑口祭文之間發現矛盾。
2. 中段需進入半封坑區找回證據，並應對殘火、毒氣與舊坑靈。
3. 真相可能是有人假死逃坑，也可能是有人借死者身份重啟黑工。
4. 結尾將改變灰鍊礦屯對舊坑的態度。
- 主要分支：重啟調查、永封、秘密補坑。
- 可延伸學習：礦坑安全、坑道辨識、礦務文書。
- 後續接軌：可連 `未刻名的鐵箱`。

## 砧尾家業
- 類型與起點：地方線，起點為 `砧尾村`。
- 核心矛盾：一門舊鍛序被後輩拿去做了不該做的東西，家業名聲和技術秘密同時陷危。
- 主要牽涉人物：老鍛家主、叛逆後輩、外地買家、替家族說情的學徒。
1. 玩家先要理解這門舊鍛序原本做什麼，為何被禁挪作他用。
2. 中段會牽出家族傳承與公會規範的衝突。
3. 玩家若只依法處理，可能壓垮整個家業；若只講情面，危險器物會繼續流出。
4. 結尾能保鍛序、毀鍛序，或將其轉入更受控的官方體系。
- 主要分支：護家業、護規範、兩邊折中。
- 可延伸學習：家系工藝、鍛序辨識、師徒倫理。
- 後續接軌：可連 `工坊資格影章`。

## 奔嶺谷的車轍
- 類型與起點：護送線，起點為 `奔嶺谷鎮`。
- 核心矛盾：一支重車隊繞開所有查驗點，卻又一路留下像刻意讓人看到的車轍。
- 主要牽涉人物：谷鎮哨長、車隊押運頭、道口稅印員、追跡騎。
1. 玩家先從轍深、轉向與停輪痕判斷車隊載的不是普通貨。
2. 中段會在風口、山道和倉點中反覆判斷真假路線。
3. 這條車隊可能是單純走私，也可能在試杜羅山路防線。
4. 結尾將影響山道通行與白炬遠堡的戒備。
- 主要分支：截車、跟車、放車做反追。
- 可延伸學習：山道護送、貨重判讀、稅點通行。
- 後續接軌：可連阿瓦隆軍需線與克羅嵐黑市線。

## 工坊資格影章
- 類型與起點：產業線，起點為 `靈爐懸心`。
- 核心矛盾：有人用被註銷的影章重新取得工坊資格，撼動了整個杜羅工坊認證體系。
- 主要牽涉人物：資格審核官、工坊老匠、影章偽刻者、受害的新晉匠師。
1. 玩家要先查清影章為何被註銷、又是如何被復用。
2. 中段會牽出工坊評定與環內保薦制度的漏洞。
3. 若玩家揭露太深，將引爆一波資格全面重審。
4. 收尾能選擇局部補洞或全面翻修。
- 主要分支：維持市場穩定、公開清洗、暗中收編影章網。
- 可延伸學習：工坊評定、印章辨識、器具登錄。
- 後續接軌：可連 `五環裂鍛`。

## 幽廊冷爐
- 類型與起點：神秘工藝線，起點為 `鑄聖幽廊`。
- 核心矛盾：一座被封存的古爐自行回火，但沒有任何工匠承認重新點過它。
- 主要牽涉人物：幽廊守爐者、古爐研究師、恐慌學徒、想趁亂試爐的偏執匠。
1. 玩家先要處理的是不穩定的回火與恐慌。
2. 中段將帶出古爐曾用於何種儀式性鍛造。
3. 某些人相信古爐是在選人，某些人認為那只是危險餘燼。
4. 結尾能讓古爐再度封存、有限重啟，或留成長期禁地。
- 主要分支：封爐、試爐、遷移研究。
- 可延伸學習：古鍛序、火脈判讀、危爐安全。
- 後續接軌：可連高位神 `鑄形者` 的地方詮釋。

## 密廷假線
- 類型與起點：情報線，起點為 `赤鐵隘城` 或 `白炬遠堡`。
- 核心矛盾：有人故意把爐影密廷的追查路徑帶偏，讓真正的違規物流從另一條線安靜穿過。
- 主要牽涉人物：密廷外哨、隘城書辦、假線引子、被誤認成內鬼的哨兵。
1. 玩家需先判斷哪條追查路徑像是真的，哪條像被設計給密廷看的。
2. 中段會同時碰到真違規物流與假違規餌線。
3. 若玩家幫密廷太多，可能會被當成可用的外協；若幫太少，會被列入觀察。
4. 結尾能重塑杜羅情報線的信任結構。
- 主要分支：扶密廷、拆密廷、借線牽更大網。
- 可延伸學習：山道耳目、偽線判讀、哨站通訊。
- 後續接軌：可連阿瓦隆港線、瑟恩密錄線。
