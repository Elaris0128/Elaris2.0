# 穹語市坊任務線總表

## 任務線
1. 假語牌：市政 / 詐術，起於 語舍大市。有人用假語牌在圓市內行大規模詐騙。 詳見 ../../quests/skelreach-false-language-plate.md。
2. 失契板：契約 / 調查，起於 語牌街。一塊關鍵契板在交割前失蹤。 詳見 ../../quests/skelreach-lost-contract-board.md。
3. 沉核碎件：遺物 / 探索，起於 半沉貨場。半沉坡撈出一塊可能來自失落文明核心機構的殘件。 詳見 ../../quests/skelreach-sunken-core-fragment.md。
4. 洗貨夜埠：灰線 / 交易，起於 暗潮小埠。有人利用夜埠把遺跡貨洗成普通舊貨。 詳見 ../../quests/skelreach-night-dock-wash.md。
5. 誓臺失真：宗教 / 異常，起於 市誓臺。原本可靠的誓臺在某次裁定中出現錯判。 詳見 ../../quests/skelreach-oathstage-glitch.md。
6. 錯譯：知識 / 外交，起於 多舌樓。一次翻譯失準可能引發多方衝突。 詳見 ../../quests/skelreach-broken-translation.md。
7. 寄箱暗線：後勤 / 灰線，起於 暗潮小埠。一批寄箱從小埠進來後就消失了。 詳見 ../../quests/skelreach-hidden-box-line.md。
8. 拆古者：文物 / 犯罪，起於 半沉貨場。有人專門把完整古器拆散後分批出售。 詳見 ../../quests/skelreach-antique-dismantlers.md。
9. 黑潮買家：情報 / 議價，起於 後價巷。一群外地買家正在高價收特定文明殘片。 詳見 ../../quests/skelreach-black-current-buyers.md。
10. 市語之爭：社會 / 文化，起於 語舍大市。某支口譯群體和市坊執牌系統爆發衝突。 詳見 ../../quests/skelreach-market-tongue-war.md。
11. 鬼艇：夜港 / 怪事，起於 暗潮小埠。一艘沒有任何登記的艇總在潮退後出現。 詳見 ../../quests/skelreach-little-dock-ghost-boat.md。
12. 星盞線索：跨區 / 遺跡，起於 半沉貨場。一串殘件線索指向星盞群島更深處的失落遺跡。 詳見 ../../quests/skelreach-starlantern-trace.md。

## 跨區提示
- 南境的任務很容易接到外海、小島、東境交易線與失落文明探索。
