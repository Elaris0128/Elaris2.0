# 天冠洋區域包

## 區域定位
- 天冠洋是一片由浮冰、裂帶、極光窗口、冰洞與可探索陸棚共同構成的極寒海域。
- 這裡的危險不只來自怪物，還來自 `冷 / 裂 / 霧 / 假穩地 / 長距離補給斷鍊`。
- 玩家在這裡最常碰到的，不是單純戰鬥，而是 `找路 / 保暖 / 判冰 / 拖人 / 決定值不值得再往前一段`。

## 主題焦點
- 浮冰帶的活地圖與短窗航路
- 冰簾裂帶、冰洞與冰下垂降
- 白冠陸棚的極地遠征與失帳營地
- 巨型冰海魔獸、冰陸魔物與高值極寒素材
- 極光窗口、舊觀測紀錄與真假航圖
- 近岸小島、岸冰群獸與白夜錯路
- 元素異象、霜雷收核與命名巨獸
