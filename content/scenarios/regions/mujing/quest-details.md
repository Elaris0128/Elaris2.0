# 暮鏡任務細化包

## 使用方式
- 本檔整理 [questlines.md](questlines.md) 對應任務線的摘要、分支與接軌方向。
- 每條任務線都已拆成獨立任務檔，請同步參照 `../../quests/` 目錄。

## 雪塚裡少了一件不該少的東西
- 類型與起點：雪塚回收線，起點為 `裂雪原大誓圈 / 雪塚邊帶`。
- 核心矛盾：一件應陪著死者長眠的器物被人從雪塚中取走，讓整個安魂鏈開始不穩。
- 主要牽涉人物：深盞、盤骨、骨帳姨、測骨師微啞。
- 主要分支：安魂優先。 / 追查優先。 / 用器物釣出整條盜塚線。。
- 可延伸學習：雪塚禮制、禁物回收、安魂陪行。
- 後續接軌：可接北境死後世界與灰線骨物交易。。

## 夢裡先回來的人
- 類型與起點：輪夢與人格失衡線，起點為 `深暮氈坡`。
- 核心矛盾：一名參與輪夢的人明明肉身還在沉睡，卻有一個先回來的「他」開始在營地活動。
- 主要牽涉人物：岄綺、深盞、黑絮、失眠斥候阿黯。
- 主要分支：人優先。 / 制度優先。 / 追源優先。。
- 可延伸學習：輪夢守行、精神負荷辨識、夢中回路基礎。
- 後續接軌：可接權柄回聲與雪女影線。。

## 鏡嚎雪窟的第三個回音
- 類型與起點：洞窟探索線，起點為 `鏡嚎雪窟迷宮`。
- 核心矛盾：原本只會回兩種聲的雪窟突然出現第三個回音，表示迷宮內多了一套不屬於任何探隊的動線。
- 主要牽涉人物：盤骨、遂契、裂牙織、空面客。
- 主要分支：開放探索。 / 封鎖保密。 / 僅供特定組織使用。。
- 可延伸學習：迷宮回音辨識、雪窟攀行、封路術。
- 後續接軌：可接星紋古道迷宮與探索公會線。。

## 空面逐行者貼上了誰
- 類型與起點：類人威脅線，起點為 `風爪原營地`。
- 核心矛盾：像雪怪又像被剝臉旅人的空面逐行者開始模仿營地裡的人，逼得大家互相猜疑。
- 主要牽涉人物：眠汐、遂契、狼灰涔、空面客。
- 主要分支：調停與解離。 / 迅速獵殺。 / 放任它繼續追真正欠誓者。。
- 可延伸學習：類人異相辨識、營地排查、誓背對談。
- 後續接軌：可接人際信任與高層隱瞞線。。

## 四裔聯誓後的獵界重劃
- 類型與起點：政治與荒獵線，起點為 `裂雪原大誓圈`。
- 核心矛盾：一次聯誓後的獵界重劃把幾條命脈壓到同一條狹窄雪線上，任何一方都可能先餓死。
- 主要牽涉人物：烏棘、裂骨、骨帳姨、獵響谷。
- 主要分支：公平重劃。 / 向強勢方傾斜。 / 用共同危機迫使停戰。。
- 可延伸學習：獵界談判、荒隊調度、暮鏡議誓規矩。
- 後續接軌：可接雪華與薩德原的遷徙壓力線。。

## 魂霧蘚不再安靜
- 類型與起點：安魂與儀式素材線，起點為 `深暮氈坡`。
- 核心矛盾：平常只在送魂時輕動的魂霧蘚開始主動纏人，像在把某些不該留下的東西留住。
- 主要牽涉人物：深坡蔓、深盞、灰藥眠、黑坡僧。
- 主要分支：儀式純度優先。 / 保留素材價值。 / 用它引出更深靈異。。
- 可延伸學習：魂霧蘚處理、安魂儀材辨識、禁制園圃規矩。
- 後續接軌：可接盧米昂安魂與北境死後世界。。

## 裂雪原上的第二道腳印
- 類型與起點：邊境追跡線，起點為 `風爪原營地`。
- 核心矛盾：一隊返營獵手的腳印旁總會多出第二道同樣的腳印，卻沒人承認隊伍多了一個人。
- 主要牽涉人物：遂契、失眠斥候阿黯、狼灰涔、眠汐。
- 主要分支：保守封路。 / 追跡到底。 / 用它去找更大的異常。。
- 可延伸學習：雪地追跡、異常步態辨識、外原危線管理。
- 後續接軌：可接星紋古道失路與夜路線。。

## 折夢火坡的黑焰修行
- 類型與起點：修行線，起點為 `折夢火坡`。
- 核心矛盾：有人在火坡用會吞光的黑焰修行，讓一批無眠者既穩定又危險。
- 主要牽涉人物：黑絮、深盞、白蘚默修人、黑坡僧。
- 主要分支：保留修法。 / 制度收編。 / 視為禁修。。
- 可延伸學習：無眠修行、高壓心志鍛鍊、權柄回聲辨識。
- 後續接軌：可接北境修行者網與宗教深線。。

## 深暮氈坡的無眠者
- 類型與起點：聚落壓力線，起點為 `深暮氈坡冬聚地`。
- 核心矛盾：一群因夢災不敢再睡的人開始讓冬聚地運作失衡，卻也成了最可靠的夜守者。
- 主要牽涉人物：岄綺、骨帳姨、黑絮、灰藥眠。
- 主要分支：收編為夜守。 / 治療優先。 / 送出聚地。。
- 可延伸學習：收容管理、睡眠與精神調節、夜守制度。
- 後續接軌：可接靜燈庇護會與北境收容體系。。

## 雪女影在錯誓路口
- 類型與起點：類人異常與談判線，起點為 `白隱冰嶺守望點`。
- 核心矛盾：錯誓路口反覆出現像雪女的影子，把心裡最想回去的人引向冰崖。
- 主要牽涉人物：眠汐、雪女影練者、狼灰涔、無眠者代表。
- 主要分支：談判與送歸。 / 獵殺。 / 利用她追出真兇。。
- 可延伸學習：異常對談、背誓辨識、危段封控。
- 後續接軌：可接死後世界與祖靈拒收線。。

## 荒獵分圈遺失的獸骨券

- 核心矛盾：記錄荒獵配額的獸骨券被整批調包，讓整個分圈幾乎在暴雪前打起來。

- 主要分支：按舊券。 / 按當前危機重配。 / 找出幕後重審制度。。
- 可延伸學習：荒境配額制度、獵隊談判、危機物資重配。
- 後續接軌：可接盧米昂配給與星紋補給線。。

## 白毛馱群被誰驚散
- 類型與起點：生活魔物與邊荒線，起點為 `風爪原營地 / 深暮氈坡`。
- 核心矛盾：一群負責過冬搬運的白毛馱群在無暴雪的夜裡集體驚散，像是被看不見的東西驅趕。
- 主要牽涉人物：荒絨妥、長絨燧、裂骨、空面客。
- 主要分支：物資優先。 / 獸群優先。 / 封路優先。。
- 可延伸學習：馱運管理、生活魔物安撫、危線封控。
- 後續接軌：可接盧米昂與雪華的生活魔物線。。
