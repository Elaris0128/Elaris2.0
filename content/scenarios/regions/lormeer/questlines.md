# 洛溟港任務線總表

## 任務線
1. 假貨單：港務 / 調查，起於 半月主港。有一批貨在帳上入港，實際從未靠岸。 詳見 ../../quests/lormeer-false-manifest.md。
2. 失夜舟：護送 / 搜尋，起於 北潮驛棚。一艘夜渡小艇與其護貨人同時失蹤。 詳見 ../../quests/lormeer-missing-night-boat.md。
3. 沉箱：沉物 / 灰線，起於 溺霧庇灣。有人從庇灣撈起一只不該在此出現的封箱。 詳見 ../../quests/lormeer-drowned-crate.md。
4. 失名港簿：宗教 / 名錄，起於 舊潮祠。一本記著失蹤者真名的副簿被偷走。 詳見 ../../quests/lormeer-lost-name-ledger.md。
5. 夜潮掉包：犯罪 / 交易，起於 潮棚後港。灰貨與正貨在後港被人對調。 詳見 ../../quests/lormeer-nighttide-swap.md。
6. 黑礁近路：探索 / 海路，起於 溺霧庇灣。一條只在特定潮夜開啟的近礁路再度浮現。 詳見 ../../quests/lormeer-black-reef-route.md。
7. 封箱舊債：人情 / 契約，起於 半月主港。一筆舊封箱債牽出多年前的失蹤船隊。 詳見 ../../quests/lormeer-sealed-cargo-debt.md。
8. 斷潮鐘：治安 / 異象，起於 夜渡燈塔。警潮鐘無故在無潮夜自鳴。 詳見 ../../quests/lormeer-broken-harbor-bell.md。
9. 鹽路爭段：民生 / 利益，起於 北潮驛棚。兩支鹽車隊為優先出線權對峙。 詳見 ../../quests/lormeer-salt-route-fight.md。
10. 古灣板片：遺物 / 探索，起於 溺霧庇灣。有人從庇灣撈起帶龍紋的古船板。 詳見 ../../quests/lormeer-ancient-bay-plates.md。
11. 倒灌棚：災害 / 救援，起於 北潮驛棚。一場異常倒灌讓驛棚下層封死。 詳見 ../../quests/lormeer-flooded-shelter.md。
12. 外洋誓船：跨區 / 航海，起於 半月主港。一艘來自外海的誓船帶來不該在南境出現的貨。 詳見 ../../quests/lormeer-outer-ship-oath.md。

## 跨區提示
- 南境的任務很容易接到外海、小島、東境交易線與失落文明探索。
