# 魈陽古域任務分支導引

## 分支重點
- `炁炎不續`：可走 `保正統 / 追火脈 / 查人為`。
- `供養線缺口`：可選 `先救補給 / 先抓偷線者 / 先保井點`。
- `焦碑下的第二名`：可偏 `護名 / 公開 / 先查來歷`。
- `井火錯位`：可走 `護井 / 封路 / 借機找更深熱脈`。
- `熔脈集場的灼客`：可選 `市集誘查 / 正面盤問 / 私下放線`。
- `裂炭坡寨的空火圈`：可偏 `守線重建 / 反釣路客 / 清查內應`。
- `熔誓壇的背灰者`：可走 `宗教審理 / 私下安處 / 借灰找真主使`。
- `年輕承名者失步`：可選 `護人 / 保儀式 / 查黑焰`。
- `遠行引火者的黑路`：可走 `封黑路 / 利用黑路 / 重劃外路規矩`。
- `灰跡聽火的假口供`：可偏 `審口供 / 追背稿者 / 借假話釣真線`。
- `枯脈深處的熱息蓮`：可選 `保樣本 / 保火脈 / 保秘密`。
- `古域第二火印`：最後會整合你在酋庭、熔誓壇、供養隊與灰跡系統中的立場。
