# 魈陽古域任務後果與接軌

## 核心原則
- 魈陽的任務後果最常改變 `火脈正統 / 承名制度 / 補給安全 / 外路開放度 / 長老與酋庭信任`。
- 玩家若長期偏某一方，會更容易得到更深的承認，也更可能被另一方視為不穩定因素。

## 任務後果
- `炁炎不續`：成功穩住古域核心威望；失敗會讓外路與內部都開始懷疑酋庭。
- `供養線缺口`：成功能改善民心與補給；失敗會讓井點與營地物價更緊。
- `焦碑下的第二名`：成功開啟更深名錄線；失敗則讓承名制度更容易被質疑。
- `井火錯位`：成功能穩定資源節奏；失敗會讓熱路和井點風險提高。
- `熔脈集場的灼客`：成功可穩住集場與外路交易；失敗會讓外客勢力滲透更深。
- `裂炭坡寨的空火圈`：成功恢復守線信任；失敗讓遠路夜行更危險。
- `熔誓壇的背灰者`：成功可提升熔誓壇權威；失敗則讓宗教對立升高。
- `年輕承名者失步`：成功能增加年輕承名者對玩家信任；失敗會讓成年禮壓力擴大。
- `遠行引火者的黑路`：成功重掌路權；失敗則外路與黑路更難清。
- `灰跡聽火的假口供`：成功能提升灰跡系統對玩家信任；失敗會讓聽火人對外人更緊。
- `枯脈深處的熱息蓮`：成功獲得資源與新知；失敗則可能引出更危險的熱脈異變。
- `古域第二火印`：會重塑玩家在古域各權威體系中的位置。
