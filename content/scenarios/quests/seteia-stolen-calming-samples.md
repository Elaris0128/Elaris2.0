# 任務：被偷帶出去的安撫樣本

## Metadata
- id: quest-seteia-stolen-calming-samples
- region: 瑟堤亞棲國 / 渥恩城
- type: sample-smuggling-investigation
- difficulty: intermediate
- recommended-level: 3-5
- status: playable

## Premise
一批本來應該只在棲國內部流轉的安撫樣本被人偷帶出去。對外界來說那可能是高價藥材，對瑟堤亞來說卻是被抽走的一段照護能力。

## Entry Points
- 枝銘發現流向對不上
- 苔翹懷疑有人在霧蘚長廊做了非法分流
- 鏡寂絮外圍人手不足，想借玩家先查外圈

## Core NPCs
- 枝銘
- 苔翹
- 鏡寂絮
- 蘚緹
- 某名外商或陪行者

## Quest Structure

### Phase 1: 確認缺的是什麼
### Phase 2: 追樣本、追名冊、追導印
### Phase 3: 判斷是灰線走私還是被授意帶出
### Phase 4: 追回、放線或反釣

## Key Clues
- 樣本批次和導印記錄不吻合
- 有人只偷拿最適合外地市場的部分
- 某段外來通行名冊有短暫空白

## Recommended Checks
- 調查 / 生態知識 / 交涉 / 潛行

## Failure States
- 樣本完全流入外地市場
- 棲國對外來者更收緊

## Success Outcomes
- 保住對外合作窗口
- 打開監察、學庭與跨國樣本線

## Rewards
- 瑟堤亞信任
- 樣本情報或照護資源

## GM Notes For ChatGPT
- 不要把它演成普通走私，重點是「被帶走的其實是照護能力」
