# 任務：試峰下消失的獵旗

## Metadata
- id: quest-krolarn-rainscar-banner
- region: 克羅嵐聯盟國 / 斷風試峰營地
- type: trial-conflict
- difficulty: beginner
- recommended-level: 2-4
- status: playable

## Premise
斷風試峰營地一面用於試獵的獵旗在夜裡消失。若在公開試獵前找不回來，兩個氏族都會認定對方動了手腳，而這種事在克羅嵐往往比真打上一架更麻煩。

## Entry Points
- 試峰教導者請玩家幫忙找旗，避免外來者見笑
- 某位年輕獵手私下說自己看見不該出現在旗棚附近的人影
- 河牙會埠送來的補給隊帶來一卷染了異味的舊旗布
- 玩家在營地夜守時目擊有人匆忙擦掉旗杆上的血痕

## Core NPCs
- 試峰教導者：希望事情在大會發現前解決
- 圖格爾：對試獵公正極度敏感
- 洛絮：能聽懂不同氏族互相指責時真正的含義
- 牙煞的外圍影耳：懷疑這不是單純搗亂
- 某位少年獵手：他看到的東西可能是全案的起點

## Quest Structure

### Phase 1: 試峰查旗
玩家先要確認旗是真的被偷走，還是被人換了位置或換了布。

#### 可做的事
- 檢查旗棚、繩結和固定桿
- 詢問夜守、獵手與營地工匠
- 比對試獵前登記與旗布紋樣

#### 可獲得線索
- 旗沒有被硬拔，而是被懂規矩的人完整拆下
- 現場留下了不屬於試峰的油脂與獸味
- 一段腳印故意通往錯方向

### Phase 2: 三條追線
玩家可優先選擇：
- 追氏族對立線
- 追營地內應線
- 追補給與外來物資線

#### 隱藏真相方向
獵旗消失不是單純偷旗，而是有人想藉這場試獵挑起氏族間的「程序性失信」，好讓之後更大的會盟安排被動搖。

### Phase 3: 試獵前夜
若玩家拖到試獵前夜，營地張力會明顯升高。

#### 可能事件
- 一面替代旗突然被拿出來，說法卻前後不一
- 某位少年獵手失蹤
- 有人試圖在重甲丘散布「對方早就承認了」的假話

### Phase 4: 公開或壓下
玩家最後可以：
- 公開指出操手，保住試獵正當性
- 只追回旗，暫時壓下衝突
- 順著假旗與錯證，釣出更大的操盤者

## Key Clues
- 被完整拆解的繩結
- 非試峰常用的油脂與獸味
- 刻意導錯的足跡
- 舊旗布上的次氏族染料
- 試獵名冊和夜守輪值對不上的空檔

## Recommended Checks
- 觀察 / 追跡：找旗、比對足跡、辨認獸味
- 交涉：安撫獵手、說服氏族代表別提早翻桌
- 生態 / 生活知識：辨識戰獸與旗布的保養痕跡
- 潛行：夜查旗棚與補給區

## Failure States
- 玩家讓兩個氏族先公開互罵，試獵難以維持公正
- 關鍵少年獵手被更高層的人帶走
- 玩家錯抓替罪羊，後續聯盟對外來者戒心升高

## Success Outcomes
### 公開修復
- 保住試獵與氏族面子
- 提升試峰與巨蹄營地對玩家的信任

### 暗中釣線
- 讓真正操盤者以為自己仍安全
- 後續更容易接到 `裂丘的假骨兆` 或 `環影斷訊`

### 保守止血
- 避免當場爆發
- 但某些氏族仍會對玩家保留

## Rewards
- 試峰信物或獵旗授權小物
- 聯盟試煉圈信任提升
- 解鎖後續任務：`骨語錯兆`、`環影斷訊`

## GM Notes For ChatGPT
- 這條任務重點是「名譽與程序」而不是單純找失物
- 要讓玩家感覺到試旗本身是聯盟秩序的一部分
- 若玩家善於安撫雙方，就讓他感受到自己能影響會盟前的空氣
