# 任務：禁誓森林的暗號石

## Metadata
- id: quest-avalon-signal-stones
- region: 阿瓦隆王國 / 銀鋒城外圍 / 禁誓森林邊界
- type: exploration-sidequest
- difficulty: beginner-to-low-mid
- recommended-level: 1-3
- status: playable

## Premise
阿瓦隆王國嚴格限制夜間進入某些森林與誓禁區域，但近日有巡邏隊在邊界石上發現陌生刻痕。這些符號既不像普通走私記號，也不像阿瓦隆官方的巡防標示。若放著不管，它可能只是下一條黑路；若太快上報，它也可能被更高層的人抹掉。

## Entry Points
- 法連將其作為低階探索委託交給玩家
- 哈姆特用老兵口吻提醒玩家「邊界石開始說別人的話了」
- 塔維恩暗示那不是單純賊路，而是某種投點記號
- 莉歐妲要求確認是否涉及禁誓或異端儀式痕跡

## Core NPCs
- 哈姆特：提供野外與邊境判讀
- 法連：給任務包裝與基本報酬
- 塔維恩：若玩家回報異常，可導入情報線
- 莉歐妲：若符號與誓約禁制相關，將介入

## Quest Structure

### Phase 1: 邊界調查
玩家抵達禁誓森林邊界後，可進行：
- 檢查暗號石刻痕
- 尋找周圍足跡與火堆殘痕
- 詢問附近巡邏兵或樵夫
- 跟著斷續記號往森林邊緣深入

#### 可獲得線索
- 記號分成兩種筆跡，不像同一批人留下
- 其中一組符號像是通行指引，另一組更像警示或修正
- 附近存在被刻意折斷的枝幹與壓低的草徑
- 某些石面刻痕有微弱環印殘留，不像純民間符號

### Phase 2: 路線判讀
玩家需要決定怎麼解讀這些記號：
- 把它視為走私路線，追查接應點
- 把它視為情報投點，尋找回收者
- 把它視為異端或違誓儀式外圍記號
- 假裝沒看懂，先埋伏等待夜裡有人來看石頭

### Phase 3: 林間接觸
夜間或深入後，玩家可能會遇到：
- 一組低階走私搬運者
- 一名偽裝成樵夫的耳目
- 一位試圖覆蓋舊記號的神秘旅人
- 一頭受異常氣味刺激而躁動的契約獸或野獸

### Phase 4: 真相拆分
這些暗號石不是單純單一勢力所留，而是至少兩股勢力在同一條林線上爭奪控制權：
- 一方想建立穩定的灰色運輸線
- 另一方則想利用這條線投放情報或釣魚

玩家可以：
- 把整條線交給軍方清剿
- 幫情報方保留釣魚價值
- 自己留下路線情報作後續籌碼
- 誤判並導致某方提早撤離

## Key Clues
- 兩套不同節奏的刻痕
- 帶有環印殘痕的石屑
- 樵夫工具上不合常理的保養痕跡
- 夜裡固定時間出現的觀測者
- 舊邊境補給線的殘留路標

## Recommended Checks
- 調查：比對刻痕與地面痕跡
- 求生 / 野外：辨識人為路線與天然獸徑
- 觀察：分辨誰在看玩家、誰在看石頭
- 交涉：逼問搬運者或安撫驚慌證人
- 潛行：夜間埋伏與反跟蹤

## Failure States
- 玩家過早暴露，導致整條線被清空
- 玩家誤將情報投點當成單純賊路，失去更深層線索
- 玩家在禁區內留下明顯痕跡，被官方追責
- 玩家與野獸衝突失控，引發額外危險

## Success Outcomes
### 軍方路線
- 哈姆特與邊境線信任提升
- 解鎖北線或巡防任務

### 情報路線
- 塔維恩對玩家評價上升
- 解鎖更深的灰色調查任務

### 宗教路線
- 莉歐妲介入，開啟禁誓與異常誓印調查

### 自留路線
- 玩家獲得一條非法或半合法情報優勢
- 未來風險也隨之上升

## Rewards
- 金錢或邊境補給
- 一份路線圖或可重複利用的林線情報
- 與哈姆特、塔維恩或莉歐妲其中一人的關係提升
- 解鎖後續：失蹤的前鋒、塔影的耳錄、邊境密令線

## GM Notes For ChatGPT
- 這條任務要讓玩家第一次感受到「世界不是非黑即白」
- 森林本身不是最危險的，最危險的是誰在這裡留下痕跡、又想讓誰看見
- 若玩家很細心，就讓雙方勢力競逐的感覺更明顯
- 若玩家偏衝動，就用追逐、誤判和夜間遭遇把任務推起來
