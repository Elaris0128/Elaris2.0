# 任務：被改過的路標石

## Metadata
- id: quest-sadren-fallen-waymark
- region: 薩德原聯盟 / 日蹄井村
- type: travel-route-investigation
- difficulty: beginner
- recommended-level: 1-3
- status: playable

## Premise
最近多支旅隊被引去錯的水點，還有人差點死在補水線外。草原上路標不只是方向，而是活命的秩序，所以這件事比一般破壞更嚴重。

## Entry Points
- 阿綠希望玩家幫忙確認哪幾根標石被換過
- 某支差點渴死的旅隊要求有人替他們討回公道
- 風策局外圍人手不足，只能先找外人幫忙看路
- 玩家在灰尾換市聽見有人打聽不存在的井名

## Core NPCs
- 阿綠：井場管理者，最清楚補水秩序被動的後果
- 瑪舒雅：路譜守望者，能從幾個刻痕看出是誰的手法
- 野祿契：若旅糧也被動手，會成為很重要的輔助線人
- 烏賽寧的外線人：會觀察玩家處理這件事的方式
- 某名倖存旅隊車手：知道錯標帶來的真正恐懼

## Quest Structure

### Phase 1: 看標與驗路
玩家先要找出哪些標石是被人錯動、哪些又是故意留下的假線索。

#### 可做的事
- 比對路標刻痕與風化程度
- 詢問旅隊與井場管理人
- 走一小段錯路，確認它真正通往哪裡

#### 可獲得線索
- 有些標石是新換的，有些只是被輕微轉角
- 錯路並不完全通往絕路，而是會把人帶去特定補水外帶
- 有人熟悉草原旅線，不像外地亂搞

### Phase 2: 查受益者
玩家可以優先追：
- 誰從錯送旅隊獲利
- 誰有能力夜裡改標石又不被看見
- 哪支旅隊或商路最近特別想控制補水順位

#### 隱藏真相方向
錯標石的目的不只是害人迷路，而是重新塑造某幾條路的「自然優先權」，讓接下來旱季前的路權和補水秩序被迫朝特定方向傾斜。

### Phase 3: 夜間路線埋伏
如果玩家選擇守夜或故意放消息出去，就會看到真正動標的人再度出手。

#### 可能事件
- 另一根標石也被轉過角度
- 有人故意把旅隊往乾井帶
- 風策局外線觀察你會不會亂動真正的舊標

### Phase 4: 公開修正或暗中釣人
玩家最後可：
- 立刻重立標石，先救人
- 假裝沒看懂，順著錯路抓更大的操手
- 把事交給風策局，但承擔之後被他們盯上的可能

## Key Clues
- 刻痕深淺不一致的路標面
- 指向特定乾井與舊路的錯角
- 旅隊口中的不存在井名
- 某支隊伍總能避開錯路的反常現象

## Recommended Checks
- 觀察 / 野外知識：辨認路標、風化、補水距離
- 交涉：安撫旅隊、套話換市商人
- 追蹤 / 潛行：夜守標石、跟蹤動手者
- 生存：親走錯路並安全回來

## Failure States
- 玩家修錯真正的舊標，導致更多人迷路
- 夜間埋伏太高調，操手暫停行動
- 錯路先害死人，整個草原對外來者更不信任

## Success Outcomes
### 立刻止血
- 路線恢復安全
- 日蹄井村與路譜守望者信任提升

### 釣出幕後
- 能往更深的路權、水權與旱季布局查下去
- 更容易接到 `風策局的假風訊` 或 `旱季前的第五柱`

### 風策局合作
- 獲得草原高層注意
- 但玩家行動會更被記錄

## Rewards
- 草原路線信任
- 低階導路權或護送便利
- 地方旅糧、路標刻片或後續風訊

## GM Notes For ChatGPT
- 這條任務重點是讓玩家理解：在草原上，路標就是秩序的一部分
- 不要把它只演成找石頭，真正張力在於誰有資格決定別人走哪條路
- 若玩家先救人，要給即時的情感回饋
