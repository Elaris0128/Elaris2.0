# 任務：海膽葡萄的安全採收

## Metadata
- id: quest-seteia-seaurchin-grapes
- region: 瑟堤亞棲國 / 霧蘚長廊
- type: ecological-harvest
- difficulty: beginner
- recommended-level: 1-3
- status: playable

## Premise
一批本該送往照護點的海膽葡萄成熟過快。若處理不好，不是整串炸爛，就是整片枝架都要報廢，而瑟堤亞最不喜歡外來者把本來能溫和處理的事弄成一地狼藉。

## Entry Points
- 苔翹需要臨時人手進林採收
- 林契議價窗希望有人先確認是不是有人碰過枝架
- 導印臺願意讓玩家跟一次低擾採集，作為測試
- 柊霧獸醫指出最近有旅隊坐騎被刺殼炸傷

## Core NPCs
- 苔翹：懂採息、去刺與價值調停
- 蘚緹：決定玩家有沒有資格被放進正確的林線
- 柊霧獸醫：從受傷的坐騎判斷是不是有人亂動過
- 某名外商採購：可能只是急，也可能是故意
- 在地採集照護者：知道枝架平常的呼吸節奏

## Quest Structure

### Phase 1: 先學怎麼碰
玩家不能直接去採，必須先學安全處理。

#### 可做的事
- 學去刺、安撫與封樣流程
- 詢問往年採收節奏
- 檢查哪幾串成熟得不自然

#### 可獲得線索
- 某些果串成熟速度不屬於正常風季
- 有幾串外殼刺向被人刻意引成爆裂角度
- 其中一處枝架留有外來繩纖與油痕

### Phase 2: 進林採收或追人
玩家可優先選擇：
- 先保全可收果料
- 追查誰碰過枝架
- 沿樣本流向查高價買家

#### 隱藏真相方向
這次問題不只是過熟，而是有人在測試如何讓高值魔植於「合法採收前」提前爆裂，以便壓低表面價值，再從灰線收走真正有用的部分。

### Phase 3: 林線變局
進入指定林線後，張力會明顯上升。

#### 可能事件
- 一串果突然連鎖爆裂
- 一名外來陪行者想偷帶走還沒封樣的果核
- 林線中的其他共棲生命對某位同行者表現出異常排斥

### Phase 4: 保貨、抓人或放線
玩家最後可：
- 先保住照護點需要的果料
- 直接抓操手
- 放掉下手的人，順著高價樣本線往外查

## Key Clues
- 刺殼角度被刻意引偏的果串
- 外來繩纖與油痕
- 成熟速度異常的枝架段
- 受傷坐騎留下的共鳴殘跡

## Recommended Checks
- 生態 / 採集知識：辨認能碰與不能碰的果串
- 觀察：檢查刺殼角度、樣本痕跡
- 交涉：與導印臺、採集者與外商周旋
- 靈巧 / 生存：實際去刺、封樣與避爆

## Failure States
- 玩家讓枝架大面積爆裂，照護點供給失敗
- 玩家帶出未封樣樣本，導致棲國對外來者更不信任
- 真正下手者趁亂脫身

## Success Outcomes
### 照護優先
- 提升霧蘚長廊與照護圈信任
- 更容易開啟低擾採集與照護學習線

### 追查優先
- 能把問題帶去 `被偷帶出去的安撫樣本` 或更大的高價樣本外流線
- 監察部與學庭都可能開始注意玩家

### 平衡處理
- 同時保住供應與留住調查線
- 最適合作為瑟堤亞中前期的代表任務

## Rewards
- 採集與照護圈信任
- 海膽葡萄安全處理權或配方線索
- 解鎖後續任務：`被偷帶出去的安撫樣本`、`綠苞溪聚的採樣者`

## GM Notes For ChatGPT
- 這條任務的重點是讓玩家感受到瑟堤亞的規矩不是冷，而是為了避免傷害
- 採收過程要有技巧感，不要只是骰成功就結束
- 若玩家尊重程序，要給很正面的反饋
