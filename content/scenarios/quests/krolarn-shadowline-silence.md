# 任務：環影斷訊

## Metadata
- id: quest-krolarn-shadowline-silence
- region: 克羅嵐聯盟國 / 河牙會埠
- type: intelligence-recovery
- difficulty: intermediate
- recommended-level: 3-5
- status: playable

## Premise
一支環影部隊留下半句暗碼後失去聯絡。更糟的是，那半句暗碼像是故意留給別人看的。

## Entry Points
- 牙煞外圍人手不足，找玩家補位
- 葛薩從會埠貨流裡察覺不對
- 河牙誓衛撿到一枚不該掉在碼頭的影耳標片

## Core NPCs
- 牙煞
- 葛薩
- 魁牙
- 一名失聯影耳的親屬或同行者

## Quest Structure

### Phase 1: 解讀半句暗碼
### Phase 2: 查會埠貨線與邊界足跡
### Phase 3: 判斷是救人還是保線
### Phase 4: 回收、反釣或切斷

## Key Clues
- 半句暗碼有雙重讀法
- 失聯小隊曾碰過誓貨與邊界線
- 河埠有人在等某個不該出現的交接

## Recommended Checks
- 潛行 / 追跡 / 暗號解讀 / 交涉

## Failure States
- 整條外線被拔乾淨
- 玩家被反標記

## Success Outcomes
- 打開更深的聯盟情報線
- 可直接銜接 `裂丘的假骨兆` 或跨國邊界主線

## Rewards
- 環影信任
- 暗號與邊界情報資源

## GM Notes For ChatGPT
- 要把壓力放在「你不知道救的是人還是圈套」
