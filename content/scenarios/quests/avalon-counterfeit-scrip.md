# 任務：偽幣之流

## Metadata
- id: quest-avalon-counterfeit-scrip
- region: 阿瓦隆王國 / 銀鋒城
- type: urban-financial-investigation
- difficulty: intermediate
- recommended-level: 3-5
- status: playable

## Premise
阿瓦隆市面上出現一批做得極好的偽券。麻煩不是它們夠像，而是它們被刻意投放進幾個最會互相影響的地方：糧鎮、港口和王都。

## Entry Points
- 雷維斯發現檔錄裡有幾份對不上來源的券號
- 瑟蘭在商會窗口攔到一批被多家店退回的票券
- 梅列絲從港口送來一份可疑貨款清單
- 玩家在下城拱市購物時被店家指出手上的找零不對

## Core NPCs
- 雷維斯：從文契與券號角度切入
- 瑟蘭：從市場與跨國流通角度切入
- 維榮・暮鴉：懷疑這批偽券只是更大操作的一層
- 某名糧商：可能只是貪，也可能被迫配合

## Quest Structure

### Phase 1: 辨券
- 確認哪些偽券屬同一批、哪些只是跟風仿作
- 比對券號、紙質、印序與流入路徑

### Phase 2: 查流向
- 追王都市場
- 追誓禾鎮糧貨
- 追誓港貨款
- 追商會換券窗

### Phase 3: 市場驚動
- 偽券消息傳開後，會出現搶換、拒收與囤貨
- 玩家要選先穩市面，還是先抓投放者

### Phase 4: 結案站邊
- 交給王室與商會
- 交給暮環司
- 留一批假券做反釣

## Key Clues
- 券號段集中在幾條糧貨線
- 有些偽券刻意做得「只差一點」，像是測試店家辨識能力
- 港口貨款與糧鎮貨價同步異常

## Recommended Checks
- 文書 / 觀察 / 交涉 / 潛行

## Failure States
- 市場過早恐慌
- 偽券頭先撤乾淨
- 玩家自己被列為可疑持券者

## Success Outcomes
- 穩定市價
- 打開跨國票據與偽印線
- 提升王室、商會或情報線信任

## Rewards
- 票據流向資料
- 商會折抵
- 解鎖後續任務：`港盟偽券`、`暮環港線`
