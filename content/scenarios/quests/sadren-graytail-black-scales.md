# 任務：灰尾換市黑秤

## Metadata
- id: quest-sadren-graytail-black-scales
- region: 薩德原聯盟 / 灰尾換市
- type: market-shadow-investigation
- difficulty: intermediate
- recommended-level: 3-5
- status: playable

## Premise
灰尾換市表面公平，實際卻有人用黑秤慢慢抽乾某支族柱的貨與命脈。問題在於，這種做法如果抓錯時機，整個換市都會先亂。

## Entry Points
- 灰尾姊私下提醒玩家最近秤感不對
- 野祿契發現旅糧重量和記價總對不上
- 風策局想知道是不是有人借換市轉敏感貨

## Core NPCs
- 灰尾姊
- 野祿契
- 嵐索
- 瓦克

## Quest Structure

### Phase 1: 識秤與比貨
### Phase 2: 找出黑秤如何運作
### Phase 3: 夜市與敏感貨物流動
### Phase 4: 公開翻桌、暗中拔線或放長線

## Key Clues
- 同攤不同客的秤感不一樣
- 黑秤只在特定貨類和特定隊伍上出現
- 幼體與未記錄樣本的流向和黑秤時間吻合

## Recommended Checks
- 觀察 / 交涉 / 市價知識 / 潛行

## Failure States
- 換市先炸鍋
- 操手斷尾逃掉

## Success Outcomes
- 穩住換市
- 打開草原灰線與敏感貨物流向

## Rewards
- 換市信任
- 灰線情報

## GM Notes For ChatGPT
- 讓玩家感覺這是「市場秩序被人慢慢搬歪」的任務
