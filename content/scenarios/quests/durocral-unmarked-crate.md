# 任務：未刻名的鐵箱

## Metadata
- id: quest-durocral-unmarked-crate
- region: 杜羅山邦 / 白炬遠堡
- type: logistics-investigation
- difficulty: beginner
- recommended-level: 2-4
- status: playable

## Premise
白炬遠堡查到一口規格完美的鐵箱。問題不在箱子，而在它沒有任何刻印。對杜羅來說，這幾乎等於有人讓正式工坊的東西失去了名字。

## Entry Points
- 卡魯姆請玩家幫忙驗箱，因為正式流程已卡住
- 妲萊懷疑這口箱子跟最近的錯號事件有關
- 某名商隊護衛表示自己一路上都被人盯著這口箱子
- 坡堡外棚有人開價想私下買走它

## Core NPCs
- 卡魯姆：驗契官，代表關堡程序
- 妲萊：火號手，從邊線異常聯想到這口箱子
- 奧芬石掌：能從防護與重量判斷箱子的原始用途
- 不具名押運書記：看過路線，但未必願意全說
- 爐影密廷外圍耳目：若事情夠大，會開始出現

## Quest Structure

### Phase 1: 驗箱
玩家先要確認鐵箱本身的製作層級與目前狀態。

#### 可做的事
- 檢查箱體結構、鉚釘、封扣與刮磨痕
- 比對關城貨單與工坊樣式
- 詢問護衛、驗契官和修具匠

#### 可獲得線索
- 箱體出自正式工坊體系，但刻印被整齊抹除
- 鎖扣與把手規格並不適合普通商貨
- 箱底沾過不同地層的灰，不像只走一條山路

### Phase 2: 查路線還是拆箱
玩家要先決定優先：
- 追製造與刻印來源
- 追押運路線
- 直接開箱
- 假裝不開箱，釣收貨人

#### 隱藏真相方向
這口箱子是某個本不該離開杜羅正式體系的物件外殼，它本身也許不是核心，但它本該裝過什麼，才是幕後真正怕被查到的東西。

### Phase 3: 夜間爭奪
只要玩家拖延到夜裡或消息外漏，就會有新的人來碰箱子。

#### 可能事件
- 有人來高價買箱
- 箱上的抹痕被人試圖再加工
- 關堡外棚起小火，明顯想製造混亂
- 密廷外圍人先一步來問：你打算把箱子交給誰

### Phase 4: 交回、拆開或放線
玩家最後可選：
- 交回正式工坊體系
- 交給關堡與軍線
- 留在自己手裡釣收貨人
- 與密廷合作，但承擔之後被記住的代價

## Key Clues
- 被抹除的刻印痕
- 不合理的鎖扣規格
- 箱底混雜地灰
- 押運名單中的時間空白
- 與工坊留樣不完全一致的細節

## Recommended Checks
- 調查 / 工藝知識：辨認工坊工法、抹印方式
- 交涉：與關堡、工匠、押運書記對話
- 潛行：盯收貨人或夜間接觸箱子的人
- 力量 / 工具：必要時安全拆箱

## Failure States
- 玩家直接拆壞關鍵證物
- 真正收貨人察覺後改線
- 關堡把箱子過早封死，線索只能剩下文書

## Success Outcomes
### 正規移交
- 提升關堡與工坊體系信任
- 後續更容易接到正式護送與工坊調查任務

### 暗線追查
- 提前進入密廷注意範圍
- 後續更容易接到 `密廷假線` 或技術外流支線

### 釣魚成功
- 找到真正的接貨者或次級中間人
- 能把問題一路帶到 `石契法典的斷頁`

## Rewards
- 杜羅關堡信任
- 工藝體系或密廷體系注意度
- 一件實用鐵件、封樣工具或後續線索

## GM Notes For ChatGPT
- 這條任務重點是讓玩家感受到杜羅把「名字與刻印」看成秩序的一部分
- 箱子本身要有份量感，不要像普通走私箱
- 若玩家選擇放線，不要立刻給全貌，先讓他抓到次級人物
