# 任務：星軌全權令

## Metadata
- id: quest-therne-startrack-decree
- region: 瑟恩聯邦國 / 瑟彌亞
- type: political-crisis
- difficulty: intermediate
- recommended-level: 4-6
- status: playable

## Premise
一場事故讓聯邦不得不提前討論是否啟動 `星軌全權令`。問題是，這道機制一旦啟動，就代表某些正常程序要暫時讓位。

## Entry Points
- 伊瑟菈要玩家去核對事故資訊
- 托文席爾希望玩家先穩住部分議席代表
- 密錄局透露事故可能不是意外

## Core NPCs
- 伊瑟菈
- 托文席爾
- 露黎恩
- 梅瑞狄雅
- 某位反對全權令的城邦代表

## Quest Structure

### Phase 1: 確認危機是否真足以啟動全權令
### Phase 2: 說服、查證或拖延
### Phase 3: 議席對撞與外部施壓
### Phase 4: 啟動、限縮或否決

## Key Clues
- 事故報告的時間戳不一致
- 某些議席早就準備好備案
- 外部勢力似乎希望聯邦提早集中權限

## Recommended Checks
- 交涉 / 文書 / 調查 / 洞察

## Failure States
- 聯邦內部分裂加劇
- 玩家被視為替某派跑腿

## Success Outcomes
- 決定聯邦接下來的危機節奏
- 打開更高層政治與情報線

## Rewards
- 高層通行與議場信任
- 解鎖後續任務：`七曜席次裂縫`、`協章裂口`

## GM Notes For ChatGPT
- 重點是讓玩家感覺自己在碰一個「程序是否要讓位給生存」的問題
