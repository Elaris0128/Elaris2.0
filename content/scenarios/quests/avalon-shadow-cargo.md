# 任務：誓港的幽影貨櫃

## Metadata
- id: quest-avalon-shadow-cargo
- region: 阿瓦隆王國 / 東海誓港
- type: main-quest-entry
- difficulty: beginner
- recommended-level: 1-3
- status: playable

## Premise
東海誓港查獲一批貼著臨時封條、來源不明的貨櫃。官方認為這可能是單純的走私，但現場痕跡顯示，貨櫃內可能藏著更敏感的東西：偽造稅契、被轉運的誓環器物、甚至能牽動教廷與情報機關的禁制物。

## Entry Points
- 法連從冒險者公會正式發放調查委託
- 梅列絲直接要求玩家協助觀察貨區
- 塔維恩私下接觸玩家，表示這批貨不宜交給表面程序處理
- 玩家主動追查受驚水手，意外捲入任務

## Core NPCs
- 法連：提供任務合法包裝
- 梅列絲：代表秩序與港務審查
- 塔維恩：代表情報與灰色處理方式
- 無名水手：第一個可變動的證詞來源
- 倉區看守長：容易被買通，也可能只是替罪羊

## Quest Structure

### Phase 1: 現場接觸
玩家先接觸貨櫃區，能做的事包括：
- 觀察封條與貨櫃外觀
- 詢問現場守衛與稅官
- 追蹤逃走的水手
- 偷取或合法查驗一份貨單

#### 可獲得線索
- 封條是臨時補貼，原始印記被刻意覆蓋
- 貨單上的品項與貨櫃重量不吻合
- 至少一個貨櫃有內層隔板
- 水手提到「環印箱」與「夜裡有人換箱」

### Phase 2: 線索交叉
玩家需要在 2 到 3 條線索中選擇優先順序：
- 查貨單來源
- 跟進水手去向
- 盯守夜間搬運
- 請梅列絲合法開箱
- 先把情況回報給塔維恩或法連

#### 隱藏真相方向
這批貨並非單純走私，而是有人利用港口常規貨運，把一批偽裝成祭儀器物的敏感物資送進阿瓦隆。這些物資可能與偽幣、密函或失竊誓契案形成後續串線。

### Phase 3: 夜間轉運事件
若玩家選擇埋伏、追蹤或拖延到夜間，就會進入任務第一次高張力場景。

#### 夜間事件可能內容
- 有人試圖把一只貨箱轉移到未登記的小船
- 一名港區內應者試圖燒毀帳冊
- 水手想逃離誓港，卻被另一批人先追上
- 塔維恩出現，暗示官方內有人不想讓真相進到白天程序

### Phase 4: 結案選邊
玩家最終可把成果交給不同對象，導向不同結果：
- 交給梅列絲：合法結案，獲得港務與公會信任
- 交給塔維恩：換來更深層情報線，但會被暮環司盯上
- 交給教廷：若查出祭儀或誓約禁制物，將觸發宗教線
- 自己留下部分證物：能換取後續優勢，也埋下風險

## Key Clues
- 偽造稅契副本
- 重新覆蓋過的封條碎片
- 記有特殊環印代碼的木箱內壁
- 受驚水手的不完整證詞
- 港務簽名與實際值勤名單不一致

## Recommended Checks
- 觀察 / 調查：檢查封條、貨單、痕跡
- 交涉：安撫水手、說服稅官、套話守衛
- 潛行：夜間監視、追蹤搬運者
- 戰鬥：必要時處理低階打手或逃亡者

## Failure States
- 水手死亡或失蹤，導致證詞線中斷
- 貨單與帳冊被燒毀，證據轉為間接
- 玩家過早驚動幕後方，使夜間轉運改道
- 玩家被官方當成可疑人物，任務難度上升

## Success Outcomes
### 秩序結案
- 取得梅列絲與港務體系信任
- 公會對玩家評價上升
- 後續可更容易接正式調查委託

### 陰影結案
- 取得塔維恩私下認可
- 開啟暮環司支線
- 玩家獲得一條未公開的高價值線索

### 混合結案
- 同時保住表面秩序與部分暗線情報
- 最適合作為第一章通往第二章的結局

## Rewards
- 金錢
- 港區通行便利
- 一件初期實用物資或證物
- 與至少一位關鍵 NPC 的信任關係提升
- 解鎖後續任務：偽幣之流、誓契失竊案、塔影的耳錄

## GM Notes For ChatGPT
- 這條任務的重點不是戰鬥，而是讓玩家學會：觀察、選邊、保存證據、承擔後果
- 第一次不要把真相一次講完，只透露「這批貨比走私更麻煩」
- 若玩家偏愛正規玩法，就讓梅列絲線更紮實
- 若玩家偏愛灰色玩法，就讓塔維恩線更有吸引力
- 若玩家想自由亂闖，也要讓港口秩序與律法後果立刻出現
