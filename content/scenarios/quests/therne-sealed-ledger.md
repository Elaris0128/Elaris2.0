# 任務：失頁的條契簿

## Metadata
- id: quest-therne-sealed-ledger
- region: 瑟恩聯邦國 / 洛菲達
- type: institutional-investigation
- difficulty: beginner
- recommended-level: 2-4
- status: playable

## Premise
洛菲達外約院的一本條契簿被替換了幾頁。麻煩不在於有人動手，而在於替換者留下的文風和印封過於完美，像是故意要讓人發現有人能做到這種事。

## Entry Points
- 蕾薇安請玩家幫忙確認兩份看起來幾乎一致的條契副本
- 賽莫在河港查驗時扣到一本不該流到外港的小簿子
- 伊蓮瑟帶著一份潮損抄本來求助，表示其中一頁的口氣不對
- 玩家在白印街聽見有人收購舊印紋和濕損頁角

## Core NPCs
- 蕾薇安：條約翻譯師，能分辨文字裡細微的不自然
- 賽莫：河港查驗員，負責把可疑文件攔在正式流程外
- 伊蓮瑟：流動修典師，知道哪些抄本曾經被轉手
- 露黎恩：若玩家驚動到足夠高的層級，密錄局會開始對此案感興趣
- 不具名抄手：真正摸過原件的人，但不一定是主使者

## Quest Structure

### Phase 1: 確認失頁
玩家要先分辨哪幾頁被換過，以及是原件被抽換還是副本被污染。

#### 可做的事
- 比對原件、副本與河港轉送版本
- 檢查封蠟、頁序、抄寫手勢與水痕
- 詢問最近接觸過條契簿的人

#### 可獲得線索
- 有兩頁的頁角裁切方式不屬於洛菲達本地
- 封蠟是真的，但蓋印時間和頁面潮痕對不上
- 替換頁的字句過度完美，像故意照著公證語氣偽造

### Phase 2: 找出替換鏈
玩家需要決定先追哪一段流程：
- 追抄手
- 追保管櫃點
- 追河港轉件
- 追誰最早從錯誤頁面獲利

#### 隱藏真相方向
這次失頁不只是黑工抄手接單，而是有人測試聯邦公證鏈在多點轉抄下，能不能被「合法地篡改」。

### Phase 3: 河港與白印街交叉
一旦玩家查到白印街或河港，就會進入第一個真正會驚動別人的節點。

#### 可能事件
- 白印街有人急著收回做壞的頁角紙
- 河港倉側有人要把一本補頁簿沉進河裡
- 蕾薇安和賽莫對該不該立刻公開起爭執
- 密錄局外圍耳目開始旁聽玩家說話

### Phase 4: 選擇交付對象
玩家可以把成果交給不同單位：
- 交給法理與公證系：保全制度，提升聯邦正式信任
- 交給密錄局：快速擴線，但會被捲進更深的文書與涉外調查
- 暫不全交：保留一部分頁面資訊，為後續更大案件留底

## Key Clues
- 被重新裁切過的頁角
- 與潮痕不一致的封蠟
- 用字過於標準的偽頁
- 河港轉件名冊缺漏的半日空檔
- 白印街交易過的舊印紋樣

## Recommended Checks
- 觀察 / 調查：檢查頁序、蠟痕、紙質與抄寫習慣
- 交涉：說服抄手、查驗員、保管者開口
- 潛行：跟蹤取件者、夜查白印街
- 文書 / 知識：辨認公證語氣、翻譯慣用句與聯邦格式

## Failure States
- 玩家太早公開，導致真正的替換鏈被切斷
- 關鍵抄手逃走或失聯，線索轉為間接
- 玩家把所有資料交給單一單位，導致另一條調查路完全關閉

## Success Outcomes
### 制度結案
- 提升洛菲達、艾綴塔與瑟彌亞的制度圈信任
- 後續更容易接到正式法務、公證與涉外任務

### 密錄結案
- 露黎恩體系開始正式注意玩家
- 可更早接入跨國偽文書、偽券與樣本封存問題

### 保留結案
- 玩家手上留有能證明更大問題的碎片
- 後續在 `協章裂口` 或 `港盟偽券` 線中會更有主動權

## Rewards
- 聯邦正式委任或抄錄權限
- 可用於後續調查的副本資料
- 與法理或密錄體系至少一方的信任提升
- 解鎖後續任務：`艾綴塔雙版本協議`、`港盟偽券`

## GM Notes For ChatGPT
- 這條任務重點是讓玩家感受到瑟恩的危險，常常不是刀，而是能被承認的文字
- 不要太早公開幕後黑手，先讓玩家覺得「流程本身也可能成為武器」
- 若玩家偏愛正規程序，就讓法理線更有成就感
- 若玩家偏愛暗查，就讓密錄線更快接到東境大網
