# 任務：白炬錯號

## Metadata
- id: quest-durocral-whitebeacon-false-signal
- region: 杜羅山邦 / 白炬遠堡
- type: military-warning-investigation
- difficulty: intermediate
- recommended-level: 3-5
- status: playable

## Premise
白炬遠堡的夜間火號被人調換了一個位次。只差一個位次，卻足以讓另一段山線以為該進行完全不同的動作。

## Entry Points
- 妲萊請玩家幫忙確認是不是記錄錯誤
- 遠堡守軍發現另一側山線的回應不正常
- 坡棚商隊抱怨因錯號被迫改道

## Core NPCs
- 妲萊
- 卡魯姆
- 奧芬石掌
- 某位年輕火號替班手

## Quest Structure

### Phase 1: 查火號紀錄與視線位置
### Phase 2: 找出誰有機會動手
### Phase 3: 判斷錯號目的
### Phase 4: 公開追責或借假號反釣

## Key Clues
- 更換號序的時段很短
- 假訊號和某支車隊時間一致
- 有人故意把責任推給替班新手

## Recommended Checks
- 觀察 / 軍事知識 / 交涉 / 潛行

## Failure States
- 山線再次誤判
- 真正操手先切斷下一節點

## Success Outcomes
- 穩住白炬遠堡信號系統
- 打開邊堡軍務與密廷外線

## Rewards
- 關堡信任
- 軍線便利

## GM Notes For ChatGPT
- 把張力放在「這不是大爆炸，而是精準的一格偏移」
