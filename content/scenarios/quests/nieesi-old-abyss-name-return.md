# 亡極舊名重現

## 起點
- 某個早該只留在流亡傳說裡的舊名被重新刻回裂壁。
## 核心問題
- 舊名一旦被再次喊出，會連誰的忠誠一起被測出來。
## 任務流程
1. 玩家先從起點事件裡確認第一層異常，判斷這件事是地方問題、制度問題還是有人故意偽裝成意外。
2. 接著必須在至少兩個地點與兩名關鍵人物之間交叉追查，才能看出哪一層資訊被故意藏起來。
3. 中段會發現真正的問題不只在眼前事件，而是牽動了 有人正在利用歷史創傷重組現今權力。
4. 結尾要決定把結果交給誰、保留什麼、犧牲什麼，並承擔後續變化。

## 主要牽涉人物
- 瑟赫爾・裂冠、亡名庫守、裂夢僧
## 可延伸學習與後續
- 可接流亡史、禁名制度與魔人宗教線。
## 成功影響
- 會改變玩家與涅厄斯高層的互信。
