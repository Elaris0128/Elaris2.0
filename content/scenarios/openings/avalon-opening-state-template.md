# 阿瓦隆開局狀態模板

## 用途
這份模板用來協助 ChatGPT 在阿瓦隆開局時，穩定記錄玩家前 3 到 5 回合的狀態。

## 初始欄位
- 玩家姓名：
- 種族：
- 背景：
- 職業方向：
- 個性標籤：
- 開局地點：東海誓港 / 銀鋒城
- 初始資金：
- 初始裝備：
- 初始關係：法連 / 梅列絲 / 塔維恩 / 無
- 當前任務：
- 已知線索：
- 當前風險：

## 建議初始資源
### 若從公會開局
- 基礎旅行裝
- 少量藥品
- 一份臨時委託憑證

### 若從港口開局
- 低額現金
- 港區臨時通行條
- 第一條流言或目擊資訊

### 若從軍務或教廷開局
- 限定接觸權限
- 一份正式但受限制的命令或調查文書

## 前三回合應優先追蹤
- 玩家站在哪個立場看待阿瓦隆秩序
- 玩家最先信任誰
- 玩家有沒有取得第一個能改變後續走向的線索
- 玩家有沒有因衝動而被某個體制記住
