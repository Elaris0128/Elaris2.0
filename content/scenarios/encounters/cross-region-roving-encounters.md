# 跨區流動人物與團體遭遇框架

## 用途
- 這份表用來處理 `不綁定單一地區` 的流動人物與團體遭遇。
- 典型對象包括：冒險者團、跨區商人團、樣本學者、碑文抄錄隊、危材買手。
- 人物索引可對照：
  - [世界流動人物與團體索引](/C:/Users/rabbi/Desktop/Elaris/content/world/07-characters/world-wanderers/README.md)

## 快速使用法
1. 每當隊伍完成一段官道、航段、遠征段、高空接駁或危區推進時，可檢一次。
2. 先看 `基礎遭遇率`。
3. 再加上 `修正值`。
4. 若觸發，依 `場域類型` 選對應團體。

## 基礎遭遇率

| 場域 | 基礎遭遇率 |
|---|---|
| 王國官道 / 大路 / 站道 | `8%` |
| 跨國灰線 / 邊境轉路 | `12%` |
| 港灣 / 泊地 / 河口交易帶 | `14%` |
| 海上長航 / 遠洋補給窗 | `10%` |
| 熱門遺跡 / 遠征季遺構 | `18%` |
| 高空中繼 / 浮島接面 / 空航補站 | `11%` |
| 邊荒 / 極地 / 危區外線 | `7%` |

## 修正值
- 隊伍攜帶高價貨、危材、遺物、樣本：`+4%`
- 當地正值遠征季、商季或開航窗：`+3%`
- 玩家主動打聽委託、找嚮導、找買手：`+5%`
- 當地封航、戒嚴、內亂或近期大屠殺：`-4%`
- 隊伍聲名很高、正在被公開通緝或攜帶醒目旗號：`+3%`

## 常見互動結果
- 接到臨時委託
- 被要求同行一段路
- 被先行探路或先行買手截胡
- 買到或看到半真半假的情報
- 因同一份遺物、樣本、貨物或路線起競爭
- 在危區被迫互助

## 稀有流動人物層
- 若基礎遭遇已觸發，可再檢一次這層，判斷會不會遇到 `更稀有的個體人物`。
- 建議流程：
  1. 先抽一般流動團體。
  2. 若玩家聲名高、攜帶高價貨或正處於大事現場，再檢 `稀有人物層`。

| 類型 | 建議再觸發率 |
|---|---|
| 世界級知名人物 | `3%` |
| 灰線人物 | `7%` |
| 公會流動講師與評議員 | `5%` |

## 稀有人物適用場域

### 世界級知名人物
- 高機率：遺跡封存、遠征季核心地點、世界級拍賣桌、大航窗
- 中機率：高空中繼、大港、大型事故現場
- 低機率：一般官道、普通聚落

### 灰線人物
- 高機率：灰線、暗泊、夜拍、黑市、封航前後
- 中機率：港灣、邊境轉路、沉貨回收區
- 低機率：正式公會窗口、公開廣場

### 公會流動講師與評議員
- 高機率：公會總部外站、遠征考核、封樣爭議、重大事故善後
- 中機率：大港、公會合作城、跨區中繼站
- 低機率：純民間小路
