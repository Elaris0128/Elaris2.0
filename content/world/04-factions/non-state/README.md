# 非國家性質組織索引

## 目的
這個目錄收納 Elaris 中已經具名、可跨地影響世界的非國家性質組織。

它們不直接等於主權機關，但會長期影響：
- 任務
- 執照
- 交易
- 跨國物流
- 探索與回收
- 災後照護
- 灰線流通

## 使用原則
- 世界級骨架先寫在這裡。
- 世界級組織通常需要明寫 `總部 / 地方網或分院 / 入會或認證方式`。
- 區域級與地下組織不一定有正式分部，但至少要寫清楚 `核心據點 / 影響圈或外圍窗口 / 吸收或進場方式`。
- 各國承認、限制與灰線活動差異，搭配 `../non-state-organization-country-relationship-matrix.md` 一起看。
- 各國之後只需要補 `分部樣貌 / 章程差異 / 合法性差異 / 當地窗口人物`。
- 若同一組織有公開面與暗線面，可在同一檔中分開寫，不必拆成互相矛盾的兩份設定。
