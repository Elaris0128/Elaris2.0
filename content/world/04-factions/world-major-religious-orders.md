# 世界級主要宗教組織

## 目的
這份文件整理 Elaris 目前已經明確長成 `世界級宗教制度` 的主要共同體，並把它們分成兩類：
- `可成為國教核心的人類制度宗教`
- `非國教類的世界宗教組織`

這些組織不是要先替神系下定論，而是先回答：
- 哪些信仰已經形成跨地區制度
- 哪些宗教角色不只活在單一國家
- 神不降臨、凡人不能直接見神的前提下，信仰如何仍然影響世界

## 通用前提
- 這些組織都不以「神明親身現身」作為合法性來源。
- 它們的正當性主要來自：
  - 神諭
  - 夢兆
  - 見證結果
  - 儀式延續
  - 聖地回應
  - 長期社會信任
- 同一高位神聖核心，在不同地區可以有不同名字、圖像與教義表層。
- 世界級、跨地、帶有章程與教階的大型宗教組織，通常更容易由 `人類制度文明` 發展出來。
- 其他種族多半保留自己的 `神話故事 / 祖靈敘事 / 地方祭儀 / 生態信仰 / 口傳禁忌`，不一定需要被整理成中央化宗教制度。
- 人類宗教組織可以研究、吸收、詮釋其他種族的信仰，但不代表它們就能真正代表那些信仰。
- 棲裔文明應優先被理解成 `誓王、受名、誓網與棲地共生` 的信仰制度，而不是人類式宗教。

## 一、可成為國教核心的人類制度宗教

### 1. 誓光見證庭
- 性質：世界級高位見證與誓約共同體。
- 主導文明傾向：`強烈人類主導`
- 神聖核心：偏向 `源神 / 見證 / 正統 / 王權正當性 / 誓約神學` 的大型宗教網。
- 組織型態：教團網型 + 國教合作型。
- 主要功能：授誓、繼承見證、出征誓約、停戰見證、重大裁決背書。
- 常見據點：誓堂、王都大庭、邊關誓階、盟誓廣場。
- 常見神職：主祭者、見證者、守誓者、誓錄官。
- 常見服務：婚誓與盟誓見證、繼承正名、誓印認證、重大判決背書。
- 正當性來源：古約、長期見證紀錄、王權與地方共同承認。
- 常見衝突：與世俗法官、地方祖靈信仰、反王權派與偽誓網衝突最深。
- 世界意義：很適合串起制度型國家、王室合法性、軍誓與誓約術學。
- 補充：它很適合成為 `法雷恩人` 這類有流亡、被迫害與正統主張記憶的國教核心。
- 地方化延伸：在瓦雷溫社會中，這類誓約與見證神學常會被 `維拉特教`、軍誓傳統與誓守家系語言吸收，但瓦雷溫本身仍不是單一國教國度。

### 2. 序律編院聯席
- 性質：世界級秩序、條律與神學編註共同體。
- 主導文明傾向：`強烈人類主導`
- 神聖核心：偏向 `律神 / 宇宙秩序 / 可解釋法則 / 教義編纂`。
- 組織型態：教團網型 + 學院化宗教型。
- 主要功能：編註經典、校對教義、制定儀式標準、判定異端與教義偏差。
- 常見據點：編院、經典樓、註解堂、教律庫、誓條評議室。
- 常見神職：典籍解釋者、教義評議者、禁忌管理者、異端審理者。
- 常見服務：教義解讀、儀式規格校正、正本認證、宗教爭議調停。
- 正當性來源：成文經典、註解傳統、跨地教義互認與學術威望。
- 常見衝突：與秘儀圈、地方聖地派、草根祭司與政治權力會互相制衡。
- 世界意義：很適合承接法奧式制度宗教、學院神學、律神術學與異端事件。
- 地方化延伸：在法奧人社會中，它最典型的在地形態就是 `誓智殿系統`；舊有七座主神，後續應優先整理成法奧版本的七座誓智或神學位格，而非另一套獨立國教。

## 二、非國教類的世界宗教組織

### 1. 天幕巡禮會
- 性質：世界級巡禮、夢兆與遠路神諭共同體。
- 主導文明傾向：`人類主導，但跨種族參與度高`
- 核心主張：傾向主張 `萬象起於一位創世主`，其他神聖存在多被視為其回聲、從屬秩序或較晚顯化的神聖層級。
- 神聖核心：偏向 `創世 / 天象 / 遠光 / 夢兆 / 旅途啟示` 的流動信仰。
- 組織型態：巡禮與行旅型。
- 主要功能：解夢、讀兆、巡禮導引、旅途安撫、遠行死者記名。
- 常見據點：巡禮宿站、星台、遠路祭棚、風口觀夜亭。
- 常見神職：巡禮者、夢兆解釋者、路祭者、長夜守燈者。
- 常見服務：旅程祝禱、夢兆判讀、遠行護名、失路者收容、巡禮路線指引。
- 正當性來源：夢錄、路碑記錄、巡禮路與地方靈驗傳說。
- 常見衝突：與地方官署的路權、和以制度為主的國教體系常有摩擦。
- 世界意義：很適合連結旅人文化、創世敘事、星象術學與長路故事。

### 2. 祖燼安魂盟
- 性質：世界級安魂、墓守與亡者紀錄共同體。
- 主導文明傾向：`人類主導，但祖靈文化可跨種族接觸`
- 神聖核心：偏向 `祖靈 / 葬祭 / 記名 / 亡者見證 / 血脈延續`。
- 組織型態：祖靈會型 + 地方優先聯盟型。
- 主要功能：葬祭、安魂、無名者記錄、戰後收屍、祖譜保存、死者通路維護。
- 常見據點：安魂堂、祖墓區、悼名廊、收骨院、無名碑林。
- 常見神職：安魂者、墓守、悼名者、祖譜守錄者、夢驗記錄者。
- 常見服務：葬祭、失蹤者記名、遺骨送返、無名者安置、祖譜見證、亡者異象記錄。
- 正當性來源：長期葬祭信任、祖墓守護權、地方共同體默認與戰後需求。
- 常見衝突：與戰亂政權、遺體走私、禁忌研究圈與偷葬者衝突明顯。
- 世界意義：很適合連結祖靈信仰、災後整理、戰場餘波與家系世界觀。
- 補充：它帶有明顯 `半研究型` 色彩，會保存夢兆、死者殘響與葬祭紀錄，但不等於所有祖靈文化都屬於祖燼安魂盟。

### 3. 林棲息育園
- 性質：世界級生命網、生態守護與棲地保育共同體。
- 主導文明傾向：`人類與林境外緣守護群體主導，非棲裔正統信仰`
- 神聖核心：偏向 `生命網 / 守地存在 / 生態回應 / 棲地誓約`。
- 組織型態：生態共生型 + 保育型。
- 主要功能：封林、止獵、棲地照護、祭水、共鳴穩定、生命循環教育。
- 常見據點：聖樹圈、靜養池、棲地祭台、護林站、共鳴水庭、育植園。
- 常見神職：聖地守護者、生態祭司、止災者、棲地照護者、育植看守。
- 常見服務：生態禁忌解釋、採集許可、受傷生命安置、災後棲地回復、保育區導引。
- 正當性來源：生態穩定成效、地方靈驗、聖地延續與共同體依賴。
- 常見衝突：與過度開採、軍需擴張、遺跡開發與地方飢荒壓力常有對撞。
- 世界意義：很適合對接林境民族、水域圈層與魔物生態治理。
- 補充：它可以研究、模仿或試圖理解瑟堤亞的生命網與棲地秩序，但 `不等同於瑟堤亞棲國自己的信仰制度`。

### 4. 靜燈庇護會
- 性質：世界級庇護、照護與災害收容宗教共同體。
- 主導文明傾向：`人類主導，但對外最開放`
- 神聖核心：偏向 `祝護 / 安撫 / 守夜 / 收容 / 災時不棄人` 的信仰實踐。
- 組織型態：教團網型 + 慈護服務型。
- 主要功能：收容流民、病傷照護、災後宿站、孩童與孤弱安置、守夜慰靈。
- 常見據點：庇護院、夜燈宿站、災民棚、靜室、照護院。
- 常見神職：守夜者、安撫者、照護者、庇護院主持者。
- 常見服務：災時住宿、傷病暫養、孤幼照護、安撫儀式、守夜看顧。
- 正當性來源：長期慈護口碑、地方捐獻、災後實績與弱者依賴。
- 常見衝突：與地方財政、疫病恐慌、偽慈善圈與人口販運勢力衝突。
- 世界意義：很適合作為亂世、流亡、災害與低潮劇情中的安全節點。

### 5. 輪迴續生會
- 性質：世界級轉生、輪迴與來世延續共同體。
- 主導文明傾向：`人類主導，但常吸引邊地與混合文化參與`
- 神聖核心：偏向 `輪迴 / 續生 / 前世殘痕 / 來世準備 / 靈魂循環`。
- 組織型態：教團網型 + 巡禮驗證型。
- 主要功能：轉生徵候判讀、輪迴故事整理、前世記憶見證、臨終引渡與再生祝禱。
- 常見據點：續生堂、回夢室、引渡庭、輪印壁、來生記名所。
- 常見神職：引渡者、回夢記錄者、輪印見證者、續生講者。
- 常見服務：臨終陪行、輪迴教導、前世記憶記錄、幼年異夢判讀、家系續生見證。
- 正當性來源：輪印傳說、前世記憶案例、臨終夢兆與長期引渡紀錄。
- 常見衝突：與只重一次性正統的信仰常有爭論，也會和祖燼安魂盟在「應該被記住」或「應該被放行」之間產生分歧。
- 世界意義：很適合承接前世、轉生、宿命、再會與延續型敘事。

## 不一定會形成世界級宗教組織的信仰類型

### 獸人氏族神話圈
- 性質：不是正式世界級宗教組織，而是大量彼此有相似結構、但不統一成中央教團的氏族神話與祭儀傳承。
- 常見核心：祖靈、戰祖、狩獵神話、骨器、戰歌、墓地、部族誓印。
- 維持方式：長者口傳、戰歌者、墓守、巫祭、成年試煉與氏族故事。
- 常見特徵：每個部族都有自己的版本，不追求單一正典。
- 與世界組織的關係：可能在安魂、停戰、共同葬祭時和 `祖燼安魂盟` 接觸，但不會因此被完全吸納。

### 棲裔誓王共生制
- 性質：不是人類式世界級宗教組織，而是棲裔文明自己的信仰與秩序制度。
- 常見核心：誓王、受名、誓網、棲地共生、自然循環、群體記憶。
- 維持方式：賜名、見證、誓階、語孢紀錄、棲地照護與群體承接。
- 常見特徵：更像文明秩序與生命觀本身，而不是把信仰單獨切成一個宗教機構。
- 與世界組織的關係：可被外來宗教觀察、誤解、借題發揮或合作，但不應被視為任何人類教團的分支。

### 魔人禁約神話譜
- 性質：以禁忌、契約、災厄記憶與高代價神話維持的族群信仰傳承。
- 常見核心：禁約、毀滅性代價、墮落者故事、深層契印、極端試煉。
- 維持方式：災厄傳說、家系戒律、危險儀式與禁名敘事。
- 常見特徵：不喜歡被外來教義簡化為正邪二分。

### 森馳裔林徑傳承
- 性質：以林境、奔行地脈與古路記憶維持的地方神話與共生祭儀。
- 常見核心：林地律動、追跡祖靈、古樹見證、獵與不獵的界線。
- 維持方式：林歌、口傳路譜、守徑者傳承、季節性共生儀式。
- 常見特徵：信仰直接綁在棲地與路徑上，很難被中央教團完整收編。

### 矮人祖匠火脈傳承
- 性質：山體、鍛火、祖匠與火脈相連的工藝信仰傳承。
- 常見核心：祖匠、火脈、熔爐、成品見證、礦山禁忌。
- 維持方式：工坊誓詞、鍛成儀式、祖匠譜系與火室祭具。
- 常見特徵：更像工藝與生命秩序一體，不一定需要跨地宗教教階。

### 地精聚落小神與匠火習俗
- 性質：聚落型、工坊型的小規模實用信仰。
- 常見核心：灶火、攤位、交易順利、手藝守護、聚落小神。
- 維持方式：工坊供位、攤販口彩、節期交換儀式、地方守護故事。
- 常見特徵：極度在地化，常常一個聚落就有一套自己的說法。

### 海人潮裔回潮傳統
- 性質：潮汐、海流、深水記憶與形態輪轉的海洋信仰傳統。
- 常見核心：回潮、深海召喚、海墓、航向、潮裔輪轉。
- 維持方式：潮歌、海葬、船首祭、航季儀式與海流口傳。
- 常見特徵：更依賴海上共同體與航路經驗，而非陸上教廷。

### 精靈古誓星樹傳承
- 性質：古老高位神話、誓約森林與星樹記憶交織的長時信仰傳承。
- 常見核心：古誓、星樹、長夜記憶、古老領域、誓名。
- 維持方式：長年儀式、古語典籍、守林者與高位祭儀。
- 常見特徵：雖可形成高學理神學，但未必願意變成人類式的跨國教廷。

### 翼人高穹兆詩傳承
- 性質：天空、風路、視界與預兆詩歌構成的高空信仰傳承。
- 常見核心：風路、光翼、視界、雲海徵兆、高穹回聲。
- 維持方式：兆詩、飛巡儀式、高地祭壇與觀風者傳承。
- 常見特徵：重視預兆與詩性詮釋，不一定接受低地制度教義。

### 巴布窟族洞靈回音祭系
- 性質：洞域、回音、地下守地存在與深層禁忌構成的地底祭系。
- 常見核心：洞靈、回音、封坑、深層見證、地下避禍。
- 維持方式：洞口祭、回音辨路、礦壁刻記與禁層故事。
- 常見特徵：高度地方化，信仰與生存幾乎無法切開。

## 世界級宗教組織之間的常見關係
- `誓光見證庭` 與 `序律編院聯席`：常合作，也常為解釋權競爭。
- `天幕巡禮會` 與兩大國教核心：常爭論創世主、正統與啟示該由誰解釋。
- `祖燼安魂盟` 與 `靜燈庇護會`：在災後最常聯手。
- `祖燼安魂盟` 與 `輪迴續生會`：常爭論亡者應被牢記、安置，還是應被放行、續生。
- `林棲息育園` 與多數主權體系：在採集、開發與封林問題上長期拉扯。
- `林棲息育園` 與棲裔誓王共生制：可合作，也常因外部詮釋過度而被警惕。

## 使用原則
- 目前明確形成世界級制度的宗教共同體，先分成 `兩大國教核心型` 與 `五大非國教型`。
- 其他種族與人類邊地文化，可以強烈信仰神聖存在，但不必強行長成世界級宗教組織。
- 後面正式建立神系時，可讓不同高位神、律神或地方神聖存在分別掛到這些共同體上。
- 同一國家可以同時承認多個宗教組織，也可以只正式承認其中一部分。
- 若後續展開個別組織主檔，建議搭配 `organization-card-framework.md` 與 `religious-organization-framework.md` 一起使用。
- 若要看這些宗教各自的正式神話版本，應搭配 `religions/README.md` 一起使用。
- 若要看這些宗教的詳細組織卡、教階、聖地與跨宗教關係，也應搭配 `religions/README.md` 一起使用。
- 若要看非人類種族與邊地族群的正式神話傳承名稱，應搭配 `../05-races/racial-myth-and-faith-integration-map.md` 一起使用。
