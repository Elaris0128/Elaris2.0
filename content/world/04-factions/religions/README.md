# 宗教神話版本索引

## 目的
這個目錄收納 Elaris 主要宗教的 `正式神話版本`。

這裡處理的不是：
- 世界真實層的神系骨架
- 各神的完整個體卡

這裡處理的是：
- 各宗教如何理解 `萬初者` 與 `七大高位神`
- 各宗教如何講述創世、權柄、凡人與神聖之間的關係
- 宗教之間彼此在哪些地方會產生正統衝突

## 使用原則
- 先用 `../../01-foundation/creator-and-high-gods.md` 確認通用神名。
- 再用這裡的檔案看各宗教怎麼講述祂們。
- 後續再往下展開各種族神話、律神、昇格神與守地神。
- 若要看各宗教更常接到哪些下位神，搭配 `religion-lower-deity-matrix.md` 一起使用。
