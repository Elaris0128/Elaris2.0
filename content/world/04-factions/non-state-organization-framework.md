# 非國家性質組織通用框架

## Purpose
這份文件用來先把 Elaris 中 `不屬於主權機關，但足以跨地影響世界` 的組織類型固定下來。

它回答的是：
- 這些組織如何在不同國家之間存在
- 它們如何不靠主權本身仍取得權威
- 它們如何成為玩家真正常接觸的世界入口

## 定義
非國家性質組織指的是：
- 不直接等於王室、議會、官署、國家軍團或地方統治機關
- 但能持續提供 `資格 / 服務 / 交易 / 調度 / 保護 / 壟斷 / 仲介 / 研究 / 救援 / 探索`
- 並在多個城市、地區甚至多國之間留下制度痕跡

這類組織可以：
- 被國家承認
- 和國家合作
- 被國家限制
- 在灰色地帶自行存活
- 被不同國家用完全不同方式對待

## 為什麼要先做這一層
- 國家包回答的是 `政治秩序`
- 非國家性質組織回答的是 `日常運作`
- 玩家更常透過它們接到：
  - 執照
  - 任務
  - 補給
  - 通行
  - 跨國人脈
  - 專業升階
  - 灰線交易
- 如果不先定這一層，後續國家內容很容易只剩官方機構，世界會少掉流動性

## 主要類型

### 1. 專業公會
- 以職業資格、技術標準、作品認證與共同市場維持存在。
- 例：鍛造師、藥師、誓紋技師、靈廚。

### 2. 學術與研究共同體
- 以知識、樣本、檔案、評議與師承維持存在。
- 例：魔法師公會、靈息學會、檔案與觀測聯席。

### 3. 商業與物流共同體
- 以貨運、票據、押運、倉儲、互保與價格仲介維持存在。
- 例：中立商會、航運團、關口會館、地區交易聯席。

### 4. 探索與回收共同體
- 以測繪、回收、危險標記、失蹤搜救、遺跡探勘與前哨維持存在。
- 例：探索公會、回收隊、測繪團。

### 5. 照護與庇護共同體
- 以醫援、安置、庇護、送亡、災後照護與弱勢收容維持存在。
- 例：藥師公會、靜燈庇護會的外圍服務網、巡行醫帳。

### 6. 生態與育契共同體
- 以保育、馴養、棲地協調、伴行生命照護與資源守護維持存在。
- 例：誓獸育契公會、林境照護共同體、封林巡護網。

### 7. 契約護行與自由武裝共同體
- 不等於國家軍隊，但會提供護行、護商、地帶巡防、特約鎮守與武力仲介。
- 例：護商團、自由守路團、山道護契隊。

### 8. 文化與記錄共同體
- 以口傳、書錄、碑刻、見證、故事收集、地名保存與禮制維持存在。
- 例：記錄人聯席、巡唱者、碑師圈、旅路見證人。

### 9. 灰線與地下共同體
- 以違禁物流、轉手、洗貨、秘密委託、情報轉接與非法保護維持存在。
- 例：走私網、黑市中介、祕密運線。

### 10. 非國教型宗教共同體
- 不掌握國家主權，但能跨國建立巡禮、庇護、安魂、續生或保育網。
- 這類雖屬非國家性質組織，但宗教細節應交由 `religious-organization-framework.md` 處理。

## 非國家性質組織最重要的六個問題
- 它靠什麼在沒有主權的情況下仍被需要。
- 它跨地活動時靠的是執照、互保、人情、武力還是名望。
- 它的據點是公開分部、特許窗口、寄宿設施還是暗線節點。
- 它和國家碰撞時，誰會先讓步。
- 它在戰亂、災害、封關或失衡事故時，是否仍能運作。
- 玩家若不是正式成員，能否透過付費、合作、聲望或特許臨時取得其資源。

## 承認與存活模式
- `章程特許型`：持有多國承認的正式章程或執照。
- `互認網型`：不同地區的分會互相承認身份與票據。
- `契約寄宿型`：借用當地勢力名義存在，靠契約與人情站穩。
- `市場自然型`：因實際需求與壟斷技術而被默認存在。
- `灰區默許型`：大家知道它存在，但明面不承認。
- `地下滲透型`：沒有正式承認，只靠節點與暗線維持。

## 和國家的常見關係
- `合作外包`
- `受許可監管`
- `互相依賴`
- `表面服從、實際自主`
- `灰色容忍`
- `競爭與壓制`
- `敵對與驅逐`

## 組織細節應優先寫的欄位
- 正式名稱與常用稱呼
- 它不是國家，但為什麼大家離不開它
- 活動範圍與據點形式
- 如何認證成員
- 如何收費、抽成、養活自己
- 如何和主權、宗教、地方傳統協商
- 平時提供什麼，危機時又會做什麼
- 玩家最常從哪個窗口接觸到它

## 世界級與區域級的最低清晰標準
- `世界級組織`：至少應明寫 `總部 / 地方網或分院形式 / 入會或認證方式`。
- `區域級組織`：至少應明寫 `核心據點 / 影響圈、外圍窗口或地方接點 / 吸收或進場方式`。
- `地下組織`：若沒有正式分部，也應說明它靠什麼節點存活，例如暗橋、撤離線、交易窗、短泊點或安全屋。
- 這個標準的目的是讓不同尺度的組織都夠清楚，但不會被迫長得一模一樣。

## 對玩家的意義
非國家性質組織通常比國家機關更常成為玩家入口，因為它們更容易提供：
- 第一份工作
- 基礎身份
- 專業認證
- 補給與收購
- 路線情報
- 災害救援
- 灰線交易
- 跨國任務線

## 使用原則
- 在擴寫國家細節前，先決定該地最重要的非國家性質組織是哪幾種。
- 一個國家不需要什麼都自己承包，很多功能應由非國家性質組織補上。
- 世界級與區域級組織應先有共通骨架，再讓各國寫地區章程差異。
- 之後若要做具名組織，可搭配 `organization-card-framework.md` 與 `non-state-organization-archetype-library.md` 一起使用。
