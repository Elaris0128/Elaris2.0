# 職業文化名字池

這份名字池用來快速替 `職業 NPC` 命名。  
它不是嚴格姓氏學，而是偏 `好用、像本地人、能快速生成` 的工作池。

使用原則：

- 人名維持 `1-4 字混用`
- 不同文化的人名長度不必一致
- 同一份職業在不同文化裡，名字氣味應該不同

## 快速命名公式

如果你要臨場快速命名，可以用：

`[文化字根] + [職業字根] + [個人氣質字]`

例子：

- 翼人判風手：`風 + 岫 + 衡 = 風岫衡`
- 幽語送魂師：`潮 + 骨 + 笙 = 潮骨笙`
- 星盞勘界手：`燼 + 界 + 修 = 燼界修`
- 黯鉚估價手：`票 + 衡 + 岑 = 票衡岑`

## 最後原則

- 重要 NPC 可以更完整、較穩重。
- 常駐背景 NPC 可以偏綽號感。
- 灰線人物的名字可以更像外號。
- 公會、官署、王庭系統的人名通常比邊海與外島更正式。
