# 職業 NPC 模板庫

這份模板庫是 `可直接套用的職業人物雛形`。  
如果你不想從零捏 NPC，可以先從這裡抽一個，再換名字、地區和任務。

## 港務、法務與秩序

### 卡在程序裡的簿手

- 適用職業：
  `文簿官 / 航權司記 / 泊庭核名手 / 稅簽官`
- 表面印象：
  禮貌、冷靜、看起來不近人情。
- 實際困境：
  他知道程序有洞，但一旦承認，整條鏈都會追到自己頭上。
- 適合任務：
  找假牌、補失頁、查誰動過簿冊。

### 快撐不住的控場者

- 適用職業：
  `港泊管事 / 港巡兵 / 關卡查驗官 / 岬哨守燈手`
- 表面印象：
  很兇、語速快、動不動就叫人後退。
- 實際困境：
  現場人手不夠，而且他其實已經連值很多輪。
- 適合任務：
  協助維持秩序、補空缺、去截某個快跑掉的人。

### 把規矩當最後防線的人

- 適用職業：
  `司燈者 / 守墓者 / 祖物看守人 / 長記守庫者`
- 表面印象：
  講話很慢、不喜歡被打斷。
- 實際困境：
  他知道只要讓一步，後面就會連鎖崩掉。
- 適合任務：
  守住禁線、查祖物、見證或拒絕某次請求。

## 海運、高空與遠征

### 知道路但不想帶路的人

- 適用職業：
  `引水人 / 路線踏勘手 / 浮流判風手 / 深窟實測手`
- 表面印象：
  寡言、看地比看人多。
- 實際困境：
  他知道這條路能走，但也知道代價是什麼。
- 適合任務：
  借道、測路、撤退、封線。

### 裝備比隊友可靠的老手

- 適用職業：
  `航具維養工 / 索橋保線手 / 空艇航手 / 船長`
- 表面印象：
  會先看器具，再看人。
- 實際困境：
  他懷疑事故不是自然損壞，而是人為。
- 適合任務：
  查斷裂、搶修、改裝、護送。

### 第一個看到異常的人

- 適用職業：
  `巡翼哨手 / 岬哨守燈手 / 巡空護航者 / 港巡兵`
- 表面印象：
  繃得很緊，眼睛總在看邊角。
- 實際困境：
  他看到了不對勁，但還沒有足夠證據說服上級。
- 適合任務：
  警報、追跡、試探、救失聯者。

### 撐住整支隊伍的人

- 適用職業：
  `遠征後勤員 / 旅灶長 / 外緣採樣手 / 打撈手`
- 表面印象：
  手上永遠拿著清單或工具。
- 實際困境：
  他已經算過，現有資源只能救一半的人。
- 適合任務：
  補給、取樣、搶回物資、回收。

## 學術、樣本與古文明

### 覺得大家都太草率的研究者

- 適用職業：
  `學者 / 標本師 / 古文讀解手 / 抄錄員`
- 表面印象：
  說話很精確，有時候讓人煩。
- 實際困境：
  他掌握的線索很重要，但需要有人替他跑現場。
- 適合任務：
  對讀、查偽、補證、把某樣東西帶回來。

### 手上握著關鍵真相的人

- 適用職業：
  `古文讀解手 / 學者 / 長記守庫者`
- 表面印象：
  看起來只是做文書。
- 實際困境：
  他知道某段文字會改變整個事件解讀，但公開會出大事。
- 適合任務：
  護送文稿、驗真、決定是否公開。

### 先看樣本再看人的人

- 適用職業：
  `標本師 / 外緣採樣手 / 深窟實測手 / 製藥師`
- 表面印象：
  很專業，也很容易讓人覺得無情。
- 實際困境：
  眼前的樣本可能比眼前的人更重要。
- 適合任務：
  搶樣、封樣、辨認錯樣、處理污染。

## 藥療、照護與料理

### 心很軟但嘴很硬的人

- 適用職業：
  `藥師 / 療傷手 / 隔棚照護員 / 送魂師`
- 表面印象：
  說話不客氣，手卻很穩。
- 實際困境：
  他想幫人，但知道多救一個就可能害整棚。
- 適合任務：
  救人、分流、送行、守住潔淨序。

### 需要冒險材料的人

- 適用職業：
  `製藥師 / 靈廚 / 旅灶長 / 供饌手`
- 表面印象：
  很會評估材料，對味道和狀態敏感。
- 實際困境：
  他需要的關鍵主材合法取得幾乎不可能。
- 適合任務：
  取材、試做、追查被換掉的材料、保住某批成品。

### 比祭司更懂儀式後勤的人

- 適用職業：
  `供饌手 / 旅灶長 / 司燈者 / 抄錄員`
- 表面印象：
  看起來像幫手，其實是流程真正的支柱。
- 實際困境：
  主事者出錯時，最後還是會落到他頭上。
- 適合任務：
  補供物、重排流程、在儀式崩盤前補洞。

## 商業、灰線與地下情報

### 看似普通其實很會算的人

- 適用職業：
  `商人 / 估價手 / 旅店掌櫃 / 掮客`
- 表面印象：
  和氣、實際、很懂做生意。
- 實際困境：
  他有一筆帳不能被查到。
- 適合任務：
  運貨、追帳、估值、找失物。

### 只說半句話的人

- 適用職業：
  `情報掮客 / 黑鍛匠 / 假旗手 / 灰線借名人`
- 表面印象：
  話不多，但總能講到重點。
- 實際困境：
  他知道太多，也知道誰不能得罪。
- 適合任務：
  查人、借路、換名、追黑貨。

### 看起來像店家其實是節點的人

- 適用職業：
  `旅店掌櫃 / 商人 / 估價手 / 巡演藝人`
- 表面印象：
  好相處，像在招呼客人。
- 實際困境：
  他其實在替好幾方保東西、保人或保消息。
- 適合任務：
  接頭、藏人、交換、拖時間。

## 生產、採收與地方生活

### 對季節比對人更敏感的人

- 適用職業：
  `漁戶 / 採藥人 / 狩窗手 / 果林看守人`
- 表面印象：
  常盯著天色、潮色、果色或草勢。
- 實際困境：
  這一季的窗口出了不該有的偏差。
- 適合任務：
  採收、找失蹤者、查偷採、判窗口。

### 被地方壓力夾住的勞動者

- 適用職業：
  `礦工 / 鍛匠 / 珠骨工 / 裝具匠`
- 表面印象：
  很實在，不喜歡空話。
- 實際困境：
  上面趕工、下面缺料，事故已經快壓不住。
- 適合任務：
  查缺料、救礦坑、補件、追壞掉的來源。

## 最快用法

如果你想立刻抽一個 NPC，可以這樣用：

1. 從這份模板庫挑一型。
2. 去 [職業文化名字池](profession-cultural-name-pools.md) 換名字。
3. 去 [職業語氣、裝具與外觀提示矩陣](profession-speech-gear-and-visual-cue-matrix.md) 補口氣和外觀。
4. 去 [職業人物模板框架](profession-character-template-framework.md) 補秘密與壓力。

這樣很快就能生出 `不是只有工作名稱` 的角色。
