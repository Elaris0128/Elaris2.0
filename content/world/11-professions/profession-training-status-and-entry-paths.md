# 職業訓練、入行方式與職級規則

Elaris 的職業不是只有「會不會做」，還包含 `誰承認你能做`。

## 常見入行方式

### 1. 家傳

最常見於：

- 海人守域職
- 諾魯因送魂與潮療職
- 小海島採果、看汀、曬棚職
- 船團長期維養工
- 地方祭器、骨串、珠工

特徵：

- 起步早
- 規矩多
- 不一定有正式文憑
- 在地方上反而比公會資格更有用

### 2. 師徒

最常見於：

- 鍛匠
- 黑鍛匠
- 靈廚
- 藥師
- 旅灶長
- 引水人

特徵：

- 以跟班、看手、借工開始
- 常用 `出師件 / 成手件 / 初次獨立接案` 當關卡
- 師門名聲會直接影響接案能力

### 3. 公會考核

最常見於：

- 藥師公會
- 冒險者公會
- 探索公會
- 魔法師公會
- 誓紋技師公會
- 靈廚公會

特徵：

- 有分級、掛名、暫牌、正牌
- 可以跨區通用
- 常帶有 `可承接項目上限`

### 4. 國家授印或地方任命

最常見於：

- 城衛兵
- 港巡兵
- 稅簽官
- 文簿官
- 港泊管事
- 哨站守望手

特徵：

- 權限來自政權
- 失職可能直接失印或被降格
- 常和地方曆法、印章、制服、誓名綁定

### 5. 宗教誓名或聖所承認

最常見於：

- 祭司
- 司燈者
- 送魂師
- 祖物看守人
- 供饌手

特徵：

- 合法性來自神殿或祭序
- 常用潔淨度、誓名、守戒作為資格
- 可能不承認外地同類職的等價資格

### 6. 航團、守域圈或戰時吸收

最常見於：

- 巡空護航者
- 封路兵
- 深巡隔潮手
- 船團護運手
- 外洋隔離守

特徵：

- 常在危機期擴編
- 實戰與存活率比證書更重要
- 很容易形成地方色彩濃厚的副職稱

### 7. 灰線吸收

最常見於：

- 掮客
- 灰簿手
- 偽旗抹號手
- 黑頁買手
- 洗印手

特徵：

- 幾乎不會公開招生
- 以借工、試手、傳口令、處理第一筆高風險貨物作為入門
- 最重視保密與可替換性

## 常見職級

| 職級 | 含義 | 常見用法 |
| --- | --- | --- |
| 見習 | 還在跟班或輪值學習 | 公會、工坊、船團 |
| 掛名 | 已可被機構承認，但權限有限 | 公會、學院 |
| 正職 | 可獨立接一般工作 | 多數機構 |
| 主手 | 能帶學徒或領小組 | 工坊、藥棚、靈廚 |
| 執牌 | 持正式牌照，可承接高風險或高等級項目 | 公會 |
| 講師 | 可授課、判定、評分 | 學院、公會 |
| 評議 | 有裁定與修規權限 | 公會、聯席 |
| 守域職 | 只在特定區域有效的承認職位 | 海人、翼人、諾魯因 |
| 灰線自由手 | 沒有正牌，但在灰線內被承認 | 黑市、海盜、掮客圈 |

## 玩家可直接使用的對讀方式

玩家或角色在建構職業時，可以先選 `通用職業`，再從下列三項補足地方味：

1. `文化職稱`
2. `入行方式`
3. `制度掛靠`

例子：

- 通用職業：藥師
- 文化職稱：潮骨療手
- 入行方式：家傳 + 師徒
- 制度掛靠：幽語海岬哨域與諾魯因骨鏈

例子：

- 通用職業：引水人
- 文化職稱：熱泉路引者
- 入行方式：守域圈內部承認
- 制度掛靠：海人熾潮之脈守域圈

例子：

- 通用職業：鍛匠
- 文化職稱：黑鍛匠
- 入行方式：師徒 + 灰線吸收
- 制度掛靠：黯鉚列島黑鍛工坊

## 判斷一個文化稱呼是不是獨立職業

可用三個問題判斷：

1. `他是否做的是另一種工作，而不只是另一種叫法？`
2. `他是否有不同的入行方式或合法性？`
3. `他是否在地方上被賦予額外義務？`

只要有兩項以上成立，就值得在世界觀裡把它當成 `文化職業` 專門記錄。
