# 職業索引

這一層用來整理 Elaris 的世界通用職業、文化職業稱呼、入行方式與社會位置。

職業層的讀法分成三層：

1. `世界通用職業`
   用來回答「這個人在世界機制上是做什麼的」。
2. `文化職業稱呼`
   用來回答「在不同地區、族群、制度裡，這份工作被怎麼理解、怎麼命名」。
3. `地方職守與機構職位`
   用來回答「這個人是不是掛在某個國家、公會、神殿、守域圈、灰線團體底下」。

使用原則：

- 一個世界通用職業，可以對應多個文化稱呼。
- 一個文化稱呼，不一定只做一件事，常常會兼任地方義務。
- 世界級公會職業通常有標準考核。
- 區域型、家傳型、灰線型職業更常見 `家系傳承 / 師徒吸收 / 誓名掛靠 / 臨時借工`。
