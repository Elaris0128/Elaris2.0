# 物品標記、合法性與權限標籤

## Purpose
- 這份文件定義 `物品在 Elaris 世界裡應該怎麼被標籤`。
- 之後不管是寫王器、海怪材料、公會器具、遺物或灰線貨，都可以共用這套標記。

## 稀有度
- `常見`
  - 一般聚落、市集、港口都可能看到。
- `少見`
  - 要到專門商人、公會或特定地區才容易找到。
- `珍稀`
  - 通常要有人脈、委託或特定時機才拿得到。
- `名物`
  - 在一個地區或職業圈有明確名號。
- `傳說`
  - 世界級高辨識器物或高位材料。
- `唯一`
  - 目前世界上只有一件，或即使有複件也不被承認。

## 合法性
- `公開`
  - 可正常持有與流通。
- `限持`
  - 可以持有，但需要資格、身分或用途說明。
- `需封樣`
  - 沒有封樣紀錄，就會失去法律價值或被直接沒收。
- `灰線`
  - 不完全非法，但出處、用途或流向不乾淨。
- `走私`
  - 明確繞開稅、檢、封樣或守域規則。
- `禁持`
  - 原則上不允許私持，通常會被查扣、封存或就地毀棄。

## 權限層級
- `通用`
  - 一般人可持有。
- `職業限定`
  - 需要特定工作身份，例如藥師、鍛匠、航手。
- `公會限定`
  - 必須是對應公會成員或授權持有人。
- `宗教限定`
  - 只能在祭司、守墓人、送魂隊等體系內使用。
- `血脈限定`
  - 家系或種族認主物，外人可見不可用。
- `認主限定`
  - 器物本身有識主或綁定條件。

## 追溯性
- `可替換`
  - 丟了就能補發或再買。
- `可記錄`
  - 有編號或發放紀錄。
- `需回收`
  - 持有者失格、死亡或職務終止後必須回收。
- `遺失追責`
  - 遺失本身就是事故，會牽出罰則與責任鏈。
- `不可外借`
  - 不得跨人轉持。

## 危險標記
- `無危`
- `輕危`
  - 可能割傷、刺激或小範圍過敏。
- `中危`
  - 需有保護程序，否則可能造成中毒、污染、誤燃。
- `高危`
  - 要專門人員、專門場域處理。
- `禁帶級`
  - 一旦失控可能引發整個地點封鎖。

## 攜行尺度
- `隨身`
- `長件`
- `箱件`
- `載具級`
- `場域級`

## 推薦簡寫格式
- 可寫成：
  - `主分類 / 稀有度 / 合法性 / 權限 / 危險標記`
- 例：
  - `職業器具 / 珍稀 / 需封樣 / 公會限定 / 中危`
  - `古器 / 傳說 / 禁持 / 認主限定 / 高危`
  - `素材 / 少見 / 灰線 / 通用 / 輕危`

## 常見搭配
- 王權與政務重器：
  - `權器 / 名物或傳說 / 限持 / 認主限定 / 遺失追責`
- 海怪活性腺體：
  - `素材 / 珍稀 / 需封樣 / 職業限定 / 高危`
- 黑市仿造公會器具：
  - `職業器具 / 少見 / 灰線或走私 / 偽權限 / 中危`
- 古文明殘器：
  - `古器 / 名物或傳說 / 限持或禁持 / 可記錄 / 中危到高危`
