## 常見成品鏈

### 陸地武裝鏈
- 喉豹牙 -> 短刃
- 裂蠍尾刺 -> 毒針
- 雪魈爪 -> 冰地鉤具
- 冰熊骨 -> 重型骨柄與營釘

### 高空器具鏈
- 飛龍翼膜 -> 高風襯膜
- 王鷹王羽 -> 號令羽與儀式箭羽
- 風鉤喙 -> 崖行鉤具
- 裂風龍風囊 -> 壓差試具

### 深淵危材鏈
- 魅粉 -> 迷香對照材
- 噬香腺 -> 誘引危材
- 寄形膜 -> 禁術研究件
- 異血結晶 -> 高危封樣樣本

### 神獸殘件鏈
- 落鱗 / 燼羽 / 角屑 / 甲紋片 / 風鬃
  -> 權器、祭器、守域補片、家系認主物

## 最容易引發劇情的類型
- `來源不清的高位殘件`
- `需要封樣但已離體過久的活性器官`
- `本來應屬守域或宗教秩序，卻出現在黑市的材料`
- `能被錯認成神獸殘件的仿品`
