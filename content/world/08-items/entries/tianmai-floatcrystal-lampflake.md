# 天脈浮晶燈片

- 主分類：貨物、奢侈品與交易品
- 副標籤：地方特產 / 易裂 / 高空貨 / 半工藝件
- 常見來源：天脈島弧浮晶棚、樣本商與泊庭轉售桌
- 外觀：
  薄如魚鱗的半透明晶片，邊緣帶淡青、乳白或淺金暈光，移動時會像水面反日般閃一下。
- 常見持有者：
  高空商人、燈匠、收藏家、外務禮品採購手。
- 主要用途：
  鑲燈、做信標、磨成碎晶粉入儀式火盆，也常被做成可拆換的高價燈面。
- 估值帶：
  中高價，完整無裂又光澤穩定的會進到高價禮貨層。
- 合法性與限制：
  可公開流通，但高純度樣片常需附來源簿頁。
- 辨識點：
  真片在暗處有延遲回光，假片多半只有單面反亮，沒有內層浮光。
- 劇情用途：
  作為高空使節禮、被黑市偷換成低純度仿片，或牽出一條樣片來源簿造假案。
