# 回燼祭風台核心件

- 主分類：遺物、古器與命名寶物
- 副標籤：古文明遺物 / 需封樣 / 高保全 / 高仿風險
- 常見來源：星盞群島回燼盞朝遺構核心、封庫與高級保全庫
- 外觀：
  多呈多角不規則石心，表面有白燼細線與古老風向刻紋，靠近火光時會出現短暫逆亮。
- 常見持有者：
  遺物保全者、遠征總監、極少數高權學營。
- 主要用途：
  判定祭風台真偽、重建古儀式結構、校正白燼區帶風向記錄。
- 估值帶：
  不宜公開估價；黑市若真出現，通常是世界級高價。
- 合法性與限制：
  需封樣、需保全簿與雙重授權，不可私藏轉售。
- 辨識點：
  真件的燼紋會在不同風向下有細微位移，仿件多半只是死紋雕刻。
- 劇情用途：
  成為多方追逐的核心遺物、引發保全派與開掘派衝突，或作為重啟古儀式的關鍵部件。
