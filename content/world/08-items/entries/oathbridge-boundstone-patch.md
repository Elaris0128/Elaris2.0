# 誓橋界石補片

- 主分類：遺物、古器與命名寶物
- 副標籤：高空古器 / 權限物 / 可修補橋界 / 需授權
- 常見來源：曦環舊橋遺構、斷橋背灣打撈件、誓橋維護庫
- 外觀：
  呈灰白石片或細長楔片，表面帶風紋與微光裂縫，邊緣有人工裁補痕。
- 常見持有者：
  高空橋守、誓紋技師、少數橋界研究者。
- 主要用途：
  用來補穩舊橋界石、暫時續接風橋、校正古橋殘存的權限紋。
- 估值帶：
  中高到極高價，依是否仍保留可用橋紋而定。
- 合法性與限制：
  屬半保全級物件，通常需橋界授權方能持有與使用。
- 辨識點：
  真補片靠近活橋界時會有細震與微鳴，假片通常只有外觀沒有回應。
- 劇情用途：
  修橋、假修橋、爭奪橋界控制權，或追查哪塊補片是從封存維護庫失竊的。
