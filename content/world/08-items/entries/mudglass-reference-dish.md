# 泥璃對照盤

- 主分類：職業、公會與專業器具
- 副標籤：藥師器具 / 封樣比對 / 可攜 / 中價專業物
- 常見來源：霧羽列島藥師公會總部、對照棚、封樣庫
- 外觀：
  混合泥璃與薄陶做成的淺盤，盤內分成多格，邊緣刻有對照序與水位記號。
- 常見持有者：
  藥師、公會學徒、巡採手、覆驗人。
- 主要用途：
  比對樣本色澤、沉積、黏性與折光，常用來抓出掉包藥材與污染樣本。
- 估值帶：
  中價。
- 合法性與限制：
  可公開持有，但正式覆驗盤通常要附序號。
- 辨識點：
  真盤在濕度變化下格線不糊，廉價仿品往往遇潮就暈開。
- 劇情用途：
  找出同名異樣本、追查病潮污染來源，或成為學徒晉級的必備器具。
