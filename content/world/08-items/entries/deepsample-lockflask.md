# 深樣鎖口瓶

- 主分類：行旅、航海與探索裝具
- 副標籤：探索裝具 / 封樣輔助 / 可攜危樣 / 需報備
- 常見來源：探索公會、藥師公會、靈穹洋深採樣線
- 外觀：
  瓶身偏細長，口部有多層鎖口環與軟襯墊，外層常加一圈保護框。
- 常見持有者：
  採樣隊、海窟研究者、遠征記錄官。
- 主要用途：
  裝載深層水樣、危材液樣或會揮散的特殊樣本。
- 估值帶：
  中高價器具。
- 合法性與限制：
  可持有，但裝載高危樣本時需附簿記或封樣資格。
- 辨識點：
  真瓶的鎖口會有分層回扣，仿瓶通常只做外圈樣子，實際氣密差很多。
- 劇情用途：
  深採樣任務、事故調查、違規帶出危樣時的重要證物。
