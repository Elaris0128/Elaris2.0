# 偽鑑印箱

- 主分類：黑市仿製、失格品與偽封樣器物
- 副標籤：灰線器物 / 高仿風險 / 需多人操作 / 限灰線
- 常見來源：黯鉚列島黑鍛圈、天脈夜線市場、港邊偽印匠
- 外觀：
  外觀看似普通驗印箱，內部卻藏有多層可替換印模、殘印粉與快乾假封層。
- 常見持有者：
  偽印匠、灰線中間人、冒名委託接手。
- 主要用途：
  快速做出臨時驗印、替換序號、模糊真印邊線。
- 估值帶：
  高價灰線器物。
- 合法性與限制：
  禁持。
- 辨識點：
  真驗印箱偏重且結構單純，偽箱常有異常夾層與不必要的小抽槽。
- 劇情用途：
  偽造港檢、冒名提貨、栽贓某公會失竊器具，或在追查時牽出真正的印模來源。
