# 星盞白燼果脯

- 主分類：藥食、消耗與一次性物件
- 副標籤：地方特產 / 季節貨 / 可公開流通 / 易被仿造
- 常見來源：星盞外緣小海島採果戶、遠征補給桌與岸頭交易
- 外觀：
  薄片果脯呈淺白金色，邊緣帶微灰燼紋，掰開時會有細小白粉落下，但入口是清甜而不是焦苦。
- 常見持有者：
  外緣採集者、遠征隊、港邊買辦、藥食商。
- 主要用途：
  快速補糖、壓暈船、做長途口糧，也能入藥做輕度安神與清口配方。
- 估值帶：
  低到中價；純白熟果做的高級批次則會上到中高價。
- 合法性與限制：
  可公開流通，但若聲稱出自淨區白燼果林，通常需附採收來源。
- 辨識點：
  真品表面粉層細而不黏，仿品常灑鹽灰冒充，入口會明顯發苦。
- 劇情用途：
  區分小海島真採集貨與盜採淨區貨，或在遠征糧短缺時成為被爭奪的小救命物。
