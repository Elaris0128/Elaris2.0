# 龍記折鱗牌

## 主分類與標籤
- 主分類：遺物、古器與命名寶物
- 副標籤：`名物 / 種族古器 / 龍族記錄件 / 高敏感`
- 常見來源：奧崙舊庫、主家長記庫外環、黑市極少數流出

## 外觀
- 牌體像薄鱗展開後的片狀牌，紋理極細。
- 真件表面在不同角度會有微弱層疊光。
- 牌背常刻有短列記序。

## 常見持有者
- 龍族長記官
- 主家史官
- 罕見的遺物流出收藏者

## 主要用途
- 記錄某段長記摘要、族誓或主家列位
- 在內部儀式與校驗場合可作旁證
- 若流出外界，會成為極高敏感的歷史證據

## 估值帶
- 不列公開市價
- 灰線若出現，屬世界級極高值

## 合法性與限制
- 屬 `高保全 / 限持`
- 非授權持有足以引發主家追索

## 真品辨識
- 真件鱗紋不是刻上去，而像從材質本身長出來
- 仿件常只模仿表面亮度，卻沒有層疊紋深度

## 常見劇情用途
- 龍族內鬥證據
- 主家秘史外流
- 黑市圍繞一塊折鱗牌展開追逐
