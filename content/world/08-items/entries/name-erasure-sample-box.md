# 抹名樣本盒

- 主分類：黑市仿製、失格品與偽封樣器物
- 副標籤：灰線器物 / 限灰線 / 危材配套 / 高風險
- 常見來源：潮牢陰面黑市、霧羽外流品、幽語海低聲灰線
- 外觀：
  小型硬盒，內裡多層軟襯可拆卸，外殼通常沒有真正序號，只有可擦除假記。
- 常見持有者：
  走私樣本商、違規採集者、黑頁買手。
- 主要用途：
  用來抹去樣本來源、打散原本的封樣鏈與保管責任。
- 估值帶：
  中高價灰線器物。
- 合法性與限制：
  禁持；被查到通常會被視為故意切斷責任鏈。
- 辨識點：
  真封樣盒講究來源與序列，抹名盒則故意留空白與可刮除層。
- 劇情用途：
  讓玩家追查某份危樣究竟從哪來，或揭露某研究報告其實全靠抹名盒洗過來源。
