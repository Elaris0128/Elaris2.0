# 危材級序盤

- 主分類：職業、公會與專業器具
- 副標籤：公會器具 / 限持 / 危材處理 / 需複核
- 常見來源：藥師公會、靈廚公會、危材封樣庫與共管桌
- 外觀：
  金屬或硬骨製的圓盤，盤面分成多層同心區，刻著危材級序與處理窗口。
- 常見持有者：
  高級封樣手、危材覆驗人、危材料理監督者。
- 主要用途：
  判定危材級別、決定可否流通、可否試用，以及是否需二次封樣。
- 估值帶：
  中高價專業器具。
- 合法性與限制：
  限持；通常需要公會授權或掛名資格。
- 辨識點：
  真盤的級序紋會與附屬封條互相對讀，仿盤常只有表面分級沒有隱層校驗。
- 劇情用途：
  作為危材責任爭議的關鍵物，或被仿造後導致整批危材流入市面。
