# 暮牙記憶宴鹽片

## 主分類與標籤
- 主分類：貨物、奢侈品與交易品
- 副標籤：`少見 / 靈廚專貨 / 宴席貨 / 風險高`
- 常見來源：暮牙海、靈廚公會外放配額、黑宴桌

## 外觀
- 多為薄片狀灰白鹽片，邊緣帶一點油光。
- 高品質鹽片會壓有細小試味紋。
- 若保存不佳，表面會發黃或出苦點。

## 常見持有者
- 靈廚公會高階廚者
- 宴席買手
- 黑宴桌試味師

## 主要用途
- 在高階宴席中微量調味，強化記憶感與回味層次
- 某些配方會拿它和危材共同處理
- 也可作為身份與地位象徵性食材

## 估值帶
- 中高值到高值
- 帶正式宴季來源者更高

## 合法性與限制
- 屬 `限流 / 需配額`
- 黑市版本風險極高，常混入替味料

## 真品辨識
- 真品入口不刺舌，回味會慢慢浮上
- 假貨往往第一口就過鹹或帶苦焦味

## 常見劇情用途
- 禁宴配方走私
- 宴席毒替案
- 名廚與黑宴桌的供貨衝突
