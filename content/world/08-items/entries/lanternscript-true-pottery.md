# 盞紋薄陶真件

- 主分類：遺物、古器與命名寶物
- 副標籤：古文明遺物 / 易碎 / 可研究 / 常被偽造
- 常見來源：星盞群島外緣遺址、學營保全庫、盜掘回流桌
- 外觀：
  極薄陶片帶淡暖白色，表面壓有細密盞紋，光斜照時能看到極淺的雙層胎紋。
- 常見持有者：
  抄錄隊、古陶研究者、黑頁買手、收藏者。
- 主要用途：
  用來比對回燼盞朝工藝期別、祭器分級與地區流布。
- 估值帶：
  普通殘片低到中價，完整或帶明確盞碼的會大幅升值。
- 合法性與限制：
  可研究流通，但高級真件通常需要登記來源與拓錄副本。
- 辨識點：
  真件薄而不脆聲，敲擊音偏沉；現代仿件聲音太亮，胎紋也太整齊。
- 劇情用途：
  當作偽造遺物案的證據、拼回古祭器、或從小陶片看出某遠征隊曾偷挖封區。
