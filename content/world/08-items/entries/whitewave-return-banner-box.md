# 白浪回旗盒

## 主分類與標籤
- 主分類：貨物、奢侈品與交易品
- 副標籤：`少見 / 禮盒 / 海港政治意味 / 可收藏`
- 常見來源：晨穗群島港禮鋪、泊權宴席、老旗隊贈禮

## 外觀
- 細長盒身常用白木或淺灰骨板製成，盒蓋壓有回旗紋。
- 內襯多為細布與薄灰紙。
- 真正高價版本會附一片小小回旗銅片。

## 常見持有者
- 港務與商會人物
- 海盜談判中間人
- 收藏港禮的達官貴人

## 主要用途
- 當作港口間互贈的體面禮盒
- 用來裝少量香貨、糖貨或簿頁小件
- 某些人會拿它暗示停戰或舊交情

## 估值帶
- 中值到中高值
- 附名港來源與老旗銅片者可到高值

## 合法性與限制
- 公開流通
- 若內裝禁貨，盒子本身也會成為證物

## 真品辨識
- 真盒內襯貼合細緻，回旗紋邊緣不毛躁
- 仿盒常把銅片做得過亮

## 常見劇情用途
- 港禮外交
- 暗號式贈禮
- 禮盒內容物被調包
