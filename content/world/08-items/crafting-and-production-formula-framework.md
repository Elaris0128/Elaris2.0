# 製作與生產公式總框架

這份是 `所有製藥、料理、鍛造` 共用的基礎規則。

## 一次製作最少要先決定的事
- `成品型態`
  - 固態
  - 液態
  - 氣態
  - 半固態
- `主方向`：選 `2` 個最想要的效果方向。
- `可接受代價`：選 `1` 個你可以接受的缺點或副作用。
- `製作規模`
  - 小批次
  - 標準批次
  - 大批次

## 共通欄位

### 正向欄位
- `主材值`：`1-6`
  - 1-2 常見材
  - 3-4 地方專業材
  - 5-6 稀有、危材或名材
- `輔材值`：每項 `0-3`
- `載體值`：`-1 到 +2`
- `催化值`：`-2 到 +2`
- `器具值`：`0-4`
- `場域值`：`-3 到 +3`
- `熟練值`：`0-6`
- `工序值`：`0-5`

### 負向欄位
- `難度`：`4-18`
- `風險值`：`0-5`
- `相斥值`：`0-5`

## 共通公式

`製作總值 = 主材值 + 輔材合計 + 載體值 + 催化值 + 器具值 + 場域值 + 熟練值 + 工序值 - 風險值 - 相斥值`

`結果差 = 製作總值 - 難度`

## 成品結果級距
- `結果差 +8 以上`
  - `傑作成功`
  - 主方向完整達成，且通常帶 1 項正向特異。
- `結果差 +4 到 +7`
  - `優秀成功`
  - 主方向穩定達成，偶爾帶微小附贈效果。
- `結果差 0 到 +3`
  - `普通成功`
  - 成品有效，但表現不特別突出。
- `結果差 -1 到 -3`
  - `缺陷成功`
  - 能用，但有明顯瑕疵、限制或副作用。
- `結果差 -4 到 -7`
  - `廢品 / 不穩定成品`
  - 可能只能拆回部分材料，或只能在小範圍短暫使用。
- `結果差 -8 以下`
  - `嚴重失敗`
  - 伴隨污染、爆裂、逆熏、崩裂、器具報廢或引來不良反應。

## 特異值

`特異值 = 場域值 + 風險值 +（若結果差為正則取其一半，向下取整）`

- `6+`
  - 出現 `特殊作用`
- `9+`
  - 出現 `強特殊作用`
  - 但通常也會附帶輕度副作用或使用限制

## 副作用值

`副作用值 = 風險值 + 相斥值 - 熟練值 - 器具值`

- `0 以下`
  - 幾乎沒有副作用
- `1-3`
  - 輕微副作用
- `4-6`
  - 明顯副作用
- `7 以上`
  - 重副作用或事故級後果

## 型態修正
- `固態`
  - 穩定度 `+2`
  - 生效速度 `-1`
- `液態`
  - 生效速度 `+2`
  - 保存性 `-1`
- `氣態`
  - 擴散與覆蓋 `+2`
  - 精準度 `-2`
  - 風險值 `+1`
- `半固態`
  - 局部控制 `+2`
  - 施用範圍較小

## 製作規模修正
- `小批次`
  - 熟練檢定較容易
  - 產量低
- `標準批次`
  - 無修正
- `大批次`
  - 主材與器具壓力更高
  - 難度 `+2`
  - 若失敗，損失量也更大

## 難度參考
- `4-6`
  - 旅用常物、日常配方、制式件
- `7-10`
  - 地方專業品、合格商用品、常規功能品
- `11-14`
  - 危材、深樣、名工件、宴席級成品
- `15-18`
  - 禁材、高位權器、血脈認主載體、古法重現

## 世界風格差異
- `藥師公會`
  - 追求低風險、可封樣、可回溯
- `靈廚公會`
  - 追求層次、餘韻、場域氣氛與可承受的危材效果
- `鍛造 / 黑鍛 / 誓紋技師`
  - 追求結構完整、可維修、導能相性與認主穩定
- `灰線製作`
  - 器具值低、熟練值不穩、風險值偏高
