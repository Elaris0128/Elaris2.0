# 製作系統與公式索引

這套文件是給 `玩家、主持人、任務設計` 直接使用的共通製作規則。

它處理三種主要生產行為：
- `製藥 / 配方調製`
- `煮飯 / 靈廚與功能性膳食`
- `鍛造 / 精煉與組裝`

也處理共同結果：
- 成功
- 缺陷成功
- 廢品
- 嚴重失敗
- 特異效果
- 副作用

## 讀法建議
1. 先看 [製作與生產公式總框架](./crafting-and-production-formula-framework.md)，決定共通公式、結果差與風險值怎麼算。
2. 再看 [製作材料標籤與評分準則](./crafting-ingredient-tags-and-material-scoring-guide.md)，決定材料該怎麼給分。
3. 再看 [型態、批次、工時與穩定矩陣](./crafting-form-batch-time-and-stability-matrix.md)，決定這件成品要怎麼做才合理。
4. 再看對應專章：
   - 藥物與配方進 [製藥、調製、熬煎與藥劑規則](./apothecary-formulation-brewing-and-remedy-rules.md)
   - 料理與危材膳食進 [料理、靈廚與功能性膳食規則](./culinary-and-spirit-cooking-rules.md)
   - 武裝、防具、工具與權器載體進 [鍛造、精煉與組裝規則](./forging-refinement-and-assembly-rules.md)
5. 再看 [成品等級、合法性與市場可流通性](./crafting-permits-grades-and-market-readiness.md)，判定能不能賣、能不能帶、需不需要封樣。
6. 最後用 [製作結果、缺陷與副作用表](./crafting-outcomes-defects-and-side-effect-tables.md) 決定成品品質與事故。
7. 若要直接開跑，參考 [配方設計流程與完整範例](./crafting-design-workflow-and-complete-examples.md)。

## 這套系統能做什麼
- 讓玩家先決定 `想做成什麼方向`，再去組主材、輔材、器具與場域。
- 允許 `成功但有缺陷` 的成品，不會只有成或不成。
- 允許 `帶特殊作用` 或 `帶副作用` 的結果。
- 支援不同型態：
  - `固態`
  - `液態`
  - `氣態`
  - `半固態`

## 世界內常見路數
- `藥師公會`：偏穩定、可封樣、可追責。
- `靈廚公會`：偏層次、餘韻、危材控制與宴席效果。
- `鍛造 / 黑鍛 / 誓紋技師`：偏結構、火候、認主與載體相性。
- `灰線與黑市`：偏快速、便宜、高風險、結果不穩。
