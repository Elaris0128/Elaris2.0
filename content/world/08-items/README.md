## 讀法
1. 先看 [物品分類框架](./item-category-framework.md)，確認一件物品應該放在哪個主分類。
2. 再看 [物品標記、合法性與權限標籤](./item-rarity-legality-and-permit-tags.md)，理解它是公開流通、限持、灰線還是禁流。
3. 若涉及市場、採購、黑市或報價，再看 [各國與各海域的貨幣、估值與價格邏輯](./currencies-valuations-and-price-logic.md)。
4. 若是玩家製作、熬煎、靈廚、鍛造或事故判定，先進 [製作系統與公式索引](./crafting-systems-and-formula-index.md)。
5. 若是武裝、藥食、黑市仿製或高辨識度名物，再進對應專章。
6. 若是地方貨物、專業器具、遺物或魔物素材，再回到各專門總表查細節。

## 補寫順序建議
- 世界級或高辨識度器物，先進 [世界物品總索引](./world-item-master-index.md)。
- 如果是會被反覆提到的名物、重器、遺物或任務關鍵物，先進 [單件物品詳檔庫索引](./notable-item-detail-library-index.md)。
- 戰鬥與執法常見裝備，先進 [武裝與防具總錄](./weapons-armor-and-combat-gear-compendium.md)。
- 藥物、食材、短效道具與保命物，先進 [藥食、消耗與一次性物件總錄](./consumables-remedies-and-single-use-items.md)。
- 行旅、航海、泊靠、採樣與遠征生存裝具，先進 [行旅、航海與探索裝具總錄](./travel-navigation-and-expedition-gear-compendium.md)。
- 灰線仿製、公會假器具、失格版與錯版物，先進 [黑市仿製、失格品與偽封樣器物](./counterfeit-black-market-and-failed-items.md)。
- 有地區味或任務流通價值的，先進 [各地區常見物件與特產](./regional-goods-and-specialties.md)。
- 有專業門檻、資格效力或封樣責任的，先進 [公會專業器具總表](./guild-tools-and-professional-apparatus.md)。
- 跟一般魔物、狩獵、解體、藥師與鍛材有關的，先進兩份世界魔物素材總錄。
- 跟海怪、海洋遠征、靈廚、航運與深洋危材有關的，再進海怪子總錄。
