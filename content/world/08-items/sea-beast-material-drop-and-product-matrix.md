## 最常見的成品鏈

### 武裝鏈
- 裂齒鯊齒 -> 倒鉤刃
- 海龍脊刺 -> 長矛與槍頭
- 刀魚吻刃 -> 解體細刃
- 大海蛇骨 -> 輕槍與儀式刺器

### 航具鏈
- 巨魷韌筋 -> 強韌索具
- 水母大膜 -> 低光防潮罩
- 冠冰眠鯨骨 -> 極寒船腹骨架
- 極鉗星蟹甲 -> 抗撞外層

### 藥師鏈
- 海蛇毒囊 -> 解毒對照與毒理藥材
- 海龍膽囊 -> 高階催化藥
- 水母光囊 -> 低光標示與藥性試驗
- 魔海草節核 -> 淨化與保鮮

### 靈廚鏈
- 鯊肝脂 -> 試味與危材料理
- 巨魷墨囊 -> 深味發酵
- 刀魚吻片 -> 誓宴切片材
- 海龍油 -> 高價危材主料

## 最容易變成黑市標的的素材
- 沒有封樣但仍保持活性的腺體
- 命名巨獸的核心部件
- 能仿製權器或祭器的背紋骨片
- 來源可疑但外觀很完好的龍角片與鯨紋板
