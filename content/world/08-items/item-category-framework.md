# 物品分類框架

## 核心原則
- 主分類用 `用途`。
- 副標籤用來補 `稀有度 / 合法性 / 權限 / 來源 / 場域`。
- 這套框架是為了讓 Elaris 的物品能同時支撐世界觀、任務、交易、市場與黑市，而不是做成只看數值的裝備表。

## 十大主分類

### 1. 武裝與防具
- 收錄：
  - 劍、槍、弓、短器、鏈器、權戰裝具
  - 輕裝、中裝、重裝
  - 環境防護型護具
- 代表用途：
  - 戰鬥
  - 執法
  - 守域
  - 海上與高空行動

### 2. 權器、印信與身分重器
- 收錄：
  - 王庭權杖
  - 認主器
  - 身分印
  - 副牌、授章、執照板
- 代表用途：
  - 授權
  - 身分證明
  - 行政權限
  - 高位象徵

### 3. 儀式、信仰與祭祀器物
- 收錄：
  - 祭牌
  - 香條
  - 送魂物
  - 守域燈
  - 祭器補件
- 代表用途：
  - 宗教儀式
  - 送魂
  - 祈願
  - 結界與守路

### 4. 行旅、航海與探索裝具
- 收錄：
  - 航具
  - 泊靠裝具
  - 勘界與抄錄裝具
  - 極地生存裝具
  - 採樣與危樣攜行器
- 代表用途：
  - 旅行
  - 海航
  - 遠征
  - 調查
  - 求生

### 5. 職業、公會與專業器具
- 收錄：
  - 公會專業工具
  - 技師校準器
  - 藥師對照盤
  - 試術記盤
  - 危材分級盤
- 代表用途：
  - 專業工作
  - 資格行使
  - 覆核
  - 責任追蹤

### 6. 封樣、研究與危材處理器物
- 收錄：
  - 封樣液
  - 樣本盒
  - 穩定器
  - 深樣鎖瓶
  - 危樣容器
- 代表用途：
  - 研究
  - 危材處理
  - 樣本保全
  - 法規責任

### 7. 遺物、古器與命名寶物
- 收錄：
  - 古文明核心件
  - 薄陶真件
  - 橋界補片
  - 古族骨盤
  - 世界級命名重器
- 代表用途：
  - 保全
  - 研究
  - 認主
  - 世界級任務線

### 8. 藥食、消耗與一次性物件
- 收錄：
  - 藥膏
  - 丸劑
  - 保命物
  - 試味物
  - 暖身與應急用品
- 代表用途：
  - 急救
  - 旅途補給
  - 儀式短效輔助
  - 灰線暫時性用途

### 9. 素材、半成品與加工件
- 收錄：
  - 魔物素材
  - 海怪素材
  - 半成品鍛材
  - 工藝件
  - 可再加工部件
- 代表用途：
  - 鍛造
  - 藥材
  - 靈廚
  - 航具維修
  - 祭器製作

### 10. 貨物、奢侈品與交易品
- 收錄：
  - 地方特產
  - 市場流通貨
  - 禮品
  - 高價樣片
  - 稀有包裝品
- 代表用途：
  - 貿易
  - 外交禮物
  - 市場氣氛
  - 灰線轉售

## 常用副標籤

### 稀有度
- `常見`
- `少見`
- `珍稀`
- `名物`
- `世界級`

### 合法性
- `公開流通`
- `需報備`
- `限持`
- `限域流通`
- `灰線`
- `禁持`

### 權限與程序
- `需封樣`
- `需認主`
- `需授權`
- `需複核`
- `可追溯`
- `高保全`

### 來源與場域
- `地方特產`
- `公會器具`
- `古文明遺物`
- `海怪素材`
- `世界魔物素材`
- `高空貨`
- `極地裝具`
- `深洋危材`

## 寫單件物品時最少要有的欄位
- 名稱
- 主分類
- 副標籤
- 常見來源
- 外觀
- 常見持有者
- 主要用途
- 估值帶
- 合法性與限制
- 辨識點
- 劇情用途

## 最常見的交叉情況
- 一件東西可能同時屬於 `職業器具 + 權限物`。
- 一件東西可能本體是 `素材`，加工後變成 `武裝` 或 `祭器`。
- 一件東西可能有 `真品 / 仿品 / 失格品 / 灰線版` 四種流通形態。
