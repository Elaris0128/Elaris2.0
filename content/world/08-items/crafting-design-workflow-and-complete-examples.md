# 配方設計流程與完整範例

這份文件把整個製作系統跑一遍，方便主持人與玩家直接照著做。

## 五步流程

### 第一步：決定你要什麼
- 成品型態
- 兩個主方向
- 一個可接受代價
- 製作規模

### 第二步：選材料
- 選 `1` 個主材
- 選 `1-3` 個輔材
- 決定載體
- 決定是否加催化件

### 第三步：選環境與器具
- 在哪裡做
- 用什麼器具
- 由誰製作

### 第四步：計算
- 算 `製作總值`
- 算 `結果差`
- 算 `特異值`
- 算 `副作用值`

### 第五步：定結果
- 用結果差定成功層級
- 用特異值定有沒有特殊作用
- 用副作用值定後遺症、瑕疵或事故

## 完整範例一：快效止暈液

### 目標
- 型態：液態
- 方向：止暈、快速起效
- 可接受代價：保存短

### 配料
- 主材：穩息根液 `主材值 3`
- 輔材：薄荷鹽葉 `1`
- 輔材：靜蘚汁 `2`
- 載體：淨水酒液 `1`
- 催化：冷滴霧油 `1`

### 製程
- 器具值：`2`
- 場域值：`1`
- 熟練值：`3`
- 工序值：`2`
- 風險值：`1`
- 相斥值：`1`
- 難度：`7`

### 計算
`3 + (1+2) + 1 + 1 + 2 + 1 + 3 + 2 - 1 - 1 = 14`

`結果差 = 14 - 7 = +7`

### 結果
- `優秀成功`
- 液態型態讓起效更快
- 保存期短仍保留為既定代價

## 完整範例二：長航壓腥湯

### 目標
- 型態：液態
- 方向：和胃、回溫
- 可接受代價：味重

### 配料
- 主材：長航魚骨 `主材值 2`
- 輔材：壓腥藻葉 `2`
- 輔材：暖鹽根 `1`
- 載體：重鹽湯底 `1`

### 製程
- 火候控制：`2`
- 器具相性：`2`
- 熟練值：`2`
- 場域值：`1`
- 相剋值：`1`
- 過火/欠熟：`0`
- 危材負荷：`0`
- 難度：`6`

### 計算
`2 + 2 + 1 + 2 + 2 + 2 + 1 - 1 - 0 - 0 = 11`

`結果差 = 11 - 6 = +5`

### 結果
- `優秀成功`
- 回溫效果好
- 味道偏重，但仍在可接受範圍

## 完整範例三：高空繫泊刃

### 目標
- 類型：鍛件
- 方向：輕量、平衡
- 可接受代價：耐熱較差

### 材料
- 主材：風骨鋼 `4`
- 副材：輕紋鋁骨 `2`
- 副材：束環纖筋 `1`

### 製程
- 爐火控制：`2`
- 鍛打整序：`2`
- 淬回火：`1`
- 裝配/銘刻：`1`
- 工具值：`3`
- 熟練值：`3`
- 雜質：`1`
- 裂損：`0`
- 相性衝突：`1`
- 危險值：`1`
- 難度：`10`

### 計算
`4 + (2+1) + 2 + 2 + 1 + 1 + 3 + 3 - 1 - 0 - 1 - 1 = 16`

`結果差 = 16 - 10 = +6`

### 結果
- `優秀成功`
- 輕量與平衡達成
- 耐熱較差保留成既定缺點

## 建議主持方式
- 玩家如果只想快速做
  - 讓他先說 `型態 + 兩方向 + 一代價`
  - 其餘由主持人協助配分
- 玩家如果想鑽研
  - 讓他自己挑材與算值
- 如果要偏敘事
  - 只保留 `結果差 / 特異值 / 副作用值` 三步
