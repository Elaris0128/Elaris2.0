# Elaris 編年史索引

## 定位
這一層收的是整個 Elaris 的時間框架，目的是把世界主史、地區主史、種族主史與各國地方曆串成可對讀的結構。

## 建議讀法
1. 先看 [世界總編年史](./world-chronicle.md)
2. 再看 [地區編年史索引](./regions/README.md) 與 [種族編年史索引](./races/README.md)
3. 若要追地方細節，再看 [各國編年史索引](./countries/README.md)

## 紀年原則
- 上古大框架使用：`神行世 / 聖跡世`
- 現世跨國通用紀年使用：`通行曆`
- 各國與各族可再保留自己的地方曆與族內紀年

## 讀取提醒
- `世界總編年史` 負責交代大時代與世界尺度轉折。
- `地區編年史` 負責交代大陸、海域與特殊區域怎麼走出自己的歷史節奏。
- `各國編年史` 才是地方曆、國別史與更細政治史的主戰場。
- 若地區與國別描述看起來不同，通常應先理解為：地區史說的是大區主線，國別史說的是地方視角。
