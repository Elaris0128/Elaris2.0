# 各國編年史索引

## 讀法
- 這裡收的是各國、聯邦、邦盟、哨域與共同體自己的編年史版本。
- 各國版編年史以 `地方曆` 為主，必要時附上 `通行曆` 對照。
- 若追溯到更古老的祖源與世界斷代，則回接 `神行世 / 聖跡世`。
