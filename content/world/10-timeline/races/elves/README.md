# 精靈分支編年史索引

## 讀法
- 這裡收的是 `精靈大分支` 自己的編年史。
- 它們仍回掛到 [精靈編年史](../elves-chronicle.md)，但會更強調各枝域自己的長記、守則、成年禮與地方斷代。
- 因為精靈壽命長，很多分支編年史會保留更明顯的 `神行世 / 聖跡世` 殘留記憶。
