# 獸人分支編年史索引

## 讀法
- 這裡收的是 `獸人主要支系` 的編年史。
- 它們仍回掛到 [獸人編年史](../orcs-chronicle.md)，但會更強調支系自己的棲地、祖靈、戰盟、遷移與地方勇名。
- 因為獸人整體較短壽、變化快，這些分支編年史會更偏重 `世代更替 / 戰盟改組 / 棲地壓力`。
