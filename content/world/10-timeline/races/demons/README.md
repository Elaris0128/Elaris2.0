# 魔人七血裔編年史索引

## 讀法
- 這裡收的是魔人 `七血裔` 各自的分支編年史。
- 七血裔不是外界簡化想像中的七種情緒，而是和 `器官、居層、資源、職責、誓秩序` 綁在一起的深淵文明分化。
