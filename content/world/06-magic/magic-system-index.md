# 魔法系統索引

## Purpose
這份文件是世界觀魔法設定與遊戲規則之間的橋接索引。它不取代原始設定，而是把 Elaris 中與玩家系統直接相關的魔法概念整理成可調用結構。

## 世界觀核心概念
- 靈息律動（Veiren）是萬物流動與回應的基礎
- 魔法來自：內在靈核 + 靈息感知 + 術式結構
- 戰技來自：對靈息的本能導引與身體回應
- 誓環與誓印是特殊且高度世界觀化的力量軸

## 遊戲規則連動

### 與職業系統連動
- 術法系職業偏重術式結構掌握
- 武戰系偏重戰技模組與肉身灌注
- 誓約系偏重誓環系統與條件性交換
- 技藝系偏重鍊成、修典、誓器等非直接爆發能力

### 與能力值連動
- 靈感：感知靈息
- 靈核：承載與輸出靈息
- 悟性：理解術式結構
- 心志：維持專注與抗干擾

### 與資源系統連動
- 靈息儲量：施展的核心燃料
- 專注：維持與精準控制
- 負荷：反噬與極限壓力

## 特殊體質連動
- 靈息極化
- 靈核障化
- 術式干擾
- 情緒共鳴
- 靈息轉質者

## 神聖來源對術式的影響
- 神祇、祖靈、守地存在與誓律觀，會影響術式如何被命名、教授、認證與限制。
- 同樣效果的術式，在不同文明裡可能屬於神學、律學、家系秘傳或地方儀路。
- 這一層不直接取代靈息屬性與術式結構，而是作為 `術學來源軸`。
