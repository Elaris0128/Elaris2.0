# 梵洛特國家與地區索引

## 讀法
- 梵洛特的地方包以 `高壓生存區` 為主，不一定都像傳統國家。
- 有些是完整國度，有些更接近共同體、哨域、危帶或文明圈層。

## 核心地方
- [幽語海岬哨域](./youyu-haijiao/overview.md)
  翼人主導的海岬索道與霧港文明。
- [南裂危帶](./nanying-belt/overview.md)
  封樣、隔離、補線與收容制度最完整的危帶前線。
- [繭影隱林](./jianying-yinlin/overview.md)
  繭影族主導、柯沙里人與喚隱裔共居的命名與共夢文明。
- [涅厄斯魔人國](./nieesi/overview.md)
  靠裂谷、井城與假名配給勉強維持的流亡魔人王國。
- [纏聲斷濱共同體](./chansheng-duanbin/overview.md)
  依靠潮窗、保運與打撈秩序活下來的沿岸共同體。
