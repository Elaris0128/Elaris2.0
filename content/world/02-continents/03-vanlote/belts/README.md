# 梵洛特生存帶索引

## 讀法
- 梵洛特的 `生存帶` 不是行政區劃，而是更接近 `哪種地形與秩序組合能讓文明活下去`。
- 一個地方包常會橫跨多條生存帶；一條生存帶也可能被數個群體共同使用。
- 若你想先理解梵洛特，不要先背國界，先看生存帶。

## 七條主要生存帶
- [灰岬索哨帶](./frost-ridge-belt.md)
  翼人主導的海岬、索道、橋塔與霧港帶，核心是燈號、鐘序與失路回收。
- [裂鱗核心帶](./scaleclast-core-belt.md)
  高差工事、斷脈硬地與核心補給點集中的硬地生存帶。
- [裂砂誓帶](./cracksilt-oathbelt.md)
  裂紋沙原、乾地長路、遷營隊列與沙原誓約維持的沙帶。
- [煙火臨界帶](./smokefire-verge-belt.md)
  熱霧、煙火、火山緣與耐熱窗口構成的高風險過渡帶。
- [繭影帶與霧林谷寨生存圈](./cocoonshade-belt.md)
  命名、守名、歌路記憶、谷寨水行與林境允準共同維繫的深霧林帶。
- [海懸斷崖帶](./coast-cliff-belt.md)
  崖海相疊、斷岸高差與沿岸哨域最強的海崖帶。
- [危帶前緣](./hazard-front-belt.md)
  封樣、收容、補線、點名與撤退節奏最密集的危帶邊線。
