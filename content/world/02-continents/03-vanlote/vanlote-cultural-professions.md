# 梵洛特文化職業與流派

## 使用方式
梵洛特的職業應以 `文化職名` 為主，再註明對應通用方向。這裡很多工作不是普通職位，而是把生存、地形、儀式、運輸與記錄責任綁在一起的身份。

## 涅厄斯魔人國
- `誓錄撤名官`：通用方向為書吏 / 法務 / 檔案官。負責名字、契約與撤記秩序。
- `真路審行者`：通用方向為嚮導 / 斥候 / 通路官。專長裂谷真路與幻路辨識。
- `井城配給手`：通用方向為後勤 / 水務 / 民生官。掌井線與庇層配給。
- `裂誓谷軍`：通用方向為軍士 / 關路衛隊。重通道封鎖、撤離與短兵突擊。

## 纏聲斷濱共同體
- `潮窗引航人`：通用方向為船長 / 領航員。專長潮窗與裂岸路徑。
- `沉艙回收師`：通用方向為探索者 / 回收者。負責沉船、沉艙與舊航路打撈。
- `保運記路人`：通用方向為商路見證 / 記錄官。負責可信船名單與保運誓規。
- `回聲護浪手`：通用方向為海防 / 救援。專長風浪救援與沿岸警戒。

## 繭影隱林
- `守名者`：通用方向為祭司 / 見證者 / 記名官。負責回名、守名與林境允准。
- `花粉路師`：通用方向為嚮導 / 採集師。專長花息、林徑與季節路線。
- `夢池看守`：通用方向為儀式者 / 藥理者。管理夢池與共夢儀式。
- `母繭護林手`：通用方向為獵手 / 守衛。維持林核安全與驅離擾林者。

## 幽語海岬哨域
- `索塔守線官`：通用方向為工務 / 守衛。專長索道、橋路與高差過線。
- `燈鐘審旗者`：通用方向為導航審核 / 港防。負責燈號、鐘序與禁航判定。
- `霧譯行話師`：通用方向為翻譯 / 情報中介。專長沿岸話語與旗號判讀。
- `岬崖補纜手`：通用方向為工匠 / 攀修者。負責裂岬設備維護。

## 南裂危帶
- `封樣官`：通用方向為檢驗 / 紀錄 / 防疫。處理樣本、異常材料與標記。
- `送渡點名人`：通用方向為安置 / 回收 / 治安。管理撤退、失蹤與名錄。
- `灰藥修補師`：通用方向為藥師 / 修護師。專長危帶藥材、創口與裝備維護。
- `守線短衛`：通用方向為邊防 / 危機應對。專長封線、撤離與隔離。

## 常住部落區帶
- `裂沙導路者`：通用方向為嚮導 / 商旅護行。沙原與伏井路線專精。
- `裂霜獵長`：通用方向為獵手 / 守衛。高寒狩獵與雪脊巡界。
- `玄谷熱穴手`：通用方向為採集 / 工匠。熱穴採收與毒煙生存。
- `灰釘篝守人`：通用方向為營地護守 / 分解回收。灰境晚間守火與清沼。
- `裂星守礦人`：通用方向為採礦 / 高地守備。階地礦路與風岫觀兆。

## 主持使用建議
- 梵洛特職業應多半帶有 `責任與代價`，而不只是技能名稱。
- 這裡很少有純粹學院型職業，更多是 `地方共同體允許你承擔什麼`。
