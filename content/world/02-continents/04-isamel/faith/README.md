# 伊莎梅爾信仰與祭儀索引

伊莎梅爾不是由單一教團統治的大陸，而是由 `葉脈命名諸傳` 串起的精靈文明網。信仰在這裡不是獨立於生活之外的殿堂制度，而是命名、守林、伴生、育養、守路與送眠的總體秩序。
