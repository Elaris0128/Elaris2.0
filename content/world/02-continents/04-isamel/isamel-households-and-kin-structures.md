# 伊莎梅爾家系、巢居與親緣結構

伊莎梅爾的精靈社會不是以單純血統高低來分層，而是以 `命名責任 / 居住節點 / 守護對象 / 代際承接` 共同構成家系。不同文明域的家系名稱、繼承規則與成年義務差異很大，但都承認一件事：一個名字只有在被共同體記住、被居所與守護對象接納之後，才算真正成立。

## 共通原則
- 家系不是封閉家門，而是圍繞某個節點、巢居、橋道、井樹或寒穴所形成的責任共同體。
- 精靈的親緣除了血親，還包括 `授名者 / 同伴學徒 / 靈誓伴照護鏈 / 送眠見證鏈`。
- 成年後離開原居所並不代表斷絕家系，而是改成對另一個節點負責；許多精靈一生會同時保有 `原名所屬` 與 `現居責任` 兩條身分線。

## 常見家系層級
- `根家`：守某株樹、某段根脈、某座寒井或某條橋索的基本家系。
- `庭群`：數個根家組成的生活圈，通常共享祭日、育伴棚與共食場所。
- `誓鏈`：以某種任務或守護職責相連的人群，例如送眠、育龍、聽潮、巡橋。
- `回名圈`：與名字修復、失序補見、名墓照看相關的見證群。

## 居住形態
- 雲冠與林庭地帶多是 `枝庭 / 冠屋 / 橋宿`。
- 濕地與迷澤地帶多是 `浮棚 / 淺台 / 泊居 / 葦橋住帶`。
- 高臺原多是 `岩棚 / 風脊牧棚 / 龍欄與試飛台`。
- 寒穴地帶則是 `井屋 / 冰橋居點 / 名墓附穴 / 霧井群`。

## 繼承與領責
- 多數文明域不是把家產交給長子，而是把 `最適合承責的人` 推到前面。
- 能否繼承橋、巢、井、龍棚與祭位，通常取決於是否完成成年禮、是否曾在危機中守住節點、以及是否能維持靈誓伴與共同體的安全。
- 若家系內無人可承責，常會由外支、學徒或被見證過的旁系接手。

## 與靈誓伴的關係
- 靈誓伴不是私有寵物，而會被寫進家系責任。
- 若某人與幼龍、飛蜥或鳳裔締結靈誓伴，相關照看、食糧、巢材與醫護通常會由整個庭群共同承擔。
- 靈誓伴死亡、失序或離巢，也會影響家系名聲與祭儀責任。

## 巴布窟族共生特例
- 雲瑟皇族與巴布窟族的共息，不是普通鄰居關係，而是被正式寫進家系與命名秩序的共生鏈。
- 某些深根家系同時由皇族記名者與穴民守護者共同承責，兩者缺一都不算完整家系。
