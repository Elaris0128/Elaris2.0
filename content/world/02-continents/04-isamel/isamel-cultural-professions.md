# 伊莎梅爾文化職業與流派

伊莎梅爾的職業以 `文化職業` 為主，不先用通用冒險職稱。這些職業都和文明域責任、靈誓伴、環境維護與命名秩序綁在一起。

## 雲瑟
- `命枝校名者`
  - 對應通用方向：學者 / 見證官
  - 獲得條件：完成正式命名校對，並在命枝典藏所留下第一份合格抄錄。
  - 專屬技法：`枝序校名`、`失名回聽`
- `共息井守`
  - 對應通用方向：祭儀守護 / 修復者
  - 獲得條件：經祠井見證，完成一次共息修復。
  - 專屬技法：`井息安定`、`根廊同頻`
- `育羽照伴師`
  - 對應通用方向：照看師 / 靈誓伴導師
  - 獲得條件：完成幼龍或小型花翎龍的照護輪值。
  - 專屬技法：`柔枝停棲`、`幼龍定驚`

## 瑟塔嶺
- `高脊馭龍手`
  - 對應通用方向：騎手 / 偵巡者
  - 獲得條件：通過試飛日並獲風臺師認可。
  - 專屬技法：`裂風掠階`、`雲脊下俯`
- `風場測譜師`
  - 對應通用方向：氣象學者 / 行路導師
  - 獲得條件：能獨立記錄一次龍風轉向。
  - 專屬技法：`逆風立點`、`星墜風譜`
- `龍鞍工師`
  - 對應通用方向：工匠 / 裝具師
  - 獲得條件：完成一套可通過試飛檢查的鞍具。
  - 專屬技法：`脊索穩骨`、`高場扣鞍`

## 菲露涅
- `羽冠護林者`
  - 對應通用方向：巡林者 / 守域者
  - 獲得條件：通過林冠試禮並完成護林輪值。
  - 專屬技法：`冠橋靜踏`、`羽蔭驅散`
- `藥脂辨病師`
  - 對應通用方向：藥師 / 生態醫照者
  - 獲得條件：能獨立辨識疫斑並製出對應藥脂。
  - 專屬技法：`脂霧封斑`、`溪葉點診`
- `幼龍試伴官`
  - 對應通用方向：靈誓伴見證者
  - 獲得條件：完成一次花翎幼龍試伴引導。
  - 專屬技法：`幼翎試名`、`安枝回鳴`

## 維塔蕾
- `漂葦橋守`
  - 對應通用方向：橋工 / 路權管理者
  - 獲得條件：能主理一段浮橋轉移。
  - 專屬技法：`潮退束橋`、`水索換結`
- `低潮棚師`
  - 對應通用方向：聚落工師 / 水邊建造者
  - 獲得條件：完成一座合格浮棚或工棚。
  - 專屬技法：`葦骨定棚`、`灣口壓樁`
- `鈴葦照伴師`
  - 對應通用方向：伴生照看者
  - 獲得條件：通過鈴葦小龍的近水照看考核。
  - 專屬技法：`低鈴安潮`、`濕翼導返`

## 雅席爾
- `澤錄守名者`
  - 對應通用方向：錄官 / 祭沼見證者
  - 獲得條件：雨季中完成一段澤錄更新。
  - 專屬技法：`沉筆不失`、`泥燈續錄`
- `眠礦封樣師`
  - 對應通用方向：素材封樣 / 危材管理者
  - 獲得條件：能安全封存一批眠礦。
  - 專屬技法：`泥層封樣`、`雨脈定箱`
- `沼光導行者`
  - 對應通用方向：濕地導路人
  - 獲得條件：在沉雨季成功帶隊返營。
  - 專屬技法：`沼面試步`、`沉澤引返`

## 歐羅瑟
- `聽潮號讀者`
  - 對應通用方向：引航者 / 訊號官
  - 獲得條件：完成一次根港回訊與礁窗引航。
  - 專屬技法：`潮窗判位`、`崖訊回響`
- `岸巡龍馭者`
  - 對應通用方向：巡岸騎手
  - 獲得條件：獲准進入岸巡飛龍輪值。
  - 專屬技法：`低崖掠巡`、`鹽霧壓港`
- `鹽木工誓師`
  - 對應通用方向：海木工匠
  - 獲得條件：打造一件合格的岸林耐鹽裝具。
  - 專屬技法：`鹽骨固榫`、`崖鉤束木`

## 奧里梵
- `晨火試行者`
  - 對應通用方向：試煉武者 / 封線手
  - 獲得條件：通過燃霧試行。
  - 專屬技法：`晨焰斷霧`、`藤火開路`
- `焰藤工師`
  - 對應通用方向：火工 / 橋具工匠
  - 獲得條件：完成一套可在試道中使用的焰藤裝具。
  - 專屬技法：`火藤束形`、`赤霧烙結`
- `蔓叢封路守`
  - 對應通用方向：防線守衛 / 路權執行者
  - 獲得條件：能在不毀環境的前提下完成封線。
  - 專屬技法：`蔓網止步`、`火露標界`

## 米蘭羽
- `霧川泊記者`
  - 對應通用方向：水路記錄官 / 導泊者
  - 獲得條件：完成一段盲沼路記。
  - 專屬技法：`靜泊抄影`、`霧牆辨潮`
- `眠羽照看者`
  - 對應通用方向：伴生照護 / 巢地守護者
  - 獲得條件：獲准值守眠羽或霧潟小龍巢地。
  - 專屬技法：`眠羽安巢`、`淺霧回鳴`
- `迷澤引泊人`
  - 對應通用方向：危地導路者
  - 獲得條件：在迷澤中帶回一隊外行者。
  - 專屬技法：`盲沼探樁`、`曲流避陷`

## 霜諾蘭
- `守眠修名師`
  - 對應通用方向：安魂者 / 名錄修復者
  - 獲得條件：完成一次送眠唱與封名記錄。
  - 專屬技法：`寒井封名`、`眠霧續記`
- `冰橋補索者`
  - 對應通用方向：橋工 / 冰路守護者
  - 獲得條件：能在風雪夜維持一段冰橋通行。
  - 專屬技法：`冰索再固`、`霜橋換步`
- `霜龍照看者`
  - 對應通用方向：幼龍照護 / 巡夜守望
  - 獲得條件：完成霜紋幼龍照看輪值。
  - 專屬技法：`寒鱗定驚`、`冬息守眠`

## 共通流派方向
- `伴生系`：照護、試伴、巢地維持、共感與安定。
- `路權系`：帶路、守橋、封線、引航、改道與交通秩序。
- `工藝系`：橋、鞍、棚、索、祭器、封樣與維修。
- `守域系`：巡守、封禁、安魂、護林、危帶穩定與失序修復。
