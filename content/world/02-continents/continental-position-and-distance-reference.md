## Purpose
這份總表把目前已完成的大陸級位置與距離資訊整理成單一入口，方便主持與後續擴寫時快速判斷：

- 哪些區域已有精細距離表，哪些仍以旅行層級為主

## 賽勒亞大陸相對位置
### 五大區域骨架
- `東境`：位於賽勒亞東側平原文明帶，與中央維爾岩脈、西部山脊、北部寒帶與南部霧林皆有明顯接縫。
- `北境`：位於大陸北側寒地前線，以雪原、冰橋、燈鏈與關隘系統構成主體。
- `西境`：位於大陸西側裂谷、山陵與隱域帶，地形破碎，高差與封路風險遠高於其他區域。
- `南境`：位於大陸南側潮林、潟湖、港灣與避難聚鎮帶，陸路、水路與潮窗航線混合。
- `中央維爾岩脈`：位於賽勒亞中央高地，是整塊大陸的脊柱與翻山過脊的核心轉運區。

### 相鄰關係
| 區域 | 主要相鄰區域 | 主體交通印象 |
| --- | --- | --- |
| 東境 | 北境、西境、南境、中央 | 平原大道、河運、大港與國界走廊 |
| 北境 | 東境、中央 | 誓燈大道、冰橋線、碑宿線 |
| 西境 | 東境、中央、南境邊路 | 裂谷工坊線、山陵重裝路、隱路 |
| 南境 | 東境、西境邊路、中央南麓 | 港路、潟湖水道、補糧線 |
| 中央 | 東境、北境、西境、南境 | 過脊重載路、古道、觀星路 |

## 賽勒亞距離層級現況
### 已較完整的區域
- `東境`：已有國土尺幅、跨國距離與主要走廊的較完整表述。
- `北境`：已有以雪道、燈鏈、冰橋與補給節點為核心的距離與行程邏輯。

### 以旅行層級為主的區域
- `西境`：已有移動節奏與主幹線，但尚未像東境那樣全面量化。
- `南境`：已有水路與港市間的大致節奏，但更重潮位與可泊點判斷。
- `中央`：已有三臺內部與過脊旅行基準，但仍偏高地行程概念表。

## 賽勒亞跨區域旅行基準
| 起點區域 | 終點區域 | 主要理解方式 |
| --- | --- | --- |
| 東境 -> 北境 | 由平原防線進入雪關與燈鏈系統 | 直線不重要，是否能守燈與補給更重要 |
| 東境 -> 西境 | 由平原轉入山前道、裂谷與隘口 | 高差與封路風險會拉長體感距離 |
| 東境 -> 南境 | 由內陸大道或霧林邊路轉入港灣與潟湖 | 季節、霧期與水位影響大 |
| 東境 -> 中央 | 由平原大道或山前轉運進入過脊路 | 重載、風雪與翻脊能力是主因 |
| 北境 -> 中央 | 由寒關、碑宿與高地轉運節點銜接 | 儀式性與補給性都很強 |
| 西境 -> 中央 | 由重裝山道與高地互市線銜接 | 工匠物流與軍用搬運最常見 |
| 南境 -> 中央 | 由補糧線與高差通道銜接 | 物資補給比速度更關鍵 |

## 梵洛特大陸相對位置
### 判讀原則
- 梵洛特不適合硬切東西南北。
- 目前以 `七條生存帶` 判讀其文明與路線，比方位切割更準確。
- 正式政體之外，還存在高風險常住區帶、季節據點與無聚落荒段。

| 寒裂群嶺帶 | 高寒山嶺與裂霜北原 | 高山、斷脊、雪裂與部落誓地 |

| 煙熔灰釘帶 | 煙啼玄谷、熔痕赤域、灰釘荒境、裂星階 | 煙谷、火脈、灰釘荒境與獸人支裔常住區 |
| 繭影幻林帶 | 幻蔓霧林與繭影隱林 | 林幕、幽池、絲蔓、繭影共同體 |

### 梵洛特住民集中帶
- `裂紋沙原`：薩魯恩人與裂蠍沙裔長住。
- `裂霜北原`：寒鱗、凍蛟、絕影諸部常住。
- `煙啼玄谷`：霧黽裔常住。
- `熔痕赤域`：熔炭龜裔常住。
- `灰釘荒境`：渣咬裔與焦燄鼬裔常住。
- `裂星階`：礦鼠裔與星獅裔常住。

### 梵洛特旅行判讀
- 梵洛特的路線不應只看距離，還要看 `是否能補線、是否能藏身、是否能換路、是否會被煙霧或林幕吃掉視野`。
- 同一段路在 `裂風季 / 濕霧季 / 灰落季 / 封樣季` 的體感距離差異很大。
- 大陸內部常存在 `聚落核心 -> 功能據點 -> 無聚落荒段` 的三層結構，因此很多路不是「城到城」，而是「聚點到補線站再到安全邊帶」。

## 使用建議
- 若是賽勒亞內部的正式國家旅行，優先先查各區域的 `distance-reference` 與 `route-matrix`。
- 若是梵洛特，優先先判斷 `目前位在哪條生存帶`，再判斷路線，而不是先問方位。
- 後續若西境、南境、中央與梵洛特補出更精細的量化距離，應以新資料覆蓋本表的概略語言。
