# 穹語市坊生態

## 生境分層
- `語棚與橋列帶`：生活型、雜食型與人為餵養種最密集。
- `半沉庫坊帶`：濕貨、腐敗種與偷食種大量聚集。
- `霧谷外灘帶`：幻導性與水域掠食種最常出沒。

## 生活型與市場伴生魔物
- `霧書肩猴`：會偷卷冊卻也能幫忙找掉落小物。危險 1 / 親近 2。
- `潮靴小鷺`：專吃棚底小害蟲。危險 0 / 親近 3。
- `半沉殼犬`：對陌生氣味極敏銳，常被當作夜市警戒獸。危險 1 / 親近 2。

## 野外威脅與水域掠食種
- `霧鰻`：潟湖常見掠食種。危險 2 / 親近 0。
- `潟湖步蟲`：夜間大量出沒會啃壞貨箱。危險 2 / 親近 0。
- `潮鏡喙鳥`：會偷走帶亮光的小物。危險 1 / 親近 0。

## 魔植、魔菌與市場依存種
- `夜根藤`：沿橋腳與濕木瘋長。危險 1 / 親近 2。
- `蘚霧木`：是穹語重要的遮霧與保濕背景樹種。危險 0 / 親近 3。
- `鹹夢菇`：可入藥也可製幻夢食材。危險 2 / 親近 2。
- `半沉蘚簾`：讓整段棚巷看起來更像同一片濕壁。危險 1 / 親近 2。

## 異常與禁忌生命
- `潮價魘`：會讓人錯把假價當真價的市場異常。危險 2 / 親近 0。
- `鹹霧聽客`：會偷聽秘密並在別處重播。危險 2 / 親近 0。
- `庫坊濕偶`：由舊貨與潮菌黏成的人形危險體。危險 3 / 親近 0。

## 生態鏈摘要
- 穹語市坊的生態核心是 `濕度 / 霧 / 貨物 / 人流 / 藏匿空間`。
- 很多生命本來不該在一起，但市場把它們暫時擠成了新的共生關係。
