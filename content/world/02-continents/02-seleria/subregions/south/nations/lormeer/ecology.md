# 洛溟港生態

## 生境分層
- `主港泊位帶`：生活型、食腐型與港害種最密集。
- `近岸外灘帶`：大型潮汐掠食種與漂流資源交會區。
- `堤外淺灣帶`：可食水產、潮植與小型危險種主要分布區。

## 生活型與港務伴生魔物
- `堤殼搬蟹`：可搬重貨但需定期去殼照料。危險 1 / 親近 3。
- `港綴燈鷗`：會叼走小件卻也能預報風向突變。危險 0 / 親近 2。
- `繩尾潮狐`：喜歡偷吃魚乾，卻會驅趕部分碼頭害種。危險 1 / 親近 2。

## 野外威脅與水域掠食種
- `潮殼鼉幼`：近岸大型掠食種幼體。危險 3 / 親近 0。
- `割水背鰭`：常在壞天氣前成群逼近外灘。危險 3 / 親近 0。
- `霧鰓鉤魚`：會卡住船底與拖網。危險 2 / 親近 0。

## 魔植、魔菌與潮路依存種
- `潮果繩灌`：果肉可食、枝皮可編繩。危險 0 / 親近 4。
- `港灣蘚`：能吸附鹽分與部分腐敗氣，港工很愛。危險 0 / 親近 3。
- `堤根鹽簇`：沿防波堤生長，常做簡易藥鹽。危險 1 / 親近 2。

## 異常與禁忌生命
- `沉舟回聲者`：沉船與無歸港工留下的危險殘響。危險 3 / 親近 0。
- `錯泊提燈客`：會把人引到不該靠近的泊位與暗潮。危險 2 / 親近 0。

## 生態鏈摘要
- 洛溟港的生態取決於 `潮窗 / 漁貨 / 港務秩序 / 外海風暴`。
- 許多魔物與人類距離很近，但只要港務失序，就會立刻轉成災害。
