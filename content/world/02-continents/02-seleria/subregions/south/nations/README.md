# 南境國家索引

## 構成
- `lormeer`：洛溟港
- `perlin`：珀林穀鎮
- `skelreach`：穹語市坊

## 南境整體氣質
- 南境澐緲荒域不像單一王國，而是由 `港市 / 穀鎮 / 市坊 / 潮林與避難聚帶` 拼接出的濕潤邊界。
- 這裡最重要的不是王權，而是 `潮路、庇護、補給、物資流動與灰線情報` 是否還能運作。

## 三個主體
- `洛溟港`：南境最大的對外港口與潮路集散城。
- `珀林穀鎮`：糧藥與谷地補給的穩定後方。
- `穹語市坊`：公開交易與灰線消息並存的市場節點。

## 使用方式
- 若你要查國家與城市主檔，先看各自 `overview.md`。
- 若你要接任務與開局，優先看對應 `content/scenarios/regions/lormeer / perlin / skelreach/`。
