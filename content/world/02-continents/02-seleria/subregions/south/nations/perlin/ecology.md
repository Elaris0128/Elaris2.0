# 珀林穀鎮生態

## 生境分層
- `穀心坡地帶`：最穩定的耕作與居住區。
- `靜溪緩濕帶`：療養、採藥與溫和水生生命聚集地。
- `迷藤外林帶`：最常接觸到外來危險與病潮的邊緣地。

## 生活型與伴生魔物
- `穗背馱鹿`：耕作、載運與護送都很實用。危險 0 / 親近 5。
- `靜溪鼠獸`：常被當作穀倉預警小獸。危險 0 / 親近 3。
- `霧葉蝶`：是藥材成熟與空氣濕度的重要指標。危險 0 / 親近 3。

## 野外威脅與擾動種
- `病沼褶犬`：病潮季最讓人頭痛的濕地獵種。危險 2 / 親近 0。
- `藤喉拖獸`：會拖走小型牲口與補給袋。危險 2 / 親近 0。
- `溪邊冷背鱷`：平時少動，飢餓時極危險。危險 3 / 親近 0。

## 魔植、魔菌與療養依存種
- `靜息穗蘚`：療養床、藥包與庇護所常備材料。危險 0 / 親近 4。
- `安眠穗`：可作舒緩與鎮夢料理。危險 0 / 親近 4。
- `濕地樹菇`：高營養、高水分的常見菌。危險 0 / 親近 3。
- `緩藤灌`：可作繩與簡易支架，也會在雨季迅速瘋長。危險 1 / 親近 3。

## 異常與禁忌生命
- `病潮拾衣人`：舊災年殘響形成的人形異常。危險 3 / 親近 0。
- `穗田低語者`：會在深夜用熟人聲音叫人離開穀區。危險 2 / 親近 0。

## 生態鏈摘要
- 珀林的生態核心是 `穀物 / 濕地水源 / 療養植物 / 庇護密度`。
- 這裡的生活型魔物很多，但一旦病潮或外來污染進入，整條生態鏈會很快從溫和轉向脆弱。
