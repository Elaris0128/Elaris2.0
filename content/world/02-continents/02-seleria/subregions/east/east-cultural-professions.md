# 東境文化職業與流派

## 使用說明
- 這份文件以 `文化職業` 為主體書寫。
- `對應通用職業` 只作備註，方便回掛到底層規則。
- 每個條目都標明 `獲得條件`，方便創角、NPC 建立與任務設計。

## 阿瓦隆王國

### 王誓騎衛
- 類別：軍事 / 王廷正規軍
- 對應通用職業：重裝守衛、守誓戰士
- 常見所屬機構：王室誓旗軍、銀鋒城衛誓隊
- 獲得條件：完成侍從期、通過授旗誓檢、需有封誓家或軍功保薦
- 常見媒介 / 裝備：重盾、長劍、誓旗短槍、誓印胸鎧
- 專屬戰技 / 術式：`誓旗衝陣`、`白塔斷線`
- 流派方向：白塔正旗派、邊關重誓派
- 社會印象：體面、可靠、但帶有王權傲氣

### 暮環影吏
- 類別：諜報 / 秘線
- 對應通用職業：情報行者、機動斥候
- 常見所屬機構：暮環司
- 獲得條件：需先有港務、書吏或邊巡經歷；通過暗號、記憶與尾隨測驗
- 常見媒介 / 裝備：記契刀、暗簿、封蠟針、夜披
- 專屬戰技 / 術式：`影誓留痕`、`暮碼移記術`
- 流派方向：港線靜錄派、封誓尾行派
- 社會印象：看似文弱，實際非常難纏

### 誓鐘司祭
- 類別：宗教 / 見證
- 對應通用職業：祭儀執行者、聖職審律者
- 常見所屬機構：七誓教廷、地方鐘座
- 獲得條件：經教廷誓學院認可，完成見證、安魂與誓鐘值守
- 常見媒介 / 裝備：誓鐘杖、記名鈴、誓帶
- 專屬戰技 / 術式：`鐘庭鎮音`、`見證留名式`
- 流派方向：鐘庭見證派、安魂靜誓派
- 社會印象：神聖、拘謹、能左右大事

### 港契衡錄官
- 類別：民政 / 港務 / 文契
- 對應通用職業：記錄學者、商旅判契者
- 常見所屬機構：港務記契所、王路稅務署
- 獲得條件：文契見習滿期、通過帳契與查驗考核、需有合法身分
- 常見媒介 / 裝備：帳冊、秤印、封條、貨驗筆
- 專屬戰技 / 術式：`港誓封檢式`、`契秤辨偽法`
- 流派方向：東海誓檢派、王路衡錄派
- 社會印象：麻煩但不可或缺

### 誓膳師
- 類別：民間 / 公會 / 照護
- 對應通用職業：靈廚
- 常見所屬機構：靈廚公會、王路旅舍、教廷膳房
- 獲得條件：完成旅灶與安膳考核，能處理至少一種誓約性食材
- 常見媒介 / 裝備：調和盒、灶刃、誓紋保鮮匣
- 專屬戰技 / 術式：`誓灶回溫膳`、`安旗定胃湯`
- 流派方向：王路灶傳、誓宴膳序派
- 社會印象：溫和、實用、很懂人情

## 瑟恩聯邦國

### 星律衛務士
- 類別：軍事 / 城邦衛務
- 對應通用職業：軍陣武士、守誓戰士
- 常見所屬機構：聯邦衛戍軍、城邦衛務署
- 獲得條件：通過聯邦軍律考核、完成隊列與術式配合訓練
- 常見媒介 / 裝備：標槍、盾書、訊號燈
- 專屬戰技 / 術式：`聯章轉位陣`、`星紋壓線術`
- 流派方向：七席衛陣派、河港截流派
- 社會印象：守規矩、硬派、效率高

### 密錄校閱官
- 類別：諜報 / 檔案審查
- 對應通用職業：情報行者、記錄學者
- 常見所屬機構：聯邦密錄局
- 獲得條件：具檔案、公證或查驗背景；通過保密與偽訊辨識測驗
- 常見媒介 / 裝備：密錄夾、隱碼鏡、校印刀
- 專屬戰技 / 術式：`頁痕回讀術`、`靜章封口式`
- 流派方向：檔案靜審派、河港暗校派
- 社會印象：冷靜、苛細、擅長把人看穿

### 序律講證士
- 類別：宗教 / 公共儀式 / 法理
- 對應通用職業：聖職審律者、學院施術者
- 常見所屬機構：序律編院聯席
- 獲得條件：完成序律課程、辯誓合格、可獨立主持見證儀式
- 常見媒介 / 裝備：誓頁杖、校章冊、證印盤
- 專屬戰技 / 術式：`條律見證光`、`序頁約束陣`
- 流派方向：講證正席派、邊城審儀派
- 社會印象：理性、正統、難以敷衍

### 公證術檢師
- 類別：學術 / 檢定 / 公務
- 對應通用職業：學院施術者、記錄學者
- 常見所屬機構：聯邦學院、城邦術檢廳
- 獲得條件：通過樣本分級與術式安全考核
- 常見媒介 / 裝備：檢式尺、樣本盒、定序筆
- 專屬戰技 / 術式：`樣本定階術`、`場域歸檢式`
- 流派方向：樣本分級派、術安歸序派
- 社會印象：專業、囉嗦、很有權威

### 調膳官
- 類別：民間 / 學術 / 醫膳
- 對應通用職業：靈廚、治療藥師
- 常見所屬機構：公學膳房、學院療膳室、靈廚公會
- 獲得條件：完成藥食與安全配比課程，需具公衛證明
- 常見媒介 / 裝備：量匙匣、保溫甕、調序板
- 專屬戰技 / 術式：`定量回息膳`、`靜脈安和湯`
- 流派方向：學院調序派、療膳實證派
- 社會印象：溫和但制度感很重

## 克羅嵐聯盟國

### 裂丘先鋒
- 類別：軍事 / 試煉戰士
- 對應通用職業：近戰前衛、狩獵戰技系
- 常見所屬機構：十獸共誓營、裂丘前鋒線
- 獲得條件：通過成年試峰、完成共誓見證、至少獨立帶隊返程一次
- 常見媒介 / 裝備：重斧、裂骨槍、獸紋護肩
- 專屬戰技 / 術式：`裂丘奔破`、`重吼震步`
- 流派方向：裂痕正獵派、馭角護群派
- 社會印象：強硬、可靠、用實績說話

### 環影伏線者
- 類別：諜報 / 邊線暗行
- 對應通用職業：情報行者、機動斥候
- 常見所屬機構：環影部隊
- 獲得條件：需有席位推薦，通過伏行、聽風與邊線存活測驗
- 常見媒介 / 裝備：短弓、伏草披、骨哨、暗印繩
- 專屬戰技 / 術式：`風耳伏線`、`影獸留痕`
- 流派方向：隱耳靜伏派、逐影斷尾派
- 社會印象：話少、準、很難追

### 骨語祭獵師
- 類別：宗教 / 祖地 / 狩祭
- 對應通用職業：祭儀執行者、遠行獵手
- 常見所屬機構：骨語長者圈、祖地祭圈
- 獲得條件：能判讀骨兆、完成送骨儀式、獲長者允准
- 常見媒介 / 裝備：骨串、祭矛、煙藥罐
- 專屬戰技 / 術式：`骨兆定獵`、`祖風回吼`
- 流派方向：冥羽夜骨派、斑影獵儀派
- 社會印象：神祕、古老、接近祖地禁忌

### 試峰教獵者
- 類別：學習 / 試煉 / 民間武職
- 對應通用職業：遠行獵手、契靈行者
- 常見所屬機構：試峰營地、氏族獵教圈
- 獲得條件：完成對應席位試煉，能獨立判風、判足跡與處理獵獲
- 常見媒介 / 裝備：獵索、剝取刀、標記骨釘
- 專屬戰技 / 術式：`追痕換步`、`試峰定手`
- 流派方向：鷹目遠獵派、蟄蛇伏追派
- 社會印象：能吃苦、會看地、很難被騙

### 營火膳手
- 類別：民間 / 補給 / 公會
- 對應通用職業：靈廚
- 常見所屬機構：靈廚公會、共誓營地、獵隊膳棚
- 獲得條件：得能處理大型獵獲、毒肉邊界與長隊補給
- 常見媒介 / 裝備：大鍋架、獸骨鉤、火坑鏟
- 專屬戰技 / 術式：`獵肉回力湯`、`大營穩火法`
- 流派方向：獵團重鍋派、風雨旅食派
- 社會印象：粗豪、能撐隊伍、深受前線喜愛

## 杜羅山邦

### 爐環重衛
- 類別：軍事 / 關堡守備
- 對應通用職業：重裝守衛、軍陣武士
- 常見所屬機構：關堡軍、爐環守隊
- 獲得條件：通過重裝與關堡輪值考核，需能守火線與搬運工事
- 常見媒介 / 裝備：重盾、爐鎚、壁槍、火號器
- 專屬戰技 / 術式：`爐環壓城步`、`鎚壁反震`
- 流派方向：關堡壓線派、坑口反衝派
- 社會印象：沉穩、結實、不愛廢話

### 密廷匿刻手
- 類別：諜報 / 內審 / 技術保密
- 對應通用職業：情報行者、鍊成工匠
- 常見所屬機構：爐影密廷
- 獲得條件：需有工坊或礦道背景，通過記號、隱刻與反滲透內審
- 常見媒介 / 裝備：匿刻錐、碎印片、火灰囊
- 專屬戰技 / 術式：`匿刻封流`、`灰印斷證術`
- 流派方向：礦道靜刻派、坊間暗審派
- 社會印象：像工具匠，實際很危險

### 誓火工祭師
- 類別：宗教 / 工儀
- 對應通用職業：祭儀執行者、誓印守護者
- 常見所屬機構：祖爐祭列、石契誓碑庭
- 獲得條件：能主持封坑、開爐、見契三類儀式中的至少兩類
- 常見媒介 / 裝備：火誓鉤、環印盤、祭炭匣
- 專屬戰技 / 術式：`爐誓鎮火儀`、`石契封裂式`
- 流派方向：祖爐正火派、封坑鎮脈派
- 社會印象：安靜、莊重、帶有技術神聖感

### 岩律鍛印師
- 類別：學術 / 工藝 / 法度
- 對應通用職業：鍊成工匠、記錄學者
- 常見所屬機構：岩律院、誓紋技師公會
- 獲得條件：需完成工坊師徒期並通過鍛印與規格驗證
- 常見媒介 / 裝備：印模槌、測紋尺、熱脈筆
- 專屬戰技 / 術式：`岩律定紋法`、`鍛印回正式`
- 流派方向：山邦標準派、遠堡實戰派
- 社會印象：專業、可靠、脾氣硬

### 爐膳匠
- 類別：民間 / 工坊補給
- 對應通用職業：靈廚
- 常見所屬機構：靈廚公會、山堡膳坊、關堡鍋所
- 獲得條件：能處理高熱灶具、長工時補給與礦區安全飲食
- 常見媒介 / 裝備：厚鐵鍋、保熱匣、熱石夾
- 專屬戰技 / 術式：`熔鹽回息羹`、`爐腹定火膳`
- 流派方向：工坊輪鍋派、堡衛長灶派
- 社會印象：耐熱、耐煩、救命性很高

## 薩德原聯盟

### 逐原騎
- 類別：軍事 / 騎行
- 對應通用職業：騎戰戰技系、機動斥候
- 常見所屬機構：逐原騎隊、五柱護路隊
- 獲得條件：通過長途騎行、井鏈辨路與回收撤離三重測驗
- 常見媒介 / 裝備：長騎矛、風旗、短弓、水囊甲
- 專屬戰技 / 術式：`逐原斷風衝`、`側旗換騎步`
- 流派方向：長途追風派、護井回折派
- 社會印象：迅捷、能扛責任、草原核心武力

### 風策聽路人
- 類別：諜報 / 路線情報
- 對應通用職業：情報行者、機動斥候
- 常見所屬機構：誓原風策局
- 獲得條件：熟悉兩條以上遷徙線，通過風訊、哨標與口信驗證
- 常見媒介 / 裝備：風片、路牌刀、短哨、信筒
- 專屬戰技 / 術式：`風跡聽線`、`草痕回訊法`
- 流派方向：井鏈靜聽派、外路折返派
- 社會印象：像普通路人，實際很會收消息

### 祖柱送魂者
- 類別：宗教 / 祖靈 / 喪儀
- 對應通用職業：祭儀執行者、誓印守護者
- 常見所屬機構：祖柱祭圈、送魂隊
- 獲得條件：須經祖靈見證，完成送魂、安名與返井儀式
- 常見媒介 / 裝備：柱繩、送魂鈴、風灰袋
- 專屬戰技 / 術式：`返野送魂路`、`柱風安名式`
- 流派方向：五柱正祭派、旅送遠行派
- 社會印象：受敬重、不宜輕慢

### 井譜識路師
- 類別：學術 / 旅路 / 民生
- 對應通用職業：記錄學者、自然共鳴者
- 常見所屬機構：井譜圈、路譜學徒團
- 獲得條件：能背誦井鏈、判草勢、繪製季節路譜
- 常見媒介 / 裝備：井譜冊、風向紗、測水棒
- 專屬戰技 / 術式：`井鏈定向法`、`旱季換路訣`
- 流派方向：水線主譜派、遠牧折路派
- 社會印象：知路、懂節奏、很被隊伍需要

### 旅灶手
- 類別：民間 / 公會 / 補給
- 對應通用職業：靈廚
- 常見所屬機構：靈廚公會、長營灶隊、逐隊伙行
- 獲得條件：需能處理水袋補給、風沙下點火與移動中備食
- 常見媒介 / 裝備：行灶板、風罩鍋、草鹽囊
- 專屬戰技 / 術式：`路火定湯`、`長騎續力餅`
- 流派方向：長營灶派、逐隊旅膳派
- 社會印象：看似後勤，實際很關鍵

## 瑟堤亞棲國

### 棲枝衛
- 類別：軍事 / 邊界護衛
- 對應通用職業：守勢戰技系、自然共鳴者
- 常見所屬機構：棲枝衛隊、邊界陪行列
- 獲得條件：需被允許進入正式衛列，完成低擾護衛與誓網壓制訓練
- 常見媒介 / 裝備：枝盾、柔矛、誓網節杖
- 專屬戰技 / 術式：`枝列緩截`、`誓網壓行式`
- 流派方向：邊界陪行派、林脈守列派
- 社會印象：安靜、克制、壓迫感強

### 暗響靜追者
- 類別：諜報 / 回聲追索
- 對應通用職業：情報行者、自然共鳴者
- 常見所屬機構：暗響監察部
- 獲得條件：需通過偽名識別、低痕跟追與誓網回聲驗證
- 常見媒介 / 裝備：靜步帶、回聲針、孢痕匣
- 專屬戰技 / 術式：`回聲追名`、`靜孢匿跡術`
- 流派方向：邊網靜追派、內林辨偽派
- 社會印象：不大出聲，但誰都知道不好惹

### 命名見證者
- 類別：宗教 / 命名秩序
- 對應通用職業：祭儀執行者、誓印守護者
- 常見所屬機構：長老議庭、命名庭
- 獲得條件：需曾參與命名儀式、受高位見證者認可、不得有重度失名紀錄
- 常見媒介 / 裝備：命名枝環、誓網晶線、見證蘚盤
- 專屬戰技 / 術式：`命名定聲`、`失名束止儀`
- 流派方向：庭證正名派、幼名護持派
- 社會印象：神聖、穩重、地位很高

### 語孢導引師
- 類別：學術 / 共鳴 / 導引
- 對應通用職業：學院施術者、自然共鳴者
- 常見所屬機構：語孢共學圈、導印師列
- 獲得條件：能穩定讀取語孢記憶，且在共鳴環境下不造成過度擾動

### 照膳者
- 類別：民間 / 照護 / 公會
- 對應通用職業：靈廚、治療藥師
- 常見所屬機構：靈廚公會、照護圈、幼枝膳庭
- 獲得條件：需學會安撫膳、低擾調理與共生食材安全處理
- 常見媒介 / 裝備：柔火盤、安膳盒、靜香匣
- 專屬戰技 / 術式：`安枝回息膳`、`靜霧調味術`
- 流派方向：幼枝安膳派、棲護緩膳派
- 社會印象：平靜、可靠、常被低估
