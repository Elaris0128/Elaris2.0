# 東境探索素材流通表

## 用途
這份表用來把東境探索點的主要掉落，接回 `商店收購 / 初步處理 / 工藝用途 / 合法性`。

## 阿瓦隆王國

### 誓王舊渠地宮
- 主要素材：`逆壓銅閘片`、`沉誓濾水石`、`撤離箱暗鎖骨架`
- 常見收購端：銀鋒城誓鋼工坊、港口稅契窗口、下城灰線回收商
- 初步處理：去潮、拆件、洗除舊誓痕、確認是否帶王室舊記號
- 常見製成物：耐潮門扣、地下濾水芯、暗鎖訓練盒、舊渠開鎖件
- 代表配方：`地宮濾水芯`
- 合法性與價格：濾水石 `中價 / 合法`，銅閘片 `中至高價 / 管制`，暗鎖骨架 `高價 / 禁制邊緣`

### 白鹿溪誓裂洞
- 主要素材：`白鹿溪藥蘚`、`誓痕石屑`、`寒地水晶滴`
- 常見收購端：村藥鋪、教廷誓務所、邊境補給站
- 初步處理：陰乾、去泥、分離誓痕與藥性、避免回聲污染
- 常見製成物：止血敷料、寒地敷包、簡易誓痕試片、巡路安神包
- 代表配方：`白鹿溪藥蘚暖粥`、`溪蘚止血包`
- 合法性與價格：藥蘚 `低至中價 / 合法`，石屑 `中價 / 需低調處理`，水晶滴 `中價 / 合法`

## 東境共通判斷
- 同一份素材，`原料價` 和 `加工後價` 可能差到兩級以上。
- 只要牽涉王室舊路、聯邦舊檔、氏族祖骨、山邦封坑、路權證物、棲國語紋，價格就不只是市場問題，而是身分與風險問題。
- 如果玩家沒有合法身分，最容易買賣的是 `基礎藥材、普通礦材、布材與工具料`；最難安全出手的是 `敏感記錄、深層機件、封樣素材與地方權屬證物`。
