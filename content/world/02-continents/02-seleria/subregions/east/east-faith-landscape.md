# 東境宗教與信仰格局

## 總覽
東境不是單一宗教區，而是 `無國教但主流宗教強勢的王國 / 有國教但尊重他信的聯邦 / 部族與祖靈傳承 / 草原誓環共同體 / 非人類文明信仰制度` 並存的地帶。

## 東境宗教走廊
- `誓約與見證帶`：阿瓦隆沿海、騎士領與邊關要塞，最明顯地把誓約神學、繼承與軍事見證綁在一起。
- `條律與正本帶`：瑟恩河港、學院與城邦行政圈，宗教與文本、法理、授證幾乎難以切開。
- `祖靈與圖騰帶`：克羅嵐濕原、試峰和獵道，以祖靈、骨歌與圖騰承認維持秩序。
- `祖匠與火脈帶`：杜羅山脈、礦廊與谷口工坊，工藝成功本身就是宗教性的回應。
- `長路與風靈帶`：薩德原公井、風柱、遷路、誓環聚與返野送魂，把宗教與生存節奏完全重疊。
- `受名與棲網帶`：瑟堤亞霧林與棲地，不以人類教義語言為中心，而是以受名、安歸和共生編排神聖秩序。

## 各國信仰定位
- `阿瓦隆`：無國教，但維拉特教影響極強，尤其軍人與誓守階層。
- `瑟恩`：序律編院聯席為國教核心，但保留城邦地方信仰。
- `克羅嵐`：氏族圖騰、戰祖骨歌與原野先靈為主，無中央國教。
- `杜羅`：祖火、祖匠與山體記憶為主，無中央國教。
- `薩德原`：以曠野誓環、五風靈、祖靈誓柱與返野送魂為主，無單一教團國教。
- `瑟堤亞`：誓王、受名、誓網與棲地共生的文明信仰制度。

## 世界級宗教組織落點
- `誓光見證庭`：阿瓦隆可運作，但非其國教；在跨國盟誓與高位見證上有影響力。
- `序律編院聯席`：瑟恩國教核心，也是東境最制度化的宗教權威。
- `祖燼安魂盟`：對戰後、礦難、古戰場與無名者處理有高實用性。
- `靜燈庇護會`：港市、城市與旅傷安置帶最容易落地。
- `林棲息育園`：可與瑟堤亞合作，但無法代表棲國信仰。
- `輪迴續生會`：在瑟恩與旅人、港市社會中較容易找到聽眾。

## 東境的共同爭議
- `誰有見證權`：阿瓦隆、瑟恩與薩德原都會把見證視為宗教性權力，但語言完全不同。
- `亡者應如何被安放`：祖燼安魂盟、地方祖靈傳承與輪迴續生會經常對同一名死者給出不同答案。
- `生命觀衝突`：誓獸育契、公會式保育、氏族獵律與瑟堤亞共生制彼此難以完全互譯。
- `聖地與資源權`：一個地方究竟是遺跡、礦脈、祖道還是聖地，往往由宗教與組織共同角力。

## 東境整體特徵
- 宗教不是背景裝飾，而是直接進入見證權、葬祭權、樣本權、旅路祝護與合法性爭議。
- 同一位高位神，在阿瓦隆、瑟恩與薩德原會被完全不同的制度和語言理解。
- 東境也是世界級公會與宗教共同體最容易碰撞的地方之一。

## 主持使用建議
- 阿瓦隆最適合拿來做 `誓約 / 繼承 / 軍誓 / 異端見證`。
- 瑟恩最適合拿來做 `教義 / 正本 / 條文 / 研究倫理`。
- 克羅嵐與杜羅最適合拿來做 `祖靈 / 榮譽 / 工藝禁忌 / 外來制度誤讀`。
- 薩德原最適合做 `旅路祝護 / 井鏈秩序 / 群行禁忌`。
- 瑟堤亞最適合做 `非人類文明信仰制度 / 生命網 / 命名權`。
