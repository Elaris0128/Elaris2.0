# 靈爐懸心在地支線

## 支線 1
- 起點：布魯格丹 或 懸爐堂
- 核心問題：一枚重要礦契的刻面被磨掉了一個關鍵符號。
- 常見阻力：工坊保密、礦脈調度與石契責任。
- 可能報酬：工坊通行、器材修整與技術認可。

## 支線 2
- 起點：瑟瑪妮 或 環誓軸廊
- 核心問題：有人在下爐市階兜售沒有登記編號的高品質盾牌。
- 常見阻力：工坊保密、礦脈調度與石契責任。
- 可能報酬：工坊通行、器材修整與技術認可。

## 支線 3
- 起點：多倫嶺錘 或 稅石庫
- 核心問題：懸爐堂某次夜間自燃的火色與往常不同，長老們卻不願公開說明。
- 常見阻力：工坊保密、礦脈調度與石契責任。
- 可能報酬：工坊通行、器材修整與技術認可。
