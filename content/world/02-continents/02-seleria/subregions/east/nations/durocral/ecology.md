# 杜羅山邦魔物生態

## 核心生境
- 山腹坑道、工業水渠、熱脈熔坑與高風關口構成杜羅的立體生境。
- 這裡的魔物生態強烈受熱度、礦物鹽、水壓與人工工程影響。

## 生活用魔物
- `坑廊短角羊`：耐寒家養種，是外層住區肉乳來源。危險 0 / 親近 5。
- `石腳鼬`：常半伴生於工坊邊緣，會獵礦蟲也會偷食。危險 1 / 親近 2。
- `岩盞苔`：濕冷石壁照明苔類。危險 0 / 親近 4。
- `灰芯松`：外圍支柱木料來源魔植。危險 0 / 親近 4。
- `熱尾洞蜥`：深廊熱度指標種。危險 0 / 親近 2。

## 伴生與邊境魔物
- `熔鹽蕈`：可食但高風險採集的熱脈依存魔菌。危險 2 / 親近 2。
- `冷脈石花`：礦植共生體，能判讀坑道失衡。危險 0 / 親近 3。
- `灰羽岩雀`：通風良好外坡與關口常見，是風勢指標。
- `熔脈背蜥`：啃食金屬鹽分的熱脈魔獸。危險 2 / 親近 1。

## 危險與災害魔物
- `灰鍊掘甲蟲`：坑道支柱破壞者。危險 3 / 親近 0。
- `裂爐石魈`：深廊擬石掠食者。危險 4 / 親近 0。
- `食鐵蠕母`：深坑裝備吞食者。危險 4 / 親近 0。
- `蒸喉炭魘`：躲在煙道與廢鍛室的擬影異常。危險 3 / 親近 0。
- `墜錘骨妖`：工坊事故殘響形成的危險聚合體。危險 3 / 親近 0。

## 水系與繁衍特徵
- 杜羅許多魔物的繁殖與熱脈開合、排水節奏及蒸氣壓變化有關。
- `灰鍊掘甲蟲` 會在木支柱最潮濕的季節大量孵化。
- `食鐵蠕母` 偏好裝備鹽分與金屬粉塵累積區，常和無人清理的舊坑共生。

## 生態鏈摘要
- 深廊鏈：岩盞苔 / 熔鹽蕈 -> 礦蟲與小型蜥類 -> 背蜥 / 石魈 -> 蠕母。
- 外坡鏈：灰芯松 / 岩草 -> 短角羊 / 岩雀 -> 高差掠食者與山嚎。
- 工坊鏈：金屬碎屑與潮煙 -> 掘甲蟲 / 炭魘 -> 骨妖等事故異常。

## 旅人注意
- 杜羅的魔物往往和工程事故、坑道管理與水熱失衡一起出現。
- 你看到一隻危險生物時，通常背後已經有更大的結構問題。
