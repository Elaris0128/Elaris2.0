# 薩德原聯盟魔物生態

## 核心生境
- 草海、低丘、井鏈、水槽與祖靈祭地構成薩德原的生命網。
- 薩德原的魔物生態和遷徙、風向、水權與夜間補給密不可分。

## 生活用魔物
- `長草羽穗`：最常見的風向指標魔植。危險 0 / 親近 5。
- `井緣涼葉`：補水與解熱用魔植。危險 0 / 親近 4。
- `灰尾藥蒿`：獸傷與旅傷常備藥草。危險 0 / 親近 4。
- `黃鬃載鹿`：副載與短停搬運用魔獸。危險 1 / 親近 4。
- `逐原羚`：偏野生，但部分族落會合作驅引。危險 1 / 親近 2。

## 伴生與邊境魔物
- `遠鳴地鶴`：天候與水氣變化指標種。
- `夜圈穴兔`：補水點外圍常見小型獵物，是草原中層攝食鏈基礎。
- `夢汀水花`：只在靜水祭地穩定開放的占兆魔植。危險 0 / 親近 2。
- `逐雷獵隼`：沿風暴邊緣狩獵的高空猛禽。危險 2 / 親近 1。

## 危險與災害魔物
- `金鬃踏雲獸`：大型草原靈獸，受驚時會形成壓倒性群行風險。危險 2 至 4 / 親近 2。
- `暮井掠飲蝠`：夜間補水點掠食與驚擾種。危險 2 / 親近 0。
- `草影裂角獸`：繁殖季極危險的大型草獸。危險 3 / 親近 1。
- `井嚎沙髏`：廢井與枯井帶異常。危險 3 / 親近 0。
- `逐原幽唱者`：以歌聲與風聲誤導旅者。危險 3 / 親近 0。

## 水系與繁衍特徵
- 薩德原許多魔物的出沒節奏直接跟井鏈水位和草勢同步。
- `暮井掠飲蝠` 常在缺水季聚向仍有水氣的補水點。
- `金鬃踏雲獸` 的繁殖與遷行帶會受雷季與草量變化影響。

## 生態鏈摘要
- 草海鏈：長草羽穗 / 井緣涼葉 -> 穴兔 / 羚群 -> 裂角獸 / 獵隼 / 大型群靈。
- 井鏈鏈：井邊藥草與水蟲 -> 掠飲蝠與夜行掠食者 -> 枯井異常。
- 祭地鏈：夢汀水花與靜水藻群 -> 小型靈性水生種 -> 占兆與異常干擾生命。

## 旅人注意
- 草原上的危險不是只有追你跑的東西，還包括會把你帶錯方向、拖離水線、耗掉補給的存在。
- 在薩德原，尊重水和風，也等於尊重整套魔物生態。
