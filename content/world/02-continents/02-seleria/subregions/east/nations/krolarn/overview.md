## Summary
克羅嵐聯盟國是由十獸氏族共構的內陸邦聯。這裡沒有穩固到窒息的中央王權，真正統合各族的是共同戰場、共誓儀式與對榮譽的高度敏感。

## Core Features
- 地形由高嶺山脈、台地草原、裂谷濕原與河澤航道交錯組成
- 透過 `十獸環誓會議` 維持盟合，不同氏族保有鮮明傳統與地盤
- 十獸不是統稱，而是 `馭角 / 逐影 / 鷹目 / 裂牙 / 斑影 / 隱耳 / 馭嶺 / 蟄蛇 / 裂痕 / 冥羽` 十個席位氏族共同構成的政治與文化骨架
- 法律與政治深受誓約、圖騰、試煉與部族見證影響
- 是東境西北側最難用書面規則完全理解的國家，但也最不容易被單一外力吞下

## Major Populations
- 以獸人各氏族為主，氏族間體貌、戰技與圖騰信仰差異很大
- 有少量人類、矮人與商旅聚落沿河埠和集鎮停留
- 祭司、獵手、戰舞者、風信使與圖騰守衛在社會中地位很高

## Adventure Design
- 適合做部族外交、試煉、獵祭、河澤衝突、榮譽調停與邊境戰事
- 很適合讓玩家第一次體驗「文化誤解不等於敵意」的東境玩法
