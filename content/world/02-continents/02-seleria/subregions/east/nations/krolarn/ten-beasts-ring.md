# 克羅嵐聯盟國十獸之環

## 定位
- `十獸之環` 是克羅嵐聯盟國最核心的共誓結構。
- 它不是十個長得像動物的氏族集合，而是十個有自己棲地、嚎語、圖騰、戰法、政治習慣與榮譽語言的席位共同體。
- 克羅嵐能維持聯盟國形狀，不靠王座，而靠這十個席位彼此制衡、彼此見證。
- 每一獸都會參照對應現實動物的 `群體性 / 狩獵法 / 領域感 / 育幼方式 / 季節節奏` 來塑造文化，但不代表獸人只是穿著皮毛的動物化身。

## 十獸之環不是什麼
- 不是十個平均分權、所有事情都一人一票的單純議會。
- 不是十種只看外貌差異的獸人亞種標籤。
- 不是所有氏族都能隨時互換職能的鬆散部落聯盟。
- 不是一套低地人能靠官印、條文和血統表完全讀懂的制度。

## 十獸如何運作
- 每一獸不只是圖騰，也是 `席位、氏族傳承、地盤責任、嚎語系統與誓約身分`。
- 聯盟大事由 `十獸環誓會議` 裁定，但不同獸席在不同議題上的話語權並不相同。
- 戰時通常由 `裂牙、裂痕、馭角、馭嶺` 等重地與重軍氏族提高聲量。
- 邊境與情報議題常由 `逐影、鷹目、隱耳、蟄蛇` 佔優。
- 祖靈、禁地、亡者、凶兆與聖地問題則常由 `冥羽、斑影、骨語系見證者` 主導。

## 十獸建模原則
- `馭角 / 馭嶺` 偏群行草食大型獸，強調隊形、護幼、遷徙記憶與路線穩定。
- `逐影 / 斑影 / 裂痕` 偏掠食獸，強調狩獵節奏、領域界線、耐心觀察與決定性撲殺。
- `鷹目 / 冥羽` 偏猛禽與夜鳥，強調視野、聽覺、制高點、單位訊號與時機判斷。
- `隱耳` 偏荒原兔群，強調耳警、伏身、曲折逃線、洞藏與弱者轉移。
- `蟄蛇` 偏蟒蛇與伏擊型爬行獸，強調耐心、蜕變、伏線、絞殺式審問與長時間監視。
- `裂牙` 偏野豬，強調正面衝擊、群體暴走、翻地奪路與短距離爆發。
- 十獸文化採用動物習性作為 `社會模型`，不是外貌換皮，所以同一獸席內仍會出現祭司、商隊主事、骨語者、戰首、教師與護幼者。

## 十獸共誓層級
- `大環主誓使`：主持會盟、完成共誓成案，決定何時讓十獸正式進入同一個答案。
- `主誓使`：代表各獸席在大會盟發聲，對外也常被視為氏族的臉面。
- `族印代表`：處理名冊、界標、獵場與實務分工，是席位落地的重要中層。
- `見證圈`：由祭司、獵場見證人、骨語者、試煉主持者構成，負責把榮譽秩序說成可執行的規矩。
- `地方火圈與聚地領頭`：真正把十獸差異落在村、營、埠、崖路與濕地裡的人。

## 十獸席位詳解

### 1. 馭角氏族
- 原型與氣質：原野野牛、角鹿，穩、重、耐壓，是最像聯盟骨架的一席。
- 現實動物特性映射：
  - 參照野牛與大型角鹿的 `群體護幼`，所以馭角最重護送、運載與隊列不崩。
  - 參照草食大型群獸的 `遷徙記憶`，因此他們對水源、草場與長程補給線的記憶最穩。
  - 參照角獸的 `正面頂撞與半弧防線`，所以戰時擅長扛第一波壓力，而不是花式突襲。
- 外觀辨識：粗厚背肩、額骨隆起或短角、胸口常掛獸骨護符，膚色多見土赭或深棕，脊背常有短鬃線。
- 人數與壽命：`5,000 至 7,000` 人，常見壽命 `70 至 90 載`，老角長者可達百載。
- 成長階段：`幼角期（0–10） / 角徒期（11–25） / 成角期（26–60） / 老角期（60+）`
- 主要棲地與路線：以 `斷風試峰`、`巨蹄踐痕原` 為核心，掌握北部中央高地到西南蹄痕原野的重隊路。
- 群體結構：多採 `領角長 / 押後長 / 護幼圈 / 重蹄列` 的大群體系，真正重要的是誰能在亂陣時穩住整條隊。
- 領域與警戒：以可長期使用的路、泉、糧草地為核心，不喜歡被切碎補給線。
- 育幼與成年：年幼者多在大群中一起被照看；成年禮重 `扛重、領隊、護行不失名`。
- 最常承接的國家職能：大會盟秩序、重隊調動、長距離搬運、草原與高地的正面守勢。
- 常見社會角色：大環主誓使、重蹄護衛、盟道車隊主事、營鎮糧皮調度者、護幼列長。
- 嚎語與稱呼：使用 `Ghar-Horn` 嚎語，厚重咽音與鼻腔共鳴很強，講話像擊胸定節拍。
- 誓儀與禁忌：重視 `角骨護符、重蹄踏誓、群隊先鋒見證`；在共隊運載與會盟序列中擅自越位，是極重的失禮。
- 對外印象：可靠，但不輕易讓外人碰聯盟核心。

### 2. 逐影氏族
- 原型與氣質：草原狼、原野狐，擅夜獵、側行與耐心觀察。
- 現實動物特性映射：
  - 參照狼群的 `輪值斥候與協同圍獵`，因此逐影特別擅長接力跟監、包抄與逼出錯位。
  - 參照狐類的 `機敏轉向與臨場變通`，所以逐影比其他獸席更能接受灰色談判與臨機偽裝。
  - 參照犬科的 `氣味標記與夜間呼應`，使他們對邊線、暗號與路權痕跡極敏感。
- 外觀辨識：尖立耳、短蓬尾、灰紋斑背、琥珀眼，動作常比話更早抵達。
- 人數與壽命：約 `4,000` 人，常見壽命 `60 至 80 載`。
- 成長階段：`幼影期（0–8） / 潛影期（9–20） / 影狼期（21–50） / 遺影期（50+）`
- 主要棲地與路線：`影息藍塹`、`裂脈獵原`，控制東南隱谷、裂谷草原與多條灰色導路。
- 群體結構：常見 `前嗅、側追、回影、守火` 四類角色，小群分工明確。
- 領域與警戒：不一定硬守固定地面，而是守 `誰能走哪條影路`。
- 育幼與成年：年少者先學低嚎與聞痕，再學真正潛行；成年禮重 `獨追不暴露、合圍不失默契`。
- 最常承接的國家職能：潛行、邊線觀察、夜路追跡、低聲翻譯與誤導敵手。
- 常見社會角色：影獵、邊霧耳目、外語轉譯者、灰路嚮導、滲透回收者。
- 嚎語與稱呼：使用 `Var-Shade` 嚎語，滑音尾挑、忽遠忽近如風鳴，外人很難只靠音量判斷距離。
- 誓儀與禁忌：重 `裂影低嚎、夜獵名誓、影息守火`；最忌在未經允許的影路上留下可追認的重痕。
- 對外印象：最懂外人，也最不容易真正相信外人。

### 3. 鷹目氏族
- 原型與氣質：高原猛禽，視野最好，也最擅長把零散消息拼成整體局勢。
- 現實動物特性映射：
  - 參照猛禽的 `制高點習性`，所以鷹目凡事先問高點在哪、風從哪裡來。
  - 參照鷹隼的 `俯衝決斷`，使他們平日很冷，但一旦出手會追求一次打準。
  - 參照大型鳥類的 `築巢與配對守點`，因此他們特別重雙人搭檔與觀測位傳承。
- 外觀辨識：耳羽或羽冠感明顯、銀瞳或淡金瞳、頸背常有細羽紋，指節與手勢比多數氏族更利落。
- 人數與壽命：約 `3,500` 人，常見壽命 `70 至 85 載`。
- 成長階段：`幼羽期（0–10） / 風眼期（11–25） / 鷹冠期（26–55） / 靜羽期（55+）`
- 主要棲地與路線：`戰歌嶺坡`、`長歌石海`，守北境孤嶺石林與西北古道遠望點。
- 群體結構：多採 `高瞳守 / 風號手 / 岩巢記錄人 / 下坡傳訊者` 的搭檔制。
- 領域與警戒：比起地盤，他們更重 `看得到哪裡` 與 `誰能先傳到訊`。
- 育幼與成年：幼者先學辨風、認距、守崖；成年禮重 `單夜判完一條路的所有風險`。
- 最常承接的國家職能：高地觀測、遠望、風號系統、路線判讀、戰前預警。
- 常見社會角色：風號長、天瞳守望者、古道引路者、遠觀記錄人、界丘判視者。
- 嚎語與稱呼：使用 `Aerk-Talon` 嚎語，尾音拖長如風鳴，嚎唱常會把地形當擴音器。
- 誓儀與禁忌：重 `迎風立羽、獵前高鳴、遠視誓語`；在沒有完成觀測的情況下先下結論，被視為失去高處資格。
- 對外印象：看得遠，也因此不容易被短期利益說服。

### 4. 裂牙氏族
- 原型與氣質：荒野野豬，正面衝擊、狠、快，一旦決定開戰就很少半途收手。
- 現實動物特性映射：
  - 參照野豬的 `短距離爆發與翻地蠻力`，所以裂牙擅長在混亂中硬開缺口。
  - 參照群豬的 `護幼與反擊凶猛`，因此他們在守營、守糧、守傷者時反而最可怕。
  - 參照野豬的 `地面感知與根挖習性`，使裂牙對泥地、坡腳與障礙突破很有天分。
- 外觀辨識：下顎獠牙、鼻樑隆起、肩甲粗斑、灰棕膚色，很多人從站姿就能看出戰首壓迫感。
- 人數與壽命：約 `3,000` 人，常見壽命 `50 至 70 載`。
- 成長階段：`幼獠期（0–6） / 初牙期（7–20） / 巨牙期（21–40） / 殘牙期（40+）`
- 主要棲地與路線：`裂吼蹄谷`、`血染角環`，控制南端深谷與戰丘凹谷的重衝路線。
- 群體結構：多採 `獠首 / 壓線隊 / 護崽圈 / 翻地工` 的重撞式分工。
- 領域與警戒：以泥牆、翻土痕與撞裂樁標記邊界，很少做太細的外交緩衝。
- 育幼與成年：幼者先學挨撞與挖根，成年禮重 `受擊不退` 與 `衝線不亂群`。
- 最常承接的國家職能：衝陣、破線、重壓突擊、戰時動員、前線壓制。
- 常見社會角色：戰首、血咬勇者、裂線隊長、重甲衝陣手、戰後埋牙見證人。
- 嚎語與稱呼：使用 `Thruk-Tusk` 嚎語，下顎顫鳴與磨牙撞牙是明顯特徵。
- 誓儀與禁忌：重 `裂牙初試、血咬列名、失誓埋牙`；最忌臨陣改名與讓他人替自己承擔前鋒羞辱。
- 對外印象：最容易被低地人誤認成只有蠻力的一席，但其實最重戰場榮譽。

### 5. 斑影氏族
- 原型與氣質：草原花豹，優雅、獵感強、擅長以最省力的方式取得勝利。
- 現實動物特性映射：
  - 參照大貓科的 `獨獵與爆發撲殺`，所以斑影不喜歡低效持久纏鬥。
  - 參照豹類的 `拖獵與藏食`，使他們擅長戰利處理、獵場裁分與追跡證物保存。
  - 參照貓科的 `靜伏試探`，所以斑影的禮節總帶一點探獵意味。
- 外觀辨識：花斑膚紋、短圓耳、金瞳、修長四肢，衣著與配飾比多數獸席更講究獵痕秩序。
- 人數與壽命：約 `2,800` 人，常見壽命 `50 至 70 載`。
- 成長階段：`幼斑期（0–7） / 影斑期（8–20） / 斑獵期（21–40） / 斑魂期（40+）`
- 主要棲地與路線：`碎咬石坪`、`獵痕環徑`，掌握東部碎岩草原與中部丘原獵環。
- 群體結構：平時多為小圈或個別獵手，只有裁獵、決鬥與大追索時才暫時合圈。
- 領域與警戒：重 `獵圈` 多於住地，對誰有資格進哪片獵圈極其敏感。
- 育幼與成年：幼者先學貼地、藏味、帶回獵物；成年禮重 `獨獵成圈` 與 `不讓獵物白死`。
- 最常承接的國家職能：追獵、獵場裁定、榮譽決鬥見證、邊獵爭議與快速小隊行動。
- 常見社會角色：獵場見證者、斑獵長、決鬥裁誓人、追獵者、分血主持者。
- 嚎語與稱呼：使用 `Karr-Spot` 嚎語，節奏輕疾斷續，像正在繞圈觀察獵物。
- 誓儀與禁忌：重 `斑痕留名、獵獲分血、影步試圈`；無資格者擅入他氏狩圈，極易引發血誓。
- 對外印象：很有禮，但那種禮往往帶著試探與壓力。

### 6. 隱耳氏族
- 原型與氣質：荒原兔，不弱，只是把生存變成聽、伏、閃與先一步察覺。
- 現實動物特性映射：
  - 參照野兔的 `耳警與瞬間折返`，所以隱耳是克羅嵐最會先聽見危險的一席。
  - 參照兔群的 `多巢與分散藏糧`，因此他們把物資轉移、幼者疏散與暗丘洞點做得最好。
  - 參照兔類的 `不以硬碰硬求生`，所以隱耳文化特別重「活著帶人回來」。
- 外觀辨識：耳長可伏、背脊灰白或赭紋、足踝強韌、尾短白尖，整體存在感不高卻很難真正忽視。
- 人數與壽命：約 `2,000` 人，常見壽命 `55 至 75 載`。
- 成長階段：`幼耳期（0–10） / 躍耳期（11–20） / 影耳期（21–50） / 隱跡期（50+）`
- 主要棲地與路線：`灰囊深丘`、`百骨風埡`，掌握西方黑灰凹原與獸骨崖谷之間的低丘躲線。
- 群體結構：多採 `耳站 / 洞藏 / 轉送 / 聽地` 的鬆群制，每條逃線都有人知道下一個洞點。
- 領域與警戒：比起「守住土地」，他們更重「保住通往安全處的網」。
- 育幼與成年：幼者從很早就開始訓練伏耳、躍步和無聲停身；成年禮重 `避災不亂、護幼不漏人`。
- 最常承接的國家職能：聽風、伏行、低丘暗道、物資避藏、孩童與弱者轉移。
- 常見社會角色：路耳哨、風埡聽地者、遷群看守、藏糧記路人、避災轉送者。
- 嚎語與稱呼：使用 `Murn-Ear` 嚎語，輕柔吐氣與拍足同鳴是辨識重點。
- 誓儀與禁忌：重 `影耳伏聽、百骨避誓、風埡靜立`；最忌在避災路上大聲爭執或擅自改動藏糧點。
- 對外印象：常被低估，但最容易在你以為無人看見時先一步知道你來了。

### 7. 馭嶺氏族
- 原型與氣質：荒原山羊，堅韌、能扛、最懂難走的坡與最不穩的路。
- 現實動物特性映射：
  - 參照山羊與高崖偶蹄獸的 `高差記憶`，因此馭嶺最擅難路、窄崖與驮運。
  - 參照角撞排序的 `頭位競逐`，所以他們尊敬真能帶隊穿險的長者，不是只看血統。
  - 參照山地群獸的 `幼者隨列學路`，讓馭嶺文化非常重帶新手上路。
- 外觀辨識：額短彎角、下顎稀鬃、淺灰赭白膚色、足踝與膝部特別厚實。
- 人數與壽命：約 `4,500` 人，常見壽命 `65 至 80 載`。
- 成長階段：`幼角期（0–8） / 嶺徒期（9–20） / 嶺首期（21–55） / 嶺鬚期（55+）`
- 主要棲地與路線：`聖焰嶺紋`、`暮角嶼丘`，主控中部高原熱丘、多丘嶺地與危崖補給線。
- 群體結構：多採 `領崖者 / 角列 / 驮背隊 / 收尾索` 的長列行進制。
- 領域與警戒：他們守的是坡、梁、崖嘴與能否過隊，不是單一聚地。
- 育幼與成年：年少者先學跟列、記足點、辨落石；成年禮重 `帶一整列人安全過險地`。
- 最常承接的國家職能：山路維護、高低差行軍、驮運、危崖哨站與艱困生存。
- 常見社會角色：嶺道守、驮隊長、崖路導者、石坡築障人、高差補給頭。
- 嚎語與稱呼：使用 `Ghram-Ridge` 嚎語，尾音顫抖如山風，石巖回音很強。
- 誓儀與禁忌：重 `角跡巡嶺、崖前定名、暮角留火`；最忌讓隊伍在自己答應能帶過的路上失足無名。
- 對外印象：不多話，但通常是最能把隊伍帶回來的人。

### 8. 蟄蛇氏族
- 原型與氣質：草原蟒，慢、靜、隱，擅長等敵人自己走進錯誤。
- 現實動物特性映射：
  - 參照蟒蛇的 `長時間伏擊`，所以蟄蛇特別擅長靜待、審問與設置延時陷阱。
  - 參照蛇類的 `蜕皮`，因此他們文化裡很重換名、藏名與身份過渡。
  - 參照爬行掠食者的 `熱點敏感與近身絞制`，使他們對壓力、呼吸和微動作很敏感。
- 外觀辨識：淺鱗光肌膚、暗紋背脊、細伏耳、縱瞳，很多成員在動作上極少浪費。
- 人數與壽命：約 `1,500` 人，常見壽命 `50 至 70 載`。
- 成長階段：`幼鱗期（0–8） / 潛蛇期（9–20） / 影蛇期（21–45） / 亡蛇期（45+）`
- 主要棲地與路線：`骨霧塚境`、`風槍裂丘`，最擅長東北丘陵霧谷、濕地伏線和碎岩裂口。
- 群體結構：多採 `伏線師 / 毒辨人 / 靜候者 / 蜕名見證` 的細線制。
- 領域與警戒：不愛吵著宣示邊界，而是讓你進到不該進的地方後才知道越界。
- 育幼與成年：幼者先學靜息、辨熱、收身；成年禮重 `連續潛伏與蜕名不露破綻`。
- 最常承接的國家職能：濕地伏線、毒與藥的邊界、耐久監視、審問與陷阱設置。
- 常見社會角色：蛇影伏守、濕原審訊者、藥毒辨識人、裂丘潛線手、靜候截流者。
- 嚎語與稱呼：使用 `Ssil-Skin` 嚎語，滑腔吐舌音與縱環節奏很強，近距離極具壓迫感。
- 誓儀與禁忌：重 `蜕誓、伏息、裂丘靜行、低聲繫名`；最忌在還沒收束的伏線上先說結論。
- 對外印象：最像陰影的一席，也最常被人誤會成不夠光明正大。

### 9. 裂痕氏族
- 原型與氣質：高原虎，是克羅嵐最強硬、也最容易被推出去成為戰時旗面的席位之一。
- 現實動物特性映射：
  - 參照虎類的 `單獵巡領`，所以裂痕特別看重誰真的有資格單扛一線。
  - 參照大型貓科的 `決定性撲殺`，讓他們在戰場上追求一擊定局，而不是細碎消耗。
  - 參照強領域猛獸的 `標記與巡線`，因此裂痕對邊界、旗面與名聲異常敏感。
- 外觀辨識：灰黑虎斑、背脊裂紋狀暗紋、肩背短鬃、金琥珀瞳，壓場感極強。
- 人數與壽命：約 `2,200` 人，常見壽命 `80 至 100 載`。
- 成長階段：`幼斑期（0–10） / 裂紋期（11–30） / 虎魂期（31–70） / 冥裂期（70+）`
- 主要棲地與路線：`靈蹄飲澤`、`狂飆喉嶺`，守北境濕原、天雷走廊與危線正面通道。
- 群體結構：平日偏 `單核帶小隊`，戰時才會短暫聚為大旗。
- 領域與警戒：最重 `誰守哪條正面線`，失守不只丟地，還丟名。
- 育幼與成年：幼者先學巡線、低伏、讀風暴；成年禮重 `獨守一段危線不失位`。
- 最常承接的國家職能：正面威懾、戰時共誓、前鋒總召、雷暴地帶與危線守勢。
- 常見社會角色：裂痕戰首、虎魂守嶺者、雷喉先鋒、重誓決行者、戰時壓線主將。
- 嚎語與稱呼：使用 `Kharr-Tiger` 嚎語，吼音延展後會突然收束，像雷前裂空。
- 誓儀與禁忌：重 `裂紋開名、雷喉立誓、虎魂裂行`；最忌在共誓旗面前退縮卻不承認退縮。
- 對外印象：代表聯盟最尖銳的一面，但也最被寄望在戰時扛住第一波。

### 10. 冥羽氏族
- 原型與氣質：夜梟，與夜、亡者、遠聽、骨語和錯誓後果的距離最近。
- 現實動物特性映射：
  - 參照夜梟的 `極靜飛行與聽位`，使冥羽特別擅長夜守、靜聽與先兆判讀。
  - 參照梟類的 `吐骨與消化殘留`，所以冥羽對骨、殘留、未說完的故事有天然關聯。
  - 參照夜鳥的 `夜域專精`，因此他們在黑暗裡比白天更像自己。
- 外觀辨識：耳羽環冠、寬大夜瞳、頸後羽環紋、修長指節，很多成員在夜裡比白天更像自己。
- 人數與壽命：約 `1,000` 人，常見壽命 `70 至 100 載`。
- 成長階段：`幼羽期（0–12） / 夜語期（13–30） / 冥鳴期（31–70） / 冥根期（70+）`
- 主要棲地與路線：`獸神古原`、`骨霧塚境`，掌握深原林帶、祖地夜路與亡者回名的核心圈。
- 群體結構：多採 `聽祭者 / 守夜者 / 回名火圈 / 夜瞳記骨` 的小圈傳承制。
- 領域與警戒：守的是夜裡的靜地、骨塚與不能亂動的回名路。
- 育幼與成年：幼者先學夜聽、辨骨、守火；成年禮重 `整夜守靜而不錯聽一個名字`。
- 最常承接的國家職能：骨語占兆、祖地禁忌、亡者回名、夜間守望、對凶兆的先期判讀。
- 常見社會角色：骨語長母、冥羽聽祭、回名守火人、夜瞳誓者、夢息巡守者。
- 嚎語與稱呼：使用 `Murn-Owl` 嚎語，低嚎夜林迴旋，尾音像從樹梢翻回來。
- 誓儀與禁忌：重 `冥羽守夜、骨聲聽兆、回名與埋名分流`；最忌在未經允許的情況下擅闖夢息地與祖骨塚圈。
- 對外印象：最神秘，也最常掌握聯盟不願對外公開的那一面。

## 十獸與克羅嵐主要聚地對照
- `獸魂裂丘`：十獸共用核心，但 `馭角 / 裂痕 / 冥羽` 的影響最大，分別對應會盟骨架、戰時旗面與骨語合法性。
- `河牙會埠`：`逐影 / 鷹目 / 裂痕 / 斑影` 最活躍，分別對應暗線、遠望、水路判讀與爭議裁定。
- `斷風試峰營地`：`馭角 / 裂牙 / 馭嶺` 最常被看見，是試煉、重隊與艱地訓練的三重核心。
- `影息藍塹`：`逐影 / 鷹目 / 隱耳 / 蟄蛇` 共同構成邊線秩序。
- `巨蹄踐痕原營鎮`：`馭角 / 裂牙 / 馭嶺` 主導軍鎮節奏，外來伴獸勢力也常先碰到這三席。
- `霧骨灘村`：`隱耳 / 蟄蛇 / 冥羽` 的氣質最重，偏生存、濕地與前兆。
- `石蹄窩聚`：`馭角 / 馭嶺` 影響最大，因為這裡最重養群、搬運與艱地守成。
- `狼歌坡村`：`逐影 / 斑影 / 鷹目` 最常出現，既有獵村感，也有風信和禁歌傳承。
- `雷痕界丘`：`裂痕 / 鷹目 / 逐影` 最常為此爭執或共守，因為這裡既關係邊線也關係誰有資格先說話。

## 十獸之間的關係
- `馭角 × 馭嶺`：常一起扛聯盟骨架、行軍與重隊秩序。
- `逐影 × 鷹目`：一內一外，一近一遠，掌握邊線與預警節奏。
- `裂牙 × 裂痕`：戰時最容易被推成主力，但也最常因榮譽順位摩擦。
- `斑影 × 隱耳`：都擅移動與觀察，只是前者偏獵，後者偏聽。
- `蟄蛇 × 冥羽`：最常一起處理禁地、凶兆、失序與不能公開的事。
- `馭角 × 裂牙`：合作時能扛住正面戰，但若榮譽順位談不攏也最容易互不相讓。
- `鷹目 × 裂痕`：常在界標與邊線上一起工作，但前者重證據、後者重決斷，風格差異大。

## 在遊戲裡怎麼用十獸
- 若寫克羅嵐高層人物，先決定他屬於哪一獸席，因為這會直接影響說話方式、榮譽語言、禁忌與政治立場。
- 若寫地方包，優先讓聚落與路線能對應至少一到兩個獸席，而不是把整個聯盟寫成同一種獸人文化。
- 若寫任務，十獸差異最適合用在 `會盟 / 路權 / 榮譽 / 獵場 / 邊線 / 祖地 / 伴獸 / 風號` 這八種衝突上。
- 若寫玩家角色，`獸人 / 十獸之環・克羅嵐共誓支` 還不夠，最好再往下補一層 `十獸席位`。
- 若寫角色細節，優先先決定這個角色更像該席位中的哪一類：
  - `護幼者 / 領路者 / 斥候 / 守崖者 / 獵裁者 / 回名者`
  - 這會比只寫「他是某某獸席的人」更有畫面。
