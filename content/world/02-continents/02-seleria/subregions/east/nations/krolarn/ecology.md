# 克羅嵐聯盟國魔物生態

## 核心生境
- 高嶺、台地、裂谷、濕原與河埠共同構成克羅嵐生態。
- 聯盟的氏族不是征服自然，而是和特定獵場、濕地、圖騰丘與風帶建立互相承認的共存關係。

## 生活用魔物
- `裂原角羊`：高地放牧與祭用來源。危險 1 / 親近 3。
- `濕蹄原牛`：濕地與泥灘負重牲口。危險 1 / 親近 4。
- `河灘灰獺`：半野生伴生種，會偷工具但也會追魚驅蟲。危險 1 / 親近 2。
- `骨風藤`：綁具與輕繩用魔植。危險 0 / 親近 4。
- `霧沼苦花`：藥性與毒性取決於採集時機的濕地魔植。危險 1 / 親近 2。

## 伴生與邊境魔物
- `雷痕蘚`：導電焦土魔蘚，是判讀雷擊帶的重要指標。
- `風脊長尾雉`：高嶺與丘地常見，也是祭羽來源。
- `灰帳裂狐`：擅長誤導追蹤與偷食，是邊境隊伍最煩的小型種。危險 2 / 親近 0。
- `霧骨螯蟹`：低灘築殼種，退潮時最活躍。危險 2 / 親近 1。

## 危險與災害魔物
- `雷痕泥鰻`：能感應地脈異常的泥水掠食者。危險 3 / 親近 0。
- `裂脊食霧犬`：低霧時成群狩獵的高風險種。危險 3 / 親近 0。
- `骨鈴藤魘`：寄生在古圖騰木上的危險魔植異種。危險 3 / 親近 0。
- `河牙巨背鼉`：深曲流頂層壓制者。危險 4 / 親近 0。

## 水系與繁衍特徵
- 河牙曲流與濕原退潮線每季都會改變，因此許多魔物的巢位與繁殖帶並不固定。
- `霧沼苦花` 與 `雷痕蘚` 都高度依賴異常泥色與雷後土壤。
- `裂脊食霧犬` 會在霧最厚時驅趕獵物往軟灘和淺坑去。

## 生態鏈摘要
- 濕地鏈：雷痕蘚 / 苦花 -> 灘蟲與小型泥棲種 -> 螯蟹 / 泥鰻 -> 巨背鼉。
- 高嶺鏈：骨風藤 / 山草 -> 角羊 / 長尾雉 -> 食霧犬與高地獵手。
- 古木鏈：圖騰木 -> 寄生藤魘 -> 夜間誤導與誘捕事件。

## 旅人注意
- 克羅嵐的危險很多時候不是體型，而是地勢、霧、退潮和本地魔物對環境的熟悉程度。
- 若本地人叫你不要踏某種泥色，通常不是迷信，而是生態記憶。
