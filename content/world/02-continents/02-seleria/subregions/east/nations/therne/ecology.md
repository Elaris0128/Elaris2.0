# 瑟恩聯邦國魔物生態

## 核心生境
- 聯邦腹地由大河、河港、高臺議區、農耕帶與西北工藝山麓組成。
- 瑟恩的生態不是野性被消除，而是大量魔物被制度化管理、觀測、記錄與利用。

## 生活用魔物
- `星鞘蘆`：河灣纖維魔植，可作紙材與固岸結構。危險 0 / 親近 4。
- `穗光穀`：南部主糧魔植。危險 0 / 親近 5。
- `聯道馱鹿`：中距離轉運主力魔獸。危險 0 / 親近 5。
- `河階墨鴨`：水質指標鳥類，港務與藥師都會觀察牠們。危險 0 / 親近 4。
- `煙紋苔`：吸附鍊金殘塵的濾化菌苔。危險 0 / 親近 3。

## 伴生與邊境魔物
- `靜脈葉`：療養丘野常見安神魔植，是醫藥與修養產業核心。
- `塔簷星燕`：與高牆與書樓共棲，會改變紙蟲與小型魔蟲密度。
- `河鏡步蜃`：常在黎明與黃昏誤導船夫。危險 2 / 親近 0。
- `抄墨紙蠹`：以舊紙與潮庫為食的小型魔蟲，是書庫維護頭痛來源。危險 1 / 親近 0。

## 危險與災害魔物
- `沉燈水魘`：夜河與失事船邊的高風險異常。危險 4 / 親近 0。
- `河牙巨背鼉`：極少深入瑟恩腹地，但洪潮期可能沿大河上行。危險 4 / 親近 0。
- `抄影紙魍`：由書庫殘留靈息與潮濕文書形成。危險 2 / 親近 0。
- `契墨鴉群`：在港務與文書城市尤其擅長造成混亂。危險 1 / 親近 1。

## 水系與繁衍特徵
- 大河洪潮會重排蘆束、黏泥與小型水棲魔物的棲地，因此瑟恩的河港生態帶有強烈季節性。
- `星鞘蘆` 需要穩定河泥和緩流，適合形成高承載生境。
- `沉燈水魘` 多和事故點、廢港石階與長期未清理的水下遺構共生。

## 生態鏈摘要
- 河灣鏈：星鞘蘆 -> 河階墨鴨 / 小型蘆蟲 -> 河鏡步蜃與夜河異常。
- 書庫鏈：紙材、潮氣與紙蠹 -> 抄影紙魍。
- 工坡鏈：煙紋苔吸附殘塵 -> 小型濾食種 -> 污染依存異常。

## 旅人注意
- 在瑟恩，很多魔物風險不在荒野，而在倉庫、書樓、碼頭和閘門邊。
- 聯邦越擅長管理生態，越容易在管理斷點養出新的異常。
