# 阿瓦隆王國魔物生態

## 核心生境
- 內陸誓道平原、東南沿海港帶、北境霜濕邊界與西境鋒石高地共同構成阿瓦隆的魔物生態圈。
- 王國對自然的態度不是排除魔物，而是把高親近度與高利用度的魔物盡可能納入秩序。

## 生活用魔物
- `誓禾穗`：主糧魔植。危險 0 / 親近 5 / 平原農帶基礎。
- `誓田耕角牛`：耕作與拖運役獸。危險 0 / 親近 5。
- `北線灰背駿`：巡邏與傳令坐騎。危險 1 / 親近 4。
- `塔崖信鷹`：塔鎮與高地訊禽。危險 0 / 親近 4。
- `燈岬鹽蘆`：沿海防潮與飼料用魔植。危險 0 / 親近 5。

## 伴生與邊境魔物
- `白鹿溪藥蘚`：溪石藥用蘚類，是醫療與旅人補給的重要來源。
- `沿潮白喙鷺`：沿海潮勢與魚汛指標種。危險 0 / 親近 3。
- `契墨鴉群`：會叼走亮面小物與封蠟，是典型邊城伴生麻煩種。危險 1 / 親近 1。
- `橋縫灰螫蟻`：愛在橋基與舊木樁築群，平時只是麻煩，一旦過盛就影響橋道維護。危險 1 / 親近 0。

## 危險與災害魔物
- `誓痕鬃狼`：常沿舊誓場、棄營與軍路狩獵。危險 3 / 親近 0。
- `夜旗骨獒`：古軍犬野化後的群居威脅。危險 3 / 親近 0。
- `霜蹄泥鬼`：北境濕地模仿求援足跡的異常種。危險 3 / 親近 0。
- `潮箱寄貝`：港務害種，會腐蝕船底與貨箱。危險 2 / 親近 0。

## 水系與繁衍特徵
- 王誓河支流周邊的魔植多依賴春汛更新與淤泥礦物。
- `潮箱寄貝` 會在潮汐與木材鹽分合適時大規模附著繁殖。
- `誓痕鬃狼` 偏好在舊營、誓碑與乾草存地附近築窩，顯示牠們對人類秩序殘留有異常依存。

## 生態鏈摘要
- 平原鏈：誓禾穗 -> 耕角牛 / 穗田齧獸 -> 誓痕鬃狼 -> 腐食蘚類。
- 河港鏈：白鹿溪藥蘚 / 鹽蘆 -> 沿潮小魚群與白喙鷺 -> 潮箱寄貝與夜河異常。
- 北線鏈：霜濕蘚群 -> 泥棲小獸 -> 霜蹄泥鬼與骨獒殘群。

## 旅人注意
- 阿瓦隆真正危險的不一定是荒野，而是被秩序半控制、半遺漏的邊線地帶。
- 在王國境內看到魔物，不代表出了文明；很多魔物本來就是文明的一部分。
