# 瑟堤亞棲國生態與棲裔分類

## 稱呼原則
- `棲裔`：被誓王賜名、納入棲國秩序的文明主體。
- `魔物`：棲國對一般自然生命與未受名個體的常用稱呼。
- `失衡異種`：誓網失衡、污染、裂變或殘骸混成後的危險存在。
- 對外檢索時，外地仍可能把瑟堤亞整體放進廣義魔物生態索引，但棲國內部不會把棲裔降回單純魔物稱呼。

## 核心生境
- 誓息霧林的樹冠、菌坡、溪網、孢霧與地底靈網晶層，是棲國真正的國土。
- 這裡的生態不是很多植物而已，而是一整套會回應誓網、情緒與共鳴節奏的活體系統。
- 結果喬木、爆裂散種藤本與木妖型魔植在這裡同樣是正常森林結構的一部分，不是傳說外掛。

## 棲裔與聚落魔物
- `誓蘚`：棲國最基礎的共鳴底層魔植。危險 0 / 親近 5。
- `靜紋藤`：安撫與低刺激編構用魔藤。危險 0 / 親近 4。
- `蘚冠果木`：近聚落結果喬木，提供低光果囊與小型傳孢者棲位。危險 0 / 親近 3。
- `綠脊水鼴`：協助翻鬆溪岸與保持流路。危險 0 / 親近 3。
- `靜翼纖鳥`：近聚落魔物，會攜帶語音殘片。危險 0 / 親近 3。
- `眠灣軟殼龜`：靜養池平穩指標魔物。危險 0 / 親近 4。
- `孢霧步鹿`：誓網平衡指標魔物，部分區域可半引導共存。危險 1 / 親近 3。

## 林中魔物與高危個體
- `語孢囊蕈`：高敏訊息承載菌種。危險 2 / 親近 2。
- `晶露花`：吸收誓晶霧氣的修復型植種。危險 0 / 親近 2。
- `海膽葡萄`：高濕枝層的爆裂散種藤本，去刺前極危險。危險 2 / 親近 1。
- `語殼寄鳴蟲`：會模仿人聲與導印口令的小型林棲魔物。危險 2 / 親近 1。
- `霧椏木妖`：具領域感的木妖型魔植，常守水脈與舊路節點。危險 2 / 親近 1。
- `誓枝影貓`：樹冠掠食魔物。危險 3 / 親近 0。

## 失衡異種與災害生命
- `迷導繭魘`：共鳴失衡後形成的危險異種。危險 4 / 親近 0。
- `食印蘚團`：會吞噬低階導印與標記的集合體。危險 2 / 親近 1。
- `孢冠裂偶`：植物、誓網殘渣與守備體混成的異體。危險 4 / 親近 0。
- `失語共鳴者`：誓網失衡後殘留人形輪廓的危險存在。危險 4 / 親近 0。

## 受名與繁衍特徵
- 瑟堤亞許多生命的繁衍不只靠季節，還會受誓網節奏與共鳴濃度影響。
- 但 `棲裔` 的成立不是繁衍學分類，而是政治與共鳴分類：只有被誓王賜名的魔物，才會從魔物被納入棲裔。
- `語孢囊蕈` 會在聲紋與濕度合適時形成可保存訊息的孢囊。
- `海膽葡萄` 以 `爆裂散種` 為主，刺籽成熟後會在受驚時整串炸裂，因此採收需要冷霧、穩枝與逆旋拆串。
- `霧椏木妖` 主要透過根網交握與緩慢分株延續個體，少數長期承擔守望責任者，會在受名後被納入棲裔。
- `迷導繭魘` 往往來自本應安撫或保護的系統失衡，而非單純野性增幅。

## 生態鏈摘要
- 底層鏈：誓蘚 / 靜紋藤 / 晶露花 / 蘚冠果木 -> 小型濾食與傳孢種 -> 步鹿 / 水鼴 / 纖鳥。
- 果串與枝層鏈：蘚冠果木 / 海膽葡萄 -> 寄鳴蟲與果食小型魔物 -> 誓枝影貓；霧椏木妖則作為領域壓制者維持枝層秩序。
- 池群鏈：安撫池植群與藻群 -> 靜灣眠鱗 / 軟殼龜 -> 過載時生成的繭魘或裂偶。

## 旅人注意
- 棲國的危險經常不是某一隻野性生命，而是整片環境開始不歡迎你。
- 若你在棲國境內把棲裔一概當成普通魔物，往往會被視為無知甚至失禮。
- 若你把失衡異種誤認成普通魔物，通常後果更糟。
- 若你想自己徒手採 `海膽葡萄`，最好先確定旁邊沒有整串連爆的第二株。
