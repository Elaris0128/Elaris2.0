# 魈陽古域生態

## 生境分層
- `焦熱荒坡`：最常見火脈獸與掠食種的地帶。
- `裂谷陰井帶`：有限水源與伏擊魔物聚集點。
- `熔誓火地帶`：部族立誓與祭火之地，也會吸引高靈息生命。

## 生活型與部族伴生魔物
- `焦鬃馱獸`：最穩定的高耐熱馱騎種。危險 1 / 親近 3。
- `炎步短馬`：速度快但脾氣烈的巡行獸。危險 2 / 親近 2。
- `熔耳砂兔`：小型肉源與預警用魔物。危險 0 / 親近 2。

## 野外威脅與掠食種
- `炎脊獵駒`：會群追弱者與落單者。危險 3 / 親近 0。
- `裂炭伏蛇`：偏愛藏在焦黑石縫間。危險 3 / 親近 0。
- `炁焰蜥`：受激時會噴出短暫高熱息。危險 2 / 親近 1。

## 魔植、魔菌與火脈依存種
- `焦脈花`：可作藥也可製燃劑。危險 1 / 親近 2。
- `焰鹽苔`：能在熱井邊形成鹽層。危險 1 / 親近 2。
- `炭根爆豆藤`：豆囊成熟時會炸裂散種。危險 2 / 親近 1。

## 異常與禁忌生命
- `崩誓炁影`：立誓失敗與戰殞重疊地出現的火紋殘響。危險 4 / 親近 0。
- `熔盆空名者`：會在夜裡重複喊出未被承認的名字。危險 3 / 親近 0。

## 生態鏈摘要
- 魈陽古域的生態由 `火 / 水源 / 炁脈 / 立誓場` 四件事決定。
- 很多物種會圍著火脈走，因此部族路線和魔物路線經常重疊。
