# 蒼淵據地與洛丹崖營生態

## 生境分層
- `霧崖高帶`：最適合攀附、滑翔與高處觀測種活動。
- `裂谷陰面`：光線最弱、異常最容易積聚的區域。
- `崖路與索橋帶`：人造結構與野生掠食共存最明顯。

## 生活型與伴生魔物
- `崖索短蹄羊`：高差運補不可或缺。危險 0 / 親近 5。
- `夜繩滑鼬`：會清除部分寄生小獸，但也會偷取細小金屬件。危險 0 / 親近 2。
- `霧哨小鸮`：常被當成預警鳥。危險 0 / 親近 3。

## 野外威脅與掠食種
- `影崖滑蛛`：黏索幾乎看不見。危險 3 / 親近 0。
- `裂谷風喰`：利用風切傷人的高空掠食者。危險 4 / 親近 0。
- `斷壁伏狐`：會故意誘使獵物踩錯重心。危險 2 / 親近 0。

## 魔植、魔菌與崖陰依存種
- `影葉藤`：擅長封路與纏繞橋索。危險 2 / 親近 1。
- `冷風藥株`：止血、止痛與鎮顫的重要藥植。危險 0 / 親近 3。
- `崖苔靜簾`：可作遮蔽材料，也會藏匿小型異常。危險 1 / 親近 2。

## 異常與禁忌生命
- `吊索聽名者`：模仿熟人聲音的崖路異常。危險 3 / 親近 0。
- `沉影回足`：會重複舊巡路者步伐的殘響。危險 2 / 親近 0。
- `崖底未葬者`：與未完成安置的屍骨與誓繩相關的人形危險體。危險 4 / 親近 0。

## 生態鏈摘要
- 這裡的生態核心是 `高差 / 低光 / 索道 / 霧 / 聲音`。
- 很多生命不靠力量壓倒人，而是靠讓你走錯一步、聽錯一句。
