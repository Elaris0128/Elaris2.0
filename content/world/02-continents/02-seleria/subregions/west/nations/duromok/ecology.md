# 杜魯摩克隱域生態

## 生境分層
- `地底主坑帶`：工坊、礦脈、運補獸與坑道害種最集中。
- `裂隙外市帶`：半地表、半地下，灰線與外來生命混入頻率最高。
- `廢工坊殘坑帶`：異常、舊機關與危險共生種密集。

## 生活型與工坊伴生魔物
- `鉤背搬坑獸`：最穩定的重載坑獸。危險 1 / 親近 4。
- `熔爪工蝠`：常棲在坑頂吃掉有害小蟲與黴粉。危險 0 / 親近 3。
- `鐵鼻短鼴`：小型探坑伴生種，擅找脆弱層。危險 0 / 親近 2。

## 掠食與災害型魔物
- `坑道咬岩蟲`：會沿支架咬穿整段坑壁。危險 3 / 親近 0。
- `煙翼蝠`：群起時能讓整坑失光。危險 2 / 親近 0。
- `熔渣蜥`：受熱脈吸引的厚鱗掠食種。危險 2 / 親近 1。
- `灰鍊掘甲蟲`：啃食金屬與繩鍊，對軍需威脅極大。危險 3 / 親近 0。

## 魔植、魔菌與危材依存種
- `爆蕈叢`：受震、受熱、受火都可能爆裂。危險 3 / 親近 0。
- `齒苔`：附著齒輪與工具汲取鐵粉。危險 1 / 親近 1。
- `黑粉甲群`：和爆蕈、礦粉與油脂環境緊密共生。危險 2 / 親近 0。
- `銹火菌膜`：覆在舊工坊牆面，能腐蝕護具與器械。危險 2 / 親近 0。

## 異常與禁忌生命
- `裂爐石魈`：最危險的廢坑伏擊者之一。危險 4 / 親近 0。
- `食鐵蠕母`：能讓整批武備憑空消失。危險 4 / 親近 0。
- `熄火誓偶`：破誓工坊事故留下的人形殘偶。危險 3 / 親近 0。

## 生態鏈摘要
- 杜魯摩克的生態圍繞 `熱脈 / 金屬 / 油脂 / 震動 / 光源` 運作。
- 越靠近利益中心，生態就越人工化，也越容易長出會吃工坊本身的魔物。
