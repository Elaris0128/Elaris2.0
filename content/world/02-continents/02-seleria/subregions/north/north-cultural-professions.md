# 北境文化職業與流派

## 使用說明
- 這份文件以 `文化職業` 為主體書寫。
- `對應通用職業` 只作備註，方便回掛到底層規則。
- 北境職業特別重視 `生存壓力、同行承認、寒地責任與環境考驗`。

## 盧米昂寒國

### 守燈關衛
- 類別：軍事 / 關防
- 對應通用職業：重裝守衛、守誓戰士
- 常見所屬機構：關樓守隊、燈鏈巡隊
- 獲得條件：完成關樓輪訓、守燈值夜、橋關封鎖演練，需有合法保薦
- 常見媒介 / 裝備：關戟、燈盾、雪鉤、厚披鎧
- 專屬戰技 / 術式：`燈線拒寒陣`、`關門斷潮擊`
- 流派方向：橋關守燈派、霜脊封線派
- 社會印象：刻板、硬派、真正保命的人

### 霜簿潛聽吏
- 類別：諜報 / 關卡潛線
- 對應通用職業：情報行者、記錄學者
- 常見所屬機構：寒國密務線、關簿內聽組
- 獲得條件：有關樓書吏、港道查驗或宿站記錄背景，通過偽件與耳目訓練
- 常見媒介 / 裝備：霜簿、封票刀、路簽匣
- 專屬戰技 / 術式：`霜頁聽差術`、`關簽回讀法`
- 流派方向：關樓內聽派、港冰暗簽派
- 社會印象：不起眼，但非常會抓錯

### 送渡安魂者
- 類別：宗教 / 守夜 / 安魂
- 對應通用職業：祭儀執行者、誓印守護者
- 常見所屬機構：安魂歌館、守渡誓庭
- 獲得條件：完成送渡、守屍、夜路引導與遺物轉交儀式
- 常見媒介 / 裝備：送渡鈴、燈誓杖、遺物封帶
- 專屬戰技 / 術式：`夜渡安魂式`、`霜燈留名術`
- 流派方向：橋誓送渡派、長夜守名派
- 社會印象：受敬重、帶悲意、可信度極高

### 雪橋算記師
- 類別：學術 / 土木 / 文契
- 對應通用職業：記錄學者、鍊成工匠
- 常見所屬機構：雪橋算院、橋路修務所
- 獲得條件：需完成橋務實習、載重與冰裂計算、現地巡測
- 常見媒介 / 裝備：算尺、冰鑿筆、橋圖冊
- 專屬戰技 / 術式：`冰載測定術`、`裂縫提前記`
- 流派方向：長橋算務派、寒潮維修派
- 社會印象：安靜、精確、很被依賴

### 寒橋修補師
- 類別：民間 / 工務 / 求生
- 對應通用職業：鍊成工匠、技藝系前線支援
- 常見所屬機構：橋路修補行、宿站工棚
- 獲得條件：需能在低溫下完成火線維修、繩橋補強與快速搭棚
- 常見媒介 / 裝備：熱鉤、霜釘槌、吊索架
- 專屬戰技 / 術式：`霜索補橋手`、`急棚立火法`
- 流派方向：橋線急修派、雪道棚匠派
- 社會印象：粗實、救命、常比官員更受歡迎

## 雪華部族

### 白隱守徑者
- 類別：軍事 / 邊林守望
- 對應通用職業：機動斥候、自然共鳴者
- 常見所屬機構：白隱邊亭、守徑列
- 獲得條件：通過邊林巡行、低聲守望與靈誓伴協同行測
- 常見媒介 / 裝備：輕弓、花印索、柔步鞋
- 專屬戰技 / 術式：`白隱折步`、`霜花靜警術`
- 流派方向：邊亭巡徑派、伴行守望派
- 社會印象：安靜、輕巧、不容小覷

### 靜聽回名者
- 類別：諜報 / 記名 / 回返引導
- 對應通用職業：情報行者、記錄學者
- 常見所屬機構：回名圈、靜聽者列
- 獲得條件：需能記住多條花徑，並在不驚擾林心的前提下找回失行者
- 常見媒介 / 裝備：回名枝片、靜香袋、細紋筆
- 專屬戰技 / 術式：`回名靜尋法`、`花徑留息術`
- 流派方向：內林回尋派、邊徑靜聽派
- 社會印象：溫和、難追、很會看人

### 護生祭伴者
- 類別：宗教 / 照護 / 成年禮
- 對應通用職業：祭儀執行者、契靈行者
- 常見所屬機構：護生圈、尋伴禮主列
- 獲得條件：完成護生期、獲長輩與靈誓伴認可、能主持尋伴前準備儀式
- 常見媒介 / 裝備：蓮節環、靈香囊、伴息繩
- 專屬戰技 / 術式：`伴息安林式`、`霜蓮護心歌`
- 流派方向：尋伴護行派、幼伴照息派
- 社會印象：柔和、神聖、很受信任

### 花徑引返師
- 類別：學術 / 引路 / 生態
- 對應通用職業：自然共鳴者、記錄學者
- 常見所屬機構：花徑學圈、引返庭
- 獲得條件：需能判花徑、識水脈、引外客不驚林，並通過長季觀察期
- 常見媒介 / 裝備：花徑圖、柔尺、引返籤
- 專屬戰技 / 術式：`霧花返路術`、`林心水脈記`
- 流派方向：水脈引返派、伴行教引派
- 社會印象：很懂路，也很懂什麼不該碰

### 寒蓮調理者
- 類別：民間 / 藥食 / 照護
- 對應通用職業：靈廚、治療藥師
- 常見所屬機構：寒蓮護生圈、靈廚公會合作點
- 獲得條件：需熟悉寒蓮、月蓮與安撫型食材的低擾處理
- 常見媒介 / 裝備：蓮片刀、溫香盒、柔火盞
- 專屬戰技 / 術式：`寒蓮回暖膳`、`伴息安胃湯`
- 流派方向：護生柔膳派、尋伴照食派
- 社會印象：安靜、細心、常是隊伍穩定器

## 暮鏡荒境

### 裂雪獵首
- 類別：軍事 / 狩獵 / 圍剿
- 對應通用職業：遠行獵手、近戰前衛
- 常見所屬機構：獵隊、裂雪圍剿圈
- 獲得條件：完成高危獵行、帶隊返還、能獨立處理類人威脅
- 常見媒介 / 裝備：重獵叉、骨刀、雪索
- 專屬戰技 / 術式：`裂雪追身`、`獵坡壓勢`
- 流派方向：荒口圍剿派、深雪追獵派
- 社會印象：危險、靠得住、常帶傷

### 灰痕聽伏者
- 類別：諜報 / 潛伏 / 邊坡耳目
- 對應通用職業：情報行者、機動斥候
- 常見所屬機構：宿火聽線、邊坡伏聽圈
- 獲得條件：需能長時間伏雪、辨風聲與識足跡，並通過失溫測試
- 常見媒介 / 裝備：灰披、骨哨、雪鏡
- 專屬戰技 / 術式：`灰痕聽伏`、`靜坡留印`
- 流派方向：風口伏聽派、回火追痕派
- 社會印象：沉默、敏銳、像雪裡的一部分

### 守火送魂者
- 類別：宗教 / 守夜 / 喪儀
- 對應通用職業：祭儀執行者、守誓戰士
- 常見所屬機構：火圈祭列、送魂骨帳
- 獲得條件：須完成守火、回收與送魂三重責任，不得在喪儀中失序
- 常見媒介 / 裝備：骨燈、火骨杖、送魂布
- 專屬戰技 / 術式：`守火留魂圈`、`荒雪送行歌`
- 流派方向：火坡正送派、骨帳安回派
- 社會印象：粗硬中帶敬畏

### 夢兆辨骨人
- 類別：學術 / 夢兆 / 骨記
- 對應通用職業：記錄學者、自然共鳴者
- 常見所屬機構：輪夢守行圈、骨兆學徒團
- 獲得條件：能分辨夢兆真假，並把骨記與實地危機對上
- 常見媒介 / 裝備：骨籤、夢灰盤、兆記線
- 專屬戰技 / 術式：`夢骨校兆法`、`寒嚎定真術`
- 流派方向：輪夢辨行派、禁窟兆讀派
- 社會印象：神神叨叨，但常真的說中

### 荒境回收者
- 類別：民間 / 回收 / 探索
- 對應通用職業：遠行獵手、鍊成工匠
- 常見所屬機構：回收隊、探索公會合作線
- 獲得條件：需有活著從遺跡或雪窟帶回人與物的紀錄
- 常見媒介 / 裝備：鉤索、封樣袋、載架
- 專屬戰技 / 術式：`雪窟拖回手`、`殘井封樣式`
- 流派方向：遺窟回收派、冰裂搬運派
- 社會印象：務實、命硬、常被低估

## 星紋古道

### 危段導路者
- 類別：軍事 / 護行 / 導路
- 對應通用職業：機動斥候、守勢戰技系
- 常見所屬機構：護行隊、宿地守望隊
- 獲得條件：需帶隊通過一段正式危路並成功回返，且獲至少兩名資深導路者掛名
- 常見媒介 / 裝備：路鉤、風尺、導燈、短槍
- 專屬戰技 / 術式：`危段換位步`、`路印壓心術`
- 流派方向：北脊導路派、迷段帶返派
- 社會印象：很被敬重，也很容易背鍋

### 宿地主理人
- 類別：民生 / 組織 / 路權
- 對應通用職業：商旅判契者、記錄學者
- 常見所屬機構：宿地火圈、共火圈議位
- 獲得條件：需守火、記名、管水、調解糾紛並獲共火圈承認
- 常見媒介 / 裝備：火簿、宿地木牌、水封罐
- 專屬戰技 / 術式：`宿火留位規`、`共火止爭式`
- 流派方向：碑宿正管派、交會營整序派
- 社會印象：看似平凡，實際是文明骨架

### 拓印回尋者
- 類別：探索 / 學術 / 回收
- 對應通用職業：記錄學者、遠行獵手
- 常見所屬機構：碑影帳場、拓印圈
- 獲得條件：會辨碑、會封樣、會在迷路段找回人或遺物
- 常見媒介 / 裝備：拓印袋、炭紙、回尋鉤
- 專屬戰技 / 術式：`碑紋回讀法`、`失路返線術`
- 流派方向：古碑拓讀派、危段回尋派
- 社會印象：像學者，也像半個送命職

### 夜路祈安人
- 類別：宗教 / 安撫 / 守夜
- 對應通用職業：祭儀執行者、誓印守護者
- 常見所屬機構：星讀夜課、祈安圈
- 獲得條件：須完成多次守夜、祈安與失路安撫，不得亂解路誓
- 常見媒介 / 裝備：夜燈、祈紋布、安心鈴
- 專屬戰技 / 術式：`夜行祈安心火`、`失路止亂歌`
- 流派方向：宿地祈安派、危路鎮心派
- 社會印象：安靜、有用、很多人暗地裡依賴

### 行囊修補僧
- 類別：民間 / 技藝 / 補給
- 對應通用職業：鍊成工匠、靈廚的旅裝支系
- 常見所屬機構：宿地修補棚、行囊僧行
- 獲得條件：需能修包、補繩、分濕乾、整隊裝，不可只會做表面活
- 常見媒介 / 裝備：修補針、扣具匣、繩油袋
- 專屬戰技 / 術式：`重包換肩法`、`風雪整裝訣`
- 流派方向：長宿修囊派、危途速整派
- 社會印象：不起眼，但老旅人最看重
