# 暮鏡荒境生態

## 生境分層
- `裂雪原`：開闊、高風、最容易爆發大型追獵。
- `白隱冰嶺`：洞穴、雪崖與風口交疊，類人威脅多。
- `深暮氈坡`：誓魂殘響、夢兆與安魂草最密集。
- `鏡霧雪盆`：視覺錯亂與月雪幻映最明顯的荒盆。

## 族群伴生與生活型生命
- `月影狼群`：與狼裔文化纏繞最深的共鳴種。危險 2 / 親近 1。
- `影尾雪狐`：具迷導與潛行能力的靈巧種。危險 2 / 親近 1。
- `暮羽夜禽`：能提早感應魂波與風勢。危險 1 / 親近 2。

## 掠食與荒境威脅
- `匿霜伏狼`：在暮鏡的伏擊性更強。危險 4 / 親近 0。
- `裂牙雪兔`：會啃食傷獸與腐屍，是荒境最煩人的小型掠食群。危險 2 / 親近 0。
- `魂裂白猞`：會追逐重傷者與誓魂異常源。危險 3 / 親近 0。

## 類人與夢兆生命
- `雪脊山吼`：近似雪怪的大型類人魔物。危險 4 / 親近 0。
- `鏡步雪女`：會借熟人聲線與步態引離隊伍。危險 4 / 親近 0。
- `空面逐行者`：在隊伍最疲憊時靠近的空面人形異常。危險 4 / 親近 0。
- `夢行白童`：拉人離火、離營、離路的白影。危險 3 / 親近 0。

## 荒境魔植與異常素材
- `魂霧蘚`：安魂、追跡與夢儀常用。危險 1 / 親近 2。
- `暮毛絨菇`：燃起來會安撫，也會誤導。危險 2 / 親近 1。
- `雪眠花`：在暮鏡常與輪夢與送別綁在一起。危險 0 / 親近 3。

## 生態鏈摘要
- 暮鏡荒境的核心不是單一食物網，而是 `誓魂、輪夢、荒獵與類人威脅` 交錯出的心理型生態。
- 這裡很多生命吃的不只是肉與植物，還吃記憶、恐懼、熟悉感與錯路者的判斷失誤。
