# 星紋古道生態

## 生境分層
- `古碑交會帶`：人與魔物都依賴碑紋與歇臺記路。
- `北脊雪道帶`：高風險長距徙路，驛點稀少。
- `斷霜古段`：碑列斷層、白口洞與失路異常最密集。
- `路口寒潭帶`：古道旁少量可採的水系魔植與補給點。

## 生活型與伴路種
- `星步馱駝`：最可靠的長線運補獸。危險 1 / 親近 4。
- `夜碑狐`：會順碑紋找到安全歇點。危險 1 / 親近 3。
- `寒羽引路鳥`：常被旅者視為好兆頭。危險 0 / 親近 3。
- `長絨雪犛`：在重貨古道線同樣常見。危險 1 / 親近 5。

## 路線威脅與掠食種
- `斷碑獵猞`：最常伏在斷碑背面。危險 3 / 親近 0。
- `匿霜伏狼`：會沿驛站外圈試探隊伍收尾。危險 4 / 親近 0。
- `裂牙雪兔`：專咬補給袋與傷者繃帶，是古道旅人最恨的小東西。危險 2 / 親近 0。
- `路紋食角蟲`：會啃蝕路標與營地繩料。危險 2 / 親近 0。

## 魔植、路徑依存種與寒潭生命
- `誓痕路藤`：沿可行路線長出細微紋理。危險 0 / 親近 3。
- `碑面寒蘚`：判斷碑石是否還可用的重要蘚類。危險 0 / 親近 3。
- `夜星露蕈`：只在特定夜相後可安全採摘。危險 1 / 親近 2。
- `霜河寒蓮`：少量生在歇潭暖口，成為極珍貴的路途補命資源。危險 1 / 親近 2。

## 異常與禁忌生命
- `路碑啃印者`：破壞古道路權的常見異常。危險 2 / 親近 0。
- `失路行囊鬼`：會假裝遺失補給吸引人偏離主線。危險 3 / 親近 0。
- `斷霜回身者`：在禁段反覆出現的半人形危險體。危險 4 / 親近 0。

## 生態鏈摘要
- 星紋古道的生態核心是 `路線本身`。只要碑列和驛站穩，許多危險種會被壓在外緣；一旦路權斷裂，整條古道就會變成迷宮。
