# 盧米昂寒國生態

## 生境分層
- `雪關燈鏈帶`：人為維持度最高，生活型魔物與守夜制度綁定最深。
- `霜河寒蓮帶`：冰河、裂冰支流與寒蓮暖湧共構的危險水脈。
- `礦脊雪坡帶`：冰銅脈、寒鐵與導材最集中的礦坡地。
- `外原盲雪帶`：大型掠食者、匿形狼群與半人形異常最容易游走的外環。

## 生活型與伴生魔物
- `長絨雪犛`：拖運重貨與保暖毛材主力。危險 1 / 親近 5。
- `燈鼻雪犬`：盧米昂最信任的守夜伴生種。危險 0 / 親近 5。
- `長鬃冰蹄鹿`：雪關巡線與高地快遞常用坐騎。危險 1 / 親近 4。
- `霜棉穴羊`：提供乳脂、絨毛與寒地帳料。危險 0 / 親近 5。

## 河流、冰湖與水邊生命
- `裂冰潛鰭獸`：冰層下最危險的長體掠食獸。危險 4 / 親近 0。
- `霧鰓長鱗鱔`：常藏在霜河盲彎與橋洞下方。危險 3 / 親近 0。
- `霜河寒蓮`：盧米昂最重要的寒水魔植之一，兼具藥用與送亡象徵。危險 1 / 親近 2。
- `裂冰鬚藻`：能纏人、纏槳也纏住拖網。危險 2 / 親近 0。

## 野外威脅與邊境魔物
- `匿霜伏狼`：會收住氣味與腳印的北境狼種。危險 4 / 親近 0。
- `裂牙雪兔`：兇暴、肉食、擅群突的小型危險種。危險 2 / 親近 0。
- `霜脊雪猞`：雪脊上的高機動掠食者。危險 3 / 親近 0。
- `雪脊山吼`：盤踞洞巢與風口的類人巨體威脅。危險 4 / 親近 0。

## 魔植、魔菌與寒地依存種
- `寒燈蘚`：驛柱、橋側與古井常見低光蘚。危險 0 / 親近 4。
- `冰髓穗草`：做保溫墊、寒包與枕卷的重要草本。危險 0 / 親近 3。
- `霜鈴莓灌`：可食，但脆響會暴露採集者位置。危險 1 / 親近 3。

## 異常與禁忌生命
- `雪誓殘影`：在失溫死地反覆出現的誓約殘響。危險 3 / 親近 0。
- `無燈行客`：會把人引離燈鏈的半人形異常。危險 4 / 親近 0。
- `鏡步雪女`：會在冰面與霜河邊借熟人步態引路。危險 4 / 親近 0。

## 生態鏈摘要
- 盧米昂的核心不是豐饒，而是 `用燈鏈、橋路、長毛馴用種與礦材工事去壓住野地`。
- 一旦燈鏈或橋路出問題，原本勉強共存的河獸、狼群與類人異常會很快壓回文明邊線。
