## 北境整體氣質
- 北境不是單一雪國，而是由 `雪林部族 / 遷徙荒境 / 古道路網` 交錯組成的寒帶文明帶。
- 這裡的共同問題不是單純天寒，而是 `路線是否可信、邊界是否還被承認、冬季窗口是否已經關閉、亡者與誓約是否仍被記住`。
- 玩家在北境常碰到的第一層壓力，通常來自：
  - 行路與回程窗口
  - 邊境重畫與通路改判
  - 冬季物資、牲群與渡點安全
  - 碑記、足跡、名誓與禁林規矩是否仍有效

## 三個主體
- `雪華部族`：以母樹、霜蓮、命名水儀與寒林守生維持秩序的雪精靈部族地。
- `暮鏡荒境`：四獸裔共存的遷徙荒境，重狩窗、回返倫理、埋骨記憶與邊境線重畫。
- `星紋古道`：以碑記、路宿、校線與導標維繫通行可信度的古道路文明。
