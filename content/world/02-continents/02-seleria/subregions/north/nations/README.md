# 北境政治體索引

## 已建立政治體包
- `lumiang`：盧米昂寒國
- `xuehua`：雪華部族
- `mujing`：暮鏡荒境
- `xingwen-road`：星紋古道

## 現況
- 北境已建立區域總覽與四個主要政治體入口。
- 四個北境政治體現已具備正式國家層主檔：`overview / politics / diplomacy / military / law-and-order / culture / social-structure / education-and-knowledge / festivals-and-rituals / religion / organizations / ecology / resources / landscapes / settlements / special-sites / trade-and-economy / quest-hooks / characters`。
- 盧米昂之外，雪華、暮鏡與星紋古道也已補上 `history` 與 `intelligence-service` 入口。

## 延展方向
- 後續可把北境雪關、冰湖、古道、部族據地與地下冰洞全部接成地方包與任務線。
- 北境的下一步重心，應該是把地方包、任務線與更多角色密度追平東境。
