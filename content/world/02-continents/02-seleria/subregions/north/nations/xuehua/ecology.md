# 雪華部族生態

## 生境分層
- `雪華母樹林心`：最穩定也最受守護的花林中核。
- `白隱花林帶`：花粉、菌群與伴生鹿群最密集的地方。
- `寒蓮溪環帶`：林水相接、蓮與藤類繁生的邊水帶。
- `冰環外林帶`：最常與外來者接觸的過渡圈。

## 生活型與伴生生命
- `雪瓣眠鹿`：雪華最重要的採藥與巡林伴生種。危險 0 / 親近 4。
- `霜枝狐`：會替林地預警，但不完全受控。危險 1 / 親近 2。
- `月瞳雪雀`：能辨誓印、辨情緒、辨陌生氣息。危險 0 / 親近 3。

## 林地與水邊魔植
- `月魄冰蓮`：雪華最具象徵性的寒水靈植。危險 1 / 親近 2。
- `霜河寒蓮`：在暖溪潭口也能形成小型蓮環。危險 1 / 親近 2。
- `雪眠花`：安神、止痛與送別常用花植。危險 0 / 親近 4。
- `白隱絨菇`：藥食兩用菌。危險 0 / 親近 4。
- `花印藤`：沿著被允許的路線生長。危險 0 / 親近 3。

## 守域與排外生命
- `白根纏行者`：會封閉不受歡迎的路。危險 3 / 親近 0。
- `蓮雪花侍`：類人的花系守域生命。危險 3 / 親近 1。
- `凍花裂鹿`：繁殖季會暴力驅離靠近母樹的外物。危險 2 / 親近 1。
- `冰弦伏豹`：喜歡白霧與落雪中的短距離暴起。危險 3 / 親近 0。

## 異常與禁忌生命
- `碎誓花偶`：在破誓、污林與錯祭處形成的小型異常。危險 2 / 親近 0。
- `寒晶回唱者`：會模仿已亡族人的詩句與腳步。危險 3 / 親近 0。
- `霜臂花偶`：植物型人形危險體，常守著錯名與壞誓之地。危險 3 / 親近 0。

## 生態鏈摘要
- 雪華的生態是 `花粉 / 蓮潭 / 菌群 / 伴生獸 / 守域植物人形` 撐起來的高感應林境。
- 這裡看似溫柔，但只要誓印秩序被破壞，花林會比荒地更快轉成排外與封閉狀態。
