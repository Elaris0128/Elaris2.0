# 北境距離參考表

## 目的
這份文件整理北境的 `雪道 / 燈鏈 / 冰橋 / 驛站 / 破冰航線` 基準，讓主持時能穩定判斷旅行時間、補給窗口與封關風險。

## 北境距離判讀原則
- 北境不優先用平地里數思考，而優先用 `能否守燈、能否補給、能否過橋、能否在夜前抵達` 判讀距離。
- 同一段路在 `白晝積雪期 / 暴雪期 / 解凍期 / 極夜期` 的體感距離會差很多。
- 因此北境記錄距離時，建議同時給出：
  - `大致路線層級`
  - `正常季節行程`
  - `惡劣季節修正`

## 北境常用行程層級
- `半日`：村與驛站、驛站與橋口之間的短補給距離
- `1 日`：安全燈鏈內的標準移動單位
- `2 日`：需要正式宿站或中繼燈塔的路段
- `3 日以上`：必須規畫燃料、橋況、守夜輪值與應急退路

## 北境主體尺度
- `盧米昂寒國`：以港、關、集鎮、橋線和誓燈大道為主，路線最穩定
- `雪華部族`：移動距離不只看地形，也看母樹圈與雪祭季節
- `暮鏡荒境`：路線常受視野、獵道與荒地異象影響
- `星紋古道`：距離感最像驛段和碑段，而不是固定行政邊界

## 盧米昂基準距離
- `冰冠港 -> 寒牙集市`：`3 日雪橇程`
- `寒牙集市 -> 霜道關隘`：`2 日騎乘 / 雪橇混行`
- `寒牙集市 -> 雪橋村`：`1 日半`
- `寒牙集市 -> 冰脈驛站`：`半日`
- `冰冠港 -> 灰燈塔聚`：`1 日半`
- `寒牙集市 -> 靜骨屯`：`1 日`

## 惡劣季節修正
- `暴雪期`：正常路程通常增加 `半日到 2 日`
- `極夜期`：若燈鏈不穩，旅行風險會高於純粹時間損失
- `解凍期`：冰橋與雪橇線可能失效，需繞路或改走破冰水道

## 主持使用原則
- 北境若沒有明確理由，不建議讓隊伍「當天衝完多個節點」。
- 只要路線涉及：
  - 封道牌
  - 冰橋
  - 塔聚
  - 夜間換燈
  - 安名送行
  就應視為兼具交通與儀式性的路段。
