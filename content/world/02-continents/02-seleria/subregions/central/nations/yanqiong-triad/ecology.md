# 巖穹三臺盟生態

## 生境分層
- `主脊高嶺帶`：高空猛禽、耐風獸與王者級最容易活動的帶狀區。
- `岩臺盆地帶`：少數穩定居住與家養魔物密集區。
- `斷橋峽隘帶`：掠食、遺跡異常與守路生命都會聚集。

## 生活型與高地伴生魔物
- `岩背角獸`：高地重載主力。危險 1 / 親近 4。
- `寒苔羊`：高脂乳與毛皮來源。危險 0 / 親近 5。
- `石脊猿`：少量可用於搬索與陡坡取物。危險 2 / 親近 1。

## 野外威脅與掠食種
- `斷橋喉豹`：伏在橋影下等待落單者。危險 4 / 親近 0。
- `風鉤崖鷲`：會把小型獵物從坡面捲下。危險 3 / 親近 0。
- `霜脊躍猊`：高差跳躍能力極強。危險 3 / 親近 0。

## 魔植、魔菌與高地依存種
- `裂風草`：纖維韌且可編索。危險 1 / 親近 3。
- `霧崖苔`：止血與保濕效果強。危險 0 / 親近 4。
- `冷峰樹菇`：常作乾糧或簡湯。危險 0 / 親近 3。
- `崖鐘藤`：風大時會發出預警般的鐘響。危險 1 / 親近 2。

## 異常與禁忌生命
- `觀臺殘響者`：古觀星台衍生的低語異常。危險 3 / 親近 0。
- `守橋盲影`：會把人帶向已崩橋段。危險 3 / 親近 0。
- `坠雲石偶`：在廢哨與落石區活動的擬岩異常。危險 4 / 親近 0。

## 生態鏈摘要
- 巖穹三臺盟的生態由 `風 / 高差 / 路權 / 少量穩定盆地` 決定。
- 真正溫和的生命大多集中在少數盆地，離開那裡後每一步都更接近掠食與失足。
