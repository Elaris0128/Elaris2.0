# 獸人分支詳解

## 定位

- 獸人不是單一外貌模板，而是一組在高壓環境中分化明顯、支系特徵強烈的種族群。
- 現行正典中的獸人正式支系，以玩家橋接與種族分支清單為準。
- 部分支系對應梵洛特，部分支系對應暮鏡荒境與其他高壓邊域。

## 正式支系

### 十獸之環

- 定位：多席位、多職責的古誓獸環支系
- 備註：`馭角 / 逐影 / 鷹目 / 裂牙 / 斑影 / 隱耳 / 馭嶺 / 蟄蛇 / 裂痕 / 冥羽` 是可再細選的席位，不是新的正式種族分支

## 使用原則

- 玩家若選 `獸人`，AI 應先列正式支系，再決定起始圈。
- 十獸之環的席位層只算再細選，不是必選第二層新分支。
- 若地區檔出現更細地方名、獵團名或席位名，預設不升格為新的正式玩家支系。

## 不要誤用舊名

下列舊稱不作為現行優先正式名：

- 霧沼裔
- 炭龜裔
- 隱號裔
- 炭鼬裔

它們最多只算舊稱、簡稱或過渡命名，不應優先用來回答現行玩家分支。
