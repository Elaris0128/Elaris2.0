# 獸人分支詳解

這份文件把獸人從單一種族名拆成真正可用的支系層。獸人不是一張長著獠牙的統一臉譜，而是一整串與棲地、嚎語、祖靈、誓約與生存方式綁在一起的血裔文明群。

## 總觀
- 上位種族：獸人
- 世界使用原則：角色卡、國家資料與地區神話可以先寫具體支系，但必須能回掛到 `獸人` 這個總入口。
- 信仰基調：偏戰祖、守地神、祖靈骨歌與地方律神，很少形成殿堂式中央教團。

## 當代東境主幹

### 十獸之環
- 代表分布：克羅嵐聯盟國
- 社會定位：東境最穩定、最制度化的獸人共同體。
- 文化氣質：共誓、榮譽、議會、席位與氏族平衡。
- 常見能力：狩獵、戰鬥、氏族誓約、群體決策與嚎語傳令。
- 角色卡建議寫法：`獸人 / 十獸之環・克羅嵐共誓支`
- 主要席位：`馭角 / 逐影 / 鷹目 / 裂牙 / 斑影 / 隱耳 / 馭嶺 / 蟄蛇 / 裂痕 / 冥羽`
- 細分原則：十獸不只是十個名字，而是十種參照現實動物習性的社會模型。例如 `馭角` 重群隊與護幼、`逐影` 重群獵與聞痕、`隱耳` 重耳警與轉移、`裂痕` 重巡領與單核壓線。
- 使用建議：若人物真正來自克羅嵐，最好再補一層 `十獸席位`，這會直接影響他的說話方式、榮譽觀、常走的路與擅長的職能。玩家創角時若選 `十獸之環・克羅嵐共誓支`，也可再細選一個席位作為第三層背景。
- 延伸閱讀：克羅嵐國家檔中的 `ten-beasts-ring.md`

## 其他已知重要支系

### 裂蠍沙裔
- 代表分布：裂紋沙原
- 外觀與體質：常帶乾裂甲殼感、耐熱與伏沙特性。
- 生存方式：擅長藏伏、裂谷穿行與沙影狩獵。
- 信仰與儀式：相信 `裂息藏針，沙魂不散`，重沙魂、誓火與穴群記憶。
- 對外關係：與長路民族時常既衝突又交易。

### 霧黽裔
- 代表分布：煙啼玄谷
- 外觀與體質：潮霧、灰斑、煙腺與裂蹼特徵鮮明。
- 生存方式：擅長低視野區域、生濕環境與吐霧掩護。
- 信仰與儀式：奉 `啼魂潛霧，裂腺守誓`，戰鬥與祭儀時會啟動祖脈氣息。
- 對外關係：容易與迷霧地帶的魔人、隱行者或邊地旅人產生複雜關係。

### 熔炭龜裔
- 代表分布：熔痕赤域
- 外觀與體質：背紋、厚甲、耐熱與火脈適應力強。
- 生存方式：擅長在高熱、熔痕、裂縫與黑岩地帶長期生存。
- 信仰與儀式：奉 `熔紋守誓，炭魂不滅`，對祖火與火脈守序非常重視。
- 對外關係：常與鍛造文明、火脈守護者或禁材黑販發生衝突。

### 喚隱裔
- 代表分布：繭影隱林
- 外觀與體質：偏隱匿、輕捷、擅偽裝與低調行動。
- 生存方式：善於隱林、繭影帶與伏蹤環境，強調先藏後動。
- 信仰與儀式：尊信 `繭生幻影，魂隨低啼`，成年後多會選定一條役務支路。
- 對外關係：與外界接觸少，但對密探、灰線或森林文明很有吸引力。

### 潮聲鳥裔
- 代表分布：纏聲斷濱
- 外觀與體質：骨架輕、鳴腔特化、對風潮與音場極敏感。
- 生存方式：靠潮聲、鳴位與海濱斷崖環境建立路徑與群體秩序。
- 信仰與儀式：重海風、鳴濱、回聲與群鳴誓約。
- 對外關係：與港埠文明既可能合作導航，也可能因海路利益而衝突。

### 渣咬裔
- 代表分布：灰釘荒境
- 外觀與體質：齒力強、擅啃蝕、善於低資源與殘渣環境求生。
- 生存方式：擅長吃荒地、舊遺構與裂隙廢帶剩下來的東西。
- 信仰與儀式：重殘料、荒境求活與祖魂殘火。
- 對外關係：常被低估，但在廢城、坑道與垃圾戰利網中非常危險。

### 焦燄鼬裔
- 代表分布：灰釘荒境
- 外觀與體質：細長、靈快、帶火焰與焦氣特徵。
- 生存方式：依沼氣、裂縫潮息與焦原火線共生，機動性極高。
- 信仰與儀式：奉 `焰不熄，魂不息`，以尾焰點燃誓火引導祖魂。
- 對外關係：容易與黑貨、火材、廢坑與危險燃物交易圈糾纏。

### 礦鼠裔
- 代表分布：裂星階
- 外觀與體質：小巧、耐坑道、對礦層與裂隙聲極敏感。
- 生存方式：擅長潛穴、探礦、低語密訊與階隙路網。
- 信仰與儀式：相信 `礦魂藏於沉默之岩`，用尾敲與低鳴保存祖魂記憶。
- 對外關係：與鍛造文明、深廊回收隊和走私礦材圈常有接點。

### 星獅裔
- 代表分布：裂星階
- 外觀與體質：更重視胸腔咆音、觀陣與正面守勢。
- 生存方式：守觀陣、護坑脈、以群吼與星痕儀式維持高地秩序。
- 信仰與儀式：相信 `星痕映於獅咆之路`，把咆哮視為通天誓魂。
- 對外關係：與礦鼠裔互補也互制，對外則常作為高壓守衛者出現。

## 北境寒荒支系

### 月影狼裔
- 代表分布：北境 `暮鏡荒境`
- 外觀與體質：更偏長腿耐寒、群追、雪夜聞痕與寒原長奔。
- 生存方式：擅長在風雪低視野中維持隊形，靠長嗥、短嚎與腳痕判隊友位置。
- 信仰與儀式：重 `戰祖骨歌、輪夢與荒原先靈`，成年標誌更偏向第一次在暴雪裡把同伴與死者都帶回火圈。
- 對外關係：對盧米昂與星紋古道有長期緊張與合作並存的邊地關係，對不守夜、不認錯的外來者最不信任。
- 角色卡建議寫法：`獸人 / 月影狼裔`

### 霜爪熊裔
- 代表分布：北境 `暮鏡荒境`
- 外觀與體質：骨架厚重、耐寒、耐飢，常見重肩、厚掌與更強的短時爆發承壓。
- 生存方式：擅長守火圈、扛風雪、硬撐回收與近距離破陣，是暮鏡最常被推上正面撐線的位置。
- 信仰與儀式：重 `守冬、守火、守骨`，在四獸裔裡最看重誰能把族人熬過最壞的一夜。
- 對外關係：對外人最先看的是你能不能在寒地裡撐住，不太吃花言巧語。
- 角色卡建議寫法：`獸人 / 霜爪熊裔`

### 影尾狐裔
- 代表分布：北境 `暮鏡荒境`
- 外觀與體質：更偏靈巧、低姿、雪霧偽裝與細微路跡辨讀。
- 生存方式：擅長繞線、探路、追蹤失蹤者、讀風與拆穿錯燈假路。
- 信仰與儀式：重 `夜路、回尋與失路者安返`，常和輪夢、送魂與邊地秘密傳聞綁得很深。
- 對外關係：比其他三裔更能和外來探路者、行旅與灰線人物周旋，但也最不喜歡被人探底。
- 角色卡建議寫法：`獸人 / 影尾狐裔`

### 暮羽雀裔
- 代表分布：北境 `暮鏡荒境`
- 外觀與體質：骨架較輕、耳喉與鳴位感更敏銳，對風向、回聲與雪原空曠感非常敏感。
- 生存方式：擅長守望、遠訊、長嗥接力與高風口哨戒，很多危險都是先由他們發現。
- 信仰與儀式：重 `夜嚎、風聽與守望先知`，在四獸裔裡最常接近兆象、輪夢與警戒儀式。
- 對外關係：對守燈者、邊關哨兵與高地巡者相對友善，但對製造噪聲與亂訊的人容忍度最低。
- 角色卡建議寫法：`獸人 / 暮羽雀裔`

## 遊戲使用建議
- 如果角色來自克羅嵐，預設優先寫 `十獸之環・克羅嵐共誓支`
- 如果角色來自克羅嵐，而且想更強調背景，可再加一層 `十獸席位`
- 如果角色來自北境暮鏡荒境，優先在 `月影狼裔 / 霜爪熊裔 / 影尾狐裔 / 暮羽雀裔` 中擇一
- 如果角色來自外域或特殊遺構，可再細分為上面列出的支系
- 地方包若要加獸人 NPC，先決定他屬於哪一個棲地支系，再決定講話方式、信仰與戰法

## 與宗教和神話的關係
- 多數獸人支系沒有單一世界宗教組織，而是靠 `祖靈骨歌 / 地方守地神 / 戰祖神話 / 嚎語儀式` 傳承
- 即使十獸之環已經制度化，它也不是人類式國教共同體
- 獸人支系之間可以共享某些高位權柄的影子，但他們更常透過地方神話去理解，而不是透過殿堂神學
