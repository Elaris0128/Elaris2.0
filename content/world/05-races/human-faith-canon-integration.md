# 人類系舊信仰資料整合規則

## Purpose
這份文件用來說明：舊版 `2.4 信仰與神祇` 資料中，哪些內容要如何被整合進目前的新正典。

它的目標不是保留兩套互相衝突的版本，而是把舊資料裡有價值的神名、儀式、語感與文化結構，接回現在已經定版的人類宗教正典。

## 一、瓦雷溫人

### 新正典定位
- 瓦雷溫人 `沒有國教`。
- 他們制度上尊重多元信仰，不應被寫成只有一個國家正統教會。

### 舊資料保留方式
- 舊稿中的 `維拉特教` 不廢除。
- 但其定位改成：
  - `瓦雷溫社會中流行度很高的主要宗教之一`
  - 尤其在 `軍人 / 騎士 / 邊關守軍 / 誓守家系 / 王都誓職者` 之間影響最大
- 也就是說，維拉特教可以很強、很有歷史、很有儀式重量，但 `不是整個瓦雷溫人的唯一正統國教`。

### 保留重點
- `誓環三環律`
- 軍誓、家誓、誓譜與誓火文化
- 維拉特教的榮光、塔環、軍籍與見證語感

### 使用原則
- 後續若引用舊稿中的維拉特教四主神，應優先重寫成：
  - 維拉特教內部對高位權柄的瓦雷溫式詮釋
  - 或維拉特教神學中的核心誓名與守護位格
- 不要再把它寫成 `瓦雷溫人唯一國教`。

## 二、法奧人

### 新正典定位
- 法奧人 `有國教`。
- 國教正式對應：`序律編院聯席`
- 同時 `尊重其他信仰`。

### 舊資料保留方式
- 舊稿中的 `誓智殿系統` 不直接廢除。
- 但其定位改成：
  - `序律編院聯席在法奧人社會中的法奧式神學結構`
  - `法奧國教內部最重要的一套誓智神學與儀式詮釋`
- 換句話說：
  - `序律編院聯席` 是國教制度本體
  - `誓智殿系統` 是法奧人版本的核心神學外觀

### 舊稿中需要調整的部分
- 舊稿的 `七座主神` 不再直接視為七位彼此獨立、等階並列的真正主神。
- 新版應優先改寫成：
  - `七座誓智`
  - `七座神學位格`
  - `七座條律詮釋位`
- 也就是說，保留名字、職能、儀式用途，但不一定保留它們作為真正獨立高位神的神格地位。

### 可以優先保留的舊素材
- 伊爾賽亞
- 賽里奧
- 維亞
- 奧芙洛
- 洛錫達
- 瑟妲羅
- 阿瑟朗
- 七誓正律
- 灰墨禱讀
- 知律環試

### 使用原則
- 如果舊稿內容與現在的 `秩序 / 條律 / 國教編院` 邏輯衝突，以新正典為主。
- 如果舊稿內容只是提供更細的法奧文化語感、儀式與學理結構，就保留並重新命名整合。

## 三、薩魯恩人

### 新正典定位
- 薩魯恩人 `只有唯一國教`。
- 國教名稱：`巡風教`

### 舊資料保留方式
- 舊稿中的 `風中之靈` 與 `裂風三誓靈` 不廢除。
- 它們應直接整合進 `巡風教` 的正典結構裡。

### 整合後的定位
- `巡風教` 是國家制度上的唯一信仰與宗教。
- 但它的信奉方式 `不是殿堂神祇型`，也不應被寫成像一般人類的教廷神殿系統。
- 它更接近：
  - 森馳裔
  - 精靈
  - 強自然靈性與路徑祭儀文化
- 也就是說，薩魯恩人的國教雖是國教，卻仍保有：
  - 風中之靈
  - 誓場
  - 語旗
  - 沙路
  - 巡行儀式
  - 非擬人化神聖

### 舊稿保留重點
- 裂息神
- 語砂神
- 聽風神
- 裂風與風聲的區別
- 辯行儀
- 巡風語旗
- 靜言曠誓

### 使用原則
- 後續寫薩魯恩人宗教時，應理解為：
  - `巡風教` 是國教名稱
  - `風中之靈 / 裂風三誓靈` 是其內部核心靈性架構
- 不要再把薩魯恩人寫成「沒有國教、只有部族風靈信仰」。

## 四、諾魯因人
- 新正典定位：諾魯因人的信仰不設計成國教，應理解為幽語海地方性的潮祭、祖誓、送魂與守路信仰複合體。
- 舊資料保留方式：舊稿中的 `潮息三面 / 魂泉雙靈` 直接保留，整合為諾魯因人的神話核心與祭儀骨架。
- 保留重點：
  - `裂潮潛誓`
  - `潮霧集宴`
  - `骨火送魂`
  - `風裂骨鳴`
  - `名斷之夜`
- 使用原則：不要把諾魯因人寫成中央教團或跨海大宗教，而要寫成依附在骨鏈、潮路、誓石與送魂程序上的地方制度。

## 五、總整合規則
- 舊版 `2.4 信仰與神祇` 可以繼續當作素材庫使用。
- 但凡遇到 `制度定位` 與新正典衝突時，以目前 `content/` 下的新檔為主。
- 若舊稿內容主要是：
  - 神名
  - 儀式
  - 語感
  - 神話畫面
  - 地方文化結構
  則應優先保留並重寫整合。
- 若舊稿內容直接衝撞：
  - 是否有國教
  - 是否唯一信仰
  - 是否多元承認
  - 是否屬於某世界級宗教組織
  則以新正典為優先。
