# 種族神話版本索引

## 目的
這個目錄收納 Elaris 各主要種族、族群與特殊文明的 `正式神話版本`。

這裡處理的不是：
- 世界真實層的神系骨架
- 世界級宗教組織的教義制度
- 各神的完整個體卡

這裡處理的是：
- 各種族如何理解 `萬初者` 與 `七大高位神`
- 各種族的族源母題、禁忌、祭儀與在地神話語感
- 各種族在 `律神 / 昇格神 / 守地神與祖靈` 這三層最常長出的方向

## 文件分組
- `core-races-myths.md`：古典主種族神話版本
- `human-lineages-myths.md`：人類系族群神話版本
- `special-peoples-myths.md`：棲裔、巴布窟族等特殊文明神話版本

## 使用原則
- 先用 `../../01-foundation/creator-and-high-gods.md` 確認通用神名。
- 再用這裡的檔案看各種族如何把同一批高位權柄轉譯成自己的傳承。
- 若要看種族和宗教制度如何接回世界層，搭配 `../racial-myth-and-faith-integration-map.md`。
- 若要看各種族更常靠近哪些下位神，搭配 `../race-lower-deity-matrix.md`。
- 若要看律神、昇格神與守地神的跨種族方向，搭配：
  - `../../01-foundation/law-god-layer-directions.md`
  - `../../01-foundation/ascended-god-layer-directions.md`
  - `../../01-foundation/guardian-and-ancestor-layer-directions.md`
