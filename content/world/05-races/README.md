# 種族索引層

這裡收納新架構下的種族總索引、支系整理與世界級細化介紹。

## 保護規則
- 不得少於 `data/manifests/race-preservation-manifest.md` 所列的 18 個既有入口
- 若補充分支，採增量方式處理
- 舊資料仍為原始來源，這裡先作為索引層與整理層

- `orc-lineages.md`：獸人分支詳解
- `dragon-lineages.md`：龍族主系、守域支系與遠系綱要
- `dragon-house-surnames-and-blood-families.md`：龍族姓氏、家族血統與正式長名
- `dragon-physiology-and-growth.md`：龍族生理構造、成長階段與弱點
- `dragon-language-memory-and-oaths.md`：龍族語言、長記憶與誓約秩序
- `dragon-remote-kin-and-subspecies.md`：龍族遠系、亞種與各地龍裔差別
