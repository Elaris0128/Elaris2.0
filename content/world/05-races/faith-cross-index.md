# 種族與神聖傾向交叉索引

## Purpose
這份文件整理目前 Elaris 各主要種族與族群，較可能偏向哪一類神聖存在與信仰結構。

這不是把每個種族鎖死成只能信一種神，而是先建立 `主傾向 / 次傾向 / 術式影響 / 常見信仰形態` 的世界通用參考。

## 神聖存在大類
- `源神`：真正高位、古老、穩定的神
- `律神`：法則、自然、秩序人格化的神
- `昇格神`：由英雄、聖者、王者或特殊存在神格化而成
- `守地神 / 祖靈`：與土地、氏族、墓地、河道、山口、生命網綁定的神聖存在

## 古典主種族

### 魔人
- 主傾向：`源神 + 禁忌律神`
- 次傾向：`昇格神`
- 正式傳承名：`淵中誓主信譜`
- 術式影響：偏高風險、高代價、契約性或禁忌性術式發展
- 常見信仰形態：禁約神話、家系戒律、災厄敘事與危險儀式，未必形成中央教團

### 森馳裔
- 主傾向：`律神 + 守地神`
- 次傾向：`祖靈`
- 正式傳承名：`曠野誓環與五風靈`
- 術式影響：自然、追跡、林境調和、場域與生態共鳴術發達
- 常見信仰形態：林徑神話、守徑祭儀、古樹見證與季節性共生傳承

### 矮人
- 主傾向：`律神 + 昇格祖匠神`
- 次傾向：`守地神`
- 正式傳承名：`鎚火岩三象傳承`
- 術式影響：鍛造、火脈、結構穩定、附著與器具術發達
- 常見信仰形態：祖匠火脈傳承、工坊誓詞、鍛成儀式與山體禁忌

### 地精
- 主傾向：`昇格神 + 地方守護神`
- 次傾向：`律神`
- 正式傳承名：`咬誓成鐵`
- 術式影響：小型工藝術、機巧輔術、交易與生活型術式發達
- 常見信仰形態：聚落小神、灶火習俗、攤位口彩與地方守護故事

### 海人
- 主傾向：`源神 + 律神`
- 次傾向：`守海存在`
- 正式傳承名：`潮魂回巡傳統`
- 術式影響：潮汐、航行、深水感應、水域防護與風浪術發達
- 常見信仰形態：潮裔回潮傳統、海葬、航季儀式與海流口傳

### 精靈
- 主傾向：`源神 + 律神`
- 次傾向：`守地神`
- 正式傳承名：`母誓之源與靈詩流息`
- 術式影響：高位神學術式、長時維持術、儀式術與古老領域術成熟
- 常見信仰形態：古誓星樹傳承、長年儀式、高位祭儀、古語典籍解讀，以及以 `靈誓伴` 為核心的成年尋伴禮

### 翼人
- 主傾向：`律神 + 天象源神`
- 次傾向：`昇格神`
- 正式傳承名：`空律三紋與風誓之道`
- 術式影響：風、光、視界、移動、預兆與高空場域術發達
- 常見信仰形態：高穹兆詩、觀風儀式、飛巡祭壇與天空預兆詮釋

### 獸人
- 主傾向：`祖靈 + 昇格戰祖`
- 次傾向：`守地神`
- 正式傳承名：`戰祖骨歌與原野先靈`
- 術式影響：狩獵、血勇、群戰、氏族誓印與戰祭儀式偏強
- 常見信仰形態：氏族神話、戰歌、墓守、成年試煉與部族儀式，不優先形成中央宗教

## 人類系族群

### 瓦雷溫人
- 主傾向：`源神 + 律神`
- 次傾向：`昇格聖者`
- 術式影響：國教式誓約術、見證術、裁決術與王權合法性術式成熟
- 常見信仰形態：多元承認型；沒有單一國教，但 `維拉特教` 是最具影響力的主要宗教之一，尤其在軍人與誓守階層中極強

### 法奧人
- 主傾向：`律神`
- 次傾向：`源神`
- 術式影響：學理化、制度化、可教學的正規術學體系發達
- 常見信仰形態：有國教，正式對應 `序律編院聯席`，但尊重其他信仰；舊有 `誓智殿系統` 應併入其法奧式神學結構

### 極光徙裔
- 主傾向：`天象源神 + 祖靈`
- 次傾向：`律神`
- 正式傳承名：`星跡五魂`
- 術式影響：星光、寒夜、旅途、夢兆、遠路導引術發達
- 常見信仰形態：流動旅路信仰、夢兆與天象祭儀，不優先形成國教

### 魈陽族
- 主傾向：`烈日律神 + 守地神`
- 次傾向：`昇格神`
- 正式傳承名：`熔誓神環`
- 術式影響：高照、灼熱、試煉、防沙、防乾竭與烈地術學偏強
- 常見信仰形態：地方烈日神話、乾地試煉儀式與群體生存禁忌

### 崖影族
- 主傾向：`守地神 + 祖靈`
- 次傾向：`律神`
- 正式傳承名：`碑靈系統`
- 術式影響：陰影、崖道、藏路、伏行、墓守與回音術偏強
- 常見信仰形態：崖墓儀式、藏路祖靈與地方氏族故事

### 巖穹族
- 主傾向：`山體律神 + 昇格祖匠神`
- 次傾向：`守地神`
- 正式傳承名：`山穹祖匠火脈`
- 術式影響：石脈、壁壘、礦材、建構、防護與工藝術偏強
- 常見信仰形態：山邦祭儀、祖匠譜系與火脈工藝信仰

### 薩魯恩人
- 主傾向：`風與旅途律神 + 流動祖靈`
- 次傾向：`守地神`
- 術式影響：旅程保命、風向感知、補給術、群行調節與短儀式術成熟
- 常見信仰形態：唯一國教，正式對應 `巡風教`；但其實踐方式非殿堂神祇型，內部保留 `風中之靈` 與 `裂風三誓靈` 的靈性結構

### 柯沙里人
- 主傾向：`水路律神 + 商約神聖觀`
- 次傾向：`地方守航神`
- 正式傳承名：`河契航燈信俗`
- 術式影響：河運、航標、護貨、查驗、印契與交易見證術成熟
- 常見信仰形態：港市聖所、商約祭儀、水路守航神話與地方習俗

### 諾魯因人
- 主傾向：`寒地源神 + 祖靈`
- 次傾向：`律神`
- 正式傳承名：`守夜寒靈傳統`
- 術式影響：守夜、嚴寒、雪地生存、寒傷處理與安魂術偏強
- 常見信仰形態：寒地祖靈、守夜傳承、生存與安魂儀式

### 法雷恩人
- 主傾向：`源神 + 律神 + 守夜祖靈`
- 次傾向：`守地神`
- 術式影響：誓燈導式、理約冰印、守關見證術、寒地封護與安魂術成熟
- 常見信仰形態：有國教，正式對應 `誓光見證庭`；神話敘事需保留 `正統主張` 與 `被迫害後重建` 的核心

## 特殊獨立族群

### 巴布窟族
- 主傾向：`洞窟守地神 + 祖靈`
- 次傾向：`禁忌小神或地底異神聖存在`
- 正式傳承名：`微光四誓名`
- 術式影響：洞域辨路、深層回音、礦洞防災、地底封護與禁忌祭式偏強
- 常見信仰形態：洞靈回音祭系、洞口祭、封坑禁忌與深層故事；其中部分洞域分支也會與雲瑟皇族共享共息共居的契儀

## 使用原則
- 這份索引回答的是 `傾向`，不是鐵律。
- 同一種族在不同國家與地區，仍可能發展出完全不同的神學表層。
- 真正建立神系時，應優先保留高位共通神，再讓各族用自己的語言與制度詮釋。
- 若要看各族傳承名、既有神話名稱與新神系的正式對接，應搭配 `racial-myth-and-faith-integration-map.md` 一起使用。
- 若要看各主要種族已展開的正式神話版本，應搭配 `myths/README.md` 一起使用。
- 若要進一步看人類族群和國教、制度宗教的正式版本，應搭配 `human-national-faith-alignment.md` 一起使用。
- 若要處理舊版人類信仰稿與新正典的整併，應搭配 `human-faith-canon-integration.md` 一起使用。
