# 龍族主系、守域支系與遠系綱要

龍族在 Elaris 裡不是「比較強的飛龍」，而是一個具備長記憶、嚴格成長秩序、主家分工與守界使命的高智慧古族。

若從奧崙本體讀，龍族會先把彼此分成 `主系 / 支系 / 遠系`，而不是先問血緣是否純正。  
對他們來說，`你替哪一段天空承責`，往往比 `你出自哪一枚卵` 更先決定你的身分。

## 一、正統三主系

### 星殞正裔
- 對應家域：`星殞高原`
- 核心責任：孵巢、育裔、熱脈調度、試翅與第一次吐息校準
- 氣質：嚴、熱、重血脈次序，最不容許假名與錯列
- 常見判準：哪一枚卵該先破殼、哪一名幼裔該延後試翅、哪一條暖脈足以養住下一季幼群
- 對外觀感：最像「純龍主血」，也最容易被誤解成傲慢保守

### 煌渦守裔
- 對應家域：`煌渦穹塢`
- 核心責任：守界、護息、觀風、祖骨長記與高空裂縫巡守
- 氣質：安靜、耐久、記錄重於辯白
- 常見判準：哪一道高風裂線需要封閉、哪份古誓仍有效、哪條空路還能承載重型飛行
- 對外觀感：最像守門者，也最接近翼人記錄裡的「不與凡語多談的大龍」

### 隱宙歸裔
- 對應家域：`隱宙群巔`
- 核心責任：回名、歸終、放逐後複核、最後裁定與殘誓收束
- 氣質：霧重、冷靜、對終局與失敗極有耐性
- 常見判準：誰還配被記名、誰該帶著最後稱號歸終、誰只適合被留在斷鏈與霧縫之間
- 對外觀感：最難親近，但其實最擅長看出偽悔與假悔

## 二、守域支系

### 塵光試翅支
- 常駐 `塵光散環`
- 由年幼龍、初獵龍與試翅監領者組成
- 不被視為獨立主系，但承擔整個龍族文明最危險的「第一次離巢」

### 幽輝斷鏈支
- 常駐 `幽輝斷鏈`
- 包含被放逐者、平誓重立者、短期互保者與少數監鏈裁記者
- 不是單純罪龍聚集地，而是龍族用來容納失序、異脈、失位與未完全斷絕責任者的危險支域

### 渺徑孤鱗支
- 常駐 `渺徑浮鱗`
- 多為孤行龍、殘裔、邊緣守望者與不再完全被三主系吸納的個體
- 對主家有用，但很少真正被主家信任

### 朧潮息傷支
- 常駐 `朧潮靜層`
- 由重傷、停翼、調息與再飛複核相關龍裔構成
- 在龍族內部地位尷尬，因為他們常代表「飛不起來但還沒死」

### 百巢承契支
- 常駐 `百巢下望島` 與下垂承契帶
- 是奧崙對外最低空、最有限、也最受監視的一條龍族支線
- 不負責統治，只負責 `育契、寄養、觀樣、引升與退件`

## 三、遠系與亞種

### 奧崙近系
- `裂風脊龍`
- `星塵細翼龍`
- `高巢暖流飛龍`
- 這些仍明顯承接奧崙主脈，但已不屬於正統三主系本裔

### 伊莎梅爾高空遠系
- `冠脊飛龍`
- `天輝風龍`
- `寒脊飛龍`
- 它們與龍族共享高空祖脈，但已經和精靈文明的守望、巡飛與高枝巢地一起演化

### 其他地區遠系
- `霜脊飛龍`
- `深潮盤龍`
- `岩脈伏龍`
- `礁火巢龍`
- 這些更常在其他地區以守域支裔、亞種王者或遠古災痕的形式出現

## 四、龍族如何看待「純血」
- 龍族當然在乎血脈，但他們真正焦慮的不是混血本身，而是 `失去可被驗證的長記與承責鏈`
- 一名不夠純的個體，若仍能接上家域、守住誓約、承完職責，往往比一名只剩高貴血脈卻守不住天空的本裔更值得保留
- 因此龍族的殘酷，常不是在驅逐弱者，而是在驅逐 `不肯承責卻又想佔位者`

## 五、龍冕鱗冠
- `龍冕鱗冠（Drakecorona Crest）` 在龍族裡不是單純聖物，而是極少數會把 `血脈、主權、長記與守界責任` 綁成同一件事的冠印象徵
- 它被視為與 `曜鱗聖幕` 深度綁定的古老遺印
- 多數龍一生不會見到它，只能從長記與古誓裡知道它的存在
- 在奧崙裡，真正能宣稱自己有資格接近它的，不會是最會說的人，而是最少犯錯、最久守界、最能背住舊誓的人

## 六、主系底下的家族姓
- 龍族不是只有 `主系 / 支系 / 遠系` 的大分類。
- 在奧崙本體裡，三主家與守域支裔底下還各自有不同 `冠姓 / 支姓`，代表更細的血統傾向與家族責任。
- 例如：
  - 星殞主家常見 `曜穹氏 / 赤熾氏 / 星垣氏`
  - 煌渦主家常見 `霆渦氏 / 玄守氏 / 裂衡氏`
  - 隱宙主家常見 `霧隱氏 / 歸寂氏 / 宵裁氏`
- 詳見：`dragon-house-surnames-and-blood-families.md`

## 七、使用建議
- 若要寫龍族核心人物，先決定他屬於哪條主系或守域支系
- 若要寫某地出現的龍裔魔物，先決定它是 `近系 / 遠系 / 殘血偽裔`
- 若要寫與其他文明互動，優先從 `百巢承契支` 與 `伊莎梅爾高空遠系` 下手
