# 魔人生理構造、器官與弱點

## 共通原則
- 魔人七族不是只有文化差異，而是都有明確的 `族裔專屬器官或構造`。
- 這些器官決定了他們的術式、社會地位、成年禮與致命弱點。
- 若器官受損、封鎖、反噬或被拆解，常會引發比一般外傷更嚴重的崩潰。

## 七族器官總表

| 族裔 | 專屬器官 | 典型弱點 |
| --- | --- | --- |
| 曜血 | `曜血核脈` | 核脈重創會導致血律崩解與自滅 |
| 鳴紗 | `魅息腺` | 腺封、腺裂會造成魅惑系崩潰與反噬 |
| 垢金 | `垢金核` | 核碎後常出現自腐、崩裂與失序礦化 |
| 幽鬩 | `影心珠` | 珠毀則奪形能力崩散，形魂皆裂 |
| 獰罡 | `血爆囊` | 囊裂會被自身血焰逆噬 |
| 緲眠 | `夢核` | 夢核受損會陷入永續夢魘或魂醒失衡 |
| 噬淵 | `噬源` | 噬源被封會導致自噬、饑裂與骨崩 |

## 年齡階段
- `曜血`：幼裔 / 昇裔 / 貴裔 / 冕裔
- `鳴紗`：幼魅 / 魅形 / 織魅 / 蠱魅
- `垢金`：幼貪 / 金核 / 金侯 / 金獄主
- `幽鬩`：幽胚 / 影襲 / 鬩主 / 鬩影王
- `獰罡`：血胚 / 血烈 / 血罡 / 罡裂主
- `緲眠`：夢胚 / 夢寄 / 夢師 / 夢淵主
- `噬淵`：噬胚 / 噬形 / 噬渦 / 淵主

## 舊深淵人口估值

## 器官的社會意義
- 曜血會以核脈完整度判斷貴脈正當性。
- 鳴紗會用魅息腺成熟度區分是否能主持魅宴。
- 垢金以金核穩定度決定是否能接手契錄與封庫。
- 幽鬩的影心珠會影響其奪形層次與寄影壽限。
- 獰罡的血爆囊越穩，越能駕馭高烈戰印。
- 緲眠的夢核穩定與否，直接關係寄魂與守夜能力。
- 噬淵的噬源越完整，越能承受吞噬與孵化反噬。

## 深淵外生存問題
- 流亡到外界的魔人，最先要處理的往往不是身份，而是器官護層失去熟悉環境後的退化。
- 因此涅厄斯這類外流共同體，會特別重視器官保養、替代藥劑、假名保全與深井環境模擬。
