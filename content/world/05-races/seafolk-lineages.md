# 海人分支、潮居文化與外海守域

## 定位
- 海人是 Elaris 的長存海居種族，不是臨時變異出的旁支，也不是單一海國人民。
- 他們真正共享的核心不是國號，而是 `潮路倫理 / 活水觀 / 可持續採收觀 / 守域責任`。
- 在海人眼裡，海不是背景，而是會記住傷痕、會回報錯誤、也會拒絕被濫用的活體。

## 正典分支
- 海人現行正典只保留三大潮裔分支：
  - `潮霧之裔`
  - `熾潮之脈`
  - `鱗盾之裔`
- 其他地方上的巡守圈、採收圈、隔離圈、白化觀測圈，都屬於 `地方制度與職責圈`，不另外算作種族分支。

## 潮霧之裔
- 主要分布：幽語海。
- 文化氣質：低聲、守霧、藏路、重回聲與暗潮判讀。
- 常見特徵：更擅長迷霧潮窟、低光礁帶、暗流改道與悶聲海窗。
- 支系專屬構造：`潮囊`、`霧鰭脈`
- 對外態度：審慎、少話，願意合作，但不喜歡被逼問老路。

## 熾潮之脈
- 主要分布：焰潮洋。
- 文化氣質：熱潮、活珊、熱泉主渠、危材慢處理與守域責任。
- 常見特徵：更能適應高熱、毒潮、熱泉裂帶與高壓危窗。
- 支系專屬構造：`熾脈核`、`鰭刺束`
- 對外態度：務實、能談交易，但一旦踩壞活路會立刻翻臉。

## 鱗盾之裔
- 主要分布：鱗盾海懸與纏聲斷濱一帶。
- 文化氣質：重殼、重鳴、重群護與礁脊防禦。
- 常見特徵：更偏甲殼感、背甲防護與潮聲共鳴。
- 支系專屬構造：`鱗盾背甲`、`潮聲鰭刺`
- 對外態度：區域性開放，對邊海防守與集體秩序看得特別重。

## 共同文化核心
- 海人很少先問「值多少」，更常先問：
- 這條水還活不活。
- 這條路下一輪的人還能不能走。
- 這一批採收會不會害整段海床、幼生帶或主槽死掉。
- 因此海人看起來常比外地人保守，其實只是把 `長久能活` 看得比 `眼前能賣` 更重。

## 家與守域
- 海人的家不是單一一棟房，而是一整組 `能住的棚 / 能養的槽 / 能走的水路 / 能採的窗口`。
- 一個家系是否完整，很多時候不是看人口，而是看那幾段老路是否仍守得住。
- 所以海人的家系感更接近：
  - 我們守哪段潮路
  - 我們靠哪口泉活
  - 我們在哪一圈礁帶長大

## 成年與承責
- 海人的成年往往不是年齡，而是 `第一次獨自守住一段活路`。
- 常見成年條件包括：
  - 正確封掉一條會害死後隊的假路
  - 在不污染主槽的前提下帶回樣本
  - 在壞潮裡處理完危材仍保住活路
  - 在群體退潮時知道何時該讓、何時該退、何時該封

## 婚配與親密
- 海人的婚配觀建立在 `共守 / 共潮 / 共路` 上。
- 願意共同照顧一口泉、守一段礁路、維持一組採收窗口，在海人眼裡往往比空泛誓言更接近真正的親密。

## 潔淨與禁忌
- 海人追求的不是表面潔淨，而是 `不把活水弄死`。
- 重禁包括：
  - 不可任意切取活珊骨
  - 不可將不明高危樣本直接倒回主水
  - 不可為了短期利益害整段採收帶失活
  - 不可在守域裡故意染路、炸路、灼路
- 海人眼裡最重的罪，往往不是偷，而是 `斷後路`。

## 對外態度
- 對懂規矩的外來採樣者、公會船與藥師：能談，也能合作。
- 對海盜：不先天敵視，但只要踩壞活珊與主泉，就會被視為壞路者。
- 對黑市買手：最不信任，因為這類人最常把一輪潮窗當一次性耗材。
- 對諾魯因人這類近深潮人類守路者：比對一般陸民更尊重，但仍保留很清楚的界線。

## 地方守域圈不是新分支
- 焰潮洋、幽語海與其他海域裡，確實會形成很多地方性的：
  - 巡守圈
  - 採收圈
  - 隔離圈
  - 白化觀測圈
  - 灰線協調圈
- 但這些都屬於地方制度，不再另外升格為海人的種族分支。
