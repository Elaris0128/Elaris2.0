# 海人語言、潮路記號與記憶方式

海人的語言不只是一套口說文字，而是一整套 `聲音 + 手勢 + 結繩 + 標記 + 口傳句` 混用的溝通系統。這是因為他們很多最重要的資訊，本來就不適合只靠紙張或石碑保存。

## 語言的多層結構
- `日常口語`
  - 用來談家庭、交易、食物、日常分工。
- `潮路術語`
  - 用來描述潮窗、危流、回扯、白化、活脈、死水等。
- `守域短句`
  - 極短，但責任感很重，常用於警告、封路與放路。
- `口傳路記`
  - 只聽文字未必懂，必須搭配水況、地形與經驗判讀。

## 為什麼不只靠文字
- 海人知道紙會濕、木會爛、石會沉，真正能活過壞潮與搬遷的資訊，往往必須存在人身上。
- 因此，他們對 `記得` 這件事的理解，和陸上種族很不同。
- 好的記憶，不是背下很多詞，而是知道在哪種潮況下該講哪句話、該停在哪裡、該怎麼讓後隊一聽就懂。

## 手勢與無聲規矩
- 在狩窗、淺棚、病水與外洋邊界，海人常盡量少喊。
- 他們會用：
  - 手勢
  - 藻結
  - 珊扣方向
  - 吊牌位置
  - 甚至工具怎麼放
  來傳達「可走 / 不可走 / 有病 / 有外人 / 先退」。

## 命名方式
- 海人對外常報個體名，對內卻更看重：
  - 哪條路養你
  - 哪圈礁收你
  - 哪家棚讓你第一次承責
- 所以很多海人名字會帶有：
  - 潮
  - 礁
  - 槽
  - 汀
  - 瑚
  - 藻
  - 鹽
  之類和環境、功能有關的字。

## 誰有資格講哪種話
- 並不是誰都能用所有守句。
- 某些封路詞、退路詞、白化帶判詞，只能由真正在那段路承責的人來講。
- 若外人偷用、亂用或誤用，海人會把那當成很嚴重的越位。

## 記憶與羞恥
- 海人很怕 `記錯會害死人`。
- 因此比起誇耀自己記得多少，他們更在乎你有沒有在不確定時承認自己不確定。
- 在海人文化裡，背錯一段老路、害後隊踩進死窗，常比單純失言更嚴重。
