## Canon Anchor
- 舊檔正典裡的 `天脈島弧` 為 `靈穹洋北緣的半固定浮島帶`，並明確存在 `漂泊市集`。
- 這裡不是曦環本體，也不受單一勢力穩定統治，而是 `浮島中立圈、樣本採收帶、空艇泊補線與灰線過渡層`。
- 曦環浮境在北段高風位與迎商季時，會與天脈島弧形成最密切的接觸。
