# 天脈島弧抵達方式與泊靠流程

## 核心原則
- 天脈島弧不是想去就能直接撞上去的浮島帶。
- `先到哪一層`、`有沒有泊權`、`是不是趕在霧瀑穩定窗內`，會決定你是抵達，還是只是接近。

## 常見抵達方式
### 1. 由曦環浮境北段外務線進場
- 最穩的方式。
- 常見路線是 `暮渦曦岑 -> 螺環泊庭 -> 天脈島弧漂泊市集`。
- 多數正式外務、借泊隊、高價輕貨和臨時護航會走這條。
- 優點是安全、容易拿到正式票面。
- 缺點是最容易被記進簿裡。

### 2. 空艇直抵霧瀑泊墩
- 適合有自己船和引升手關係的人。
- 需要先拿到 `泊權票` 或 `逆升准位`。
- 若碰上迎瀑季或浮晶季高峰，排位比距離更重要。
- 這是最像「直接抵達」的方式，但也是最花錢的方式。

### 3. 海船先到北緣低泊區，再換升
- 海船通常不能直接進市集核心。
- 常見作法是先抵達 `靈穹洋北緣的低泊點與空霧泊圈`，再用引升小艇、索籃或霧瀑窗口轉上浮島帶。
- 這類路線最吃潮色與霧瀑穩定度，也最容易被人收兩次錢。

### 4. 灰線夜路
- 走 `空霧泊點 -> 礁脈偏橋 -> 漂泊市集夜角` 這種不完全上簿的路。
- 快、靈活、好藏貨，但最容易出現假引港、假橋和假泊墩。
- 第一次來的人不該自己走。

## 抵達流程
### 正規線
- 報到
- 驗牌
- 量風
- 核貨
- 分流到 `漂泊市集 / 霧瀑泊墩 / 浮晶曬場`

### 灰線
- 先被帶去看風，不先看票
- 先談要不要留名，不先談價
- 先確認今晚哪條橋算橋

## 什麼時候最好到
- `迎瀑季`：最容易進，但最貴。
- `浮晶季`：人最多，機會也最多。
- `靜橋季`：最穩，適合重貨與長談。
- `斷鏈季`：最危險，但灰線利潤最高。

## 玩家視角
- 第一次來，最合理的是跟著 `曦環外務 / 空艇商隊 / 被委託押送的樣本隊` 進場。
- 如果想把天脈玩成灰線地圖，則適合從 `海船低泊 -> 空霧泊點夜路 -> 漂泊市集` 這條線切入。
