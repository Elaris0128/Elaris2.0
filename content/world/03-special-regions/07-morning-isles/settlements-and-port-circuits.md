# 晨穗群島地方聚落與港埠迴路

## 主地方帶
### 東埠與港契帶
- 核心：晨穗埠城、白晶埠、潮牙寨、朝羽鐘埠
- 關鍵字：稅、燈、鹽、港牆、巡艦、正式航道、礁路鐘序

### 穗心內渠與分治帶
- 核心：潮契石庭、屯莊市集、塍羽糧階、雙旗埠頭、埠間渠聚點
- 關鍵字：雙國共治、糧契、爭簿、內水路、混合居民、封倉橋

### 西汐散島與暗泊帶
- 核心：潮汐堡、礁堡島、鷗盜村、蘆潮莊、蘆魂回燈場、葦羽潛泊、潮羽裂泊
- 關鍵字：海盜、黑市、浮棚、沼路、誓群、假燈、急泊、替箱

### 東籬鹽盤與稅路帶
- 核心：白晶埠、鹽羽監盤寮、鹽羽白盤池
- 關鍵字：白盤、封鹵井、盤差、鹽稅、潮損、樣鹽、偷盤

## 半海島居住方式
- 很多地方不是單純的岸邊聚落，而是 `架高屋 + 短橋板 + 屋腳小舟 + 潮泥空層` 一起構成的住居單位。
- 蘆隱島、葦羽潛泊和蘆魂回燈場一帶，白天像踩在泥上的棚村，夜裡則像整座村莊慢慢浮起來。
- 很多倉屋和住屋沒有清楚邊界，因為屋腳下放的是船、貨還是人，常常取決於今晚的潮和帳。
- 對晨穗人來說，地基不是不動的石，而是 `今年還願意給你踩的潮位`。

## 推薦地方迴路
- 正式秩序線：晨穗埠城 -> 朝羽鐘埠 -> 潮契石庭 -> 白晶埠
- 糧港衝突線：塍羽糧階 -> 屯莊市集 -> 潮契石庭 -> 晨穗埠城
- 黑路潛航線：潮汐堡 -> 鷗盜村 -> 葦羽潛泊 -> 潮羽裂泊 -> 蘆潮莊
- 混合玩法線：朝羽鐘埠 -> 潮契石庭 -> 屯莊市集 -> 鷗盜村 -> 潮隙礁道
- 雙帳轉運線：雙旗埠頭 -> 晨穗埠城 -> 潮契石庭 -> 雙旗暗窖
- 礁堡灰默線：礁堡島 -> 礁堡骨棧 -> 潮汐堡
- 白盤稅路線：鹽羽監盤寮 -> 鹽羽白盤池 -> 白晶埠
- 回燈沼路線：蘆魂回燈場 -> 蘆魂骨燈坪 -> 蘆潮莊
