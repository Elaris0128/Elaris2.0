## 使用方式
- 這批存在不是一般常駐角色。
- 更像 `可被請見、可被等待、可被恐懼、可被傳聞` 的活傳說。
- 很適合拿來處理：
  - 高位背書
  - 主家失語
  - 舊誓翻案
  - 旁脈請誓升級
