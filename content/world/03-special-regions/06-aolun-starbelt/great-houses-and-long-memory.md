# 長記三主家與家域分工

## 三主家

### 星殞主家
- 核心責任：誕生、孵巢、試翼與血脈第一判
- 核心地點：星殞高原、星骨孵岬、星殞潤原
- 常見偏向：不能拿幼裔去試政治賭注
- 常見冠姓：`曜穹氏 / 赤熾氏 / 星垣氏`

### 煌渦主家
- 核心責任：守界、護息、祖骨記錄與裂風修補
- 核心地點：煌渦穹塢、界霆回聲谷、穹鎖裂縫帶
- 常見偏向：過晚封鎖通常比錯封更危險
- 常見冠姓：`霆渦氏 / 玄守氏 / 裂衡氏`

### 隱宙主家
- 核心責任：回名、歸終、放逐後段處理與最後靜定
- 核心地點：隱宙群巔、隱息暗洄、霧橋群巔
- 常見偏向：失敗與終局也是秩序的一部分
- 常見冠姓：`霧隱氏 / 歸寂氏 / 宵裁氏`

## 守域支裔
- 承擔巡界、校風、驅逐、調節與引升的實務層。
- 很多高風險工作其實由支裔完成，但最終裁定權仍集中在三主家與高冠記星環。
- 常見支姓：
  - 塵光：`試翎氏 / 曬塵氏`
  - 幽輝：`斷鏈氏 / 影誓氏`
  - 渺徑：`浮鱗氏 / 孤岬氏`
  - 朧潮：`靜潮氏 / 眠翎氏`
  - 百巢：`承巢氏 / 下望氏`

## 長記的政治性
- 奧崙沒有固定皇座，真正接近「中央」的，是能把最長記憶保存得最完整的那群龍。
- 長記不是單純歷史，而是 `誰有資格證明現在的判斷不會重蹈舊錯`。

## 三主家之間的拉扯
- 星殞主家：育裔優先
- 煌渦主家：守界優先
- 隱宙主家：回名與結束優先
- 奧崙很多衝突不是爭王，而是三套責任排序互相衝突。

## 婚盟與繼序怎麼成為政治
- 星殞會把聯巢誓看成保脈與保巢的手段。
- 煌渦會先問：這樁配對有沒有削弱守界穩定。
- 隱宙則最在意：這會不會留下模糊不清的承名與繼序糾紛。
- 因此奧崙的家族政治，常常不是直接鬥爭，而是透過：
  - 聯巢誓
  - 借姓幼裔
  - 繼序簿
  - 冠印請承
  去彼此拉扯。

## 現在最尖銳的主家爭議
- `不會飛的祖脈還算不算龍`
- `海中旁脈是否永遠只能停在承契帶外環`
- `停翼者保留名字是否會動搖高空權威`
- `冠印與長記能否因功績破例外授`
