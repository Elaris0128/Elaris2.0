# 亡極徹淵文化職業與流派

亡極徹淵的職業不是中性的工作分類，而是和層位、族裔、誓位與可用性直接綁在一起的 `文化職業`。在深淵裡，一個人被怎麼稱呼，通常也代表他在哪條秩序鏈上值多少價。

## 外環幽裂

### 守夜判聲師
- 對應通用職業：斥候 / 判識者 / 低階術士
- 常見層位：外環幽裂
- 獲得條件：完成一次正式守夜輪值，並在真假命令混入時做出正確分流。
- 專屬術式 / 戰技：
  - `靜骨聽返`
  - `霧橋折聲`
- 流派方向：
  - `靜聽流`：偏守橋、留痕、慢判。
  - `裂返流`：偏追聲、回找、反滲透。

### 夢囊縫霧匠
- 對應通用職業：工匠 / 療護者
- 常見層位：外環幽裂
- 獲得條件：能獨立修補一整座夢囊棚，並讓寄魂者穩定渡過一輪夢蝕波動。
- 專屬術式 / 戰技：
  - `薄霧補囊`
  - `長息縫護`

## 裂廊上層

### 血環禮裁官
- 對應通用職業：政務官 / 神職 / 審判者
- 常見層位：裂廊上層
- 獲得條件：通過至少一場血宴或授印程序而未失禮，且被貴脈記錄承認。
- 專屬術式 / 戰技：
  - `席紋落判`
  - `血律封步`
- 流派方向：
  - `內席流`：偏禮法、議位、附錄。
  - `懸廊流`：偏橋權、通行、對外見證。

### 懸橋授印侍劍
- 對應通用職業：侍衛 / 劍士 / 儀衛
- 常見層位：裂廊上層
- 獲得條件：在橋階值守與禮儀護衛中同時守住格式與武力要求。
- 專屬術式 / 戰技：
  - `血橋三踏`
  - `印鋒留席`

## 裂廊中層

### 幻宴引席師
- 對應通用職業：社交操演者 / 幻術師 / 掮客
- 常見層位：裂廊中層
- 獲得條件：能讓一場宴局按自己的引導走完，且不被反操引。
- 專屬術式 / 戰技：
  - `紗息牽目`
  - `魅席換心`
- 流派方向：
  - `香幕流`：偏場面、美感、情緒操引。
  - `低語流`：偏密談、真話套取、逐步誘導。

### 記胃孵錄官
- 對應通用職業：記錄官 / 巢育官 / 汙染判定者
- 常見層位：裂廊中層
- 獲得條件：在母巢秩序中守住一輪孵序與胃簿，不讓錯記擴大成群體失序。
- 專屬術式 / 戰技：
  - `胃簿回錄`
  - `囊壁封紋`

## 裂廊下層

### 熔戰壓陣士
- 對應通用職業：前線重兵 / 壓陣官
- 常見層位：裂廊下層
- 獲得條件：撐過一次熔戰推進與戰押交接，且隊列未在你面前崩掉。
- 專屬術式 / 戰技：
  - `焰鏈推列`
  - `重甲墜砸`
- 流派方向：
  - `戰窟流`：偏正面推進與火力壓制。
  - `裂市流`：偏戰押、護貨、硬拆通路。

### 奪形潛名手
- 對應通用職業：刺探者 / 影務者 / 偽裝者
- 常見層位：裂廊下層
- 獲得條件：在一次奪形任務中帶著假身份全身而退，且沒有留下足以逆追的聲線破口。
- 專屬術式 / 戰技：
  - `借貌伏形`
  - `碎句換稱`

## 漩淵渦心

### 封庫驗秤官
- 對應通用職業：審計官 / 封存官 / 資材官
- 常見層位：漩淵渦心
- 獲得條件：能在高價封存流程中抓出一次錯估，並保住庫環信任。
- 專屬術式 / 戰技：
  - `首秤定價`
  - `封灰覆印`
- 流派方向：
  - `秤門流`：偏估值、初審、表面秩序。
  - `內庫流`：偏深封、抹錄、內環路徑。

### 契錄抹價師
- 對應通用職業：黑帳官 / 抹錄師 / 高階交易手
- 常見層位：漩淵渦心至跨層外務
- 獲得條件：能在不讓流程崩掉的前提下改掉一筆高價紀錄，並讓所有人只抓到你留下的那層假答案。
- 專屬術式 / 戰技：
  - `斷頁抹錄`
  - `價名互換`

## 對外特例

### 暗裂路引
- 對應通用職業：走私者 / 跨域掮客
- 常見層位：跨層與外流線
- 獲得條件：至少成功走完一條深淵到梵洛特的暗裂轉運線。
- 專屬術式 / 戰技：
  - `灰簿借路`
  - `殘名過關`
