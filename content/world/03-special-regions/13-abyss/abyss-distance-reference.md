# 亡極徹淵層間距離與降層參考表

## 讀法
- 亡極徹淵的距離不以單純里程計，而以 `降層成本 / 誓約檢查 / 生還率 / 是否需要引路` 共同計算。
- 因此同一條路線，對本族、外族、流亡者與被押送者來說，實際耗時完全不同。

## 常用參考尺度
- `半日內`：同層熟路、已授權、且沒有儀式或關卡阻斷。
- `一日到兩日`：跨同層多個節點，或必須經過一次檢印、一次補給。
- `三日到五日`：跨層下降、攜帶貨物、護送活體或經過戰區。
- `五日以上`：涉及返井、黑脈、秘密路或封庫節點。

## 代表路線
### 外環幽裂
- 夢霧窩巢群 -> 幻守巢駐：`半日內`
- 幻守巢駐 -> 影縫潛域：`半日到一日`
- 夢霧窩巢群 -> 殘燭回聲塔：`半日內`

### 裂廊上層
- 曜血議會堂 -> 血宴堂：`半日內`
- 血宴堂 -> 王脈封域：`一日內`，但審印成本極高
- 白脈折橋 -> 裂印剝階：`半日內`

### 裂廊中層
- 共幻集墟 -> 魅紗幻苑：`半日內`
- 魅紗幻苑 -> 噬淵母巢：`一日到兩日`
- 十三面紗井 -> 香殼沉席窟：`一日內`

### 裂廊下層
- 獰罡熔戰窟 -> 焰影裂市：`半日到一日`
- 焰影裂市 -> 幽鬩奪形巢：`一日到兩日`
- 熔號斷梯 -> 怒骨伏炬場：`一日內`

### 漩淵渦心
- 垢金守契台 -> 渦心契市：`半日內`
- 渦心契市 -> 幽淵寶庫：`一日內`
- 沉金契窖 -> 零價墜庫：`一日到兩日`

## 跨層路線
- 外環 -> 上層：`一日到兩日`
  - 主要成本在檢印與夢蝕排除。
- 上層 -> 中層：`一日到三日`
  - 最常因授權、護送品與席位驗證延長。
- 中層 -> 下層：`二日到四日`
  - 若牽涉孵材、戰俘或影帳運送，會更久。
- 下層 -> 渦心：`二日到五日`
  - 正常情況不會讓無契牌者直接通行。

## 返井與黑路
- 流亡返裂井 -> 回召黑脈梯：`三日到六日`
- 回召黑脈梯 -> 涅厄斯暗路節點：`不固定`
  - 取決於假路是否還活著，以及哪一方先動手改寫名冊。

## 主持提示
- 深淵的距離本質上是 `誰願意讓你活著通過`。
- 若想讓亡極徹淵顯得更壓迫，與其增加里程，不如增加停留、審印、押值、降格與被重估的風險。
