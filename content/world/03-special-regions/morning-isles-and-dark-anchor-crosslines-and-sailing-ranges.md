# 晨穗群島 × 黯鉚列島跨島灰線、幫派重疊與航海範圍

## 核心判斷
- 晨穗與黯鉚離得不算遠，所以 `完全不重疊` 不合理。
- 但它們也不是同一片海盜文化，只會在 `灰貨、人手、借旗、沉船與季節性航線` 上產生交集。
- 最準確的讀法是：`本體不同，邊帶重疊，旺季互咬，淡季互用。`

## 為什麼會重疊
- 晨穗擅長 `港埠轉運、借旗、假契、私掠與半合法護路`。
- 黯鉚擅長 `礁霧伏擊、黑鍛洗貨、沉船回收、改船骨與消號`。
- 所以最常見的跨島流程是：
  - 貨先在晨穗借合法外皮
  - 再去黯鉚被拆、重鍛、洗鉚印
  - 最後以另一個身份回流到別處

## 三種重疊海域
### 中間灰渡帶
- 位在晨穗西汐散島外緣與黯鉚外環霧帶之間。
- 適合中型快船、雙底貨船與不想留下完整港簿的人。
- 真正危險的不是風浪，而是這裡很少有哪一條線是單邊說了算。

### 沉貨回收帶
- 靠近兩區共有的外海亂流與舊商道偏折處。
- 晨穗的 `鷗骨分席眾` 和黯鉚的 `回骨鐵戶` 最容易在這裡撞線。
- 這裡常發生 `一艘船在晨穗先被分骨、在黯鉚又被重估一次` 的事。

### 旗號漂白帶
- 靠近晨穗正航道外側與黯鉚灰鍛外送線之間。
- 晨穗系幫派偏好在這裡改船籍、換泊票。
- 黯鉚系幫派偏好在這裡換鉚印、改貨名。

## 晨穗幫派的常態航海範圍
### 黑帆斷桅會
- 常態：`晨穗西汐散島外緣 -> 黑風折礁 -> 中間灰渡帶東半`
- 旺季外伸：可摸到黯鉚外環東南緣，但通常不願太深進礁霧。
- 不愛去：黑礦崖正北、鐵吻礁深窗、黯鉚主霧帶核心。
- 理由：它們是港外掠線高手，不是礁霧精算師。

### 白浪回旗隊
- 常態：`礁堡島 -> 潮汐堡外礁 -> 中間灰渡帶主航線`
- 旺季外伸：能跑到黯鉚的暗鉚外送接貨點，但偏保路，不偏硬掠。
- 不愛去：囁語淵、礁靄隘口深夜窗。
- 理由：它們主要賺護航、放行與回旗保證金，太深進黯鉚會失去談價位置。

### 潮針女群
- 常態：`葦羽潛泊 -> 潮隙潛道 -> 中間灰渡帶細舟線`
- 旺季外伸：最容易和黯鉚的霧盞潛帆接頭，甚至直接把小件活貨送到暗鉚港外夜線。
- 不愛去：鐵吻礁公開伏線、黑鍛火場。
- 理由：她們吃的是小件、禁貨與活人轉運，不喜歡大船與大火。

### 鷗骨分席眾
- 常態：`盜鷗沉埠 -> 礁骨漂場 -> 沉貨回收帶東半`
- 旺季外伸：能一路摸到鏽牙岬拍價桌。
- 不愛去：高霧伏窗深處。
- 理由：他們要的是已經死透或快死透的船，不愛賭活伏擊。

### 夜鐘換班客
- 常態：`朝羽鐘埠 -> 雙旗埠頭 -> 港簿近海`
- 旺季外伸：很少親自深入黯鉚，只會透過灰簿手和泊票販遠控。
- 不愛去：外環伏線、沉船作業帶。
- 理由：他們的戰場是 `時間、鐘序與記錄`，不是硬海戰。

## 黯鉚幫派的常態航海範圍
### 斷鏈黑帆
- 常態：`鐵吻礁 -> 礁靄隘口 -> 中間灰渡帶西半`
- 旺季外伸：會試探晨穗西汐散島最外緣，但通常只切路、不久留。
- 不愛去：晨穗主港群、雙旗埠頭正面水域。
- 理由：它們擅長的是砍退路，不愛在高監視與高鐘序的港前硬吃。

### 鐵吻群爪
- 常態：`鐵吻礁深窗 -> 外環礁霧帶 -> 沉貨回收帶西半`
- 旺季外伸：偶爾會追到晨穗黑風折礁外圍，但極少越進內潮路。
- 不愛去：晨穗內水路、蘆棚細舟帶。
- 理由：群爪靠礁霧與誤判吃飯，進到晨穗內潮細路反而失優。

### 爐丘灰盟
- 常態：不以大範圍航海為主，控制的是 `爐痕丘收貨線 -> 暗鉚港夜簿 -> 外送洗貨線`
- 旺季外伸：貨能送進中間灰渡帶，少數高價洗貨可借晨穗白皮再回流。
- 不愛去：所有需要長時間現身的公開交易海域。
- 理由：灰盟重的是貨，不是旗。

### 霧盞潛帆
- 常態：`暗鉚港外夜線 -> 囁語淵接貨點 -> 中間灰渡帶細霧窗`
- 旺季外伸：最容易和潮針女群、夜鐘換班客的下層掮客接上。
- 不愛去：高火鍛場、公開撈船台。
- 理由：它們吃的是借名、消號、灰路，不是拍賣和重鍛。

### 回骨鐵戶
- 常態：`鉚骨嶼 -> 鏽牙岬 -> 沉貨回收帶西半`
- 旺季外伸：會一路追到晨穗礁骨漂場和盜鷗沉埠外圍。
- 不愛去：純掠奪線、需要高速轉向的礁窗伏線。
- 理由：他們要的是完整回收權，不是只先咬一口。

### 寂鋒盟
- 常態：沒有穩定海域，它們沿著 `有必要消號的人與貨` 移動。
- 旺季外伸：能在兩區都有點，但通常只經過、不留下旗。
- 不愛去：需要長時間公開出面的任何地方。

## 最常見的跨區合作
- `潮針女群 x 霧盞潛帆`
  - 活貨、小件禁貨、信物與假名轉運。
- `鷗骨分席眾 x 回骨鐵戶`
  - 沉船二次分價、殘骨補錄與舵件重估。
- `白浪回旗隊 x 爐丘灰盟`
  - 給高價洗貨一層看起來比較乾淨的回旗保證。

## 最常見的跨區衝突
- `黑帆斷桅會 x 斷鏈黑帆`
  - 名號近、手法像，但一邊偏港外示威，一邊偏礁霧截斷。
- `鷗骨分席眾 x 回骨鐵戶`
  - 都想先寫沉船的第一筆價值。
- `夜鐘換班客 x 霧盞潛帆`
  - 一邊改的是港內時間與泊序，一邊改的是船名與灰路歸屬。

## 主持提示
- 晨穗幫派跨海，多半是 `把東西變得像合法的`。
- 黯鉚幫派跨海，多半是 `把東西變得像另一件東西`。
- 如果玩家在兩區都活動，很適合讓他們遇到同一批貨在晨穗和黯鉚各有一套故事。
