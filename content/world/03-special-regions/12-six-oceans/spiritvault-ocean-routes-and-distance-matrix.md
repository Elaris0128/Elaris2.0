# 靈穹洋航路、距離與轉線矩陣

## 主體節點
- 曦環浮境
- 啟誓嶼
- 天脈島弧
- 晨穗群島
- 霧羽列島
- 星盞群島

## 常用距離感
- `曦環浮境 -> 啟誓嶼`：高空直轉較快，海面轉線較慢，但整體屬中短程。
- `啟誓嶼 -> 天脈島弧`：北上浮流穩時是很常走的中程補給線。
- `啟誓嶼 -> 霧羽列島`：偏東南向，樣本與藥材隊最常走的乾淨轉線。
- `啟誓嶼 -> 晨穗群島`：西南借旗與公會外海委託常見線。
- `啟誓嶼 -> 星盞群島`：偏南長程遠征線，通常需要季節窗口與中途外緣補點。

## 具名小島怎麼嵌進去
- `霧環泊島`：介在曦環與啟誓嶼之間，是最重要的中繼轉泊點。
- `白汐樣洲`：嵌在啟誓嶼東南航線與霧羽外海之間。
- `鯨背歇礁`：卡在天脈島弧與中央主航帶之間。
- `裂帆墜洲`：卡在晨穗外海與中央主航帶交界。
- `靜藻潭洲`：卡在星盞北西外緣與乾淨採樣水帶之間。
- `空索殘橋`：卡在啟誓嶼與曦環低環之間的高風危橋帶。

## 轉線原則
- 海船偏好 `晨穗 / 天脈 / 霧羽` 作為補給節點。
- 空艇偏好 `曦環 / 啟誓嶼 / 霧環泊島`。
- 樣本與高價輕貨偏好 `霧羽 / 白汐樣洲 / 天脈`。
- 黑線與假旗貨偏好 `晨穗 / 裂帆墜洲 / 中央懸流下泊帶`。
