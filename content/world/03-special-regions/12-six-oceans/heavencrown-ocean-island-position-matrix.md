# 天冠洋島鏈、具名節點與位置矩陣

## 讀法
- 天冠洋不是一整塊平整冰原，而是 `心淵 + 浮冰帶 + 裂帶 + 冰洞 + 陸棚` 疊出來的極寒洋域。

## 具名節點
- `霜冠心淵`
  - 位置：中央偏北。
  - 定位：極光與極寒暗流的核心。
- `漂冰帶`
  - 位置：南緣長帶。
  - 定位：最大面積的浮冰海。
- `凝霧弧灣`
  - 位置：東北邊緣。
  - 定位：霧灣、短泊點與低溫補給帶。
- `冰簾裂帶`
  - 位置：心淵外圍西側。
  - 定位：冰崖、裂簾、下層暗流。
- `雪影漂脈`
  - 位置：漂冰帶內側。
  - 定位：會自行位移的冰河脈鏈。
- `寒喉冰洞`
  - 位置：凝霧弧灣深端。
  - 定位：垂直冰洞與冰下氣鳴網。
- `凍層嶙背`
  - 位置：心淵與漂冰帶之間。
  - 定位：起伏壓脊、塌裂冰丘與危險越地。
- `白冠陸棚`
  - 位置：西北極帶。
  - 定位：真正可長時探索的冰陸棚與寒岸台地。

## 位置關係
- 從南向北最常見的讀法是：
- `漂冰帶 -> 雪影漂脈 -> 凍層嶙背 -> 霜冠心淵`
- 往東北則會分出去：
- `凍層嶙背 -> 凝霧弧灣 -> 寒喉冰洞`
- 往西北則會轉成冰陸探索線：
- `冰簾裂帶 -> 白冠陸棚`
