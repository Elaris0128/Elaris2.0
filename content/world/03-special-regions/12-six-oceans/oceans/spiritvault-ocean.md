# 靈穹洋

## 整體定位
- 靈穹洋位在三大陸與多個群島系統的中介帶，是海空雙向流通最密集的洋域之一。
- 這片海同時連著 `曦環浮境`、`天脈島弧`、`晨穗群島`、`霧羽列島`、`星盞群島`，也是 `啟誓嶼` 與冒險者公會總部的所在海域。

## 關鍵特徵
- 具名島嶼與無名小洲一起構成航路節點。
- 海怪種類密集，從表層魚群到深層異常生物都有自己的生態位。
- 這裡的核心衝突常是 `航權 / 封樣 / 打撈 / 借泊 / 假燈 / 灰線`。
- 啟誓嶼雖是冒險者公會總部，但整片海其實同時被 `探索公會 / 藥師公會 / 樣本線窗口 / 灰線回收者` 拉扯。

## 主要節點
- 啟誓嶼
- 霧環泊島
- 白汐樣洲
- 鯨背歇礁
- 裂帆墜洲
- 靜藻潭洲
- 空索殘橋
- 回航弧棚
- 雙燈候洲
- 藥封轉匙洲
- 影頁沉步礁
