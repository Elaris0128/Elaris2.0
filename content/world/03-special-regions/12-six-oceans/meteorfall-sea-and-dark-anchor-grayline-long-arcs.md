# 殞星海與黯鉚列島灰線長線

## 兩地差異
- [黯鉚列島](/C:/Users/rabbi/Desktop/Elaris/content/world/03-special-regions/08-dark-anchor-isles/overview.md) 擅長 `洗貨、改船、拆骨、重估值`。
- 殞星海更擅長 `找到貨、先標貨、從假圖與潮窗裡搶貨`。

## 主要長線
- `假圖 -> 先拖 -> 洗標 -> 重賣`
  灰圖從殞星海放出，貨在黯鉚列島被洗成另一種身分。
- `深貨 -> 沉灣回流 -> 夜拍 -> 黑鍛改件`
  最值錢的不是原物，而是經過多手後還能看起來乾淨。
- `外來海盜 -> 殞星短窗 -> 黯鉚夜泊`
  這條線最吃熟路與人情，不是每個海盜都能跑。

## 常見參與者
- 黯鉚端：
  - `斷鏈黑帆`
  - `回骨鐵戶`
  - `霧盞潛帆`
- 殞星端：
  - `碎晶獵旗眾`
  - `逆沉灰槳隊`
  - `黯墜灰桌`
- 邊緣插手者：
  - `鷗骨分席眾`
  - `黑帆斷桅會`

## 長線衝突
- 殞星海的人覺得黯鉚只會在後端吃肉。
- 黯鉚的人則覺得殞星海只會拿命去賭運氣。
- 但兩邊都知道，沒對方，自己賺不到最大那一層。
