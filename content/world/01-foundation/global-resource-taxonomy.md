# 世界資源分類通用框架

## Purpose
這份文件定義 Elaris 的世界通用 `資源層` 分類，讓地區資源、探索掉落、加工鏈、商店收購與職業分工都有一致的底層標準。

資源是原料與產出來源，物品則是經過用途化與加工後的成品。兩者要分開，但必須能互相對接。

## 核心原則
- 資源優先按 `來源` 分類，再按 `用途` 加標籤。
- 同一份資源可以服務多種職業與多條加工鏈。
- 資源除了價值，還要標記 `穩定性 / 危險性 / 可保存性 / 合法性`。
- 後續地區資源、魔物掉落、採集素材與遺構回收物，都應能回掛到這份總表。

## 一級分類

### 1. 地表自然資源
來自平原、丘陵、草原、荒原與一般地表環境。

常見子類：
- 土壤與黏土
- 砂礫與鹽土
- 草纖、藤材、木料
- 基礎可食作物
- 染料植物與織材植物

### 2. 水系與潮汐資源
來自河道、湖泊、濕地、海岸、潮間帶與地下水域。

常見子類：
- 淡水與礦水
- 河泥與潮泥
- 水草與浮蘚
- 可食水棲素材
- 鹽晶、鹽花與潮線沉積物

### 3. 林地、魔植與魔菌資源
來自森林、霧林、菌床、古木帶與高濕生境。

常見子類：
- 果實與種串
- 藥草與藥蘚
- 樹脂、樹膠與樹皮
- 菌傘、菌絲與孢粉
- 帶刺、帶毒、爆裂或共鳴型食材

### 4. 礦石與岩材資源
來自山脈、礦坑、裂谷、地下層與火脈。

常見子類：
- 普通金屬礦
- 稀有金屬礦
- 岩材與石板
- 導晶與能量礦
- 熱脈與符火媒礦

### 5. 魔物衍生資源
來自獸型、蟲型、水棲型、類人型或王者級魔物的自然產出或掉落。

常見子類：
- 皮、骨、角、牙、殼
- 腺體、囊袋、毒液與分泌物
- 羽、鬃、鱗與膜翼
- 繭、卵、巢材與育成副產物
- 可食肉材與高風險內臟素材

### 6. 共鳴與誓約資源
來自靈息濃區、誓痕遺留、共鳴點、封印地與誓器體系。

常見子類：
- 誓痕碎片
- 共鳴晶與穩定片
- 導印媒材
- 誓印承載物
- 靈息高濃縮樣本

### 7. 遺構與古代回收資源
來自遺跡、地下城、古戰場、廢都、水下遺構與封存設施。

常見子類：
- 古代金屬件
- 機關零件
- 失效印片
- 古文載體
- 危險封樣容器

### 8. 生產與文明加工資源
已經過初步加工，但仍屬於原料層，而不是最終商品。

常見子類：
- 皮革、線材、布料
- 切削木材、工坊板件
- 礦錠、半熟料、鍛造胚
- 乾製藥材、粉末、鹽封材
- 保鮮包材與封樣器材

## 二級功能標籤
每份資源除了來源分類外，建議再標記：
- `食用`
- `藥用`
- `工藝用`
- `儀式用`
- `探索用`
- `建設用`
- `照護用`
- `戰略用`
- `禁制研究用`

## 資源風險標籤
後續建立區域資源時，建議至少補以下風險欄位：
- 穩定性：穩定 / 易衰 / 活性高 / 共鳴不穩
- 危險性：低 / 中 / 高 / 災害級
- 保存性：短期 / 中期 / 長期 / 必須即時處理
- 處理門檻：民間可處理 / 專業可處理 / 需許可 / 禁制
- 合法性：合法 / 管制 / 需登記 / 禁運

## 加工鏈對接

### 靈廚
優先處理：
- 可食魔植
- 高風險果料
- 旅糧原材
- 藥食與安撫膳素材

### 藥師 / 調和職
優先處理：
- 藥草、藥蘚、菌材
- 腺體與萃液
- 穩定化樣本

### 鍛造 / 工藝職
優先處理：
- 礦石、岩材、金屬件
- 半熟料與回收零件
- 導晶與結構件

### 誓紋 / 儀式職
優先處理：
- 誓痕類素材
- 導印媒材
- 封樣承載物

## 資源卡最低模板
- 名稱
- 一級分類
- 二級功能標籤
- 來源生境或地點
- 採收 / 回收方式
- 穩定性
- 危險性
- 保存性
- 合法性
- 常見處理職業
- 初步處理方式
- 常見成品方向

## 與物品分類的關係
- 資源負責回答：`這是從哪裡來的`
- 物品負責回答：`這可以被用來做什麼`
- 同一資源可以對應多種物品，例如：
  - 食材可變成熱食、旅糧、糖品、安撫膳
  - 礦料可變成零件、武器、導印支架、建設件

## 設計原則總結
- 先判斷來源，再判斷價值與用途。
- 資源必須能接回職業、商店、配方與探索。
- 高價值不一定等於稀有，很多時候真正稀有的是安全處理能力。
