# 魔物生態通用框架

## Purpose
這份文件定義 Elaris 的世界通用生態邏輯。從這一版開始，Elaris 不再把「一般動植物」和「魔物」視為兩套分離概念，而是採用：世界中的自然生命整體都可放在 `魔物生態` 這個大框架下討論。

這代表：
- 植物是魔植
- 菌類是魔菌
- 昆蟲是魔蟲
- 水生物是水棲魔物
- 可家養、可耕作、可食用、可騎乘的生命也依然可能被放在魔物生態中理解

在 Elaris，`魔物` 不等於 `敵對怪物`，而是世界自然生命的通用外部分類語。

## 核心原則

### 1. 魔物是世界的一般生物概念
- 人們日常吃的、種的、養的、運輸用的、醫療用的生命，同樣可以被放進魔物體系。
- 差別不在於是不是魔物，而在於它的危險性、親近性、馴化可能、用途與棲地。

### 2. 生態先於戰鬥
- 一種生命首先應該是某個環境中的存在，其次才是可不可以打。
- 任何物種都應優先考慮：它靠什麼活、如何繁衍、和誰互動、在環境裡扮演什麼角色。

### 3. 不是每一種都需要奇觀化
- 並不是每種物種都要有誇張特殊能力。
- 有些只是穩定存在於日常生活的家養種、作物魔植、濾水魔藻或搬運魔獸。
- 特殊性應依環境、資源壓力與地區文化來分布，不要全部都做成稀有 boss。
- 同樣地，不是每個環境都一定要有王者級魔物；王者應該是少量、稀有、會真正改變地區節奏的存在。

### 4. 國家境內也有生活用生態
- 國家的城市、村莊、農地、河道、山道與港口，都應存在日常生命網。
- 野外與境內的差異不是「有沒有魔物」，而是密度、危險度與控制程度。

### 5. 外部分類與文明自稱可以不同
- `魔物` 是世界通用的學術 / 檢索 / 冒險分類語，不一定等於某個文明會如何稱呼自己。
- 若某群體已具備穩定語言、法律、政治結構與清楚的自我認同，記錄時應優先保留其 `文明自稱`。
- 因此後續資料可以同時存在兩層：
  - 外部分類：便於跨地區整理與檢索
  - 內部稱呼：尊重該文明的自稱與世界觀語感
- 代表案例是 `瑟堤亞棲國`：外地人可能粗略把它視為魔物王國，但棲國內部只有被 `誓王賜名` 的魔物，才會被歸入 `棲裔` 這個文明身份；其餘仍直接被稱為魔物。失控、污染或裂變後的危險存在則被稱為 `失衡異種`。

## 魔物分類的五大軸

### 1. 形態分類
用來回答它長成什麼樣。

- `魔植類`：喬木、灌木、藤本、花草、作物、蘚苔、藻類、孢囊植物、結果型植種、木妖型植種
- `魔菌類`：菌傘、菌毯、寄生菌、共感菌幕
- `魔蟲類`：昆蟲、節肢群、螯蟲、寄生小型種
- `獸型魔物`：四足、草食、掠食、雜食、馴養型獸類
- `鳥翼魔物`：飛禽、滑翔種、夜翔種、訊禽種
- `水棲魔物`：魚類、泥棲種、河口種、兩棲種、深水種
- `人型 / 類人魔物`：具人形輪廓但不屬穩定文明自稱的生命
- `靈性 / 共鳴魔物`：高度依賴靈息、共鳴點、誓痕而存在的生命
- `混相 / 異常魔物`：受污染、變異、融合或異常繁衍形成的特殊類型

### 2. 生態角色分類
用來回答它在食物鏈與環境循環中做什麼。

- `生產者`：吸收光、土、潮、霧、礦物或靈息來形成生質
- `授粉 / 傳孢者`：協助植物、菌群或孢種擴散
- `草食 / 取植者`
- `掠食者`
- `腐食 / 分解者`
- `清道者 / 濾淨者`
- `寄生者`
- `共生者`
- `頂層壓制者`

### 3. 資源依存分類
用來回答它需要什麼環境條件才能活。

- `日照依存`
- `潮濕依存`
- `礦物依存`
- `熱脈依存`
- `共鳴依存`
- `誓痕依存`
- `屍腐依存`
- `宿主依存`
- `季節依存`

### 4. 繁衍分類
用來回答它怎麼延續族群。

- `卵生`
- `胎生`
- `孢生`
- `分株 / 萌發`
- `蛻殼增殖`
- `寄宿繁殖`
- `共鳴裂生`
- `季節性聚育`
- `爆裂散種`
- `殘骸轉生`

### 5. 文明關係分類
用來回答它和文明的距離。

- `家養型`：可穩定飼養、種植或共養
- `伴生型`：可與聚落共存，但不完全受控
- `邊境型`：常見於聚落邊緣、道路、河道與田地
- `野生型`：主要出現在自然棲地
- `禁忌型`：具有宗教、法律或文化禁忌
- `災害型`：一旦擴散會直接威脅聚落或交通
- `文明自稱型`：已有獨立文明自稱與政治身份，記錄時應尊重本地稱呼

## 危險程度與親近程度

### 危險程度
用來判斷主持時的威脅層級。

- `0 級`：幾乎無害，多為日常家養與耕作型生命
- `1 級`：會造成小型騷擾或局部傷害
- `2 級`：對單人或小型隊伍有實際風險
- `3 級`：對聚落外圍、商路、河道會形成威脅
- `4 級`：需要專業隊伍或官方介入
- `5 級`：可視為區域級災害或主線級目標

### 親近程度
用來判斷它和智慧族群的互動傾向。

- `0 拒近`：幾乎無法接觸，見人就逃或攻擊
- `1 警戒`：可觀察，但不易穩定接近
- `2 容忍`：特定條件下可共處
- `3 可接觸`：有經驗者可引導、驅趕或短期利用
- `4 可共養`：可半馴化、半人工管理
- `5 伴生 / 家養 / 穩定共棲`：已深度融入聚落與產業，或與文明形成穩定共棲秩序

## 王者級魔物與領域主
不是所有環境都一定會出現王者級魔物，但在高承載、長期穩定、或共鳴特別強烈的生境中，確實可能孕育出會主導整片環境的 `王者級魔物`。

### 定義
- 王者級魔物不是單純比較強的大型個體，而是會改變 `路線 / 掠食秩序 / 資源流向 / 季節活躍 / 地方禁忌` 的生態領主。
- 它可以是獸王、河王、山王、樹王、巢王，也可以是 `母株 / 母藤 / 群核` 這類非動物形式。
- 王者級是生態位階，不等於文明君主，也不等於某個國家的統治者稱號。

### 常見形成方式
- `長年壓制型`：在單一生境中長期佔據頂層壓制位
- `共鳴凝聚型`：靠誓痕、靈息或共鳴點成長為領域核心
- `母株繁域型`：以植株、菌巢、母巢形式統御大片生長區
- `災後定域型`：在古戰場、裂谷、污染地帶成為新的環境支配者

### 與危險程度的關係
- 王者級魔物通常至少落在危險 `4` 以上，但 `危險 5` 不一定就是王者級。
- 判斷關鍵不是它多能打，而是 `它能不能改變周圍整片環境的行為`。

### 建議額外記錄欄位
- 王者稱呼或地方俗名
- 支配環境
- 活動規律
- 接近徵兆
- 會改變哪些物種或路線
- 若被驚動或消失，環境會發生什麼事

## 環境承載與多樣性原則

### 高承載環境
- 例如平原河谷、穩定濕地、沿海港帶、霧林邊層。
- 特徵是物種數多、功能分化明顯，也最容易出現家養與伴生種。

### 中承載環境
- 例如丘陵、普通林帶、草原遷徙線。
- 特徵是多樣性中等，季節變動會明顯影響族群結構。

### 低承載環境
- 例如高寒山脈、熔坑區、深地下、重污染地帶。
- 特徵是物種較少，但個體往往高度特化，生存方式也更極端。

### 異常承載環境
- 例如共鳴點、古戰場、誓痕區、封印裂口。
- 特徵是物種可能不穩定，會出現變異、混相、季節性爆發或短期繁盛。

## 生態鏈設計原則
- 每個環境至少要有 `生產者 -> 中層攝食者 -> 上層壓制者 -> 分解者` 的基本結構。
- 若是特殊環境，應再加入共生者、寄生者或濾淨者。
- 不是每條生態鏈都要很複雜，但不能只列掠食者而沒有前後文。

## 生活用生態
這類生命是國家與聚落日常的重要部分。

常見用途：
- 耕作
- 馱運
- 巡邏
- 濾水
- 授粉
- 採蜜
- 織材
- 藥材
- 發光
- 污物分解

因此後續在國家資料中，應常態性出現：
- 農耕魔植或作物種
- 家養魔獸或伴生載獸
- 河道工作生命
- 港務與山道用生命
- 醫療與工藝用途的伴生種
## 木妖型與結果型魔植補註
- `木妖型魔植` 是植物生命中最接近守望者、領域者與慢速移動個體的一支。它們不一定都有高智慧，但往往能辨認熟悉氣味、足音、誓痕與水脈變化。
- `結果型魔植` 可以是喬木、灌木、藤本甚至菌植混生成熟體。果實、種子、囊莢與汁液不必然安全，可能帶刺、麻感、爆裂散種，或只在特定節奏下可食。
- 當一種魔植同時具備食用價值與防禦性時，最適合成為聚落採集、料理、公會委託與地方禁忌的共同素材。
- 命名上也建議優先使用居民與旅人日常會講的通俗名，再視需要補上學術名、古稱或地方別稱。

## 每個物種至少建議記錄的欄位
- 名稱
- 形態分類
- 生態角色
- 棲地
- 資源依存
- 繁衍方式
- 危險程度
- 親近程度
- 常見用途或威脅
- 是否與特定國家、族群、宗教或地點有連結
- 若有文明自稱，應補上內部稱呼

## 設計原則總結
- Elaris 的生物世界不分「普通生物」和「魔物」兩套系統。
- 但世界通用分類與文明自稱可以並存，不必強迫所有文明都用魔物自稱。
- 魔物生態必須能同時支撐日常生活、世界觀深度與冒險遭遇。
- 越極端的環境，越適合做少量但高度特化的物種。
- 越穩定的環境，越適合做多樣、可利用、可共養的生態網。
