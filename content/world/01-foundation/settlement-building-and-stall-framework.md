# 聚落建築、人物與攤位框架

## Purpose
這份文件用來定義：一個聚落不論大小，至少應該有哪些 `建築功能`、`常駐人物功能位` 和 `市場 / 特殊攤位`。

目標不是讓每個村都長得一樣，而是讓每個地方都能：
- 有基本生活感
- 有可互動的人
- 有可消費、可打聽、可接支線的空間

## 核心原則
- 聚落越大，建築與人物越細分；聚落越小，功能越常集中在同一人或同一棟建物身上。
- `特殊攤位` 不一定天天都在，也可以是節期、集市日、風季、船期或祭期才出現。
- 建築和攤位都應該服務 `生活 / 交易 / 情報 / 任務 / 地方風味` 其中至少兩項。

## 王都 / 大城最低功能位

### 建築
- 公權建築：官署、議事廳、誓務所、導印臺之類
- 市場建築：主市、行商窗、稅契或登錄窗口
- 住宿與飲食：旅舍、食堂、酒屋、熱食點
- 工藝建築：工坊、修理處、訂製點
- 醫療或安養：藥鋪、調和院、照護點
- 宗教 / 儀式 / 見證空間
- 運輸與貨流：車隊點、港棧、獸欄、倉列

### 人物功能位
- 官署 / 記錄者
- 市場主或窗口主理
- 旅舍 / 酒屋經營者
- 主工藝師或修理匠
- 醫療 / 安撫者
- 情報流通者
- 一位能帶出地方麻煩的灰線人物

### 攤位
- 至少 3 種特殊攤位常駐或輪替
- 例如：占問、古物、書畫、樣本鑑定、旅路看盤、快速修補、夢解、契文代寫

## 城鎮最低功能位

### 建築
- 地方議屋 / 登記處 / 見證亭 三者至少其一
- 雜貨補給或小工坊
- 熱食與短宿點
- 倉棚、橋口、車棚或渡口其中之一
- 小型醫療 / 藥草 / 安撫點

### 人物功能位
- 地方管事或可代表地方發話的人
- 商店或工坊主
- 交通 / 運送 / 引路相關人物
- 藥師 / 安撫者 / 照護者
- 一位消息來源或愛管閒事的人

### 攤位
- 至少 1 至 2 種特殊攤位
- 可是固定攤，也可以是逢集市才出現的輪替攤

## 村莊最低功能位

### 建築
- 食物與休息點
- 農事、採集、漁事或運貨棚
- 地方祈願、見證石或小型祭點

### 人物功能位
- 老手 / 長者 / 村代表
- 採集者 / 漁人 / 耕作者 / 小舟手其中之一
- 一位會照顧病傷或懂基本藥草的人
- 一位記得外來者和最近怪事的人

### 攤位
- 不一定是正式市集攤，也可以是：
- 橋邊代寫桌
- 祈石旁說夢席
- 漁貨換物角
- 路過說書布

## 前哨 / 營地 / 關口最低功能位

### 建築
- 守望點
- 補給棚
- 簡易修具處
- 休息火圈或集宿屋

### 人物功能位
- 守望者 / 巡查
- 補給管理者
- 引路者 / 偵察者
- 照料役獸或運載者的人

### 攤位
- 最常見是功能型攤位
- 例如：補靴、看路、快修、試紙測水、換繩換扣

## 流動市 / 換市 / 節期市最低功能位

### 建築與場地
- 見證圈或議價區
- 系繩欄、獸繫區、車貨圈
- 熱食與休息棚
- 消息板或口信柱

### 攤位
- 小吃攤以外，特別適合出現：
- 占路攤
- 古物攤
- 舊地圖 / 書畫攤
- 样本鑑價攤
- 契文代寫攤
- 修補與改裝攤
- 故事、歌謠與口述史攤

## 特殊攤位最低欄位
- 名稱
- 類型
- 出沒地點
- 主理人
- 主要販售 / 服務
- 最常活躍的時間
- 隱藏價值

## 設計提醒
- 攤位不只賣東西，也可以賣 `判斷`、`故事`、`證明`、`美感`、`運氣`。
- 真正讓地方活起來的，往往不是主官署，而是那些居民會停下來看一眼的角落。
