# 價值、貨幣與交易通用框架

## Purpose
這份文件定義 Elaris 的世界通用 `價值 / 貨幣 / 交易` 骨架，讓各地的商店、走私、票據、配給、護送、查驗與黑市交易都能回掛到同一套底層邏輯。

它不負責寫死每個國家的幣值，而是先定義：
- 世界裡有哪些常見價值形式
- 交易成立需要哪些條件
- 商路、查驗與非法流通怎麼判定
- 玩家在不同地區拿著同一樣東西，為什麼可能遇到完全不同的待遇

## 核心原則
- 在 Elaris，`價值` 不等於 `硬幣數量`。
- 一件東西能不能交易，通常同時取決於：
  - 幣值
  - 當地承認
  - 合法性
  - 流通狀態
  - 來源是否可被見證
- 票據、誓約、實物等值物和許可，本身都可能比錢更重要。
- 同一件物品在不同地區可能不是價格不同，而是 `根本不能合法買賣`。

## 世界常見價值形式

### 1. 硬幣貨幣
最直觀的流通價值，適合日常購買、中小型交易和跨地結算。

常見特徵：
- 便於攜帶
- 便於拆分
- 適合公開市場
- 最容易被偽造、剪邊或混幣

### 2. 票據與契約值
以文件、印記、石契、稅契、保函、配額票為核心的價值形式。

常見特徵：
- 適合大宗物流與高風險交易
- 可代替實際金額流動
- 在制度型國家往往比現金更有力量

### 3. 實物等值物
以牲畜、糧食、皮革、鹽、藥材、礦料、工藝料等作為交換核心。

常見特徵：
- 多見於邊境、草原、部族與流動市集
- 對急需程度和保存狀況極度敏感
- 適合低制度信任環境

### 4. 服務與權限值
以護送、通行、導引、見證、照護、導印、修繕等服務作為價值交換。

常見特徵：
- 常與身份、關係、人脈綁定
- 很多時候不能直接標價
- 在特殊地區比現金更有實際價值

### 5. 生態與共鳴估值物
以活體樣本、導印素材、誓痕樣本、共鳴晶、受許可藥材等作為高敏感價值物。

常見特徵：
- 不只看稀有度，也看穩定度與合法採集證明
- 常受到誓網、公會、教團、軍方或研究體系管制
- 一旦脫離正規渠道，灰市價可能遠高於官方價

## 價值成立的五個條件
一份物品、素材或服務若要形成有效價值，通常要同時看：

### 1. 可辨識
- 交易雙方要知道這是什麼
- 或至少知道它大概值在哪個價格帶

### 2. 可承認
- 當地市場、公會、官署或熟人網願意承認它的價值

### 3. 可流通
- 能在當地轉手、使用或進一步加工

### 4. 可見證
- 在高風險交易中，來源、所有權與條件最好能被見證

### 5. 可處理
- 對特殊物品來說，再值錢也要有人能處理、保存、修繕或啟動

## 世界通用價格帶
前期主持與一般資料可先用價格帶，不必先精算幣值。

### 1. 低價
- 日常食物
- 基礎旅糧
- 常見繩索與燈具
- 普通書寫用品
- 常規民生耗材

### 2. 中價
- 標準護具
- 普通工具組
- 中距離通行費
- 常見專業服務
- 一般藥包與補給套件

### 3. 高價
- 稀有材料
- 正式護送
- 高階修繕
- 專門裝備
- 大宗物資或特許服務

### 4. 特許價
- 管制裝備
- 高敏感樣本
- 稀有誓器
- 關鍵情報
- 需要見證與許可的特貨

## 常見交易場景

### 1. 公開市場
- 最適合日常貨物、普通補給、常見成品
- 價格最穩，但限制也最明顯

### 2. 公會與專業窗口
- 最適合專業材料、正式委託、收購與驗貨
- 看重規格、資格與登記

### 3. 邊境與流動市集
- 實物等值物與熟人信用影響很大
- 價格浮動高，但彈性也大

### 4. 票據與契約交易
- 多用於大宗貨運、軍需、長距離商旅、工期型交易
- 常與保函、稅契、石契、導印許可一起出現

### 5. 黑市與灰線交易
- 不是所有灰線交易都完全非法
- 常見於：
  - 來源有問題但貨物本身可用
  - 物品合法但持有人無資格交易
  - 物品本身屬管制或禁制

## 合法流通等級
這裡和裝備的合法性互相對接，但更偏交易面。

### 1. 正常流通
- 可公開持有
- 可公開販售
- 可正常運輸

### 2. 限制流通
- 可在特定區域、對特定身份或用途流通
- 常需登記、報備或附帶文件

### 3. 特許流通
- 必須由機構、公會、軍方、教團或地方權力核可
- 常見於高敏材料、軍需件、特殊導印物與高價樣本

### 4. 禁止流通
- 不可公開持有、運輸、販售或研究
- 一旦出現就可能直接引發查扣、封存或追查

## 交易限制常見來源
- 國家法規
- 公會章程
- 地方習俗
- 宗教禁忌
- 生態保護
- 戰時配給
- 邊境封鎖

## 稅制與查驗
後續各地資料若提到港務、邊關、商路與走私，可優先用這套分類：

### 1. 通行查驗
- 針對人員、坐騎、車隊與貨物的基礎檢核

### 2. 稅契查驗
- 針對貨值、來源、目的地與應繳費的查驗

### 3. 安全查驗
- 針對危險樣本、封樣件、裝備、活體魔物與共鳴材料的檢核

### 4. 特許查驗
- 針對需要專門許可的高敏感貨品

## 黑市與灰線的區分

### 灰線交易
- 不一定違法
- 但通常不想被正式記錄
- 例如：
  - 避稅
  - 熟人轉手
  - 非正式估價
  - 無票據但雙方默認的短線交易

### 黑市交易
- 明確違反地方法規、公會章程或禁制規則
- 常見於：
  - 禁裝
  - 禁術物
  - 非法樣本
  - 偽造通行物
  - 戰時違禁品

## 配給與短缺
當地區進入戰時、災害期、風季、寒季或封鎖期時，可啟用：
- `正常供應`
- `限量供應`
- `配給供應`
- `軍需優先`
- `僅熟客 / 僅持證者`

## 走私判定
走私不只是不繳稅，也可能是：
- 帶著合法貨走非法路線
- 帶著非法貨走合法路線
- 貨本身合法，但來源或採集方式非法
- 持有人無權持有或運輸

## 交易卡最低模板
- 名稱
- 交易類型
- 主要價值形式
- 合法流通等級
- 常見交易場景
- 是否需要見證 / 票據 / 許可
- 典型風險
- 典型爭議

## 遊戲內呈現原則
- 玩家不應被迫自己用價格帶反推成交價。
- 世界層可保留 `低價 / 中價 / 高價 / 特許價`。
- 店面互動層則應直接顯示 `當次報價`、`可否購買`、`是否可折抵`。
- 若價格浮動，應一併說明 1 到 3 個主因，例如 `缺貨`、`寒季`、`熟客優惠`。

## 設計原則總結
- 價值要看承認與流通，不只看數字。
- 交易框架應讓商店、查驗、黑市、護送與走私使用同一套語言。
- 真正讓世界有味道的，通常不是幣值本身，而是誰有權說「這東西算值錢」。
