# 第一批守地神與地方神聖存在譜系

## 說明
守地神本質上比高位神與昇格神更綁地方，所以這一冊先做 `常見譜系卡`。等某個地點正式命名後，再把它們展開成真正的個體卡。

## 橋門守靈
- 主要附著地點：橋梁、城門、關口、邊橋
- 常見權柄：`渡行 / 守關 / 見證進出`
- 常見族群：瓦雷溫人、法雷恩人、柯沙里人、巖穹族
- 常見表現：不被允許的通行會遭遇反覆錯路、腳步變重或關門自鎖

## 潮灣守靈
- 主要附著地點：港灣、潮窟、碼頭、沉船帶
- 常見權柄：`回潮 / 避礁 / 航季互保`
- 常見族群：海人、柯沙里人、沿海城市
- 常見表現：風向、水色、魚群和船首聲會同時變化，提醒當地人海況失常

## 母樹看守
- 主要附著地點：古樹、母樹支脈、花橋、林境深處
- 常見權柄：`養育 / 棲地庇護 / 古誓守持`
- 常見族群：精靈、森馳裔、棲裔、林棲息育園
- 常見表現：傷者在其地較易穩定，但破壞者更容易被棲地整片排斥

## 封坑洞靈
- 主要附著地點：礦坑、封坑、支柱群、深層洞廳
- 常見權柄：`封層 / 避災 / 回音辨路`
- 常見族群：巴布窟族、矮人、巖穹族
- 常見表現：未被允許的人會在坑道裡反覆繞回原地，或聽見錯誤回音

## 碑路回靈
- 主要附著地點：崖道、祖碑林、古路、墓路
- 常見權柄：`記名 / 尋路 / 守墓`
- 常見族群：崖影族、極光徙裔、瓦雷溫邊地
- 常見表現：被記住的人能找到回路，被抹去的人則更容易在路上失向

## 營火宿靈
- 主要附著地點：營地石圈、旅路宿站、冬夜火坑、群行營盤
- 常見權柄：`安置 / 守夜 / 群體互保`
- 常見族群：薩魯恩人、諾魯因人、巡禮者、行商隊
- 常見表現：在其庇蔭下，陌生人比較願意停火與共宿，但背叛者會被夢兆警示

## 雪燈守靈
- 主要附著地點：雪棚、寒地墓園、冰湖邊、守夜燈塔
- 常見權柄：`守夜 / 防寒 / 送別`
- 常見族群：諾魯因人、極光徙裔、法雷恩北地聚落
- 常見表現：在極寒中維持火種、使守夜者不易失溫，也常在送別時帶來異常安靜的風雪
