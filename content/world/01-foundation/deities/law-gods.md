# 第一批律神個體卡

## 說明
這一批不是所有律神的完整總表，而是先定下目前最適合跨宗教、跨種族共用的第一批具名律神。

## 立名者
- 分支權柄：`立名 / 認名 / 入籍於秩序`
- 主要來源：`鑒誓者`
- 常見信眾：法雷恩人、瓦雷溫軍誓家系、法奧授證者
- 常見象徵：刻名石、白蠟印、第一筆名錄
- 常見表現：無名者得名、失位者被正名、偽名在儀式中自行脫落
- 常見衝突：誰有資格替別人命名、誰有資格把名字從公權秩序中刪去

## 守關者
- 分支權柄：`邊界 / 守門 / 關隘見證`
- 主要來源：`鑒誓者 + 夜引者`
- 常見信眾：邊關守軍、法雷恩雪關、瓦雷溫城塞、巖穹族山門守衛
- 常見象徵：關門印、界石、守門槍
- 常見表現：界線清晰、入侵者在門前感到壓迫、守關誓語在關牆久留
- 常見衝突：保護與排斥的界線、避難者是否能被放行

## 校印者
- 分支權柄：`查驗 / 正本 / 認證`
- 主要來源：`衡律者`
- 常見信眾：法奧編院、柯沙里查驗官、商會封樣人
- 常見象徵：對印、校環、真偽雙頁
- 常見表現：假文、偽印、偽契更容易暴露；錯序文書在其場域中難以成立
- 常見衝突：制度正本與地方口傳哪個更可信

## 養枝者
- 分支權柄：`養育 / 延續 / 棲地修補`
- 主要來源：`萬物母`
- 常見信眾：林棲息育園、棲裔照護者、精靈與森馳裔的護養祭司
- 常見象徵：新芽枝條、織根環、哺育盆
- 常見表現：幼群穩定、衰敗棲地緩慢復原、傷者更能接納療養
- 常見衝突：共養與資源短缺之間的矛盾、保育是否能凌駕飢荒

## 送渡者
- 分支權柄：`送行 / 引魂 / 正確離去`
- 主要來源：`歸輪者`
- 常見信眾：祖燼安魂盟、諾魯因守夜者、海人海葬祭司
- 常見象徵：小渡燈、回門繩、渡影舟
- 常見表現：亡者殘響趨於平靜、送別儀式更容易完成、迷失亡魂願意離場
- 常見衝突：該留下追真相，還是該放行去安息

## 回音者
- 分支權柄：`回聲 / 記憶殘響 / 失名尋回`
- 主要來源：`憶夢者`
- 常見信眾：崖影族、巴布窟族、天幕巡禮者、安魂記錄者
- 常見象徵：回音孔、空鐘、記聲石
- 常見表現：空間殘響重複、舊物重新回出舊名、夢境中反覆出現同一句話
- 常見衝突：究竟是在找回真相，還是在被過去拖住

## 引航者
- 分支權柄：`航向 / 引路 / 返程`
- 主要來源：`夜引者`
- 常見信眾：海人、薩魯恩人、柯沙里人、翼人與巡禮會
- 常見象徵：渡口燈、路旗、星潮盤
- 常見表現：迷途者重新找到主路、船隊更容易避開死水和暗礁、返程夢兆更清晰
- 常見衝突：應不應該幫所有人找到路，包括逃犯、侵略者與背誓者

## 鍛名者
- 分支權柄：`成形 / 鍛造 / 修補後重新成立`
- 主要來源：`鑄形者 + 鑒誓者`
- 常見信眾：矮人、巖穹族、地精工坊與烈地鍛群
- 常見象徵：小鎚印、焊紋、成器刻記
- 常見表現：危材更容易被馴服、器物重鑄後保住舊名、修補能被共同體承認
- 常見衝突：修補後是否還算原物、成器是否應被視作有名有責的存在

## 裂誓者
- 分支權柄：`毀誓 / 崩解 / 失序腐蝕`
- 主要來源：`毀誓與崩解`
- 常見信眾：無穩定正統教團，最常在魔人、破誓戰場、崩毀誓壇與詛咒工坊中被感應
- 常見象徵：裂開的誓環、倒燃的火、缺字石
- 常見表現：誓名脫落、器物自裂、記名牆空白、場域快速失序
- 常見衝突：它到底是神、災厄，還是被所有人共同餵大的崩壞回聲
