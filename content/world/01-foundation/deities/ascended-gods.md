# 第一批昇格神個體卡

## 說明
這一批是目前最適合作為跨地共通參照的第一批昇格神。祂們都保留明顯的 `曾經活過` 的敘事痕跡。

## 白關誓王
- 昇格來源：以身守關、死後仍維持關誓不墜的王者
- 常見接續權柄：`誓約與見證 / 海潮與遠途`
- 主要相關族群：法雷恩人、瓦雷溫邊關文化
- 常見象徵：白關、長炬、封關誓印
- 神話作用：正統守關、流亡者保名、邊疆不滅

## 灰爐祖匠
- 昇格來源：第一位讓災後殘爐重新點燃、並重建整個工坊族譜的祖匠
- 常見接續權柄：`火鍛與成形 / 記憶與夢兆`
- 主要相關族群：矮人、巖穹族、地精工藝圈
- 常見象徵：灰火、老鎚、裂爐重燃
- 神話作用：災後重建、祖匠正名、鍛火不絕

## 逐風先母
- 昇格來源：帶族群跨越死野與亂路、並立下不得棄同行者之誓的領路母首
- 常見接續權柄：`海潮與遠途 / 生命與棲育`
- 主要相關族群：森馳裔、薩魯恩人、極光徙裔
- 常見象徵：路旗、柔弓、遷路火
- 神話作用：群行護路、遷徙延續、不棄後隊

## 回潮舵主
- 昇格來源：在迷航與風暴中帶船隊返家的最後掌舵者
- 常見接續權柄：`海潮與遠途 / 死亡與輪轉`
- 主要相關族群：海人、柯沙里人、沿海港市信仰
- 常見象徵：破舵輪、返潮紋、潮燈
- 神話作用：返航、生還、海上送還與歸家

## 戰骨祖王
- 昇格來源：在大戰後收回全族遺骨、並以骨歌重新立族的戰祖
- 常見接續權柄：`死亡與輪轉 / 誓約與見證 / 火鍛與成形`
- 主要相關族群：獸人
- 常見象徵：骨冠、戰鼓、歸骨袋
- 神話作用：戰後重整、祖骨不棄、成年試煉的極端範本

## 失名長枝
- 昇格來源：為止住棲地崩壞而主動放棄自己名字，讓整片誓網得以續存的棲裔守領
- 常見接續權柄：`生命與棲育 / 記憶與夢兆 / 誓約與見證`
- 主要相關族群：棲裔
- 常見象徵：斷名枝、靜孢環、留白紋
- 神話作用：為群體失名、止災承載、受名秩序的極端犧牲

## 雪夜守母
- 昇格來源：在最長夜裡守住最後一盞公共燈火，使孩童與傷者得以過冬的守夜長者
- 常見接續權柄：`死亡與輪轉 / 記憶與夢兆 / 生命與棲育`
- 主要相關族群：諾魯因人、極光徙裔、法雷恩北境聚落
- 常見象徵：雪燈、毛披、熱石
- 神話作用：守夜安置、寒地庇護、亡者與生者都不被拋下
