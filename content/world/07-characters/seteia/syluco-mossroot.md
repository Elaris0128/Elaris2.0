# 賽盧柯・翠蘚脈

## Metadata
- id: seteia-syluco-mossroot
- region: 賽勒亞大陸 / 東境 / 瑟堤亞棲國
- faction: 誓王中樞
- role: 誓王
- status: active

## Snapshot
- 年齡：73
- 種族：棲裔
- 種族-分支：棲裔 / 受名主幹支
- 職業：棲國統治者 / 誓網中樞承載者
- 等級 / 危險度：極高權柄，中高戰鬥
- 關鍵能力：賜名權、誓網共鳴、全國誓令發布、受名秩序維持

## Personality
- 平靜、柔和，但不是可被輕慢的溫和
- 對棲國存續的判斷極長遠，不輕易被短期利益帶偏
- 對外來者不先排斥，但會先看其是否願意尊重林境秩序

## Speech Style
- 聲音像霧中傳來的鐘，不急卻能穿很遠
- 常用「共棲」「承認」「回來」這類詞，而不是命令語
- 常說：「被看見，不代表被佔有；被命名，才代表願意互相承擔。」

## Background
賽盧柯・翠蘚脈是霧林之亂後被推為重誓誓王的棲國統治者。他不是以傳統王權自居，而是以能維持受名、誓網與棲地共生秩序的核心存在被承認。

## Secrets
- 他感知到誓網深處有一段舊裂痕正在重新靠近表層
- 他保留了一項尚未公開的受名限制，避免棲國過快擴張失衡

## Game Hooks
- 作為瑟堤亞最高政治線、受名線與誓網主線的頂層入口
- 可導向林境危機、外來交涉與賜名權衝突
- 能把地方性的棲地問題直接提升為棲國級事件
