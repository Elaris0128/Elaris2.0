# 岑螢

## Metadata
- id: seteia-cenying-bough-warden
- region: 賽勒亞大陸 / 東境 / 瑟堤亞棲國
- faction: 誓網巡枝衛
- role: 巡枝衛
- status: active

## Snapshot
- 年齡：36
- 種族：棲裔
- 種族-分支：棲裔 / 巡枝衛支
- 職業：邊巡衛長
- 等級 / 危險度：中高
- 關鍵能力：邊巡攔截、低衝突制止、導路封鎖、誓網邊緣偵察

## Personality
- 克制、安靜，像一條你直到走近才發現存在的邊線
- 對暴力不是害怕，而是極有節制
- 很重視誰有沒有先給林境最後一次被尊重的機會

## Speech Style
- 像濕枝刮過護欄，不響卻清楚
- 常說「再往前一步，不一定是敵意，但一定是選擇」
- 很少重複第二次

## Background
岑螢負責霧林邊巡與低衝突攔截，是瑟堤亞軍事層最能體現柔性威懾的一種存在。

## Secrets
- 她知道某些看似天然的邊界躁動其實有人在試探
- 她放過一名本該被正式拘留的誤闖者，因為那人帶回了更大的線索

## Game Hooks
- 承接瑟堤亞軍事警戒、邊巡與導路封鎖線
- 可作為潛行、追蹤與邊界規則導師
- 能把小型越界變成棲網安全事件
