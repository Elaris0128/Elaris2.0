# 深眠長枝

## Metadata
- id: seteia-shenmian-elderbough
- region: 賽勒亞大陸 / 東境 / 瑟堤亞棲國
- faction: 誓環長老議庭
- role: 議庭首座
- status: active

## Snapshot
- 年齡：112
- 種族：棲裔
- 種族-分支：棲裔 / 受名主幹支
- 職業：長老 / 誓爭審理者
- 等級 / 危險度：中戰鬥，高制度威望
- 關鍵能力：重大誓爭審議、長老見證、舊例解讀、退役戰士統攝

## Personality
- 極穩，像一棵長時間沒有移動過的老樹
- 對年輕棲裔很嚴，但不是沒有慈心
- 相信柔性秩序比恐嚇更能讓棲國長久

## Speech Style
- 每句話都留有一段呼吸般的間隔
- 不急著駁斥別人，常先問「你打算讓誰承受後果」
- 常說：「誓網會記得你說過什麼，長老只負責提醒你那是真的。」

## Background
深眠長枝是誓環長老議庭首座，代表棲國內部那種更老、更慢、也更不容易被情勢推著跑的秩序。他既是重大誓爭的裁定者，也是受名文明得以延續的守望者之一。

## Secrets
- 他保有數份只在長老層流通的亂前記錄
- 他已經注意到某些年輕棲裔對受名秩序的不耐正逐漸擴大

## Game Hooks
- 作為瑟堤亞司法、長老政治與舊例爭議的高層入口
- 可牽出誓爭、長老內部分歧與亂前秘密
- 能把地方受名衝突連回整個棲國歷史
