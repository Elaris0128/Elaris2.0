# 蘚緹

## Metadata
- id: seteia-xianti
- region: 賽勒亞大陸 / 東境 / 瑟堤亞棲國
- faction: 霧蘚長廊導印臺
- role: 導印臺管理者
- status: active

## Snapshot
- 年齡：38
- 種族：靈蘚種
- 種族-分支：棲裔 / 靈蘚種蘚脈支
- 職業：通行管理者
- 等級 / 危險度：低戰鬥，中制度
- 關鍵能力：通行審核、來客名冊管理、貨流分級

## Personality
- 態度溫和，但程序感極強
- 對偷步驟和模糊身分的人毫不退讓
- 很懂外來旅隊最容易在哪裡犯錯
- 欣賞願意配合而不是硬闖的人

## Speech Style
- 句句都有清楚順序
- 不會提高音量，但很難被繞過
- 常說：「能進去，不代表能帶回來。」

## Background
蘚緹守著霧蘚長廊的導印臺，是很多外地人對瑟堤亞的第一道正式印象。她既是接待者，也是篩選器，知道哪些貨物只是無知，哪些貨物本身就不該被帶入林境。

## Secrets
- 她知道有人試圖讓一支旅隊在記錄裡空白一日
- 她一直在悄悄比對那支旅隊的實際貨單與導印紀錄

## Game Hooks
- 作為外來旅隊、貨物與邊界通行任務入口
- 提供名冊異常、走私與身份空白案線索
- 幫玩家合法切入霧蘚長廊內部秩序
