# 瑟堤亞角色索引

這裡收納瑟堤亞棲國目前已可投入遊戲的主要 NPC 角色卡。

## 最高領導層
- 賽盧柯・翠蘚脈：誓王
- 深眠長枝：誓環長老議庭首座
- 軟孢綰枝：首席語孢導引師
- 鏡寂絮：暗響監察部總監

## 目前角色
- 葳蘚：語孢導引師
- 科洛眠枝：誓網衛隊隊首
- 蘚緹：導印臺管理者
- 柊霧獸醫：共感安養者
- 苔翹：藥師公會霧林觀察站採息照護調停者
- 枝銘：受名記錄官
- 浮蘚：拾名漂棲者

## 使用原則
- 這批角色對應誓語導引、誓網警戒、外來通行與魔物照護玩法
- 最高領導層現已獨立成正式角色卡，可直接承接受名、誓網與棲國級危機
- 角色來源現在同時覆蓋棲國正式秩序、受許可外部組織與邊界型特殊流浪者
- 支援角色名錄另見 `supporting-cast.md`，已補宗教、政治、教育、軍務與諜報線
- 這些角色在棲國本地都屬於 `棲裔`，也就是被誓王賜名後納入文明秩序的存在
- 後續若要深挖霧林主線，可再補誓環長老與棲林游擊者
- 角色以現有地方包出現者為主，方便後續直接串劇情
