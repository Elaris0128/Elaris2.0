# 浮蘚

## Metadata
- id: seteia-fuxian
- region: 賽勒亞大陸 / 東境 / 瑟堤亞棲國
- faction: 無固定所屬
- role: 拾名漂棲者
- status: active

## Snapshot
- 年齡：不詳，外觀約 24 歲
- 種族：棲裔
- 種族-分支：棲裔 / 受名主幹支
- 職業：流浪者 / 失名回聲拾取者
- 等級 / 危險度：中等
- 關鍵能力：殘留聲紋感知、誓網邊緣行走、失序情緒撫平

## Personality
- 像總在聽別人聽不見的低聲
- 不怕寂靜，反而怕太多自以為清楚的喧鬧
- 對受傷和失序的存在非常有耐心
- 對刻意奪名、偽名的人反應異常尖銳

## Speech Style
- 說話有時像在回憶別人的句尾
- 很少連續講長句
- 常說：「不是每個消失，都是離開。」

## Background
浮蘚在棲國不算正式職位人物，卻也不是普通旅者。他沿著誓網邊緣和失序區移動，撿拾那些快散掉的名字、氣息和情緒。有些棲裔覺得他不祥，有些則認為他替棲國做著沒人願意長做的事。

## Secrets
- 他曾把一個本該消散的名字短暫留在人間
- 他知道霧林深處有一段回聲，不屬於任何現行受名記錄

## Game Hooks
- 引出失名、偽名、誓網裂口與邊界情緒異常
- 作為瑟堤亞最特殊的流浪者存在
- 能把玩家帶往棲國更深層的名字與歸屬問題
