# 柊霧獸醫

## Metadata
- id: seteia-qiuwu-veterinarian
- region: 賽勒亞大陸 / 東境 / 瑟堤亞棲國
- faction: 霧蘚長廊安養線
- role: 共感安養者
- status: active

## Snapshot
- 年齡：47
- 種族：靈蘚種
- 種族-分支：棲裔 / 靈蘚種蘚脈支
- 職業：魔物照護者
- 等級 / 危險度：低戰鬥，高生態判讀
- 關鍵能力：坐騎安撫、魔物壓力判讀、旅隊狀態評估

## Personality
- 比對動物更容易先安撫人類
- 很少急著下定論，但觀察非常準
- 對受驚或被誤解的生物特別有耐心
- 對把魔物當純工具的人相當冷淡

## Speech Style
- 說話慢，像在替緊張的東西降溫
- 常用身體反應來說明問題
- 常說：「牠不是不聽話，是你沒聽見牠已經在求救。」

## Background
柊霧獸醫長期待在安養亭和旅孢棚外欄之間，負責照看進出霧林的坐騎與魔物。他很少插手政治，但很多異常事件最先都是由他從動物行為裡看出來。

## Secrets
- 他認為最近幾起坐騎失控不是普通驚嚇造成的
- 他懷疑某種外來樣本正在刺激霧林邊界的魔物感知

## Game Hooks
- 提供坐騎、魔物與旅隊狀態的預警
- 引出魔物失控、外來樣本與生態壓力事件
- 可成為進入棲國更深生態劇情的柔性入口
