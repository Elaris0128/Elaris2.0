# 霧鎖寧

## Metadata

- region: 梵洛特大陸 / 纏聲斷濱共同體
- faction: 寂岩外港
- role: 外港守縛官
- status: active

## Snapshot
- 年齡：47
- 種族：海人
- 種族-分支：海人 / 鱗盾之裔
- 職業：守縛官 / 外港裁放者
- 等級 / 危險度：中高戰鬥與執法危險
- 關鍵能力：來船扣押、港口近戰、可信船判定、危船拒入

## Personality
- 不容易笑，笑了通常代表有人要倒楣
- 比起處罰，更重視先隔離風險

## Speech Style
- 乾脆、冷硬、幾乎沒有廢話
- 常先問「你從哪裡來」，再決定是不是談下一句

## Background
霧鎖寧守的是寂岩外港最窄的一道口。她很清楚只要放錯一艘船，整個共同體都可能為此付出命價。因此她寧可讓人恨，也不會讓不該進的人進來。

## Secrets
- 她曾私放過一艘掛錯旗但載著真難民的船
- 她對某段古潮語旗號過度敏感，像是曾親歷其害

## Game Hooks
- 外港、扣押、可信船與近岸危機的強硬入口
- 適合觸發衝突型任務或高壓談判
- 可與漂砥形成「能不能放行」的雙重判定
