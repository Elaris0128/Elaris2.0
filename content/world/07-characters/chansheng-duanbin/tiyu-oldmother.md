# 啼嶼老母

## Metadata
- id: vanlote-chansheng-tiyu-oldmother
- region: 梵洛特大陸 / 纏聲斷濱共同體
- faction: 崩濱獸巢
- role: 共議巢守
- status: active

## Snapshot
- 年齡：61
- 種族：獸人
- 種族-分支：獸人 / 潮聲鳥裔
- 職業：巢守 / 共議長鳴者
- 等級 / 危險度：高社會影響，中高危險
- 關鍵能力：沿崖秩序、回聲判位、群鳴誓儀、孤兒收編

## Personality
- 嚴厲，但不是冷血
- 討厭浪費糧與浪費能活下來的人
- 對任何不把幼巢放在前面的人都很不耐

## Speech Style
- 說話像啄石，一下下敲到要害
- 很少重複同一句警告第二次

## Background
啼嶼老母把崩濱獸巢從純粹的鳥裔主巢，撐成能收留拾生者與誓運孤兒的真正共同體核心。她懂得回聲、懂得風，也懂得哪種外來者值得冒一次收留的風險。

## Secrets
- 她保留一條只在極危時開啟的幼巢撤離路
- 她認得某段不屬於斷濱的古啼紋

## Game Hooks
- 斷濱共同體最高內部入口
- 涉及幼巢、保運、沿崖巡守與潮鳴儀式的主導角色
- 很適合作為長線信任關係的考驗者
