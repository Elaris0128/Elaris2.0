# 承芽

## Metadata
- id: xiaoyang-chengya
- region: 賽勒亞大陸 / 西境 / 魈陽古域
- faction: 炁炎酋庭
- role: 承名者
- status: active

## Snapshot
- 年齡：19
- 種族：人類
- 種族-分支：魈陽族 / 焦碑主脈
- 職業：試煉新承名者
- 等級 / 危險度：中
- 關鍵能力：短距爆發、炁紋初啟、試煉感知、火脈耐受

## Personality
- 衝勁大，還沒學會把榮耀和衝動分開
- 很在乎自己會被誰記住
- 對真正的老練者帶有近乎急躁的崇拜

## Speech Style
- 快，帶著燒得正旺的稜角
- 常說：「我不怕火，我怕火覺得我還不配。」

## Background
承芽剛通過第一階承名試煉，正站在能成為部族新火，也可能因急躁而摔碎自己的邊緣。

## Secrets
- 他在試煉裡看見一段不該出現的黑焰殘象
- 他一直在偷練一種未被允許的外路步法

## Game Hooks
- 年輕承名者與成年禮線入口
- 可引出試煉壓力、榮耀焦慮與文化衝突
- 很適合當玩家進入魈陽文化的第一個同齡視角
