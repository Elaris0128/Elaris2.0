# 灰卷

## Metadata
- id: duromok-huaijuan
- region: 賽勒亞大陸 / 西境 / 杜魯摩克隱域
- faction: 契印庫
- role: 主簿
- status: active

## Snapshot
- 年齡：42
- 種族：地精
- 種族-分支：地精 / 票契庫印支
- 職業：主簿 / 印庫管記人
- 等級 / 危險度：中
- 關鍵能力：印記校對、舊檔爬梳、貨單回推、庫房密錄

## Personality
- 安靜到像灰塵，但腦子快得像火花
- 喜歡從不重要的細節裡找真正的大問題
- 討厭別人碰亂她排好的舊檔

## Speech Style
- 說話平，常像在念卷尾批註
- 常說：「印不會說謊，但拿印的人常會。」

## Background
灰卷長年守在契印庫與舊檔壁室之間，是杜魯摩克最擅長從殘帳、舊契和印跡裡回推利益鍊的人。

## Secrets
- 她藏了一套只有自己看得懂的舊印索引
- 她知道某座封坑當年不是事故，而是刻意被關掉

## Game Hooks
- 文書、貨單與印記線入口
- 適合串連失契、偽契、走貨與封坑舊案
- 可成為玩家接觸西境文檔調查玩法的導師
