# 霧峰守徑

## Metadata
- id: vanlote-jianying-yinlin-wufeng-pathwarden
- region: 梵洛特大陸 / 繭影隱林
- faction: 霧林谷寨與曲水換徑圈
- role: 谷路守徑人
- status: active

## Snapshot
- 年齡：44
- 種族：人類
- 種族-分支：人類 / 柯沙里人
- 職業：守徑人 / 谷路巡望者
- 等級 / 危險度：中高
- 關鍵能力：霧徑辨位、溪谷巡守、外客攔檢、節令封路判讀

## Personality
- 寡言，警覺高，對「想抄近路的人」幾乎沒有耐心。
- 尊重守規矩的人，也願意保真正需要幫助的人走完危險路段。

## Speech Style
- 語句短，像一段一段釘在路上。
- 常說：「今天能過，不代表明天還能過。」

## Background
霧峰守徑常年守在幾條最容易出事的霧谷與曲水岔點。他不只是看路，更是在看誰有資格走路、哪一段該封、哪一段該放，以及誰會把一條活路走成死路。

## Secrets
- 他知道一條被正式封掉卻仍有人在用的暗徑。
- 他曾放過一批不該放行的外客，只因其中有人背著谷寨急需的藥材。

## Game Hooks
- 可帶出封路、放行、誤入禁段與谷路守規衝突。
- 很適合成為玩家穿越繭影隱林危險路段時的關鍵 NPC。
