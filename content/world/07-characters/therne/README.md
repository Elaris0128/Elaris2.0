這裡收納瑟恩聯邦國目前已可投入遊戲的主要 NPC 角色卡。

- 托文席爾：輪值首席協誓官
- 梅瑞狄雅：學術議誓官首席
- 露黎恩：聯邦密錄局局首

## 使用原則
- 這批角色對應瑟恩的制度、法理、涉外與河港玩法
- 最高領導層現已獨立成正式角色卡，可直接承接聯邦級危機與議會級事件
- 角色來源現在同時覆蓋聯邦官署、世界級組織與知識型流浪者
- 支援角色名錄另見 `supporting-cast.md`，已補宗教、政治、教育、軍務與諜報線
- 若後續擴張學院線，可再補更多教授、留學生與使節角色
- 角色先以現有地方包 NPC 為核心，避免設定斷層
