# 伊瑟菈

## Metadata
- id: therne-isela-high-axis
- region: 賽勒亞大陸 / 東境 / 瑟恩聯邦國
- faction: 星徽議會
- role: 備任星軌總執
- status: active

## Snapshot
- 年齡：44
- 種族：法奧人
- 種族-分支：法奧人 / 瑟恩正典支
- 職業：危機統籌官 / 議制執行者
- 等級 / 危險度：高權限，高程序壓力
- 關鍵能力：聯邦調度、危機授權、跨城邦協商、程序壓縮

## Personality
- 冷靜、果斷，能在一堆程序中迅速看出真正關鍵
- 不迷信緊急權，但也不會因害怕僭越而錯過時機
- 對術式事故與跨境危機幾乎沒有情緒化反應

## Speech Style
- 句子結構很乾淨，幾乎像法令摘要
- 擅長把混亂討論壓成幾個可執行選項
- 常說：「程序不是拖延，它是讓代價有名字。」

## Background
伊瑟菈是瑟恩聯邦目前最常被提及的備任星軌總執人選。她並非每時每刻都握有全權，但一旦聯邦需要快速統一資源、軍令與術式管制，她就是最可能被推上危機中樞的人。

## Secrets
- 她私下支持一套尚未公開的聯邦級禁術應急規程
- 她已預估瑟恩下一次大危機很可能來自水路與條約線同時失衡

## Game Hooks
- 作為瑟恩國家主線與危機調度線的最高入口
- 可導向跨城邦權限爭議、緊急法與聯邦存續壓力
- 讓玩家看見瑟恩「文明秩序」背後的極限運作方式
