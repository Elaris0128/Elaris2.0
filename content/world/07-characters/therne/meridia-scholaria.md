# 梅瑞狄雅

## Metadata

- faction: 星徽議會 / 學術議誓官聯席
- role: 學術議誓官首席
- status: active

## Snapshot
- 年齡：50
- 種族：法奧人
- 種族-分支：法奧人 / 瑟恩正典支
- 職業：學術高官 / 教育制度設計者
- 等級 / 危險度：高學術影響，中等戰鬥力
- 關鍵能力：術式教育整編、學院評議、知識授權、研究審查

## Personality
- 聰明、耐心，對真正的好問題有近乎溫柔的尊重
- 面對偷渡研究與學術腐敗時會變得非常尖銳
- 對天才沒有特別偏愛，對可持續的人才制度反而更執著

## Speech Style
- 會先問「證據呢」，再問「代價呢」
- 很少說絕對話，除非牽涉學生安全與禁術外流
- 常說：「知識不是誰先碰到，而是誰能負責它離開桌面之後的事。」

## Background
梅瑞狄雅掌握瑟恩聯邦最關鍵的學術授權節奏，是七曜城邦之間教育、研究與共享資源能否維持平衡的核心人物。她代表聯邦把知識做成制度而不是只做成名聲的那一面。

## Secrets
- 她正在壓下一批可能牽連多座學院的違規實驗名單
- 她懷疑有人在利用學生交換制度偷運分級樣本

## Game Hooks
- 作為瑟恩教育、學院、研究事故與知識政治的高層入口
- 可牽出禁術審核、學院派系與人才爭奪
- 能把單一實驗事故接到整個聯邦體制
