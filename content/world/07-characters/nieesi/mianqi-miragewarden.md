# 眠契

## Metadata
- id: vanlote-nieesi-mianqi-miragewarden
- region: 梵洛特大陸 / 涅厄斯魔人國
- faction: 幻紋守谷者
- role: 幻紋守谷首席
- status: active

## Snapshot
- 年齡：52
- 種族：魔人
- 種族-分支：魔人 / 緲眠
- 職業：首席守谷者 / 幻紋術師
- 等級 / 危險度：極高術式危險
- 關鍵能力：幻街設置、夢境探痕、結界維穩、真路遮斷

## Personality
- 安靜、難測、對睡夢和名字都保持異常尊重
- 對任何擅闖夢域的人毫不寬容

## Speech Style
- 語速慢，像在替每個字留出口
- 常把警告說得像提醒你自己選擇下場

## Background
眠契負責維持裂環幻都與外圍裂谷的幻紋秩序，懂得讓敵人迷路，也懂得讓恐慌中的自己人找到回來的路。她所管理的不是單純結界，而是一整套讓國都還能活著的誤導生態。

## Secrets
- 她曾讓某段真路對一名外來者短暫開啟
- 她在夢域裡見過不該再出現的深淵舊名

## Game Hooks
- 幻境、結界與真路相關任務的核心導師
- 可帶玩家進入裂星幻湖地宮或幻街禁段
- 很適合承接深層神祕與風險學習線
