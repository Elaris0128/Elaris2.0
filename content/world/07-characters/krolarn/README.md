# 克羅嵐角色索引

這裡收納克羅嵐聯盟國目前已可投入遊戲的主要 NPC 角色卡。

## 最高領導層
- 哈魯格：馭角 / 大環主誓使
- 烏芮薩：冥羽 / 骨語長母
- 圖格爾：裂牙 / 裂牙戰首
- 牙煞：逐影 / 環影部隊環首統領

## 目前角色
- 魁牙：裂痕 / 河埠誓衛
- 洛絮：逐影 / 翻譯者
- 霧婆薩連：蟄蛇 / 老藥婦
- 赫圖：隱耳 / 村中見證人
- 烏塔吉：馭嶺 / 誓獸育契公會獸紋伴契師
- 葛薩：鷹目 / 河牙誓盟使
- 狼齒：冥羽 / 骨歌浪客

## 使用原則
- 這批角色對應河埠、氏族、濕原採藥與見證制度玩法
- 最高領導層現已獨立成正式角色卡，可直接承接會盟、試煉與聯盟級危機
- 角色來源現在同時覆蓋聯盟秩序、跨國公會與地方神話型流浪者
- 支援角色名錄另見 `supporting-cast.md`，已補宗教、政治、教育、軍務與諜報線
- 若後續補更多氏族線，可再加入獵手、祭司與戰舞者角色
- 角色都取自現有地方包，方便直接引用
