# 牙煞

## Metadata
- id: krolarn-yasha-ringhead
- region: 賽勒亞大陸 / 東境 / 克羅嵐聯盟國
- faction: 環影部隊
- role: 環首統領
- status: active

## Snapshot
- 年齡：39
- 種族：獸人
- 種族-分支：獸人 / 十獸之環・克羅嵐共誓支
- 十獸席位：逐影氏族
- 職業：情報統領 / 邊影指揮者
- 等級 / 危險度：高威脅，極高滲透能力
- 關鍵能力：邊影巡查、反滲透、夜跡回收、影域調度

## Personality
- 少言、乾脆，很少在同一個地方留下第二次明顯痕跡
- 對榮譽有自己的版本：先讓部族活下來，再來談漂亮話
- 討厭把情報當作貴族炫耀的人

## Speech Style
- 話像從陰影裡伸出來的一把短刀
- 很少主動多說，但問的問題都會逼近核心
- 常說：「我不是懷疑你，我只是先替你的敵人想一步。」

## Background
牙煞是環影部隊現任環首統領，掌握克羅嵐最難被外人看見的那張網。很多聯盟內部的安全感，並不是來自戰首和大會盟，而是來自他早已把威脅剪掉了一半。

## Secrets
- 他默許一支不在正式編制內的影行小隊活動
- 他已掌握某個氏族代表與外部灰線接觸的證據，但還沒交出去

## Game Hooks
- 作為克羅嵐情報、反滲透與暗線行動的最高入口
- 可牽出氏族內鬼、河道黑貨與戰前偵巡
- 很適合和阿瓦隆、瑟恩、灰線組織任務接軌
