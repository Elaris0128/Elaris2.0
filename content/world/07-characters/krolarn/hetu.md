# 赫圖

## Metadata
- id: krolarn-hetu
- region: 賽勒亞大陸 / 東境 / 克羅嵐聯盟國
- faction: 霧骨灘村見證圈
- role: 村中見證人
- status: active

## Snapshot
- 年齡：45
- 種族：獸人
- 種族-分支：獸人 / 十獸之環・克羅嵐共誓支
- 十獸席位：隱耳氏族
- 職業：見證人
- 等級 / 危險度：低戰鬥，中社會秩序
- 關鍵能力：村規裁量、夜行安排、可信度判斷

## Personality
- 寡言、穩重、極少浪費詞
- 比起說服，更重視誰願不願意承擔說過的話
- 對不懂規矩的人不兇，但絕不縱容
- 很能看出誰只是害怕、誰是真的有壞心

## Speech Style
- 句子短，停頓長
- 常讓沉默替他施壓
- 常說：「在這裡，被見證過就別想假裝沒說過。」

## Background
赫圖是霧骨灘村最像地樁的人。很多外人只看到他沉默，卻沒意識到整個村子的夜行路、暫宿規矩與風險界線，其實很多都要經過他的點頭。

## Secrets
- 他知道村裡有人在偷偷替外來學者帶路
- 他一直懷疑那條路通往的不只是採藥地

## Game Hooks
- 作為夜行、暫宿與濕原深入的許可入口
- 讓玩家介入見證制度與村規事件
- 可引出學者盜路或禁地探索任務
