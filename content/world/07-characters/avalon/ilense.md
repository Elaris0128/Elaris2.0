# 伊蓮瑟

## Metadata
- id: avalon-ilense
- region: 賽勒亞大陸 / 東境 / 阿瓦隆王國
- faction: 誓典修纂體系
- role: 流動修典師
- status: active

## Snapshot
- 年齡：34
- 種族：瓦雷溫人
- 種族-分支：瓦雷溫人 / 阿瓦隆塔環支
- 職業：文書修復者 / 巡禮學者
- 等級 / 危險度：低戰鬥，高知識支援
- 關鍵能力：古文判讀、誓文修復、歷史鉤沉

## Personality
- 溫和、耐心、細膩
- 對知識抱有敬畏，不喜歡被人拿去當政治工具
- 喜歡觀察人的用字與停頓
- 表面柔和，其實意志不弱

## Speech Style
- 聲音輕，但不含糊
- 會用比喻和歷史片段幫助玩家理解世界
- 常說：「你以為那只是舊紙，其實很多人會為它流血。」

## Background
伊蓮瑟長期在修院、塔庫與地方誓典所之間往返，修復誓文、整理紀錄、抄寫古卷。她知道歷史裡真正危險的從來不是發黃的紙，而是被誰拿來用、又在什麼時候曝光。

## Secrets
- 她私下抄錄過一份不該完整存在的舊誓文
- 某些遺失卷軸其實不是被偷，而是被故意「移走」

## Game Hooks
- 能將 lore 轉為可玩線索
- 適合承接誓契、遺跡、失落卷軸任務
- 可作為玩家理解世界深層歷史的穩定入口
