# 蘇艾爾

- faction: 王座廷議會
- role: 外務傳令官
- status: active

## Snapshot
- 年齡：32
- 種族：瓦雷溫人
- 種族-分支：瓦雷溫人 / 阿瓦隆塔環支
- 職業：政務傳令官 / 外交文書使
- 等級 / 危險度：非戰鬥型，中低
- 關鍵能力：禮儀判讀、文書分辨、王都通行面子

## Personality
- 冷靜、克制、準確
- 對失禮與程序錯誤十分敏感
- 重視秩序，但不是僵化到沒有判斷力
- 對真正守信的人會逐漸釋出善意

## Speech Style
- 用詞正式，稱呼嚴謹
- 幾乎不說廢話
- 會先提醒規範，再給例外空間
- 常說：「先把名分釐清，我們再談後續。」

## Background
蘇艾爾出身中層塔職家系，沒有顯赫封地，卻靠近乎苛刻的準確度一路進入王座廷議會的外務線。他熟悉王都的人情與權力距離，也知道哪些命令被寫進紙上，哪些命令根本不該留字。

## Secrets
- 曾替上級處理過一封不該存在的密令副本
- 對暮環司抱有戒心，但知道自己有時離不開他們

## Game Hooks
- 派發正式主線任務
- 給玩家合法身份或通行便利
- 成為玩家接觸王權層的第一扇門
- 若玩家常違禮或亂來，他會成為穩定的壓力來源
