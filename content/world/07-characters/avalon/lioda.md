# 莉歐妲

## Metadata
- id: avalon-lioda
- region: 賽勒亞大陸 / 東境 / 阿瓦隆王國
- faction: 七誓教廷
- role: 牧誓官
- status: active

## Snapshot
- 年齡：41
- 種族：瓦雷溫人
- 種族-分支：瓦雷溫人 / 阿瓦隆塔環支
- 職業：教廷法誓審查者
- 等級 / 危險度：中
- 關鍵能力：誓約審查、宗教律判讀、異端辨識

## Personality
- 嚴肅、穩定、有威信
- 不輕易寬恕，但也不欣賞盲目獵巫
- 對真正尊重誓約的人抱有基本善意
- 對利用宗教牟利的人極端厭惡

## Speech Style
- 節奏穩，像在宣讀裁示
- 幾乎不提高聲量，但很有壓迫感
- 常說：「誓言不是裝飾，它會追上每個說出口的人。」

## Background
莉歐妲長年處理誓獸契約、違誓審理與異端指控，見過太多把信仰當工具的人。她代表的不是最溫柔的教廷面，但也因此成為極少數能在宗教與現實之間做出艱難判斷的人。

## Secrets
- 她對某些教廷內部的判例其實並不完全認同
- 她知道誓契失竊案若曝光完整真相，將動搖比想像更大的東西

## Game Hooks
- 提供宗教律與誓約支線
- 成為教廷線的重要審核者
- 可以讓玩家獲得正統認可，也能親手關上那扇門
