# 阿瓦隆角色索引

這裡收納阿瓦隆王國目前已可投入遊戲的主要 NPC 角色卡。

## 最高領導層
- 凱洛恩・鋒塔：阿瓦隆國王
- 瑟薇拉・塔環：塔環聖座總教長
- 布拉維克・伊蕾汀：北誓列侯 / 邊防總領
- 維榮・暮鴉：暮環司環首

## 使用原則
- 這批角色可直接進入開局與阿瓦隆主支線
- 最高領導層現已獨立成正式角色卡，可直接承接國家級事件
- 角色來源現在同時覆蓋王國體系、跨國組織與特殊流浪者
- 支援角色名錄另見 `supporting-cast.md`，已補宗教、政治、教育、軍務與諜報線
- 若未來你有正式角色名稱或既定人物，可直接替換這些功能位
- 角色功能位先保留，避免玩法斷層
