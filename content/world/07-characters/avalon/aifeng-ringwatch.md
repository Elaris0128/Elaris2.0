# 暮鴉艾馮

## Metadata
- id: avalon-aifeng-ringwatch
- region: 賽勒亞大陸 / 東境 / 阿瓦隆王國
- faction: 暮環司
- role: 港線耳目
- status: active

## Snapshot
- 年齡：32
- 種族：瓦雷溫人
- 種族-分支：瓦雷溫人 / 誓港暗線支
- 職業：暮環司監聽員
- 等級 / 危險度：中
- 關鍵能力：港線監看、異常往來辨識、灰貨追索、低調跟監

## Personality
- 不愛被記住，也不太相信別人有必要認識自己
- 對細節異常的耐心幾乎可怕
- 願意救人，但不會先解釋自己為何出手

## Speech Style
- 說話像貼著牆角走
- 常說「你先別回頭，讓我把這件事看完」
- 很少完整回答，除非他確認你值得知道

## Background
艾馮是暮環司在誓港最穩的一對眼之一，長期盯著灰貨、可疑神職往來與不該出現在港邊的影子。

## Secrets
- 他追著一條從教廷邊線通到港倉的暗訊號已經半年
- 他認得某位高層人物偽裝後的走路方式

## Game Hooks
- 承接阿瓦隆諜報、港線黑貨與教廷暗線任務
- 可作為高風險監視與跟監導師
- 能把港口支線接進王都情報主線
