# 瑟蘭

## Metadata
- id: avalon-seran
- region: 賽勒亞大陸 / 東境 / 阿瓦隆王國
- faction: 中立商會聯席 / 東海誓港券廊
- role: 券約評價官
- status: active

## Snapshot
- 年齡：33
- 種族：瓦雷溫人
- 種族-分支：瓦雷溫人 / 阿瓦隆塔環支
- 職業：商券判價師 / 票據見證人
- 等級 / 危險度：低戰鬥，高交易與情報
- 關鍵能力：貨價判讀、票據驗真、走貨流向重組

## Personality
- 有禮，卻不軟弱
- 很會讓人自己把真正想賣的東西說出口
- 對假清高的貴族和假窮的黑販同樣沒耐心
- 認為交易的體面是文明的外皮，裂開時最能看到真相

## Speech Style
- 用詞乾淨俐落，帶港口式的精準節奏
- 常先肯定對方一句，再補一刀價格現實
- 常說：「帳能作假，流向不會。」

## Background
瑟蘭出身港務書記家，後來轉進中立商會聯席的券廊系統。她比王國稅官更懂票據如何繞過邊檢，也比自由商更懂哪種貨一旦進錯倉就會讓整條商路一起出事。在阿瓦隆，她是最典型的外部組織人物，卻又帶著王國人特有的節制與誓感。

## Secrets
- 她暗中替一條救濟糧線作過不完全合法的票據掩護
- 她知道東海誓港某批軍需貨並不是送往帳面上的目的地

## Game Hooks
- 幫玩家判斷貨物值不值得碰、能不能安全脫手
- 引出幣券、走貨、假封樣與軍需轉運線
- 作為阿瓦隆非國家組織人物的代表入口
