# 薇拉辛

- faction: 七誓教廷
- role: 誓鐘司祭
- status: active

## Snapshot
- 年齡：43
- 種族：瓦雷溫人
- 種族-分支：瓦雷溫人 / 阿瓦隆塔環支
- 職業：司祭 / 公開見證主理
- 等級 / 危險度：中高
- 關鍵能力：誓鐘序列、公開見證、群眾安撫、儀節判準

## Personality
- 沉靜而嚴整，對儀式秩序有近乎偏執的保護欲
- 能理解民間恐懼，但不容許有人藉混亂竄改誓序
- 在教廷內部向來被視為穩鐘，不輕易站隊

## Speech Style
- 說話像鐘聲，一句一句敲出段落
- 常說「先讓見證成立，之後才談誰對誰錯」
- 不提高音量，但能讓周圍自然安靜下來

## Background
薇拉辛掌管銀鋒城最重要的誓鐘序列與公開見證次序。她不是阿瓦隆教廷最顯眼的人，卻是讓大型誓儀不崩掉的關鍵支柱。

## Secrets
- 她知道教廷內有人動過誓鐘排次，想讓某些名字被晚一刻看見
- 她暗中保存了一份不該被刪掉的戰亡名冊副本

## Game Hooks
- 承接阿瓦隆宗教、教廷內壓與公開見證任務
- 可帶出被改動的祭序、失名與群眾騷動
- 能作為高可信度的儀式導師與誓約判準者
