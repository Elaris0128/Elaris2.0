# 哈姆特

## Metadata
- id: avalon-hamut
- region: 賽勒亞大陸 / 東境 / 阿瓦隆王國
- faction: 霜誓關退役軍系
- role: 邊防老兵
- status: active

## Snapshot
- 年齡：58
- 種族：瓦雷溫人
- 種族-分支：瓦雷溫人 / 阿瓦隆塔環支
- 職業：退役軍官 / 民間嚮導
- 等級 / 危險度：中
- 關鍵能力：野外求生、邊境判讀、老兵直覺

## Personality
- 冷硬、寡言、討厭空話
- 對年輕軍官與文官都有保留
- 對真正扛得住事的人會釋出罕見的信任
- 外冷內熱，但只在生死關頭才看得出來

## Speech Style
- 句子短，像把多餘的字都刪掉
- 常用「我看過更糟的」「先看地，再看人」這類說法

## Background
哈姆特在霜誓關待了大半輩子，曾參與多次邊境危機，也見證太多年輕士兵被制度、命令與冬境吞掉。他退下來後不再完全相信官方敘事，但仍舊把阿瓦隆當成自己守過的家。

## Secrets
- 他知道某條舊補給線從未真正廢棄
- 他見過一個本該不存在於邊境的王國密令符號

## Game Hooks
- 野外與邊境任務的重要引路人
- 可讓玩家更早接觸軍方真實面
- 能把任務從表面的魔獸問題拉向更深的陰影
