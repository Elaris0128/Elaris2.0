# 維榮・暮鴉

## Metadata
- id: avalon-veyron-ringcrow
- region: 賽勒亞大陸 / 東境 / 阿瓦隆王國
- faction: 暮環司
- role: 環首
- status: active

## Snapshot
- 年齡：49
- 種族：瓦雷溫人
- 種族-分支：瓦雷溫人 / 阿瓦隆塔環支
- 職業：情報首腦 / 影務統籌者
- 等級 / 危險度：高威脅，極高隱蔽性
- 關鍵能力：港線反間、暗名調度、影證比對、耳目整編

## Personality
- 客氣得像沒有敵人，實際上把所有人都先當成待驗目標
- 不喜歡純粹殘酷，而偏好能留下後續價值的處理方式
- 對國家忠誠，但忠誠的對象更接近阿瓦隆本身而不是某一個人

## Speech Style
- 語速慢，像在等別人自己說出破綻
- 會把最重的話藏在最平的句子裡
- 常說：「你不必對我坦白，對自己的影子坦白就夠了。」

## Background
維榮是暮環司現任環首，主導港線、邊線與王都內層的反間整編。他不常公開現身，但在阿瓦隆真正知道情報如何改變授印、邊防與神職關係的人裡，他是最危險的一位。

## Secrets
- 他默許部分灰線流通，目的是釣出更深的港務同盟
- 他手裡握有對王室與聖座都足夠敏感的私密見證

## Game Hooks
- 作為暮環司主線、港線諜報與內廷監視的頂層入口
- 可牽出王國情報網、失名行動與黑箱授意
- 能把地方小案連到整個阿瓦隆的深層權力鬥爭
