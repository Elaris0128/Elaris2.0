# 梅列絲

## Metadata
- id: avalon-merelis
- region: 賽勒亞大陸 / 東境 / 阿瓦隆王國
- faction: 東海誓港稅契體系
- role: 港誓稅官
- status: active

## Snapshot
- 年齡：37
- 種族：瓦雷溫人
- 種族-分支：瓦雷溫人 / 阿瓦隆塔環支
- 職業：稅契審查官
- 等級 / 危險度：低戰鬥，中情報
- 關鍵能力：帳冊核對、走私辨識、貨物流向判讀

## Personality
- 現實、敏銳、幾乎不被表面說詞打動
- 知道制度如何運作，也知道制度怎麼被鑽漏洞
- 不喜歡浪漫化違法行為
- 對能看懂細節的人很有耐心

## Speech Style
- 語速平穩但壓迫感強
- 常用反問句逼對方自己露餡
- 常說：「文件不會說謊，人會。」

## Background
梅列絲在港口稅契線待了很多年，見過太多披著合法封條的非法貨物。她不屬於高權核心，卻掌握大量第一手流通資訊，因此在港口、商會、情報單位與黑市之間都有人盯著她的一舉一動。

## Secrets
- 她手上留有一份從未正式上報的黑名單
- 曾經放走過一批貨，只為釣出更大的魚

## Game Hooks
- 發放港口調查型任務
- 提供貨物、稅契、黑市的初步線索
- 可成為玩家合法行動的保護傘，也可能用玩家當誘餌
