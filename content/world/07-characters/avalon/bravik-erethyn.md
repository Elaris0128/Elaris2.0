# 布拉維克・伊蕾汀

## Metadata
- id: avalon-bravik-erethyn
- region: 賽勒亞大陸 / 東境 / 阿瓦隆王國
- faction: 伊蕾汀侯系
- role: 北誓列侯 / 邊防總領
- status: active

## Snapshot
- 年齡：47
- 種族：瓦雷溫人
- 種族-分支：瓦雷溫人 / 阿瓦隆塔環支
- 職業：封侯 / 軍政領主
- 等級 / 危險度：高戰力，高政治影響
- 關鍵能力：邊防軍調度、騎兵統御、封地徵召、前線見證

## Personality
- 務實、強硬，討厭王都用漂亮句子替前線止血
- 對部下照顧得很重，但換來的是絕對服從
- 對王室忠誠，前提是王室真有在守北線

## Speech Style
- 句子像軍令
- 不愛隱喻，只問能不能守、誰來守、要守多久
- 常說：「誓不是拿來掛牆上的，誓要有人在雪裡站著。」

## Background
布拉維克出身伊蕾汀侯系，掌北線邊防哨塔與騎兵軍團，是阿瓦隆真正最接近戰爭前沿的高層人物之一。他與王室關係密切，卻也代表前線貴族對王都改革的現實壓力。

## Secrets
- 他懷疑北境某條軍需線已被人長期標記給外部勢力
- 他保留一支名義上不存在的老誓旗隊，以備王都失手時自救

## Game Hooks
- 作為阿瓦隆軍事線與北線危機的高層入口
- 可導向前線補給、授旗爭議、封侯與王權拉扯
- 讓玩家有機會直接接觸王國的戰爭決策層
