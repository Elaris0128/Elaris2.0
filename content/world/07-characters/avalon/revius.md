# 雷維斯

## Metadata
- id: avalon-revius
- region: 賽勒亞大陸 / 東境 / 阿瓦隆王國
- faction: 王座廷議會檔誓院
- role: 王誓檔錄官
- status: active

## Snapshot
- 年齡：41
- 種族：瓦雷溫人
- 種族-分支：瓦雷溫人 / 阿瓦隆塔環支
- 職業：文職官 / 誓文核驗官
- 等級 / 危險度：低戰鬥，高制度壓力
- 關鍵能力：誓文核對、授印權限調查、檔冊追索

## Personality
- 嚴謹、慢熱、不喜歡被人催
- 對制度很忠誠，但不是盲從
- 真正讓他動怒的不是造假，而是有人拿誓名當戲法
- 對願意補正程序的人還算寬容

## Speech Style
- 講話像逐條翻頁
- 常把人的口頭承諾重新說成可以被記錄的句子
- 常說：「你想留下什麼，先決定你願不願意被記下來。」

## Background
雷維斯待在阿瓦隆權力最不耀眼卻最致命的角落。檔誓院不握兵權，也不敲鐘立法，但很多王國內部能不能成真，都得先過他們手上的一層誓文與印記。他是典型的國家人物，代表阿瓦隆把秩序寫進紙與名分的那一面。

## Secrets
- 他正在私查一批授旗誓文中的舊印異常
- 他懷疑某位下級貴族在利用戰時失檔重寫自己的家系記錄

## Game Hooks
- 讓玩家接觸阿瓦隆的正式程序、授印與舊檔調查
- 引出家系造假、誓文遺失與名分爭奪事件
- 作為王國制度線與軍政線的交會入口
