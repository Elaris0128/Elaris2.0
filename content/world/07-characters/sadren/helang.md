# 赫朗

## Metadata
- id: sadren-helang
- region: 賽勒亞大陸 / 東境 / 薩德原聯盟
- faction: 風勒騎隊訓練圈
- role: 年長騎隊教頭
- status: active

## Snapshot
- 年齡：49
- 種族：人馬族
- 種族-分支：森馳裔 / 薩德原人馬支
- 職業：騎隊教頭
- 等級 / 危險度：中戰鬥，中訓練
- 關鍵能力：坐騎判讀、護送評估、長途奔襲規劃

## Personality
- 對人兇，對坐騎與幼者很有耐心
- 最討厭華而不實的勇敢
- 很懂什麼叫真的撐得住路程
- 對不放棄的人有硬派的尊重

## Speech Style
- 像在訓練場上發號施令
- 常用簡短斷句逼人集中精神
- 常說：「撐不住就早說，草原不會替你心軟。」

## Background
赫朗多年帶騎隊跑過東境邊線，知道哪種天候會讓人逞強送命，也知道哪種隊伍表面整齊實際一拉就散。很多需要評估護送難度的隊伍，最後都會來找他。

## Secrets
- 他懷疑最近幾起失聯並不是單純迷路
- 他從失聯隊伍留下的蹄印看出有人在刻意驅趕路線

## Game Hooks
- 提供草原護送、訓練與追擊任務
- 協助玩家評估隊伍是否適合遠行
- 可引出失聯、誘導與伏擊劇情
