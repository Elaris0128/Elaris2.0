# 蘇芮瑪

## Metadata
- id: sadren-surima
- region: 賽勒亞大陸 / 東境 / 薩德原聯盟
- faction: 逐原五柱議誓制
- role: 議誓使
- status: active

## Snapshot
- 年齡：36
- 種族：人馬族
- 種族-分支：森馳裔 / 薩德原人馬支
- 職業：議使 / 誓約見證人
- 等級 / 危險度：中等
- 關鍵能力：跨族協調、路權仲裁、誓名糾紛判定

## Personality
- 直、穩、很會讓吵架的人重新記得自己為什麼結盟
- 不怕翻舊帳，但討厭人只挑對自己有利的那段記憶
- 對外來人不苛刻，但要求很清楚
- 一旦認定你在踐踏誓名，就不再給緩衝

## Speech Style
- 句子像在草地上立柱
- 不愛抽象說理，會直接把問題變成能不能見證
- 常說：「你要我信，可以。先說這句話你敢不敢讓風記住。」

## Background
蘇芮瑪負責的是薩德原最不像官、卻最像國家的部分。她在五柱議誓制裡專做跨族誓名、路權、補水順序與隊伍責任的協調。這種角色在草原上不靠城牆，而靠被所有人承認她有資格開口。

## Secrets
- 她正在追查一組被人故意偽造成合法誓名的外路契約
- 她私下相信某段被廢棄古路的權利本來就不該被抹掉

## Game Hooks
- 引出草原盟誓、路權、水權與隊伍責任糾紛
- 讓玩家接觸薩德原真正的政治核心
- 可作為跨部族主線與大任務的正式入口
