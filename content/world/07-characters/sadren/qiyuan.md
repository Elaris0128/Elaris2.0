# 祁原

## Metadata
- id: sadren-qiyuan
- region: 賽勒亞大陸 / 東境 / 薩德原聯盟
- faction: 無固定所屬
- role: 無柱行者
- status: active

## Snapshot
- 年齡：不詳，外觀約 34 歲
- 種族：人馬族
- 種族-分支：森馳裔 / 薩德原人馬支
- 職業：流浪者 / 路兆背負者
- 等級 / 危險度：中等
- 關鍵能力：古路感知、荒線生存、失隊蹤跡判讀

## Personality
- 看起來像對所有隊伍都保持距離
- 不討厭人，只是不願意被任何一根柱子留住
- 對假風兆和假消息有極強敵意
- 對真正迷路的人反而異常溫和

## Speech Style
- 講話像風換方向前的低壓
- 常把方向和名字放在同一句裡
- 常說：「有些人不是走丟，是被從路裡拿掉了。」

## Background
祁原不屬於任何柱圈，也不固定待在任何營地。很多人只在最壞的時候碰見他，比如錯路、失隊、夜裡燈火不對的時候。有人說他是壞兆，也有人說草原讓他替那些無人送回的人多走一段。

## Secrets
- 他知道一條沒有寫進任何公開路譜的返野舊線
- 他曾替某位死在外路上的重要人物守過一夜，卻從沒說出那人的名字

## Game Hooks
- 引出失隊、荒線、古路與返野送魂相關任務
- 作為薩德原最帶神祕色彩的特殊流浪者
- 能把玩家帶進草原不公開的記憶區
