# 烏賽寧

## Metadata
- id: sadren-wusaining-windstrategy
- region: 賽勒亞大陸 / 東境 / 薩德原聯盟
- faction: 誓原風策局
- role: 總策監
- status: active

## Snapshot
- 年齡：45
- 種族：人馬族
- 種族-分支：森馳裔 / 薩德原人馬支
- 職業：情報首腦 / 邊策監理者
- 等級 / 危險度：高威脅，高機動隱蔽能力
- 關鍵能力：邊境風訊、路印校核、內叛監視、行動遮線

## Personality
- 像乾冷夜風，沒有多餘動靜卻處處都在
- 不迷信秘密本身，只在乎秘密值不值得用
- 對聯盟內部的自滿特別不放心

## Speech Style
- 會把問題拆成方位、時機和誰最後看到
- 不太用感情字眼，卻總讓人聽出危險
- 常說：「草原上沒有人真正消失，只有線索被誰先踩過。」

## Background
烏賽寧是誓原風策局現任總策監，專管邊境風訊、古路動態與內部叛意。對薩德原來說，他的重要性在於草原雖然開闊，卻不能因此失去對遠處異動的提前感。

## Secrets
- 他暗中允許少數灰線人物活動，只為追更深的外路干預
- 他正在查一串被偽造成祖靈預兆的假風訊

## Game Hooks
- 作為薩德原情報線、古路線與內部風訊失真問題的最高入口
- 可牽出偽路標、古道爭奪與跨境探子
- 很適合和北境、阿瓦隆邊線以及灰線任務接軌
