# 烏嵐赫

- faction: 逐原五柱議誓制

- status: active

## Snapshot
- 年齡：51
- 種族：人馬族
- 種族-分支：森馳裔 / 薩德原人馬支
- 職業：聯盟輪值領首 / 高層議誓者
- 等級 / 危險度：高權威，高機動
- 關鍵能力：五柱共議整編、路權與水權總裁定、跨族停爭

## Personality
- 寬廣、沉穩，像很大的草原風，不急卻總在
- 對盟誓的理解不是浪漫，而是能不能讓人活著走下一季
- 對說漂亮話卻不願承責的人非常厭煩

## Speech Style
- 講話像立在路中央的一塊老石
- 不愛裝神祕，會直接把代價掀到桌面上
- 常說：「能不能一起走，不看你今天說得多好，看看霜季時你還在不在。」

## Background
烏嵐赫是薩德原現任輪值總誓首，負責在五柱之間維持議誓秩序與對外統一聲音。他不是永久君主，而是草原願意暫時把重話與重責交到誰手上的答案。

## Secrets
- 他知道某段被掩埋的古路若被重新承認，將會重畫不少部族利益
- 他正在拖延一場過早爆發的柱間正面衝突

## Game Hooks
- 作為薩德原最高政治線與五柱共議的主要入口
- 可牽出水權、路權、輪值秩序與聯盟存續問題
- 能把地方衝突直接提升為聯盟級決策
