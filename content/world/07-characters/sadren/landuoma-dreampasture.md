# 蘭朵瑪

## Metadata
- id: sadren-landuoma-dreampasture
- region: 賽勒亞大陸 / 東境 / 薩德原聯盟
- faction: 夢牧柱
- role: 夢牧柱首 / 誓靈導母
- status: active

## Snapshot
- 年齡：63
- 種族：人馬族
- 種族-分支：森馳裔 / 薩德原人馬支
- 職業：高位祭儀者 / 柱首
- 等級 / 危險度：中戰力，高神秘威望
- 關鍵能力：祖靈導祭、觀兆、送魂、柱間見證

## Personality
- 溫和，但不是柔弱；她的堅持藏在很長的耐心裡
- 對祖靈與返野儀式極度敬重
- 覺得許多現代爭執都是因為人忘了死後還要被誰看見

## Speech Style
- 像在講一條會從夢裡繼續走下去的路
- 不怕沉默，很多判斷都會在沉默後說出
- 常說：「風不是帶走名字，風是在找名字該回去的地方。」

## Background
蘭朵瑪是夢牧柱首，也是薩德原最重要的高位祭儀人物之一。她的聲音不一定決定所有政務，但在祖靈、送魂、柱間誓見與長線遷徙的事上，沒有多少人願意和她正面對著幹。

## Secrets
- 她知道一條被祖靈故意遮住的古舊誓路
- 她對某批近年無法順利返野的亡者很不安

## Game Hooks
- 作為薩德原宗教、祖靈、安魂與夢兆線的最高入口
- 可引出返野送魂、古路禁忌與柱首對立
- 能把死後世界與草原現實生活自然接起來
