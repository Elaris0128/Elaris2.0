# 瑪舒雅

## Metadata
- id: sadren-mashuya
- region: 賽勒亞大陸 / 東境 / 薩德原聯盟
- faction: 路譜守望者群
- role: 路譜守望者
- status: active

## Snapshot
- 年齡：35
- 種族：人馬族
- 種族-分支：森馳裔 / 薩德原人馬支
- 職業：路譜師 / 引路者
- 等級 / 危險度：低戰鬥，高移動規劃
- 關鍵能力：草原導航、補水節點記憶、路標異常辨識

## Personality
- 溫和，但對胡亂改路的人沒有容忍度
- 很會觀察別人是不是在說假路
- 把草原看成一張會呼吸的圖
- 對有禮貌的旅人相當照顧

## Speech Style
- 句子不快，像在替人校正方向
- 常用地勢與風勢比喻人心
- 常說：「路不會自己消失，是人先把它忘了。」

## Background
瑪舒雅的工作不是單純帶路，而是確保整條路譜仍然可靠。她長年在風勒原心與外圍補水節點之間比對路印，對草原哪一段開始不對勁有極高敏感度。

## Secrets
- 她正在悄悄比對一組被人換過的新路標記號
- 她懷疑有人故意把某條古路從聯盟記憶裡抹掉

## Game Hooks
- 提供長程旅行、追跡與補給規劃任務
- 引出偽路標、失蹤商隊與古路事件
- 作為草原區域最可靠的地圖入口
