# 薩德原角色索引

這裡收納薩德原聯盟目前已可投入遊戲的主要 NPC 角色卡。

## 最高領導層
- 烏嵐赫：輪值總誓首
- 蘭朵瑪：夢牧柱首 / 誓靈導母
- 克薩爾：踏曜柱首 / 奔蹄首領
- 烏賽寧：誓原風策局總策監

## 目前角色
- 瑪舒雅：路譜守望者
- 赫朗：年長騎隊教頭
- 阿綠：井場管理者
- 圖薩嬤：草藥婆婆
- 蘇芮瑪：逐原五柱議誓使
- 野祿契：靈廚公會風灶契膳師
- 祁原：無柱行者

## 使用原則
- 這批角色對應草原遷徙、井鏈補給、騎隊訓練與草原生存玩法
- 最高領導層現已獨立成正式角色卡，可直接承接五柱、古路與聯盟級戰備事件
- 角色來源現在同時覆蓋聯盟議誓體系、在地化跨國公會與荒線流浪者
- 支援角色名錄另見 `supporting-cast.md`，已補宗教、政治、教育、軍務與諜報線
- 後續若往祖靈線延伸，可再補祭司、觀兆師與遠行使者
- 角色一律承接現有地方包，不另開脫節人脈
