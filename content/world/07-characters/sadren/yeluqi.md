# 野祿契

## Metadata
- id: sadren-yeluqi
- region: 賽勒亞大陸 / 東境 / 薩德原聯盟
- faction: 靈廚公會 / 長路風灶線
- role: 風灶契膳師
- status: active

## Snapshot
- 年齡：28
- 種族：人馬族
- 種族-分支：森馳裔 / 薩德原人馬支
- 職業：靈廚 / 旅膳調理師
- 等級 / 危險度：低戰鬥，高補給與生存
- 關鍵能力：長路旅糧設計、缺水條件下調膳、草原應急食材判讀

## Personality
- 很會照顧人，但不喜歡被當成只會煮飯
- 對浪費水和亂碰乾糧封包的人特別兇
- 習慣一邊做事一邊看隊伍氣氛有沒有歪掉
- 覺得一隊人能不能活著走完，先看他們吃得像不像一隊

## Speech Style
- 語速快，帶草原營地那種邊做邊講的節奏
- 常用食材狀態形容人的精神
- 常說：「你不是沒餓，你是快散了。」

## Background
野祿契屬於靈廚公會，但在薩德原，他的公會身分要先被翻成草原聽得懂的價值：能不能讓人和坐騎一起走下去。他因此被很多隊伍視為真正可靠的長路人物，也讓靈廚公會在薩德原有了很強的在地味。

## Secrets
- 他替某支失聯過的商隊保留了一套從未公開的斷糧配方
- 他懷疑有人正在用低價旅糧做掩護，偷偷運送不該過草原的東西

## Game Hooks
- 引出旅糧、缺水補給、毒食材處理與長路隊伍管理
- 作為薩德原非國家組織人物的地方化代表
- 很適合接旅隊、護送與失聯追查任務
