## 區域與國別角色
- 各國、各海域、各特殊區域角色卡分散收錄於本目錄下的子資料夾。
- 角色若已達到世界級知名度、持有專屬器物，或會跨區反覆出場，優先回掛至世界級條目。

## 收錄原則
- 一般地方角色不必強行配專屬武器或寶物。
- 優先配有專屬器物的人，通常是統治者、總教長、首席官、傳說級守望者、公會總掌、世界級仲介者或高風險前線的關鍵持有者。
- 專屬器物不一定是武器，也可以是王權重器、誓器、封樣器、權印、簿冊、匙組、儀盤、權杖或私藏遺物。
