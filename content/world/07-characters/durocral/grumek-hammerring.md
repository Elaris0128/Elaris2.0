# 格魯梅克

## Metadata
- id: durocral-grumek-hammerring
- region: 賽勒亞大陸 / 東境 / 杜羅山邦
- faction: 鎚誓環
- role: 鎚誓環首
- status: active

## Snapshot
- 年齡：118
- 種族：矮人
- 種族-分支：矮人 / 杜羅鍛脈支
- 職業：法環首腦 / 石契裁定者
- 等級 / 危險度：中戰力，極高制度權力
- 關鍵能力：石契審定、違誓裁決、誓章修訂、地方法軸整編

## Personality
- 很硬，像一條不願為任何人多彎一次的鐵線
- 不喜歡感情用事，但其實對山邦名聲看得極重
- 對外人不排斥，前提是對方尊重契與序

## Speech Style
- 句子總像裁決書第一條
- 很少附帶安慰，卻會把權責講得一清二楚
- 常說：「你可以後悔，但你不能把後悔當成沒立過契。」

## Background
格魯梅克掌鎚誓環，是杜羅山邦所有大誓、律例與背誓裁定背後最重的一隻手。他不帶兵，但許多軍械、工坊與宗族衝突最後能不能成立案件，都得過他這一關。

## Secrets
- 他正在重查幾樁曾被岩律院刻意壓低的老案
- 他懷疑有人在利用外地票據規避山邦的石契見證

## Game Hooks
- 作為杜羅律法、契約與宗族審理線的最高入口
- 可牽出假契、石碑竄改與軍械責任案
- 能讓玩家直接接觸山邦最核心的法理壓力
