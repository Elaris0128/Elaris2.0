# 杜羅角色索引

這裡收納杜羅山邦目前已可投入遊戲的主要 NPC 角色卡。

## 最高領導層
- 鐸格蘭：環軸總鎚
- 格魯梅克：鎚誓環首
- 瑟姆拉：爐誓環首
- 灰喀爾：爐紋議佐

## 目前角色
- 布魯格丹：稅石庫監印官
- 瑟瑪妮：火符導鍛師
- 卡魯姆：關堡驗契官
- 妲萊：白炬火號手
- 洛鐸瑪：探索公會聯席深廊判錄官
- 塔格魯：封坑監巡官
- 墨灰：拾火客

## 使用原則
- 這批角色對應杜羅的礦材、鍛造、關堡與山線預警玩法
- 最高領導層現已獨立成正式角色卡，可直接承接五環、法契與技術外流主線
- 角色來源現在同時覆蓋山邦官署、跨國回收組織與災後流浪者
- 支援角色名錄另見 `supporting-cast.md`，已補宗教、政治、教育、軍務與諜報線
- 後續若擴大深坑主線，可再補工班長、家族長與封坑守衛
- 角色以現有地方包為主，不額外漂移設定
