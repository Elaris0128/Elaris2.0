# 洛鐸瑪

## Metadata
- id: durocral-luodoma
- region: 賽勒亞大陸 / 東境 / 杜羅山邦
- faction: 探索公會聯席 / 深廊回收站
- role: 深廊判錄官
- status: active

## Snapshot
- 年齡：37
- 種族：矮人
- 種族-分支：矮人 / 杜羅鍛脈支
- 職業：回收判錄員 / 深廊協調人
- 等級 / 危險度：低戰鬥，高風險判讀
- 關鍵能力：遺構回收分級、危險工序排序、報酬與折損估值

## Personality
- 實際、穩、對誇口非常不耐
- 會先看工具和靴底，再決定要不要跟你談深廊工作
- 不討厭冒險者，但討厭不懂自己在撿什麼的冒險者
- 對真正守流程的人有一種山邦式的尊重

## Speech Style
- 語氣像在敲礦簿邊角
- 很少說多餘形容詞
- 常說：「你帶得回來，才叫你的。」

## Background
洛鐸瑪雖然屬於探索公會聯席，但在杜羅待久了，行事比很多本地判錄官還像山邦人。他長年卡在公會回收權、山邦封線權和黑市收購之間，知道哪種東西該封、哪種該賣、哪種連碰都不該碰。

## Secrets
- 他替某次官方回收案藏過一件未上報的異常零件
- 他懷疑白炬遠堡最近流出的不是廢料，而是被拆散的舊遺構核心

## Game Hooks
- 引出深廊回收、探索權、遺構分級與走私材料線
- 讓玩家接觸杜羅的非國家組織人物，但保留強烈山邦風格
- 是深坑、回收與黑貨任務的重要入口
