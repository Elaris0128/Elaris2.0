# 墨灰

## Metadata
- id: durocral-mohui
- region: 賽勒亞大陸 / 東境 / 杜羅山邦
- faction: 無固定所屬
- role: 拾火客 / 灰炭步者
- status: active

## Snapshot
- 年齡：不詳，外觀約 50 歲
- 種族：巖穹族
- 種族-分支：巖穹族 / 山邦共鍛支
- 職業：流浪拾火者 / 廢爐清路人
- 等級 / 危險度：中等
- 關鍵能力：餘火判讀、廢坑生存、舊爐共振辨識

## Personality
- 看起來像什麼都不在乎，其實記得每一座死爐的脾氣
- 對災難有種不討好任何人的平靜
- 不會主動教訓人，但會讓人自己意識到愚蠢
- 對還能修的東西懷有異常執著

## Speech Style
- 像煤灰一樣輕，卻總落在關鍵位置
- 常用火溫和礦渣形容人事
- 常說：「還冒著氣的，不一定還活著。」

## Background
墨灰沒有固定工班，也不屬於任何公會。他沿著被封掉的坑和死去的工坊行走，把還危險的東西挑出來，把還能救的東西背走。有些人說他在撿漏，有些人說他在替舊火收屍。

## Secrets
- 他隨身帶著一小塊來自大災爐的黑芯
- 他知道某條被宣布死去的火脈，其實還在極深處呼吸

## Game Hooks
- 引出封坑外緣、舊爐殘響、遺構廢熱與亡火傳聞
- 作為杜羅的特殊流浪者，把玩家帶向深層山邦記憶
- 能串接探索、公會、灰線與災害遺留線
