# 妲萊

## Metadata
- id: durocral-dalai
- region: 賽勒亞大陸 / 東境 / 杜羅山邦
- faction: 白炬遠堡火號線
- role: 白炬火號手
- status: active

## Snapshot
- 年齡：29
- 種族：矮人
- 種族-分支：矮人 / 杜羅鍛脈支
- 職業：火號觀線手
- 等級 / 危險度：中戰鬥，中偵察
- 關鍵能力：遠距觀測、山線預警、異常光訊辨識

## Personality
- 寡言、警醒、對遠處細節異常執著
- 比起人，更相信光、風與地勢
- 不輕易下結論，但一旦確定就很難被說服
- 對能守夜的人有天然好感

## Speech Style
- 音量低，句子少
- 喜歡先說方位再說事件
- 常說：「那束光不屬於任何一座塔。」

## Background
妲萊站在白炬高處太久，久到她對夜裡的山線比對白天的臉更熟。很多人以為火號只是傳訊，她卻知道一束錯誤的光能讓整條邊境線緊張起來。

## Secrets
- 她多次看見同一束不屬於官方火號的白光
- 她懷疑那道光和某條走私或封坑路線有關

## Game Hooks
- 提供山線巡查、夜行追光與邊境警報任務
- 幫玩家預判山路危險與觀線盲區
- 可引出跨關堡的秘密訊號網
