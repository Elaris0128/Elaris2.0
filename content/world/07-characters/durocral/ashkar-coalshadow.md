# 灰喀爾

## Metadata
- id: durocral-ashkar-coalshadow
- region: 賽勒亞大陸 / 東境 / 杜羅山邦
- faction: 爐影密廷
- role: 爐紋議佐
- status: active

## Snapshot
- 年齡：83
- 種族：矮人
- 種族-分支：矮人 / 杜羅鍛脈支
- 職業：情報總監 / 內控審影官
- 等級 / 危險度：高威脅，高隱蔽權限
- 關鍵能力：工藝外流追蹤、礦脈監控、內部審影、密語交接

## Personality
- 表面粗，內裡細，特別擅長在別人低估他時做事
- 不迷戀權力，但很在意誰有資格碰山邦的核心技術
- 對外來的友善與信任始終分開處理

## Speech Style
- 像煤火，表面不亮，越扒越燙
- 話不繞，卻喜歡讓別人自己補出最危險的那半句
- 常說：「祕密不是藏起來，是放在正確的人也不敢亂碰的地方。」

## Background
灰喀爾是爐影密廷現任爐紋議佐，掌管山邦對內對外最敏感的監察網。對杜羅而言，他的重要性不亞於一整支軍團，因為一條洩漏的礦脈圖或鍛序，比一場正面戰更能毀掉未來。

## Secrets
- 他早已掌握一條跨國黑貨線，但故意放著沒全收
- 他懷疑五環之中有人在利用密廷的老規矩替自己掩護

## Game Hooks
- 作為杜羅情報、技術外流與灰線追查的最高入口
- 可牽出軍械走私、密廊交接與宗族黑帳
- 非常適合和東境跨國黑市與探索回收線接軌
