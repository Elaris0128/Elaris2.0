# 卡魯姆

## Metadata
- id: durocral-karum
- region: 賽勒亞大陸 / 東境 / 杜羅山邦
- faction: 白炬遠堡契驗所
- role: 關堡驗契官
- status: active

## Snapshot
- 年齡：51
- 種族：矮人
- 種族-分支：矮人 / 杜羅鍛脈支
- 職業：驗契官
- 等級 / 危險度：低戰鬥，中制度壓力
- 關鍵能力：貨單審核、假印判讀、邊境風險評估

## Personality
- 嚴厲、刻薄、幾乎看誰都先懷疑三分
- 對假印與特權最敏感
- 雖然不討喜，但很少冤枉人
- 對守住底線有近乎固執的自豪

## Speech Style
- 像審件一樣一句一句核對
- 很喜歡讓對方自己講出矛盾
- 常說：「文件先過，人才有資格過。」

## Background
卡魯姆守在白炬遠堡最容易讓外人頭痛的位置。每支商隊、每批礦材、每張通行契印幾乎都逃不過他的眼。他不擅長討好人，但他守住了很多本來可能失控的口子。

## Secrets
- 他手上壓著幾筆不敢公開追查的假印案件
- 他懷疑這些假印背後牽到某個固定合作的邊境節點

## Game Hooks
- 作為山線盤查、假印與通關任務入口
- 提供邊境黑路與可疑商隊的情報
- 可迫使玩家用合法方式處理問題
