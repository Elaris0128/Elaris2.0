## 讀法
1. 先看 [世界魔物分類總表](/C:/Users/rabbi/Desktop/Elaris/content/world/09-creatures/world-monster-classification-master-index.md)，快速確認一個物種的 `區域 / 遭遇量 / 特性 / 攻擊模式 / 要害 / 危險程度 / 素材 / 遭遇率`。
2. 再看 [世界魔物分類、智慧程度與生態角色框架](/C:/Users/rabbi/Desktop/Elaris/content/world/09-creatures/monster-taxonomy-intelligence-and-role-framework.md)，判斷牠算哪一類、聰明到什麼程度、在環境裡扮演什麼角色。
3. 如果要跑遭遇或任務，接著看 [世界遭遇熱區與地區魔物矩陣](/C:/Users/rabbi/Desktop/Elaris/content/world/09-creatures/world-region-encounter-matrix-and-hotspots.md) 和 [世界魔物遭遇機率、季節窗口與群體規則](/C:/Users/rabbi/Desktop/Elaris/content/world/09-creatures/monster-encounter-probability-seasonality-and-packrules.md)。
4. 如果要直接跑單一怪物，進 [單一魔物詳檔庫索引](/C:/Users/rabbi/Desktop/Elaris/content/world/09-creatures/monster-entry-library-index.md)。
5. 如果要跑戰鬥、狩獵或撤退，接著看 [世界魔物弱點、行為模式與應對指南](/C:/Users/rabbi/Desktop/Elaris/content/world/09-creatures/monster-weakpoints-behaviors-and-response-guide.md)。
6. 如果要處理馴養、合法性、解體與採收，再看後面兩份實務指南。
7. 如果是神獸古族，再回到 [神獸古族具體位置矩陣](/C:/Users/rabbi/Desktop/Elaris/content/world/09-creatures/sacred-beast-location-matrix.md)、[神獸古族季節路線、停棲窗口與禁忌圈](/C:/Users/rabbi/Desktop/Elaris/content/world/09-creatures/sacred-beast-seasonal-routes-and-taboo-zones.md) 和 [神獸個體專章與行動檔](/C:/Users/rabbi/Desktop/Elaris/content/world/09-creatures/sacred-beast-individual-dossiers.md)。
8. 如果要快速開團，直接用 [各地區實抽遭遇表與直接擲用規則](/C:/Users/rabbi/Desktop/Elaris/content/world/09-creatures/regional-encounter-tables-and-direct-rolls.md) 與 [魔物任務模板與任務鉤子](/C:/Users/rabbi/Desktop/Elaris/content/world/09-creatures/monster-mission-templates-and-quest-hooks.md)。
