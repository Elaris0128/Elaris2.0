# 六大洋跨海域生態鏈

## 共通邏輯
- 六洋的海怪不是隨機散布，而是沿著 `潮窗`、`洋流`、`露礁期`、`沉船帶` 和 `古術殘場` 聚集。
- 很多航手其實是藉由生物反應來判讀海路。

## 典型生態鏈
- `露礁期 -> 小型濾食群聚 -> 提燈鰻、剪背魚、白喙鷗增多 -> 中大型掠食海怪逼近`
- `沉船帶 -> 腐索與骨艙共生物 -> 繞骨螺群與回收買手同時靠近 -> 假燈嚎鱗獸在外圍等失航船`
- `熱噴海床 -> 盲鯺螺與紅浪鹽鰭繁殖 -> 熔脈翻潮蜥沿熱線巡游 -> 灰線採收隊趁灰熄季入場`
- `極寒冰眼 -> 霜背短鬚鯨與裂冰靜羽鷗出現 -> 冰海拖桅巨影活動跡象上升`
- `雲瀑與空海轉運帶 -> 飛鰭與白脊鯨靠近 -> 吞帆母與裂喉鰩跟著雙層風場偏移`
