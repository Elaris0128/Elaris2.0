# 梵洛特魔物生態總索引

## 索引說明
梵洛特的生命設計應先從 `環境壓力` 與 `共同體怎麼活` 去理解，再挑單一生物。這裡很多物種看起來殘酷，但它們和住民一樣，都在適應路斷、水少、熱裂、風蝕與命名秩序。

## 一、聚落與生活用魔物
- `裂霜長鬃犛`：高寒役獸。危險 1 / 親近 4 / 拉運、守寒與高地巡補。
- `伏沙井背駝`：裂紋沙原常見載運獸。危險 1 / 親近 4 / 耐沙、耐渴、能記井線。
- `熔背殼駝`：熱谷與赤域重載獸。危險 2 / 親近 3 / 能扛熱與灰。
- `骨帆拖犬`：斷濱回收與拖網用獸。危險 1 / 親近 4。
- `花幕簾鹿`：繭影隱林邊層常見伴行獸。危險 1 / 親近 3。
- `索塔峭羽`：海岬哨域短距送物禽。危險 0 / 親近 3。
- `炭盲馱牛`：南裂危帶運樣與拖補給用獸。危險 1 / 親近 3。

## 九、生態鏈讀法
- 梵洛特若出現高價資源，請同步放入 `守域種 / 偷食種 / 回收種 / 假訊號種`。
- 若某地長期靠井、潮窗、索道或封樣制度活著，該地的魔物也應該直接破壞或模仿這些秩序。
- 梵洛特的類人魔物通常要和 `失名、失路、失責、失控` 有關，而不是單純模仿人形。
