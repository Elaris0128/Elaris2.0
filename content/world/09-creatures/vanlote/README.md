## 讀法
梵洛特的生態要先看 `哪條生存帶讓什麼東西活下來`，再看單一物種。這塊大陸的很多掠食者、分解者與生活用魔物都直接貼著路權、水權、熱穴、井線與沿岸回聲秩序運作。
