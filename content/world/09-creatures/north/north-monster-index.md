# 北境魔物生態總索引

## 一、生活與馴用魔物
- `長絨雪犛`：北境最穩定的拖運與保暖主力。危險 1 / 親近 5 / 雪橇、重貨、毛料。
- `垂鬃暖角駝鹿`：能在長距離低補給路線維持隊伍速度。危險 1 / 親近 4。
- `霜棉穴羊`：毛量厚、皮下脂豐，是寒地衣料與乳脂來源。危險 0 / 親近 5。
- `燈鼻雪犬`：分辨冰裂、偽路與匿形生物的守夜犬。危險 0 / 親近 5。
- `雪瓣眠鹿`：雪華部族常見伴生與採藥同行種。危險 0 / 親近 4。
- `長鬃冰蹄鹿`：盧米昂與古道運補常用坐騎。危險 1 / 親近 4。

## 二、雪原與伏擊掠食者
- `裂牙雪兔`：體型不大但性情凶暴的肉食性兔種，擅長群突與啃咬。危險 2 / 親近 0。
- `匿霜伏狼`：能貼雪、收聲、藏味的伏擊狼。危險 4 / 親近 0。
- `霜脊雪猞`：高風雪脊上的單體掠食者。危險 3 / 親近 0。
- `雪下藏角獸`：長時間埋伏在雪面下，暴起撞擊。危險 3 / 親近 0。
- `崩雪角鼬`：會啃穿雪層與冰殼，引發連段塌陷。危險 2 / 親近 0。

## 三、霜河、寒潭與水生魔植
- `裂冰潛鰭獸`：會在薄冰下高速追擊的長體掠食水獸。危險 4 / 親近 0。
- `霧鰓長鱗鱔`：喜棲冰下縫流與黑潭口。危險 3 / 親近 0。
- `霜河寒蓮`：北境代表性水生魔植，可入藥、入祭，也會守護花心。危險 1 / 親近 2。
- `裂冰鬚藻`：會纏住腳踝與舟具的寒水藻帶。危險 2 / 親近 0。
- `霧籽浮萍`：表面無害，成熟時會釋放令視野變形的霧籽。危險 1 / 親近 1。
- `寒鏡伏鱗`：會借冰面反光迷惑獵物的水生種。危險 2 / 親近 0。

## 四、林域、花系與菌群生命
- `月魄冰蓮`：只在特定暖湧寒潭短時綻放。危險 1 / 親近 2。
- `白隱絨菇`：藥食兩用菌，北境常見安全糧補之一。危險 0 / 親近 4。
- `雪眠花`：安神、止痛、送亡與夢境儀式常用花植。危險 0 / 親近 4。
- `霜鈴莓灌`：果實可食，但脆響會招來掠食者。危險 1 / 親近 3。
- `蓮雪花侍`：類人的花系守域生命，常在寒蓮水邊出現。危險 3 / 親近 1。
- `白根纏行者`：半植物型封路生命，會主動改變小徑。危險 3 / 親近 0。

## 五、礦脊、洞巢與迷宮型生命
- `銅脊鑿甲蟲`：以礦脈邊層與支柱為食。危險 2 / 親近 0。
- `寒鐵咬尾蝠`：倒掛在冷礦洞深處，對火味極敏感。危險 2 / 親近 0。
- `藍脊回音蜥`：會記住坑道聲響並模仿。危險 2 / 親近 0。
- `井口聽手`：貼壁棲息的類人洞居魔物，專獵落單探洞者。危險 4 / 親近 0。
- `礦霜空殼民`：埋城與廢礦中會緩慢活動的人形殘殼。危險 3 / 親近 0。

## 六、誓魂異常與類人威脅
- `雪脊山吼`：近似雪怪的大型類人魔物，會佔據洞巢與風口。危險 4 / 親近 0。
- `鏡步雪女`：能在風雪與月光中借熟人聲線引路的人形異常。危險 4 / 親近 0。
- `空面逐行者`：沒有固定面貌，只會在隊伍最疲憊時貼上來。危險 4 / 親近 0。
- `夢行白童`：會用熟悉聲音把人從燈鏈與碑列旁引走。危險 3 / 親近 0。
- `冰殼遺步`：被極寒固定下來的半人形危險體。危險 4 / 親近 0。
- `霜臂花偶`：看似優雅，其實擅長把人拖入花坑或凍泥的植物人形。危險 3 / 親近 0。

## 七、王者級存在
- `裂雪月鏡王狼`：北境雪原王者，能迫使整片掠食線改道。
- `霜河蓮冠母鰩`：寒蓮大水系的水中王者。
- `白隱花母樹`：雪華林域的王者級魔植。
- `暮鏡魂鹿王`：誓魂荒境與輪夢殘響的核心王者。
- `古碑引路王駝`：星紋古道王者級長行獸，能救路也能帶人入禁段。

## 八、神獸古族痕跡與亞種邊影
- 說明：以下是北境最常被記錄到的 `神獸古族` 支裔、亞種與遠系魔物邊影，不屬於一般聚落日常常見種。
- `霜脊飛龍`：龍族支裔。危險 5 / 親近 0 / 活動於極寒高脊與天裂風口，多被視為自奧崙群星帶遠行而來。
- `霜翎幼龍裔`：龍族遠系幼裔。危險 3 / 親近 1 / 極罕見，偶爾會被雪華部族記錄為可締結 `靈誓伴` 的重誓型個體。
- `冰背玄龜`：玄龜族支裔。危險 4 / 親近 1 / 常見於寒河源頭、沉堤護池與裂冰湖心，多屬海洋主家域外散支裔。
- `雪脊嵐虎`：嵐虎族支裔。危險 4 / 親近 0 / 常在雪林高線與裂谷雪脊留下孤巡痕跡，被視為群島獵場的試煉遠裔。
- `星泉走麟`：麟族支裔。危險 3 / 親近 2 / 常與古碑路、雪橋、泉眼和迷路者傳說相關，和翼人古路圈最常被拿來對照。
- `霜燼鸞裔`：鳳族遠系亞種。危險 3 / 親近 1 / 常在暖湧寒潭、地熱花林與雪後回光帶出現，不形成北境本土大群系。
