# 東境魔物生態目錄

## Purpose
這個資料夾整理東境的 `魔物生態`，而不再只整理野外威脅。從現在開始，東境的植物、菌類、昆蟲、家養種、水棲種、掠食種、類人異常與共鳴生命，都視為同一套魔物生態網的一部分。

## 核心使用方式
- 若要先理解東境有哪些生命型態，先看 `east-monster-index.md`
- 若要看東境整體食物網與跨環境生態，先看 `east-ecological-webs.md`
- 若要看聚落與日常生活相關的魔物，先看 `domestic-and-settlement-monsters.md`
- 若要看魔植、魔菌與魔蟲，先看 `plant-fungi-and-insect-monsters.md`
- 若要看類人與異常威脅，先看 `humanoid-and-anomalous-monsters.md`
- 若要看自然環境中的王者級魔物，先看 `east-monster-sovereigns.md`
- 若要看環境分布，再看各生境檔案

- `east-monster-index.md`：東境整體魔物生態索引

- `frontier-and-roadside-creatures.md`：平原、商路與邊線生境
- `wetland-river-and-marsh-creatures.md`：河港、濕原與泥灘生境

## 使用提醒
- 東境整體仍採用 `魔物生態` 作為外部檢索語。
- 但瑟堤亞棲國是特例：棲國文明主體自稱 `棲裔`，而且只有被誓王賜名的魔物才屬於棲裔；其餘仍直接稱為魔物，失控者則稱 `失衡異種`。

## 設計原則
- 東境不再使用「一般生物」和「魔物」分開寫法。
- 國家境內也必須有生活用魔物，而不是只有野外怪物。
- 每個生境都應有生產者、中層攝食者、上層壓制者與分解者。
- 危險程度與親近程度要一起看，不能只看會不會打人。
- 環境不只要有「住在這裡的物種」，還要有 `誰吃誰、誰傳播誰、誰把路線改掉` 的邏輯。
