# 東境魔植、魔菌與魔蟲目錄

## Purpose
這份文件整理東境生態鏈中的生產者、授粉者、分解者與小型環境維持者。它們往往不像大型掠食種那麼顯眼，卻是整個環境能否穩定存在的關鍵。

## 一、生產者與農帶骨架
- `誓禾穗`：阿瓦隆高產主糧魔植。危險 0 / 親近 5 / 平原農耕核心。
- `邊鎖麥`：王道平原、路肩與田界常見的固土魔禾。危險 0 / 親近 4 / 能減少雨後路陷。
- `星鞘蘆`：瑟恩河灣纖維魔植，兼具紙材與固岸功能。危險 0 / 親近 4。
- `井緣涼葉`：薩德原補水與解熱用魔植。危險 0 / 親近 4。
- `白鹿溪藥蘚`：溪石藥用蘚類，常做基礎傷藥。危險 0 / 親近 4 / 溪岸濾水與止血並存。
- `雷痕蘚`：克羅嵐焦土低地的導電蘚群。危險 1 / 親近 1 / 會把雷痕濕地固定成特定生境。

## 二、結果型、喬木、灌木與藤本
- `蘚冠果木`：瑟堤亞邊層霧林常見的結果喬木，成熟時會垂下微光 `低光果囊`。危險 0 / 親近 3 / 夜行補給、授粉樞紐與弱光照明。
- `潮刺莓灌`：東境林緣與濕潤山徑常見的結果灌木，果實酸甜，枝刺帶輕麻。危險 1 / 親近 2 / 可食、釀汁、吸引小型傳孢者。
- `海膽葡萄`：誓息霧林與高濕陰林的爆裂散種藤本魔植，串籽如葡萄，果殼長滿海膽般硬刺。危險 2 / 親近 1 / 需冷霧、穩枝與逆旋採果，否則碰觸即炸裂。
- `靜脈葉`：聯邦療養丘地常見安神魔植。危險 0 / 親近 3。
- `骨風藤`：高嶺綁具與輕繩重要來源。危險 1 / 親近 2。
- `夢汀水花`：祭地靜水才穩定生長的稀有水花。危險 1 / 親近 1。
- `誓蘚`：瑟堤亞底層共鳴植種。危險 0 / 親近 5。
- `靜紋藤`：安撫與低刺激編構用藤類。危險 0 / 親近 4。
- `霜泥泡葵`：裂谷濕原甜味誘食型水陸魔植。危險 2 / 親近 1 / 常形成假安全踏點。
- `幽瓣收聲花`：霧林與安撫步道常見吸音花種。危險 1 / 親近 2 / 常被靜修地與照護者使用。

## 三、木妖型與棲地塑形魔植
- `霧椏木妖`：深霧林與靜溪根網常見的木妖型魔植，會緩慢挪移根群並守看熟悉水脈。危險 2 / 親近 1 / 對亂砍、火煙與錯誓氣味反應劇烈。
- `坡守梣樹`：阿瓦隆北線與瑟恩丘坡常見的低活動木妖。危險 1 / 親近 2 / 主要會在坡體被掏空時出手護坡。
- `焦脈柱藤`：杜羅熱脈邊緣的纏柱魔藤。危險 2 / 親近 1 / 會沿礦柱與支架攀附，既能穩固也能壓垮老構件。

## 四、魔菌與分解網
- `熔鹽蕈`：杜羅熱脈可食魔菌，礦鹽依存。危險 2 / 親近 2。
- `語孢囊蕈`：瑟堤亞可保存聲紋的高敏魔菌。危險 2 / 親近 2。
- `煙紋苔`：瑟恩工藝坡吸附鍊金殘塵的濾化菌苔。危險 1 / 親近 2。
- `食印蘚團`：會吞食低階導印的危險菌蘚集合體。危險 2 / 親近 1。
- `銅塵絨蕈`：高嶺風口與舊鍛台附近的粉絨菌。危險 1 / 親近 1 / 會回收金屬粉屑。
- `泥針腐幕`：裂谷濕原常見的腐幕菌席。危險 1 / 親近 0 / 是濕地主要分解層之一。

## 五、授粉、傳孢與小型魔蟲
- `鈴穗風蜂`：王道農帶與果灌邊層主要授粉者。危險 1 / 親近 2 / 對煙與焦味特別敏感。
- `絮光采孢蠅`：霧林、濕帶與蘆灘常見傳孢者。危險 0 / 親近 2 / 會沿光點與氣味遷移。
- `霧簾織蛾`：霧林濕絲築巢種。危險 1 / 親近 1 / 其蛹殼常成為安靜材料。
- `語殼寄鳴蟲`：會模仿聲音的共鳴魔蟲。危險 2 / 親近 1。
- `灰鍊掘甲蟲`：杜羅坑道支柱破壞者，也是分解型工業害蟲。危險 3 / 親近 0。
- `泥針分解蠊`：濕地與舊戰場常見的小型清道蟲。危險 1 / 親近 0。

## 六、寄附、侵蝕與小型擾動者
- `潮箱寄貝`：雖非蟲類外觀，但在功能上屬典型聚附寄生種。危險 2 / 親近 0。
- `誓泥殼蟲`：平原道路與糧倉旁常見的殼蟲。危險 1 / 親近 1 / 會把誓塵與泥屑結成硬殼。
- `暮井掠飲蝠`：兼具吸水點寄生壓力與夜行捕食特性。危險 2 / 親近 0。

## 生態角色提示
- 結果型魔植不等於安全採集，很多高糖或高價值果串本身就是防禦結構的一部分。
- 木妖型魔植常扮演棲地塑形者、守域者或水脈保護者，不一定是掠食者，卻很容易成為領域衝突核心。
- 魔菌常兼具分解、訊息承載、污染吸附與共生功能。
- 魔蟲多半負責授粉、寄生、濾淨、腐食或細部破壞。
- 後續設計一個環境時，至少要先決定該地的生產者、授粉者與分解者是誰。
- 一個環境若只有大型掠食者，卻沒有菌類、蟲群與果植，主持時會很快失去「這裡真的活著」的感覺。
