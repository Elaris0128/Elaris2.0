# 東境環境王者級魔物

## 說明
東境的自然環境不只會孕育一般魔物、生態鏈與區域災害，也能孕育 `王者級魔物`。這些存在不是單純比較強的大型個體，而是會實際影響一整片環境的路線、資源、掠食秩序、季節行為與在地禁忌的 `生態領主`。

在東境，居民對它們的稱呼通常很通俗，不一定都叫「王」。常見叫法有：`河王`、`山王`、`鬃王`、`樹王`、`母株`、`母藤`、`丘主`。

## 核心規則
- 王者級魔物可以出現在自然環境中，不需要依附地下城、遺跡或人工設施才成立。
- 王者級魔物不等於文明君主，也不等於瑟堤亞的 `誓王`。前者是生態位階，後者是文明統治身份。
- 並不是每個生境都一定要有王者級魔物；只有資源夠集中、共鳴夠穩定、或物種壓制足夠長期的地方，才比較容易形成。
- 有些王者是單一個體，有些則是 `母株 / 母藤 / 巢王 / 群核` 這種以整個生命核心存在的形式。
- 王者級魔物被擊退或消失，不一定會讓環境立刻安全，反而可能讓該生境暫時失衡。

## 東境已知或流傳中的王者案例

### 白冠誓鹿王
- 生境：王道平原帶與阿瓦隆南北誓田外緣
- 類型：獸王型 / 巡行王者
- 危險程度：4
- 親近程度：1
- 概要：一頭冠角如白枝的古老鹿王，會沿誓田、廢田與舊授旗路線巡行。牠不常主動襲人，但會強烈壓制周遭草食與巡邏魔獸的路徑。
- 接近徵兆：成片誓禾倒伏、白蹄印在清晨發亮、附近役獸異常安靜。

### 墨潮河王鼉
- 生境：大河與河港帶
- 類型：水棲王者 / 河域壓制者
- 危險程度：5
- 親近程度：0
- 概要：潛伏在深河轉彎與舊碼頭淤底的大型王鼉。牠會改變同水域掠食種的活躍區，也會讓某些老船夫突然整季都不肯走同一段水路。
- 接近徵兆：浮木卡岸異常整齊、河鳥提前離水、河面在無風時出現黑色環紋。

### 玄砧熔角王
- 生境：高嶺山堡帶與深熱脈邊層
- 類型：幻想獸王 / 熱脈領主
- 危險程度：5
- 親近程度：0
- 概要：體表像熔黑礦層的巨型角獸，會在熱脈移位與深層工序失衡時現身。杜羅部分老匠師把它視為活著的熱脈警報。
- 接近徵兆：岩壁出汗、鐵件自行發燙、坑廊小型熱棲種集體遷移。

### 霜泥王花母株
- 生境：裂谷濕原帶
- 類型：魔植王者 / 母株型
- 危險程度：4
- 親近程度：0
- 概要：一株深埋在濕原泥盆下的大型食植母株，會放出帶甜味的誘引花粉，並藉分株控制周圍小型泥沼植物與蟲群。
- 接近徵兆：濕地甜香過重、同一片泥面重複鼓起、沼蟲突然朝一個方向聚集。

### 踏雲鬃王
- 生境：草原遷徙帶
- 類型：群王型 / 遷徙壓制者
- 危險程度：4
- 親近程度：1
- 概要：一頭在大群逐原獸與高階草原魔獸間擁有異常號召力的巨型鬃獸。牠不只是強，還會讓整條遷徙線的其他生命一起改道。
- 接近徵兆：遠方獸群集體轉向、地面震動先到、逐雷獵隼在高空盤得異常密集。

### 霧椏王樹
- 生境：誓息霧林深層
- 類型：木妖王者 / 活地貌領主
- 危險程度：5
- 親近程度：1
- 概要：被部分導引者視為霧林自我防衛意志的王樹型存在。它不一定會移動，但能透過根網、枝椏與孢霧間接改變整片林線的可通行性。
- 接近徵兆：導印失準、熟路忽然變慢、枝影在無風時仍自行偏轉。

### 海膽葡萄母藤
- 生境：霧林邊層至半深層採集區
- 類型：結果型魔植王者 / 母藤型
- 危險程度：4
- 親近程度：0
- 概要：海膽葡萄並不只有普通藤株，少數極老母藤會控制一整片爆裂散種區，讓周邊採集線、伴生魔植與防刺魔蟲形成完整領域。它更像是某塊採集區的地頭王者。
- 接近徵兆：附近普通藤串同步結刺、種串未碰即微震、伴生魔蟲突然整片離枝。

## 設計提醒
- 王者級魔物應該優先改變 `環境感`，其次才是戰鬥難度。
- 它們最適合拿來做：路線改變、採集封鎖、季節異常、地區禁忌、護送危機、區域級任務。
- 不必每一隻都做成純戰鬥 boss；有些更適合做成需要理解、迴避、談判、安撫或繞路處理的存在。
