# 奧崙高空生態鏈

奧崙的生態不是「龍在頂端，其他都是食物」這麼簡單。  
這裡更像一個被高空壓力、熱脈、水脈、長記文明與放逐制度共同雕出來的生態網。

## 一、能量來源
- `星瀾源脈` 與高空冷泉提供最上層純淨水源
- `星殞潤原` 與暖脈裂谷提供熱量、礦質與孵育條件
- `朧潮靜層` 提供庇傷、回溫與低層霧潮養分

## 二、主要生產者與基礎群
- 暖脈苔
- 霧滴苔膜
- 裂礦灰菌
- 雲緣鹽皮藻
- 孵霧絨膜
- 星瀾微藻
- 幽潮鹽衣
- 低泊灰膜

## 三、中層獵物
- 暖脈巢羊
- 雲脊飛游群
- 碎礁礦獸
- 低泊鹽囊魚
- 抱殼棘蜓
- 懸泉礦背螈
- 風脊剪尾獸
- 回溫盲鰭獸

## 四、拾荒者與分解者
- 祖骨啄隼
- 靜潮息螺
- 裂礦灰菌
- 斷索霧蛛
- 孵煙巢鼠
- 霆裂長嘴鷸
- 沉夢潮蟬
- 索棚吸霧蝠
- 巢腐鳴虻
- 暖脈蛻蚓
- 霧囊黏蛭
- 索霉牙蝠

## 五、上層掠食者與秩序壓制者
- 正統本裔龍族
- 守域支裔
- 近環暖流飛龍
- 少量裂風脊龍

## 五點五、稀有高位龍裔與旁脈介入
- 高位龍裔平時很少現身，但一旦靠近某條生態帶，代表秩序問題已經被提高到主家以上。
- 海中旁脈與伏地亞龍不常直接改寫奧崙核心生態，卻會透過證物、病源、潮痕與外務衝突改寫低空壓力。
- 因此奧崙的生態異常，常常會一路長成政治異常。

## 六、文明如何改寫生態
- 奧崙不讓任何地方自由演化到失序
- 某些巢羊群會被刻意保留，供試翅期使用
- 某些靜潮息螺區會被封起，只供回溫池採藥
- 某些斷鏈霧蛛巢被故意留下，當作放逐路線的天然屏障
- 某些高風獵群會依年輕龍數量被輪調保留
- 某些低空樣本區會故意維持在「可觀測但不完全乾淨」的邊界
- 某些庇傷灣則被靜層長者壓低採集量，避免病翼者失去最後可活的生態帶

## 六點五、季節與病巢如何扭曲生態鏈
- 暖穩季會放大孵霧絨膜與抱殼棘蜓，直接影響幼裔食階。
- 試翅季若過密，會把 `雲脊飛游群 -> 風脊剪尾獸 -> 年輕龍` 這條鏈拉到失衡。
- 雷灼季會讓 `霆屑薄菌 -> 霆裂長嘴鷸 -> 裂風影鰩` 短時間暴增。
- 朧潮回溫季則會讓 `靜潮息螺 / 回溫盲鰭獸 / 霧囊黏蛭` 同時變得有價值又危險。
- 一旦病巢起來，真正先崩的常是暖苔、護熱膜與低階採收物，而不是上層龍族本身。

## 七、最像奧崙的生態特色
- 幼裔的成長本身就是生態工程的一部分
- 放逐制度會改變某些邊地物種的繁衍機會
- 龍族長記會決定哪些獵物該被保、哪些該被清掉、哪些不准碰

## 八、玩法上最好用的三條生態線
- `季節線`：風壓、暖脈與遷徙讓某些請求能不能被准，根本不只是政治。
- `病巢線`：污染、寄生與緩慢失序，會把一個小醫療問題拖成整段家域責任。
- `素材線`：奧崙沒有單純的採集，只有被育裔、庇傷、守界與外務切分過的責任資源。
