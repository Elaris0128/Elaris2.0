## 使用說明
- 奧崙生態的主角是龍族，但奧崙不是只有龍。
- 這裡還有被高空暖流養活的獵物群、撿食祖骨與殘鱗的回收物種、朧潮靜層的共生物，以及與下垂承契帶有關的低空樣本生物。
- 奧崙的重點不是「有很多怪」，而是每一層環境都會被龍族的育裔、守界、庇傷與外務秩序重新塑形。
